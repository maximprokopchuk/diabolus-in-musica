# Питание педалей

## Зачем правильное питание?

Питание педалей — **один из самых недооценённых аспектов педалборда**. Плохое питание приводит к:
- Фоновому шуму (hum 50/60 Гц)
- «Цифровому» шуму (высокочастотное жужжание)
- Потере высоких частот
- Нестабильной работе педалей
- Взаимодействию между педалями (цифровые «грязнят» аналоговые)

---

## Типы блоков питания

### 9V батарея (Крона)

Стандартный источник питания для большинства педалей.

**Плюсы:**
- Абсолютная изоляция от сети
- Нет шума (если батарея свежая)
- Ноль земляных петель

**Минусы:**
- Разряжается (~6–8 часов работы)
- Дорого и неудобно при большом количестве педалей
- Падение напряжения при разряде влияет на звук (иногда специально используется — «dying battery sound» для fuzz)

**Когда использовать:** германиевый fuzz, педали на сцене без стабильного питания, педали-одиночки.

### Одноблочный адаптер (Single Adapter)

Простой 9V/1А адаптер от производителя педали.

**Плюсы:**
- Дёшево
- Просто

**Минусы:**
- Один адаптер = один провод на каждую педаль
- Нет изоляции между педалями
- Качество питания зависит от адаптера

### Daisy Chain (Цепочка)

**Daisy chain** — один адаптер, от которого через разветвитель питается несколько педалей.

```
Адаптер 9V/500mA → [Разветвитель] → Педаль 1
                                   → Педаль 2
                                   → Педаль 3
                                   → Педаль 4
```

**Плюсы:** дёшево, компактно.

**Минусы:**
- **Шум земляной петли** — все педали имеют общую землю. Если одна создаёт шум, он слышен во всех.
- **Ток** — суммарное потребление не должно превышать мощность адаптера
- **Цифровые педали грязнят аналоговые** — через общую землю

Daisy chain работает нормально для простых аналоговых педалей. Для профессионального применения не рекомендуется.

### Isolated PSU (Изолированный блок питания)

**Isolated PSU** — блок питания с несколькими **гальванически изолированными** выходами. Каждый выход — как отдельная батарея.

```
Isolated PSU:
  ├─ Output 1 (изолирован) → Педаль 1
  ├─ Output 2 (изолирован) → Педаль 2
  ├─ Output 3 (изолирован) → Педаль 3
  └─ Output 4 (изолирован) → Педаль 4
```

Нет общей земли между выходами → нет земляных петель между педалями.

**Лучшие изолированные PSU:**

| Модель | Выходы | Особенности |
|--------|--------|-----------|
| **Strymon Zuma / Ojai** | 5–9 выходов | Высококачественный, Strymon-стандарт |
| **Cioks DC7 / DC10** | 7–10 выходов | Профессиональный, европейский |
| **Voodoo Lab Pedal Power 2+** | 8 выходов | Классика, надёжный |
| **MXR Iso-Brick** | 10 выходов | Доступный, надёжный |
| **Truetone 1 SPOT Pro** | 12 выходов | Компактный, много вариантов |
| **Walrus Audio Aetos** | 8 выходов | Boutique качество |

## Шум земляной петли (Ground Loop)

**Ground Loop** — ситуация, когда два устройства соединены по земле двумя путями (прямым аудиокабелем и электрическим питанием). По этой петле протекает небольшой ток → шум 50/60 Гц (гудение).

### Источники земляных петель

1. **Усилитель + педалборд** подключены к разным розеткам с разным «нулем»
2. **Компьютер + педалборд** через звуковую карту
3. **Педали с daisy chain** (общая земля) + длинные кабели

### Диагностика

1. Отключите все педали — тишина или шум?
2. Подключайте педали по одной — шум появляется?
3. Попробуйте другую розетку или другой кабель питания

### Решения

| Метод | Применение |
|-------|-----------|
| **Изолированный PSU** | Главное решение — устраняет земляные петли между педалями |
| **DI Box с трансформаторной развязкой** | Разрывает петлю на аудиосигнале |
| **Хамм-бастер (Hum Eliminator)** | Специализированный трансформаторный изолятор |
| **Все устройства из одной розетки** | Уменьшает разницу потенциалов земель |
| **Star-wiring (звёздная разводка)** | Все земли сходятся в одну точку |

## Специальные требования по питанию

### Педали с повышенным током

Некоторые педали потребляют значительный ток:

| Педаль / Тип | Потребление |
|-------------|------------|
| Простые аналоговые (MXR, Boss Drive) | 2–10 мА |
| Цифровые педали (TC Electronic, Boss DD) | 50–100 мА |
| Strymon (BigSky, Timeline) | 250–300 мА |
| Процессоры (Helix, Axe-Fx) | 500+ мА |

**Каждый выход изолированного PSU** имеет ограничение по току. Проверяйте совместимость.

### Педали 12V / 18V

Ряд педалей (Radial, некоторые Boss) требуют 12V или 18V. Некоторые педали (например, Ibanez Tube Screamer) звучат лучше при 18V вместо 9V — увеличивается headroom.

**Решение:** PSU с настраиваемым напряжением или несколькими выходами разного напряжения. Многие профессиональные PSU (Cioks, Strymon Zuma) имеют 12V и 18V выходы.

### Педали на переменном токе (AC)

Некоторые vintage-эффекты и ряд педалей требуют **9V AC** (не DC). Стандартные PSU дают DC — нужно проверять тип.

---

## Нумерация токов — как считать

```
Педаль 1: 10 mA
Педаль 2: 8 mA
Педаль 3: 100 mA
Педаль 4: 250 mA
Итого: 368 mA → нужен выход с запасом, минимум 400 mA
```

**Правило:** суммируйте ток всех педалей и выбирайте PSU с запасом 20–30%.

---

## Порядок подключения на педалборде

1. Подключите все кабели питания до включения в сеть
2. Сначала включите PSU в сеть
3. Затем включите усилитель
4. При выключении — обратный порядок

> **Инвестиция в хороший PSU** окупается: изолированный блок за $150–200 устраняет большинство шумовых проблем и работает 10+ лет. Это один из самых «умных» апгрейдов педалборда.
