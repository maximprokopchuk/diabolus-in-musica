# FX Loop (Эффект-петля)

## Что такое FX Loop?

**FX Loop (Effects Loop, эффект-петля)** — разрыв в сигнальной цепи усилителя **между преампом и оконечным каскадом**. Позволяет включать педали в этот разрыв.

```
Гитара → [ПРЕАМП] → [FX SEND] → Педали → [FX RETURN] → [ОКОНЕЧНЫЙ КАСКАД] → Кабинет
```

Педали, включённые в FX Loop, обрабатывают сигнал **после формирования тона в преампе**.

---

## Зачем нужен FX Loop?

### Проблема: модуляция и задержка перед перегрузом усилителя

Если дилей включён **перед** входом усилителя, а усилитель перегружается:

```
Гитара → [Delay] → [Перегруженный преамп]
```

Повторы дилея вместе с оригинальным сигналом поступают в перегруженный преамп → **повторы тоже перегружаются** → звук «мутный», «залитый».

### Решение: дилей в FX Loop

```
Гитара → [Перегруженный преамп] → [FX Send] → [Delay] → [FX Return] → Оконечник
```

Повторы генерируются **после** преампа → чистые, точные → «профессиональный» разделённый звук.

---

## Что ставить в FX Loop?

### Рекомендуется в FX Loop

| Педаль | Причина |
|--------|---------|
| **Дилей** | Чистые повторы без дополнительного перегруза |
| **Ревер** | «Пространство» вокруг готового перегруженного тона |
| **Хорус** | Модуляция готового тона |
| **Фейзер** | То же |
| **Тремоло** | Управление реальным выходным сигналом |
| **EQ** | Коррекция тона усилителя после преампа |
| **Noise Gate** | Gate «слышит» полный преамп-сигнал |

### Перед входом усилителя (не в Loop)

| Педаль | Причина |
|--------|---------|
| **Overdrive / Distortion** | Должны взаимодействовать с преампом |
| **Wah** | Должен «видеть» гитарный сигнал |
| **Компрессор** | Управляет динамикой гитарного сигнала |
| **Fuzz** | Взаимодействует с гитарным импедансом |

## Уровни сигнала в FX Loop

**Критически важный момент:** FX Loop работает на другом уровне сигнала, чем педали, рассчитанные на прямое подключение гитары.

### Уровни сигнала

| Уровень | Значение | Где встречается |
|---------|---------|----------------|
| **Instrument Level (~-20 dBu)** | Уровень гитары | Вход педалей (Guitar In), вход усилителя |
| **Line Level (-10 dBV / +4 dBu)** | Студийный уровень | FX Loop профессиональных усилителей |

**FX Loop большинства ламповых усилителей** работает на **Line Level** — значительно выше инструментального уровня.

### Что происходит при несоответствии

| Педаль | Уровень | В Line-level Loop | Проблема |
|--------|---------|-----------------|---------|
| Дешёвый дилей | Instrument level | «Задавленный», не раскрывается | Маленький gain-диапазон |
| Overdrive | Instrument level | Перегружается | Слишком горячий сигнал |
| Профессиональная стойка | Line level | Работает корректно | Совместимость |

**Решение:** проверьте уровень FX Loop вашего усилителя. Многие педали работают нормально, но некоторые требуют согласования. Ряд усилителей имеет переключатель уровня FX Loop (-10 dBV / +4 dBu).

## Серийный vs Параллельный FX Loop

### Serial (Последовательный) FX Loop

Стандартный тип. Весь сигнал проходит через педали в Loop полностью.

```
Преамп → FX Send → [Педаль 1] → [Педаль 2] → FX Return → Оконечник
```

100% сигнала идёт через педали. Если педаль выключена (bypass), сигнал всё равно проходит через её буфер или механический switch.

### Parallel (Параллельный) FX Loop

Сигнал делится на два пути: прямой (dry) и через педали (wet). Затем смешиваются.

```
Преамп → [Split] → Dry path ──────────────────┐
                 → Wet path → [Педали] → [Mix] → Оконечник
```

Регулятор **Mix** определяет соотношение dry/wet:
- Mix 0% → только dry (педали не слышны)
- Mix 100% → только wet (только педали)
- Mix 50% → равный баланс

**Преимущества параллельного Loop:**
- Dry сигнал всегда присутствует → сохраняется «живость» и атака
- Педали «добавляются» к тону, не заменяют его
- Популярно в профессиональных усилителях (Mesa/Boogie)

**Недостатки:**
- Более сложная работа с дилеем (повторы смешаны с dry → меньше контроля)
- Требует тонкой настройки уровней

## FX Loop Send как DI-выход

Ряд музыкантов использует **FX Send** как удобный **линейный выход** преампа:

```
Гитара → Преамп усилителя → FX Send → DI Box → Звукорежиссёр
                                     ↓
                               Кабинет (для сценического мониторинга)
```

Это даёт чистый преамп-сигнал на FOH без микрофона. В связке с IR (импульсными характеристиками кабинета) — полноценный студийный DI.

---

## Практические советы

1. **Пробуйте педали в обоих местах** — некоторые хорусы звучат лучше перед усилителем, другие в Loop
2. **Проверьте уровень Loop** — если педаль «задавлена», попробуйте поднять Level или Output
3. **Noise Gate в Loop** — ISP Decimator в FX Loop «слышит» полный преамп-сигнал и управляет gate точнее
4. **Два усилителя? Используйте Loop для routing** — FX Send → стерео-ревер → оба усилителя
5. **Запись через FX Send** — хороший способ записать сигнал без кабинета для последующего reamping
