# True Bypass vs Buffered Bypass

## Что такое Bypass?

**Bypass** — режим педали, когда она **выключена**: сигнал должен проходить «мимо» неё, не подвергаясь обработке. Способ реализации bypass принципиально влияет на звук, особенно в длинных цепочках.

---

## True Bypass

### Принцип

При true bypass сигнал физически коммутируется через механический переключатель, **минуя все электронные компоненты педали**.

```
True Bypass (OFF):
Вход → [Переключатель] → Выход
       ↕ (механически замкнуто напрямую)
       [Схема педали] (физически отключена от тракта)
```

**Нет электроники в тракте** = нет окрашивания, нет потери высоких частот.

### Достоинства True Bypass

- **Прозрачность**: выключенная педаль не влияет на тон вообще
- **Отсутствие «тонального загрязнения»** даже от многих педалей
- **Нет проблемы с фоном** от питания через выключенную педаль

### Недостатки True Bypass

**Длинные цепочки:**

Гитарный сигнал — высокоимпедансный (~250 кОм). При прохождении через длинный кабель и многие механические контакты происходит **потеря высоких частот** из-за ёмкостной нагрузки (кабели имеют ёмкость ~100 пФ/м).

| Длина кабелей | Потеря HF |
|--------------|----------|
| 2–3 м | Практически незаметна |
| 5–6 м | Небольшая |
| 10+ м | Заметная (педалборд → сцена → усилитель) |
| 15+ м (стандарт сцены) | Существенная потеря «воздуха» |

При большом количестве true bypass педалей + длинные кабели → звук «темнеет», теряет верхние.

**Щелчки и «пыхи»:**
Механический переключатель при коммутации может создавать щелчки, особенно на высоких gain-сигналах.

---

## Buffered Bypass

### Принцип

При buffered bypass педаль содержит **буфер (buffer)** — простой усилитель с единичным коэффициентом усиления (1:1), который:
- Принимает высокоимпедансный сигнал
- Выдаёт низкоимпедансный (< 1 кОм)

```
Buffered Bypass (OFF):
Вход → [Буфер → Unity Gain Amp] → Выход
       (схема работает, даже когда педаль выключена)
```

### Роль буфера

Низкоимпедансный выход буфера практически **нечувствителен к ёмкости кабелей** — высокие частоты не теряются независимо от длины кабелей.

```
Сигнал с буфером:    ──────────────────── (плоская АЧХ)
Сигнал без буфера:   ─────────______ (падение ВЧ)
```

### Достоинства Buffered Bypass

- **Сохраняет высокие частоты** в длинных цепочках
- **Нет щелчков** при переключении
- **Стабильный импеданс** для остальных педалей

### Недостатки Buffered Bypass

- **Плохой буфер** может «окрашивать» тон (добавлять жёсткость, резкость)
- Некоторые педали (fuzz, wah) **требуют высокоимпедансного сигнала** — буфер перед ними меняет их характер
- Буфер потребляет питание даже при выключенной педали

---

## Проблема «слишком много буферов»

**Слишком много буферизованных педалей** подряд может:
- Добавить тонкое окрашивание (особенно дешёвые буферы)
- Создать интерференцию между буферами
- Изменить взаимодействие с fuzz

**Правило:** 1–2 качественных буфера в цепи лучше, чем 10 дешёвых.

## Специальные буферы

Ряд педалей — **только буферы**, без звукового эффекта:

| Педаль | Характер |
|--------|---------|
| **Klon Centaur (выключенный)** | Легендарный тёплый буфер |
| **Xotic EP Booster (минимальный gain)** | Мягкий, тёплый характер |
| **TC Electronic Bonafide Buffer** | Прозрачный, студийный |
| **Boss CS-3 (в bypass)** | Стандартный Boss буфер |
| **Dunlop CBM10 Buffered Cable** | Специализированный кабель-буфер |

**Klon Centaur-буфер** стал легендарным: многие считают, что он добавляет «волшебный» тон даже в bypass-режиме.

## Практические рекомендации

### Для коротких цепочек (3–5 педалей, короткие кабели)

True bypass достаточно. Потерь практически нет.

### Для больших педалбордов (10+ педалей, длинные кабели)

1. Поставьте **один качественный буфер в начало цепи** (или используйте педаль с хорошим буфером как первую)
2. Остальные педали — true bypass
3. Германиевый fuzz **перед буфером** (или сразу из гитары)

### Типичная профессиональная схема

```
Гитара → Klon/EP Booster (буфер) → Fuzz → True Bypass педали → Дилей (Boss/buffered) → Ревер → Усилитель
```

### Тест «tone suck»

Сыграйте гитарой напрямую в усилитель → запомните звук.
Затем включите весь педалборд в bypass → сравните.
Если звук «потемнел» — есть tone suck. Причина: либо плохой буфер, либо слишком много true bypass без буфера.

## True Bypass Looper

**True Bypass Looper** — педаль, которая содержит несколько независимых loop'ов с true bypass. Позволяет включать/выключать группы педалей одной кнопкой и создаёт единую входную/выходную точку.

- RJM Mini Amp Gizmo, Loop-Master, Lehle
- Комбинируется с буферизацией всего педалборда

> **Главный вывод:** «true bypass = хорошо» — миф для простых случаев. В профессиональном педалборде оптимально: один хороший буфер в начале + true bypass для остальных, или качественный буферизованный педалборд (Strymon, Boss). Качество буфера важнее самого факта его наличия.
