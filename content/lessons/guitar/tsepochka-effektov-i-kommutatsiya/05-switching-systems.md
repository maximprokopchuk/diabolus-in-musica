# Switching Systems (Системы коммутации)

## Зачем нужны switching systems?

При большом количестве педалей управление ими становится сложным:
- Нужно одновременно включить несколько педалей и выключить другие при смене звука
- Переключение занимает время → паузы в игре
- Риск ошибки при сложных цепочках

**Switching system** решает это: одна кнопка → мгновенная смена комплекса активных педалей (пресет).

## Loop Switcher

### Принцип работы

**Loop Switcher** — устройство с несколькими независимыми loop'ами (петлями). Каждый loop может содержать одну или несколько педалей.

```
Гитара → Loop Switcher
          ├─ Loop 1: [Компрессор] (включён/выключен независимо)
          ├─ Loop 2: [OD1] (включён/выключен)
          ├─ Loop 3: [Distortion] (включён/выключен)
          ├─ Loop 4: [Хорус] (включён/выключен)
          └─ Loop 5: [Дилей] (включён/выключен)
         → Усилитель
```

Нажатие пресет-кнопки одновременно включает/выключает нужные loop'ы.

### Преимущества Loop Switcher

- **Мгновенное переключение** — один нажим, все loop'ы меняются одновременно
- **True Bypass** — выключенные loop'ы полностью отключены от тракта
- **Программируемые пресеты** — сохранить состояние loop'ов как «Bank 1, Preset 2»
- **Порядок педалей в сигнальной цепи** оптимален (педали подключены в нужном порядке внутри loop'ов)

### Классические Loop Switcher

| Устройство | Выходы | Особенности |
|-----------|--------|-----------|
| **Voodoo Lab Hex** | 6 loops | MIDI, программируемый |
| **RJM PBC/6X** | 6 loops | Профессиональный, MIDI |
| **Strymon Zuma + MIDI** | Зависит от версии | Интеграция с педалями Strymon |
| **Loop-Master** | 2–10 loops | Boutique, custom |
| **Lehle D.Loop** | 1–2 loops | Прозрачный, high quality |
| **Gigrig G2** | 8 loops | Модульная система |

## MIDI-управление

### Что такое MIDI?

**MIDI (Musical Instrument Digital Interface)** — стандартный протокол передачи цифровых команд между музыкальными устройствами. Команды MIDI не содержат звука — только инструкции.

**Релевантные MIDI-команды:**
- **Program Change (PC)** — смена пресета (0–127)
- **Control Change (CC)** — управление параметром (0–127)

### MIDI в педалборде

Педали с MIDI-интерфейсом принимают команды и переключают пресеты/эффекты:

```
MIDI Контроллер → [MIDI cable] → Педаль 1 (Strymon Timeline)
                              → Педаль 2 (Strymon BigSky)
                              → Педаль 3 (Line 6 M9)
                              → Loop Switcher (RJM PBC)
```

Нажатие одной кнопки на контроллере → все педали переключаются на свои пресеты одновременно.

### MIDI Thru / Daisy Chain

MIDI-устройства обычно имеют **MIDI IN** и **MIDI THRU**. Сигнал принимается на IN, и ретранслируется на THRU без изменений → можно подключить цепочкой:

```
Controller → In[Педаль 1]Thru → In[Педаль 2]Thru → In[Педаль 3]...
```

### MIDI-контроллеры для педалборда

| Устройство | Особенности |
|-----------|-----------|
| **RJM Master Control** | Профессиональный, полная интеграция |
| **Disaster Area DMC.micro** | Компактный, мощный |
| **Morningstar MC6/MC8** | Гибкий, современный UI |
| **Boss MS-3** | Встроенный switcher + MIDI |
| **Line 6 HX Stomp** | Процессор + MIDI controller |
| **Fractal MFC-101** | Для Axe-Fx, полная интеграция |

## Preset-системы

### Понятие пресета

**Пресет** — сохранённое состояние системы: какие педали включены, какие параметры, какие пресеты загружены на педалях.

Переключение пресета = мгновенная смена всего звука.

### Пресеты в педалях

Многие современные педали поддерживают пресеты:
- **Strymon** (Timeline, BigSky, Mobius) — до 300 пресетов, переключаются MIDI
- **Boss** (многие) — до 128 пресетов
- **Line 6 Helix** — до 1024 пресетов с полным патчем
- **TC Electronic** — через TonePrint app

### Пресеты в Loop Switcher

Loop Switcher хранит состояние loop'ов как пресет:
- Пресет 1 (Чистый): Loop 1 ON (компрессор), Loop 5 ON (дилей)
- Пресет 2 (Ритм): Loop 1 ON, Loop 2 ON (OD), Loop 4 ON (хорус)
- Пресет 3 (Соло): Loop 2 ON, Loop 3 ON (лид drive), Loop 5 ON, Loop 6 ON (буст)

### Integrated Systems (Интегрированные системы)

**Полностью интегрированная система:**

```
MIDI Контроллер → Loop Switcher (пресет) + Педали с пресетами
                  ↑
          Всё синхронизировано: одна кнопка меняет всё
```

**Примеры интегрированных систем:**
- **Line 6 Helix** — всё в одном: процессор + контроллер
- **Fractal Axe-Fx + MFC-101** — пресеты всего сетапа
- **Kemper + Remote** — профилировщик + педальный блок
- **Gigrig G2 + MIDI** — полная модульная система

## Простые vs Сложные системы

### Для начинающих / небольшого педалборда

Не нужна сложная система:
- Педали включаются ногой по отдельности
- Максимум — простой 2–4 loop switcher без MIDI
- Стоимость: $50–150

### Для среднего педалборда (8–15 педалей)

- Loop Switcher на 6–8 loop'ов
- Программируемые пресеты без MIDI
- Возможно, один педальный MIDI-контроллер
- Стоимость: $200–500

### Для профессионального touring сетапа

- Полная MIDI-интеграция
- Loop Switcher + MIDI + Программируемые педали
- Возможно, A/B switch для смены усилителей
- Резервные входы/выходы
- Стоимость: $500–2000+

## A/B / A/B/Y Switchers

**A/B Switcher** — переключает гитару между двумя усилителями:
- A: первый усилитель
- B: второй усилитель

**A/B/Y Switcher** — переключает ИЛИ одновременно:
- A: первый усилитель
- B: второй усилитель
- Y (Both): оба усилителя одновременно

**Педали:** Radial BigShot ABY, Lehle Dual, Boss LS-2.

**Проблема фазы:** при подключении к двум усилителям через Y может возникнуть фазовая отмена. Качественные A/B/Y переключатели (Radial, Lehle) имеют фазовый инвертор на одном из выходов.

## Практические рекомендации

**Начинайте с простого:**
1. Оптимизируйте порядок педалей и питание
2. Добавьте loop switcher только когда смена звуков станет реальной проблемой
3. MIDI — только когда у вас 10+ педалей и сложные смены пресетов

**Документируйте систему:**
- Схема подключений (сигнал + питание + MIDI)
- Список пресетов и что в них входит
- Настройки каждой педали в каждом пресете

> **Принцип минимализма:** самая лучшая система — та, которая работает, не ломается и не отвлекает от музыки. Профессионалы используют сложные системы потому, что их работа требует этого. Для домашних репетиций и небольших концертов — 5–7 педалей, хорошее питание и простой порядок дадут превосходный результат.
