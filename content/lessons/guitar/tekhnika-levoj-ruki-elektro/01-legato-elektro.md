# Легато (электрогитара)

## Что такое легато на гитаре

**Legato** (от итал. «связно») — техника игры, при которой ноты звучат плавно, без разрыва между ними, создавая ощущение единого потока.

На гитаре легато достигается за счёт **hammer-on** и **pull-off** — техник левой руки, при которых звук извлекается без удара медиатором.

## Hammer-On (удар по струне)

**Hammer-on** (буквально «молоточный удар») — нота извлекается резким ударом пальца по ладу без удара медиатором.

### Техника:
1. Медиатор ударяет первую ноту (например, 5-й лад)
2. Следующий палец резко «падает» на следующий лад (7-й лад)
7. Второй лад звучит от удара пальца — без медиатора

**Ключевой момент:** удар должен быть **кончиком пальца**, строго перпендикулярно грифу, максимально близко к ладу (но не на ладе).

### Обозначение в табулатуре:
```
e: 5h7    (5 → hammer-on → 7)
```

### Сила удара:
- Удар должен быть достаточно сильным, чтобы струна прозвучала
- Чем выше усиление (gain) — тем легче звучит hammer-on
- На чистом звуке требуется более сильный удар

## Pull-Off (снятие пальца)

**Pull-off** — обратная техника: нота извлекается резким снятием пальца со струны, слегка «сдёргивая» её в сторону.

### Техника:
1. Зажаты оба лада (например, 7-й и 5-й)
2. Медиатор ударяет 7-й лад
3. Палец на 7-м ладу резко снимается с лёгким отклонением в сторону
4. Звучит 5-й лад — нота от снятия пальца

**Ключевой момент:** снятие пальца — не просто поднятие вверх, а небольшое движение вниз или в сторону (как маленький pull на струне). Без этого нота не прозвучит.

### Обозначение:
```
e: 7p5    (7 → pull-off → 5)
```

## Трели

**Трель** — быстрое чередование двух нот с помощью hammer-on и pull-off:

```
e: 5h7p5h7p5h7p5h7  (трель на 5–7)
```

Трели — важный выразительный приём в блюзе, роке и классике.

**Пример:** открывающая трель в «Eruption» Эдди Ван Халена.

## Legato Runs (легато-пассажи)

**Legato runs** — протяжённые линии, где медиатор ударяет только первую ноту на каждой струне, а остальные звучат через hammer-on/pull-off.

### Пример Am pentatonic legato run:
```
e: ---5h8-----------
B: ------5h8--------
G: ---------5h7-----
D: -----------5h7---
A: -------------5h7-
E: ---------------5-
   Медиатор только на первой ноте каждой струны
```

Это даёт **плавный, певучий** характер линии — в отличие от alternate picking, где каждая нота «укусистая».

**Мастера легато:**
- **Joe Satriani** — плавные, вокальные линии
- **Allan Holdsworth** — экстремальное легато, практически без медиатора
- **Eddie Van Halen** — легато как часть тэппинг-техники
- **Brett Garsed** — современный мастер легато-фьюжн

## Allan Holdsworth — мастер легато

**Аллан Холдсворт** довёл легато до крайности: он играл целые соло почти без удара медиатором. Его подход:

- Медиатор ударяет только первую ноту каждой фразы
- Все остальные ноты — через hammer-on/pull-off
- Длинные, плавные линии через несколько октав
- Уникальный «синтезаторный» характер гитарного звука

Это требует **очень сильной** левой руки и высокого усиления.

## Синхронизация при легато

В легато синхронизация рук работает иначе, чем в picking:

- **С медиатором:** удар и нажатие пальца — одновременно
- **Hammer-on:** нажатие пальца **до** возврата медиатора
- **Pull-off:** снятие пальца выполняет функцию «удара»

**Типичная ошибка:** «мёртвые» ноты при переходе между струнами — старый палец остаётся на предыдущей струне и заглушает новую.

## Упражнения для развития легато

### Упражнение 1: Хроматика легато
```
e: 5h6h7h8  8p7p6p5  (вверх и вниз хроматика)
```
Все ноты — только hammer/pull, без медиатора после первого удара.

### Упражнение 2: Трели на каждом пальце
```
Указательный + средний: 5h6 (p6h5) × 8
Средний + безымянный:   6h7 (p7h6) × 8
Безымянный + мизинец:   7h8 (p8h7) × 8
```

### Упражнение 3: Легато через 3 струны
```
G: 4h5h7
D: 4h5h7  ← медиатор только первый удар
A: 4h5h7
```

> **Совет:** легато на электрогитаре с перегрузом значительно проще, чем на чистом. Начинайте практику с умеренным gain (drive). Когда легато зазвучит чисто на перегрузе, перейдите на более чистый звук — это потребует более точного удара и большей силы пальцев.
