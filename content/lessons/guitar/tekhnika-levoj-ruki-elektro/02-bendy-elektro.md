# Бенды (электрогитара)

## Что такое бенд

**Бенд (bend)** — техника, при которой палец поперечно сдвигает струну, повышая её высоту звука. Это один из самых выразительных приёмов на электрогитаре, придающий ей «певучий», вокальный характер.

Правильный бенд — это **голос гитариста**. По бенду можно определить стиль и характер игры так же верно, как по тембру голоса певца.

## Типы бендов по интервалу

| Тип бенда | Сдвиг высоты | Обозначение |
|----------|-------------|------------|
| **Четверть тона (quarter tone)** | +50 cents | b1/4 |
| **Полутон (semitone)** | +100 cents (1 лад) | b1/2 |
| **Целый тон (whole tone)** | +200 cents (2 лада) | b1 |
| **Полтора тона (tone and a half)** | +300 cents (3 лада) | b3/2 |
| **Два тона (double whole)** | +400 cents | b2 |

Бенд на целый тон — самый частый. Бенды более 1,5 тона требуют лёгких струн (0.09–0.10) и практики.

## Техника выполнения бенда

### Позиция пальцев:
1. **Основной палец** (обычно безымянный) нажимает ноту
2. **Вспомогательные пальцы** (указательный и средний) поддерживают позади на грифе
3. Все три пальца работают **вместе**, как единый рычаг
4. Движение идёт от **запястья и всей кисти**, не только от пальца

### Направление бенда:
- **Вниз** (на нижних струнах D, A, E): сдвиг к полу
- **Вверх** (на верхних струнах G, B, e): сдвиг к потолку
- На 3-й струне (G): можно в обе стороны — выбор зависит от контекста

### Точность интонации:
Бенд на целый тон должен точно попасть на ноту — не ниже, не выше. Проверка: сыграйте ноту, на которую бендите, обычным способом (на 2 лада выше). Теперь бендите — должно звучать так же.

## Pre-Bend (предварительный бенд)

**Pre-bend** — струна сгибается **до** удара медиатором. Гитарист сначала делает бенд в тишине, затем ударяет по уже согнутой струне.

```
Табулатура: (b1)5    ← скобка означает pre-bend
```

**Эффект:** нота сразу звучит на высоте бенда — нет нарастания высоты.

**Применение:** драматические вступления соло, имитация «вою» при «раскрытии» бенда вниз.

## Release Bend (спуск бенда)

**Release bend** — после pre-bend струна медленно опускается обратно на исходную высоту.

```
Табулатура: (b1)5r   ← r = release
```

**Эффект:** высота «стекает» вниз — очень певучий, блюзовый звук.

**Пример:** медленное блюзовое соло, где каждая нота «вздыхает».

## Bend and Release

Сначала бенд вверх, затем отпустить:

```
Табулатура: 7b1r    ← бенд вверх, потом назад
```

Комбинация часто используется в блюзе: нота «поднимается» и «падает» как голос.

## Unison Bend

**Unison bend** — две струны звучат одновременно: одна — открытая нота, другая — бенд до той же высоты. Создаёт напряжённый, «воющий» эффект.

```
e: 5----
B: 5b1  ← B струна бендится до e-5 (A)
```

Обе струны должны звучать **одинаково** — это требует точного бенда.

**Классический пример:** Эрик Клэптон в «Crossroads», Дэвид Гилмор в «Comfortably Numb».

## Pre-bend + Unison

Сочетание: B струна согнута заранее, затем обе ноты берутся вместе.

Это даёт немедленный «удар» unison без нарастания.

## Double Stop Bends

**Double stop bend** — бенд на двух струнах одновременно:

```
G: 7b1
D: 9b1  ← оба пальца бендят на тон вверх
```

Звучит очень мощно, используется в хард-роке и соул-блюзе.

## Знаменитые мастера бенда

| Гитарист | Особенность |
|---------|------------|
| **B.B. King** | Точные, вокальные бенды в верхнем регистре, вибрато после бенда |
| **David Gilmour** | Медленные, певучие, долгие бенды с точной интонацией |
| **Stevie Ray Vaughan** | Тяжёлые струны (0.13!), мощные бенды с огромной динамикой |
| **Eric Clapton** | Выразительные unison бенды, блюзовая интонация |
| **Jimi Hendrix** | Нестандартные бенды, смешение с wah-педалью |
| **Gary Moore** | Эмоциональные бенды с долгим вибрато |

## Распространённые ошибки

| Ошибка | Следствие | Решение |
|--------|---------|---------|
| Бенд одним пальцем без поддержки | Слабый, неточный бенд | Использовать 3 пальца как рычаг |
| Неверная интонация | Фальшиво | Проверять целевую ноту отдельно |
| Слишком резкий бенд | Нет певучести | Контролировать скорость подъёма |
| Бенд без вибрато в конце | Статичный звук | Добавлять вибрато на пике бенда |

> **Упражнение:** медленно бендите на тон, считая до 4 на подъёме и 4 на спуске. Слушайте каждую промежуточную высоту. Потом добавьте вибрато на пике — это основа «поющего» бенда.
