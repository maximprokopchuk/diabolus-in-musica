# Тэппинг (электрогитара)

## Что такое тэппинг

**Tapping** (тэппинг) — техника, при которой пальцы **правой руки** нажимают на лады грифа, извлекая звук через hammer-on. Это расширяет диапазон, позволяет играть очень широкие интервалы и создавать линии, невозможные обычными методами.

Тэппинг — одна из самых зрелищных техник на электрогитаре.

## История тэппинга

Тэппинг существовал в классической гитаре давно, но в рок-музыку его ввёл **Харви Мэндел** и другие в 1970-е. Однако настоящую революцию совершил **Эдди Ван Хален**.

## Eddie Van Halen — «Eruption»

**«Eruption»** (1978, Van Halen) — 1 минута 42 секунды соло, полностью построенного на тэппинге. Это изменило мир рок-гитары навсегда.

**Характеристики техники EVH:**
- Правый указательный (или средний) палец тэппит высокий лад
- Левая рука играет нижние ноты
- Быстрые перемещения по грифу
- Тремоло-бар в сочетании с тэппингом
- Медиатор зажат в зубах или под пальцем

**Ключевые записи с тэппингом:**
- «Eruption» — классика тэппинга
- «Hot for Teacher» — тэппинг в контексте
- «Mean Streets» — уникальный тон через ноготь тэппинг

## Базовая техника тэппинга

### Однопалцевый тэппинг (1-finger tap):

```
Правая рука: T = tap (указательный или средний)
Левая рука: h = hammer-on, p = pull-off

e: T12p8h5  T12p8h5  T12p8h5
   (tap 12-й, pull-off 8-й, hammer-on 5-й)
```

### Как выполнять:
1. Правый палец резко **ударяет** по струне на высоком ладу (как hammer-on)
2. Затем **тянет** струну при отрыве (pull-off к ноте левой руки)
3. Левая рука добавляет ноту через hammer-on или удерживает

### Позиция правой руки:
- Медиатор зажат между большим и указательным — средний тэппит
- Или медиатор между большим и средним — указательный тэппит
- Некоторые держат медиатор в кулаке (EVH-стиль)

## Многопальцевый тэппинг

**Multi-finger tapping** — тэппинг несколькими пальцами правой руки. Позволяет играть полноценные арпеджио и аккорды с использованием обеих рук.

### 8-Finger Tapping (Touch Guitar style):

Все 8 пальцев (по 4 с каждой руки) тэппят по грифу, как по клавишам фортепиано.

**Представители:**
- **Stanley Jordan** — джазовый гитарист, сделал этот стиль своим основным
- **Emmett Chapman** — создатель stick-guitar (специального инструмента)
- **Jeff Loomis** — металлический вариант

### 2-3 пальца правой руки:

```
Правая: T1 (указательный), T2 (средний), T3 (безымянный)

e: T3-15  T2-12  T1-8    (три пальца правой)
           + нижние ноты левой рукой
```

Создаёт широкие арпеджио на 3+ октавы.

## Знаменитые тэппинговые приёмы

### Tapping Harmonics:
Правый палец тэппит точно над 12-м (или 5-м, 7-м) ладом от зажатой ноты — звучит натуральный флажолет.

### Pedal Point Tapping:
Постоянная повторяющаяся нота правой рукой, левая играет мелодию:
```
e: T12-T12-T12-T12  (постоянный тэп)
B: 5-7-8-7          (мелодия левой)
```

### Slide Tapping:
Правый палец не только тэппит, но и скользит по грифу как слайд.

### Tapping Runs (скоростные пассажи):
```
e: T17p12h9  T17p12h9  (скорость = arpeggio sweep speed)
```

## Оборудование для тэппинга

**Звукосниматели:**
- Активные (EMG) дают ровный уровень — хорошо для тэппинга
- Высокий gain помогает тихим hammer-on в верхнем регистре

**Гитара:**
- Низкий action (высота струн) — меньше усилий для тэппинга
- Мягкие, тонкие струны (0.09) — легче тэппить в верхнем регистре

**Усилитель:**
- Умеренный gain — не слишком много (грязь) и не мало (тихо)
- Компрессор помогает выровнять динамику

## Упражнения для начинающих

### Упражнение 1: Простой тэп-паттерн
```
e: T12p8h5  (начните с 60 BPM, по 8 раз)
Правая: tap 12-й
Левая: pull 8-й, hammer 5-й
```

### Упражнение 2: Скольжение тэппинга
```
e: T12p8h5  T14p10h7  T15p12h9
(перемещение тэппинга по три лада вверх)
```

### Упражнение 3: Двухструнный тэппинг
```
e: T12p9
B: T10p7
(чередование двух струн правой рукой)
```

> **Практический совет:** начните тэппинг с медленного паттерна, добиваясь, чтобы все три ноты (тэп, pull-off, hammer-on) звучали одинаково громко. Самая частая проблема — pull-off звучит тише. Тренируйте отдельно: тэп и сразу резкий pull с небольшим отклонением струны.
