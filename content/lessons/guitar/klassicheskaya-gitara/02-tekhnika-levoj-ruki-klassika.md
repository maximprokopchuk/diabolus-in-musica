# Техника левой руки (классика)

## Роль левой руки

Левая рука в классической гитаре выполняет три функции:
1. **Точность:** ноты должны звучать чисто, без призвуков
2. **Легато:** плавные переходы между нотами
3. **Экономия:** минимальные движения для максимального контроля

## Классическая постановка

### Большой палец:
- Располагается **позади грифа**, примерно напротив 2-го или 3-го пальца
- Не «обхватывает» гриф сверху (как в рок-технике)
- Это позволяет пальцам нажимать ноты вертикально, с максимальной силой

### Пальцы:
- Нажимают **кончиками** (подушечками + ноготь, но ноготь левой руки — коротко стрижен)
- Дуга пальца — достаточная для того, чтобы не касаться соседних струн
- Нажатие — **сразу позади лада** (металлической порожки), не на сам лад и не далеко от него

### Запястье:
- Слегка выдвинуто вперёд (от корпуса гитары), создавая пространство для пальцев
- Не касается грифа (в большинстве техник)

### Локоть:
- Свободно опущен, не прижат к телу
- Создаёт «угол» для удобного положения пальцев

## Нумерация пальцев

| Номер | Палец |
|-------|-------|
| 1 | Указательный |
| 2 | Средний |
| 3 | Безымянный |
| 4 | Мизинец |

В нотах аппликатура левой руки обозначается **цифрами** (без букв).

## Легато (Hammer-on и Pull-off)

### Hammer-on (лига восходящая):
Звук извлекается **ударом пальца левой руки** без участия правой:

```
Нотация: лига от нижней ноты к верхней
Таб:     7h9  (7-й лад, хаммер на 9-й)
```

**Техника:**
1. Правая рука извлекает нижнюю ноту
2. Следующий палец левой руки **резко ударяет** по следующему ладу
3. Сила удара должна быть достаточной для самостоятельного звука

### Pull-off (лига нисходящая):
Звук извлекается **снятием (срывом) пальца** левой руки:

```
Нотация: лига от верхней ноты к нижней
Таб:     9p7  (9-й лад, pull-off на 7-й)
```

**Техника:**
1. Оба пальца уже стоят (верхний и нижний)
2. Верхний палец **срывает** струну внутрь ладони при снятии
3. Нижний палец уже держит струну → она звучит

### Комбинация (слур):
```
7h9p7h9p7... — трель
```

## Вибрато в классической гитаре

В классической гитаре используется **боковое вибрато** (вдоль струны), а не вертикальное:

- Палец нажат на лад
- Рука движется вперёд-назад **вдоль струны** (не перпендикулярно)
- Это изменяет натяжение → высоту ноты

Скорость и глубина вибрато — часть выразительности.

## Барре

**Барре** — указательный палец прижимает все (или несколько) струн одновременно, заменяя «нулевой порожек».

### Полное барре (Grand barré):
Все 6 струн прижаты одним пальцем.

```
C.VII (барре на 7-м ладу) = Am в позиции с открытыми струнами, но на 7-й позиции
```

### Полубарре (demi-barré):
2–4 струны прижаты указательным.

### Техника барре:
| Проблема | Решение |
|---------|---------|
| Некоторые струны не звучат | Прижмите палец ближе к металлическому порожку |
| Болит рука | Проверьте угол указательного пальца; давите ребром, не плашмя |
| Нет силы | Большой палец позади грифа = рычаг; подтягивайте шею к себе |
| Нечистый звук | Проверьте, что другие пальцы не касаются струн барре |

## Принцип экономии движений

Каждый раз, когда вы снимаете палец с лада, рука тратит время и энергию на его повторную постановку. **Принцип экономии:**

- Не снимайте пальцы без необходимости
- Если нота повторяется позже — оставьте палец на ладу
- Подготавливайте следующую ноту, пока звучит предыдущая

Этюды для отработки левой руки:
- Сеговия: хроматические упражнения
- Villa-Lobos: Étude №1 (арпеджио с фиксированными пальцами)
- Giuliani: 120 arpeggio studies Op.1

> **Постановка левой руки — это не «правило для всех».** Анатомия у каждого разная: длина пальцев, гибкость суставов, форма ладони. Классическая постановка — не догма, а **оптимальная отправная точка**, проверенная столетиями. Адаптируйте под себя, но только осознанно, с пониманием последствий.
