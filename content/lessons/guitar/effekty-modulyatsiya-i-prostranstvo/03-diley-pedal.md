# Дилей (педаль)

## Что такое дилей?

**Дилей (Delay)** — эффект задержки: педаль воспроизводит повтор (repeat) исходного сигнала через заданный промежуток времени. От простых одиночных повторов до бесконечных нарастающих слоёв — дилей один из самых универсальных и музыкальных эффектов.

### Параметры дилея

| Параметр | Функция |
|---------|---------|
| **Time / Delay** | Время задержки (мс) — как далеко «эхо» |
| **Repeats / Feedback** | Количество повторов (при максимуме — бесконечно) |
| **Mix / Wet** | Уровень эффекта относительно оригинала |
| **Subdivisions** | Ритмическое деление задержки (четверть, восьмая, triplet) |
| **Tap Tempo** | Установка времени в ритме («топ» ногой) |

---

## Типы дилея

### Tape Delay (ленточный)

**Принцип:** физическая магнитная лента движется мимо головки записи и головки воспроизведения. Расстояние между головками определяет время задержки.

**Звуковой характер:**
- Тёплый, «тёмный», с постепенным ухудшением качества повторов
- Каждый повтор — чуть темнее и «ватнее» предыдущего
- Лёгкое pitch-wobble (колебание высоты) от неравномерного движения ленты
- Органичное, «живое» ощущение

**Оригинальные устройства:**
- **Maestro Echoplex EP-3** — легендарный, использовался Hendrix, Van Halen
- **Roland Space Echo RE-201** — культовый, богатый звук
- **Watkins Copicat** — британский вариант

**Современные педали-эмуляции:**
- **Boss RE-20 Space Echo** — точная эмуляция RE-201
- **Strymon El Capistan** — профессиональный, студийный уровень
- **EHX Deluxe Memory Man** (с tape-character)
- **Catalinbread Belle Epoch** — эмуляция Echoplex EP-3

### Analog Delay (аналоговый)

**Принцип:** задержка создаётся через цепочку конденсаторов (Bucket Brigade Device, BBD). Сигнал «передаётся» от конденсатора к конденсатору.

**Звуковой характер:**
- Тёплый, но не такой тёмный, как tape
- Повторы постепенно «тают» — естественное затухание
- Короткий максимальный delay time (обычно до 600–1000 мс)
- Лёгкая модуляция добавляет «живость»

**Классические педали:**
- **MXR Carbon Copy** — стандарт аналогового дилея
- **Boss DM-2 / DM-2W** — культовый Boss analog delay
- **Electro-Harmonix Memory Man** — тёплый, атмосферный
- **Way Huge Aqua-Puss** — компактный, качественный
- **TC Electronic Gauss Tape Echo** — современный

### Digital Delay (цифровой)

**Принцип:** сигнал оцифровывается, хранится в памяти и воспроизводится через заданное время. Точность — полная.

**Звуковой характер:**
- Повторы идентичны оригиналу — «стерильный», «холодный»
- Большой диапазон delay time (до нескольких секунд)
- Точные subdivisions (восьмые, шестнадцатые, триоли)
- Идеально для ритмически точных паттернов

**Классические педали:**
- **Boss DD-3 / DD-7 / DD-8** — стандарт, надёжный
- **TC Electronic Flashback** — широкие опции, toneprint
- **Strymon Timeline** — флагман, бесчисленные алгоритмы
- **Line 6 DL4** — культовый, часто используется как лупер

## Subdivisions (деления)

Ритмическая синхронизация дилея с темпом — ключ к профессиональному использованию.

| Деление | BPM 120 → время мс |
|---------|------------------|
| **Четверть (1/4)** | 500 мс |
| **Восьмая (1/8)** | 250 мс |
| **Шестнадцатая (1/16)** | 125 мс |
| **Пунктир восьмая (dotted 1/8)** | 375 мс |
| **Триоль четверти** | 333 мс |

**Dotted eighth delay** (пунктирная восьмая) — классический «U2-delay» Эджа. Повторы создают синкопированный ритмический рисунок поверх прямого бита.

**Формула расчёта:** Time (ms) = 60000 / BPM × subdivision_factor

---

## Self-Oscillation

**Self-oscillation** («самовозбуждение») — когда Feedback поднят до максимума, повторы начинают нарастать и усиливаться, создавая бесконечный нарастающий шум — «лавину».

Используется как **выразительный приём**:
- Резкое включение самовозбуждения перед следующим куплетом
- Построение нарастающей «волны» в конце песни
- Шумовые экспериментальные текстуры

**Важно:** педали различаются по поведению при максимальном feedback. Аналоговые при самовозбуждении дают более «музыкальный» результат; цифровые могут дать резкий неприятный шум.

## Классические приёмы использования дилея

### Slapback Delay

**Время:** 50–120 мс, **1 повтор**, Feedback минимальный.
Характерный звук рокабилли и кантри 1950-х. «Слышимое эхо» с одним коротким повтором.

### Ambient Delay

**Время:** 400–1200 мс, **много повторов**, Mix высокий.
Создание атмосферных «облаков» звука. Характерно для пост-рока, прогрессива, ambient.

### Rhythmic Delay (ритмический)

**Время:** синхронизировано с BPM, **3–5 повторов**, Mix умеренный.
Повторы становятся частью ритмического рисунка. The Edge (U2) — классический пример.

### Infinite Loop / Freeze

Feedback 100% = бесконечный петлинг ноты. Педаль Strymon Timeline / Empress Echosystem имеют «Freeze» режим — нажатие кнопки «замораживает» текущий повтор бесконечно.

## Позиция в цепи

Обычно дилей ставится **после модуляции, перед ревером**:
```
Гитара → Drive → Модуляция (хорус/фейзер) → Delay → Reverb → Усилитель
```

Дилей в **FX Loop** (эффект-петля усилителя) предпочтительнее для очистки: усилитель перегружает исходный сигнал, дилей создаёт чистые повторы.

> **Совет по использованию:** начните с Tap Tempo синхронизированным с темпом трека. Dotted eighth delay с 3 повторами — универсальная настройка, которая работает в большинстве музыкальных контекстов и создаёт ощущение «ширины» без загромождения.
