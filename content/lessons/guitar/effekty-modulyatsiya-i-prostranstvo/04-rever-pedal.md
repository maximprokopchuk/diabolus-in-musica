# Ревер (педаль)

## Что такое ревер?

**Ревербератор (Reverb)** — эффект, имитирующий **акустику пространства**. Когда звук отражается от стен, пола и потолка, мы слышим множество задержанных копий, сливающихся в диффузный «хвост». Ревер добавляет ощущение пространства, объёма и «воздуха».

### Параметры ревера

| Параметр | Функция |
|---------|---------|
| **Decay / Reverb Time** | Длина хвоста (как долго «отражения» слышны) |
| **Pre-delay** | Задержка перед началом ревера (создаёт ощущение расстояния) |
| **Mix / Wet** | Соотношение оригинала и эффекта |
| **Tone / Damping** | Яркость хвоста (высокое Damping = тёмный хвост) |
| **Size** | Виртуальный размер «помещения» |

## Типы ревера

### Spring Reverb (пружинный)

**Принцип:** сигнал проходит через металлическую пружину; волны движутся по пружине и отражаются — создаётся характерный «drippy», «плещущий» звук.

**Звуковой характер:**
- Характерный «splashy», «капельный» звук
- Узнаваемый «boing» при встряске
- Тёплый, ретро-характер

**Оригиналы:** встроены в усилители Fender (Twin Reverb, Deluxe Reverb). Fender Spring Reverb — один из самых узнаваемых гитарных звуков.

**Применение:** кантри, сёрф-рок, рокабилли, блюз.

**Педали:**
- **Boss FRV-1 63 Fender Reverb** — официальная Fender spring sim
- **Strymon Flint** (spring + tremolo)
- **EarthQuaker Devices Afterneath** (ambient, не совсем spring)

### Hall Reverb (концертный зал)

**Принцип:** имитация акустики концертного зала — большое пространство, длинный хвост, ранние отражения.

**Звуковой характер:**
- Длинный, «торжественный» хвост
- Богатые ранние отражения
- Создаёт ощущение большого пространства

**Применение:** оркестровая музыка, баллады, медленный рок, торжественные моменты.

**Пресеты:** в большинстве цифровых педалей есть Hall пресет.

### Plate Reverb (пластинный)

**Принцип:** оригинально — большая металлическая пластина (plate) в звукоизолированном боксе. Звук передаётся на пластину, волны бегут по ней и записываются пьезодатчиками.

**Звуковой характер:**
- Гладкий, «сливочный» хвост
- Плотный, без ранних отражений
- Яркий, детальный

**Применение:** стандарт студийного продюсирования (EMT 140 — легендарная пластинная реверберация). Вокал, барабаны, гитары в поп и роке.

**Педали:** большинство цифровых содержат Plate алгоритм.

### Room Reverb

**Принцип:** имитация небольшой комнаты — студии, репетиционного пространства.

**Характер:**
- Короткий хвост
- Чёткие ранние отражения
- Добавляет «живость» без размытости

**Применение:** «natural» гитарный звук, когда нужно немного пространства без явного эффекта.

### Shimmer Reverb

**Принцип:** к стандартному хвосту ревера добавляется **октавный сдвиг вверх** (pitch shift +1 octave), который тоже реверберирует — создаётся бесконечно нарастающий, «ангельский» звук.

**Характер:**
- «Небесный», «парящий»
- Синтетический, нереальный
- Может нарастать до бесконечности

**Применение:** ambient, post-rock, worship music, пост-метал.

**Педали:**
- **Strymon BigSky** — лучший shimmer на рынке
- **EarthQuaker Devices Afterneath** — уникальный алгоритм
- **TC Electronic Hall of Fame 2** — доступный shimmer

### Ambient / Infinite Reverb

**Принцип:** хвост ревера продолжается практически бесконечно. Нажатие педали «замораживает» текущий хвост.

**Характер:**
- Создаёт «облака» звука
- Нота превращается в текстуру
- Слои аккумулируются

**Педали:**
- **Strymon BigSky** (Bloom режим)
- **EHX Cathedral** — доступный, множество алгоритмов
- **Boss RV-6 / RV-500**

## Применение ревера в разных контекстах

| Контекст | Тип ревера | Настройки |
|---------|-----------|-----------|
| **Clean blues** | Spring | Decay короткий-средний, Mix 20–30% |
| **Classic rock** | Spring / Room | Умеренный, не перегружать |
| **Ballad, slow rock** | Hall / Plate | Длинный decay, Pre-delay 30–60 мс |
| **Metal (lead)** | Room / Short Hall | Короткий, только для «живости» |
| **Ambient / Post-rock** | Shimmer / Infinite | Длинный, высокий Mix |
| **Country** | Spring | Classic Fender spring sound |

## Ревер и дилей вместе

Ревер и дилей часто используются одновременно, но с соблюдением правила:

**Порядок:** Delay → Reverb

Если поставить Reverb → Delay, повторы дилея будут нести в себе реверберированный сигнал — звук станет «мутным» и «загроможденным».

**Pre-delay** (задержка перед хвостом ревера) — важный параметр для «пространства». 30–50 мс Pre-delay не дают хвосту ревера «слипаться» с исходным сигналом, создавая ощущение расстояния.

> **Практический совет:** начните с короткого Spring Reverb (Mix 20–25%) — это добавит «живости» звуку без явного «пространства». По мере роста уверенности добавляйте Hall или Plate для баллад. Shimmer — только в точно выверенных местах, не «всегда включён».
