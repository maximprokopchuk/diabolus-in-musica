# Хроматические приёмы

## Хроматика в импровизации

**Хроматические ноты** — ноты вне текущего лада. При грамотном использовании они добавляют «перчинку» и джазовый оттенок.

Правило: хроматические ноты **разрешаются** в диатонические (ладовые) или аккордовые тоны.

## Хроматический подход (chromatic approach)

Подход к целевой ноте **на полутон** снизу или сверху:

```
Целевая нота G (квинта C):
Снизу: ...F# → G  (F# — хроматический подход)
Сверху: ...Ab → G  (Ab — хроматический подход)
```

На гитаре: просто сыграйте ноту на 1 лад ниже (или выше) целевой и «соскользните» в неё.

## Энклозура (enclosure)

**Энклозура** — целевая нота «окружается» нотами сверху и снизу:

```
Целевая G:
Верхняя энклозура: Ab — F# → G  (сверху хром. + снизу хром.)
Диатоническая:     A — F → G    (сверху диат. + снизу диат.)
Смешанная:         Ab — F → G   (сверху хром. + снизу диат.)
```

Энклозура — один из самых характерных приёмов бибопа.

## Хроматические проходящие ноты

Заполнение промежутков между диатоническими нотами хроматически:

```
Диатонически: C — D — E
С хроматикой: C — C# — D — D# — E
```

На гитаре: просто идите по ладам последовательно.

## Блюзовая хроматика

В блюзе хроматика — часть стиля:

```
Блюзовый ход в Am:
A — C — C# — D — E (♭3 → 3 → 4 → 5)
          ^ blue note / хроматический проход
```

## Когда использовать хроматику

| Ситуация | Подходит? |
|----------|-----------|
| На слабой доле, мимоходом | Да — звучит естественно |
| На сильную долю, быстро разрешая | Да — создаёт напряжение |
| На сильную долю, задерживая | Осторожно — может звучать «мимо» |
| Несколько хроматических нот подряд | Да, если есть ясная цель (целевая нота) |

> **Правило одного лада:** если хроматическая нота звучит «неправильно» — вы просто на 1 лад от «правильной». Сдвиньтесь на полтона в любую сторону, и вы дома.
