# Педальная сталь на гитаре

## Что такое педальная сталь

**Педальная сталь-гитара** (pedal steel guitar) — горизонтальный инструмент, где металлический стержень (bar) скользит по струнам. Педали и колени-рычаги меняют строй отдельных струн во время игры, создавая характерные бенды и «плавающий» тон.

Звук педальной стали — **определяющий** для кантри-музыки. Обычная гитара может имитировать его через:
- Унисон-бенды
- Предбенды
- Открытые строи
- Слайд-технику

## Унисон-бенд (Unison Bend)

**Унисон-бенд** — взять одну ноту стандартно + ту же ноту бендом снизу. Обе звучат одновременно, создавая «стальной» эффект.

### Пример на G major:

```
B: 5 (нота D, открытая) — не двигать
G: 4 → 5 (бенд C → D, на тон вверх)
```

Обе ноты — D, но нижняя достигает нужной высоты через бенд.

**Ещё пример:**
```
e: 5 (нота A) — держать
B: 3 → 5 (бенд D → E на тон вверх)
```

Когда бенд достигает высоты верхней ноты, звучит унисон — затем отпустить медленно.

## Pre-bend (предбенд)

**Pre-bend** — согнуть ноту **до** удара правой рукой, затем постепенно отпустить вниз.

```
G: 5 (бенд делается ДО удара)
Правая рука ударяет — нота уже «вверху»
Медленно отпустить вниз → звучит как «плач» стальной гитары
```

Это ключевой приём педальной стали на обычной гитаре.

## Открытые строи для steel-звука

### Open G (Открытый G):

```
Стандарт: E A D G B e
Open G:   D G D G B D
```

Аккорд G major звучит на открытых струнах. Слайдом (стержнем, трубкой или бутылкой) можно играть в духе настоящей стальной гитары.

**Используют:** Ry Cooder, Keith Richards (Stones), Duane Allman, Bonnie Raitt

### Open D:

```
Open D: D A D F# A D
```

Мажорный аккорд без зажатия. Слайдер + Open D = Blues/Country звук.

### Open E:

```
Open E: E B E G# B E
```

Аналог Open D, но на тон выше. Используется в блюзе и кантри.

## Ключевые приёмы имитации pedal steel

### Chord bend (аккордовый бенд):

Взять неполный аккорд и согнуть одну или две ноты:

```
G аккорд: (D-G-B открытые струны)
Бенд G-струны: 0 → полтон вверх (G# implication)
Или бенд D-струны: 0 → G → G# (полутоновые изменения)
```

### Triple stop с бендом:

```
e: 3  (G)
B: 3  (D)
G: 4 → 5  (бенд C → D)
```

Все три звучат, нижняя «подтягивается» к нужной высоте.

### Slow release (медленный отпуск):

После бенда **медленно** отпустить — нота «опускается» как в pedal steel.

## Музыканты — мастера steel-имитации

| Музыкант | Инструмент / стиль | Характерный приём |
|---------|-------------------|------------------|
| **Clarence White** | Telecaster + B-bender | Механический b-bender (педаль в гитаре) |
| **James Burton** | Telecaster (Elvis) | Ручные бенды и унисоны |
| **Danny Gatton** | Telecaster | Slide + fingerstyle |
| **Vince Gill** | Разные | Плавные предбенды |
| **Brad Paisley** | Telecaster | Комбо слайд + regular |

## B-bender (палочка-рычаг)

**B-bender** — механизм внутри Telecaster (или другой гитары), позволяющий поднять B-струну на целый тон нажатием на ремень гитары плечом.

Это **механическая имитация** педальной стали. Clarence White первым разработал такой механизм.

Современный вариант: **G-bender** (та же идея для G-струны).

> **Практический совет:** Начните с unison bends на сложений B+G-строках. Это самый «читаемый» steel-приём. Затем освойте pre-bend с медленным отпуском — это придаст вашему кантри-звуку подлинную «steel guitar» атмосферу. Слушайте записи педальной стали (Lloyd Green, Buddy Emmons) и старайтесь копировать именно фразировку, а не отдельные ноты.
