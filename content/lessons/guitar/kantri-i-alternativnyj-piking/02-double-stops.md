# Двойные ноты (Double Stops)

## Что такое double stops

**Double stops** — одновременное звучание **двух нот** на гитаре. Название пришло из скрипичной техники: «стоп» = прижатая струна.

В кантри double stops — основа мелодического языка. Они создают плотный, «деревенский» звук, напоминающий банджо, фидл или мандолину.

## Терции (Thirds)

**Терции** — самый распространённый double stop в кантри. Интервал в три ступени гаммы.

### Терции в G major на струнах B и e:

| Лады | B-струна | e-струна | Ноты |
|------|---------|---------|------|
| 0-0 | B | E | B-E (терция в G) |
| 1-0 | C | E | C-E (большая) |
| 0-2 | B | F# | B-F# (малая) |
| 2-3 | C# | G | — (уменьшённая) |
| 3-3 | D | G | D-G (малая) |
| 5-5 | E | A | E-A (малая) |

### Диатонические терции вверх по грифу (G major, струны 2-1):

```
Лады: 0-0 / 1-0 / 3-0 / 3-2 / 5-3 / 5-5 / 7-5 / 7-7
```

Играйте вверх и вниз — это базовый кантри-мелодический паттерн.

## Сексты (Sixths)

**Сексты** — интервал в шесть ступеней. Более «певучий» и «сладкий» звук, чем терции.

В кантри и блюзе сексты часто играются на **несмежных струнах** (через одну):

### Сексты в G major на струнах G и e:

| Лады (G / e) | Ноты | Тип |
|-------------|------|-----|
| 0 / 0 | G — E | большая секста |
| 2 / 0 | A — E | малая |
| 4 / 0 | B — E | малая |
| 5 / 3 | C — G | большая |
| 7 / 5 | D — A | большая |

**Паттерн «двигающиеся сексты»** — любимый приём Duane Allman, Chuck Berry:
```
G5:   0-0 (G-e)
A5:   2-0
B5:   4-2  ← (или с бендом!)
C:    5-3
D:    7-5
```

## Октавы

Октавы — double stop из одной ноты в двух регистрах. Сделал знаменитым Wes Montgomery в джазе, но в кантри тоже используется:

```
G: e-3 + D-5 (пропустить A-струну)
A: e-5 + D-7
```

Играются медиатором (по обеим нотам) или медиатор + палец.

## Характерные кантри-паттерны с double stops

### «Banjo roll» с double stops:
```
G —— B —— D —— G —— B
(медиатор, палец, медиатор, палец, медиатор)
```

### Double stop bend (ключевой приём кантри):

Взять две ноты, согнуть **нижнюю** вверх на полтон или тон, пока верхняя остаётся:
```
B:  3  (неподвижная)
G:  4  → 5  (бенд вверх на полтон)
```
Создаёт характерный «steel guitar» эффект — имитация педальной стали.

### Лики с двойными нотами:

**Классический кантри-лик:**
```
e: —5—3——————
B: ——————5—3—
G: ——————————
Смена струн на каждую долю = движение вниз по грифу
```

## Double stops и гармония

| Интервал | Характер | Применение |
|---------|---------|-----------|
| Унисон | Плоский, «органный» | Педаль-эффект |
| Большая/малая терция | Плотный, певучий | Мелодии |
| Кварта | Открытый, сильный | Рифы |
| Квинта | Пустой, «ковбойский» | Power chords |
| Большая/малая секста | Сладкий, лирический | Медленные части |
| Октава | Чистый, усиленный | Мелодическое соло |

> **Совет:** Диатонические терции и сексты — это «готовые мелодии». Возьмите любую известную кантри-мелодию и сыграйте её **параллельными терциями** — услышите классический фидл-звук. Brad Paisley часто использует этот приём для аранжировки «country twin guitars» одной гитарой.
