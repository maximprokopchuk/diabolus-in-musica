# Кантри-бендинг

## Бендинг в кантри vs блюзе

В блюзе бенды «плачут» — они выражают боль и грусть. В кантри бенды **«светлые»** — они создают прыгающий, игривый, «сияющий» характер. Принципиальное различие — в использовании **мажорной пентатоники** и мажорных интервалов.

| Параметр | Блюз-бенд | Кантри-бенд |
|---------|---------|------------|
| Лад | Minor pentatonic | Major pentatonic |
| Характер | Тёмный, «плачущий» | Светлый, игривый |
| Таргет-нота | Малая терция, септима | Большая терция, секста |
| Вибрато | Медленное, «тяжёлое» | Быстрое или отсутствует |

## Мажорная пентатоника в кантри

**Мажорная пентатоника** (1-2-3-5-6) — основа кантри-соло:

### G мажорная пентатоника на грифе:
```
e: ——3—5——
B: ——3—5——
G: ——2—4——
D: ——5——
A: ——2—5——
E: ——3—5——
```

Отличие от G minor пентатоники: **большая терция (B)** вместо малой (Bb) и **большая секста (E)** вместо малой (Eb).

## Ключевые кантри-бенды

### Бенд на большую терцию:

Из второй ступени (A) → большая терция (B):
```
G: 2 → 4 (целый тон вверх, из A в B)
```
Или из флатовой терции → большая:
```
G: 3 → 4 (из Bb → B, полутон)
```

### Бенд на большую сексту:

```
G: 4 → 5 (из C → D — квинта, не секста)
```
Более точно: бенд по гамме мажорной пентатоники.

### Бенд на мажорный нонаккорд («Nashville sound»):

```
B: 5 (нота E)
G: 4 → 5 (бенд C → D) + звучание с E = major 2nd / 9th
```

## Prebend (предбенд)

**Prebend** — согнуть струну **до** удара. Услышан звук «падает» с высоты, а не поднимается.

```
G: прижать лад 4, согнуть до высоты лада 6
   ударить правой рукой (звучит нота E, хотя держите на 4-м)
   медленно опустить вниз до нормального лада 4 (нота C)
```

Этот «нисходящий скольжение» — квинтэссенция steel-guitar имитации.

## Unison Bend (унисон-бенд)

Уже рассматривали в контексте steel-guitar. В кантри-лике:

```
e: 5 (нота A) — неподвижно
B: 3 → 5 (бенд D → E) — нет, до A:
   правильно: B: 5 → 7 (бенд E → F#? нет)
```

Классика:
```
B: 5 (E) — держать
G: 5 → 6 (D# → E — унисон с E)
```

Когда G-струна достигает E — унисон. Медленно отпустить.

## Классические кантри-лики

### Лик 1 (G major, простой):
```
Темп = 120-140
e: ———5—3————————
B: ——————5—3—5b6r5—
G: ——————————————4—
```
(b = bend, r = release)

### Лик 2 (A position, double stop bend):
```
B: 2 (неподвижно, нота C#)
G: 2 → 4 (бенд A → B)
Результат: B + B = унисон → обратно C# + A (малая терция)
```

### Лик 3 (Brad Paisley style — быстрый):
```
Hammer-on + chicken pick + bend:
e: h5—————5———
B: ——3—5——3—5b6—
G: —————4——————
Медиатор + средний палец (hybrid)
```

## Вибрато в кантри

В кантри вибрато **быстрое и узкое** — или отсутствует на быстрых нотах. На длинных нотах используется «сердечное» вибрато с движением кисти:

```
Медленный темп: умеренное широкое вибрато (после бенда)
Быстрый темп: без вибрато или очень короткое
```

## Ключевые музыканты

| Музыкант | Фирменные бенды | Стиль |
|---------|----------------|-------|
| **Brad Paisley** | Chicken pick + prebend | Современный виртуозный |
| **Brent Mason** | Double stop bends | Studio master |
| **Albert Lee** | Фигурации пентатоники | Классический кантри-рок |
| **Vince Gill** | Лиричные бенды | Лирический кантри |
| **Buck Owens** | Telecaster twang + бенды | Bakersfield sound |

> **Практический совет:** Возьмите G мажорную пентатонику и попробуйте **каждую ноту согнуть** из соседней снизу. Это даст вам «словарь» базовых кантри-бендов. Затем добавьте chicken picking — снэп-удары для характерного звука. Слушайте Brad Paisley «Throttleneck» — там весь язык кантри-бендинга в концентрированном виде.
