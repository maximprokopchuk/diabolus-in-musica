# Аранжировка для гитары

## Гитара как оркестр

Гитара уникальна тем, что может одновременно играть **бас, гармонию и мелодию** — быть «мини-оркестром». Это основа **fingerstyle** и **chord melody** аранжировок.

## Fingerstyle аранжировка

### Три слоя

```
Слой 1 (бас):     Большой палец — струны 4, 5, 6
Слой 2 (гармония): Средние пальцы — струны 2, 3, 4
Слой 3 (мелодия):  Указательный/безымянный — струны 1, 2
```

### Пример: простая аранжировка «Happy Birthday» (фрагмент)

```
    C                    G7
e|------0------0---|------1------1---|
B|----1------1-----|----0------0-----|
G|--0------0-------|--0------0-------|
D|-----------------|-----------------|
A|-3-----------3---|---------2-------|
E|-----------------|--3--------------|
```

Бас (нижние струны) + мелодия (верхние) + гармония (средние) — три голоса одновременно.

## Chord Melody

**Chord melody** — техника, где мелодия встроена **в верхний голос аккордов**. Каждая нота мелодии гармонизируется аккордом.

### Принцип

1. Запишите мелодию
2. Определите гармонию (аккорды)
3. Для каждой ноты мелодии найдите воицинг аккорда, в котором эта нота — **верхняя**

### Пример: мелодия C — D — E — F над Cmaj7

```
Мелодия C:     Мелодия D:     Мелодия E:     Мелодия F:
e|--8--|       e|--10-|       e|--12-|       e|--13-|
B|--8--|       B|--8--|       B|--8--|       B|--8--|
G|--9--|       G|--9--|       G|--9--|       G|--9--|
D|--10-|      D|--10-|       D|--10-|       D|--10-|
```

Аккорд Cmaj7 в разных обращениях, мелодия всегда наверху.

## Техники для аранжировки

| Техника | Применение |
|---------|-----------|
| **Harp harmonics** | Колокольчики, интерлюдии |
| **Percussive slap** | Ритмическая перкуссия корпусом |
| **Natural harmonics** | Эфирные звуки, интро/аутро |
| **Тэппинг** | Полифония, быстрые пассажи |
| **Alternate tunings** | Расширенный диапазон, педальные ноты |

## Шаги создания аранжировки

1. **Выучите мелодию** — пойте и играйте
2. **Определите гармонию** — аккорды, функции
3. **Найдите тональность** — удобную для гитары (с открытыми струнами)
4. **Распределите голоса** — бас на 4–6 струны, мелодию на 1–2
5. **Добавьте гармонию** — заполните средние струны
6. **Упростите** — уберите лишнее, оставьте самое важное

## Мастера гитарной аранжировки

| Гитарист | Стиль |
|----------|-------|
| Tommy Emmanuel | Fingerstyle, кантри, попурри |
| Chet Atkins | Fingerstyle, chord melody |
| Joe Pass | Джазовый chord melody |
| Andy McKee | Современный fingerstyle, тэппинг |
| Sungha Jung | Поп-аранжировки |

> **Начните просто:** возьмите знакомую мелодию, сыграйте её на 1-й и 2-й струне, добавьте бас большим пальцем на сильные доли. Это уже аранжировка! Постепенно усложняйте, добавляя средние голоса.
