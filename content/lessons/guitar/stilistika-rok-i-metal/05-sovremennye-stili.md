# Современные стили

## Djent

**Djent** — субжанр прогрессивного метала, возникший в начале 2000-х. Название — звукоподражание: «djent» — характерный звук приглушённой низкой струны при мощном перегрузе.

### Характеристики djent

| Параметр | Описание |
|---------|---------|
| **Строй** | Drop A, Drop G, Drop F (очень низко) |
| **Гитары** | 7- и 8-струнные (нижние строи) |
| **Ритмика** | Полиритмия, «синкопированные» пауэр-аккорды |
| **Palm mute** | Экстремальное PM — основа ритмического звука |
| **Продакшн** | Огромный, сжатый звук, чёткие барабаны |

### Пионеры djent

| Группа / Музыкант | Вклад |
|------------------|-------|
| **Meshuggah** | «Изобретатели» — полиритмия на 8-струнных |
| **Periphery** | Современный djent с мелодическими хорусами, Misha Mansoor |
| **Tesseract** | Лирический djent, мелодический вокал, атмосфера |
| **Animals as Leaders** | Инструментальный djent, Tosin Abasi — 8-струнная гитара |
| **Bulb (Misha Mansoor)** | Создал звук и эстетику djent как явления |

### Ритмические паттерны djent

```
Типичный 4/4 djent паттерн:
16th: 1-e-and-a 2-e-and-a 3-e-and-a 4-e-and-a
PM:   PM PM ---  PM PM ---  PM --- PM  PM PM ---
      X  X   o   X  X   o   X   o  X   X  X   o
```

«o» = акцентированный открытый удар, «X» = palm-muted. Синкопа создаёт характерный пульс.

## Math Rock

**Math rock** — жанр, возникший в конце 1980-х. Акцент на **инструментальной технике, ритмической сложности и нестандартных размерах**. Часто без вокала.

### Характеристики

| Параметр | Описание |
|---------|---------|
| **Гармония** | Пост-тональная, часто без аккордовых функций |
| **Размеры** | 7/8, 11/8, постоянно меняющиеся |
| **Тон** | Чаще чистый или лёгкий перегруз (vs heavy metal) |
| **Техника** | Тэппинг, сложные арпеджио, непредсказуемые фразы |
| **Текстура** | Диалог двух гитар без ритм/соло деления |

### Ключевые группы

| Группа | Страна | Особенности |
|--------|--------|------------|
| **Battles** | США | Зацикленные паттерны, электроника + инструменты |
| **Toe** | Япония | Лирические мелодии, невероятная чёткость |
| **TTNG** | Великобритания | Fingerstyle, интимный звук |
| **Yvette Young (Covet)** | США | Тэппинг + нежные мелодии |
| **Chon** | США | Фьюжн + math, мажорные гармонии |

### Техника в math rock

**Tapping** (двуручный тэппинг) — правая рука касается грифа, создавая ноты хаммер-оном. Позволяет большие интервалы (децимы, ундецимы), невозможные одной рукой.

## Post-Metal

**Post-metal** — слияние пост-рока (Godspeed You! Black Emperor, Explosions in the Sky) и метала. Медленное, атмосферное, «эпическое» развитие.

### Характеристики

| Параметр | Описание |
|---------|---------|
| **Структура** | Нарастание (build-up) к «взрыву» (climax) |
| **Темп** | Медленный — средний |
| **Вокал** | Часто отсутствует или используется как текстура |
| **Атмосфера** | Обширные звуковые пейзажи, реверберация |

### Ключевые группы

| Группа | Особенности |
|--------|------------|
| **Neurosis** | Основатели; тяжело, тёмно, шаманское |
| **ISIS** | Взаимодействие тяжёлых риффов и атмосферных секций |
| **Cult of Luna** | Скандинавский пост-метал, эпические построения |
| **Amenra** | Бельгийский, минималистичный, трагический |
| **Deafheaven** | Пост-метал + блэк-метал + шугейзинг, «Sunbather» |

## Metalcore

**Metalcore** — слияние хардкора и метала (1990-е). Более доступный, чем чистый метал.

### Характеристики

| Параметр | Описание |
|---------|---------|
| **Вокал** | Чередование скрима и чистого вокала |
| **Структура** | Стандартная (куплет-припев), иногда брейкдаун |
| **Гитара** | Drop D/C, palm-mute, пауэр-аккорды, мелодические чистые секции |
| **Брейкдаун** | Медленная, максимально тяжёлая секция в конце |

### Ключевые группы

| Группа | Стиль |
|--------|-------|
| **Killswitch Engage** | Мелодичный metalcore, образцовые соло |
| **Parkway Drive** | Австралийский, агрессивный, масштабный |
| **Architects** | Прогрессивный metalcore, Sam Carter — вокал |
| **Bring Me the Horizon** | Эволюция от metalcore к alt-rock/pop |

## Сравнение современных стилей

| Стиль | Тяжесть | Сложность | Мелодия | Темп |
|-------|---------|---------|---------|------|
| **Djent** | Очень высокая | Высокая | Средняя | Средний |
| **Math rock** | Низкая | Очень высокая | Высокая | Переменный |
| **Post-metal** | Высокая | Умеренная | Атмосферная | Медленный |
| **Metalcore** | Высокая | Умеренная | Высокая | Средне-быстрый |

> **Современные жанры нельзя «выучить» из учебника** — их нужно слушать часами. Начните с одной группы: Periphery для djent, Toe для math rock, ISIS для post-metal, Killswitch Engage для metalcore. Слушайте альбом целиком, потом разбирайте приёмы. Это единственный путь.
