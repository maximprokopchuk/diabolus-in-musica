# Создание мелодии

## Что делает мелодию запоминающейся

**Мелодия** — это то, что слушатель насвистит в душе. Хорошая мелодия:
- Запоминается с первого-второго прослушивания
- Имеет **ритмический характер** (не только высоту, но и ритм нот)
- Содержит **противоречие** — вопрос и ответ, подъём и спуск
- Уместна в **диапазоне голоса** (если это вокальная мелодия)

## Hook

**Hook** («крючок») — самый запоминающийся момент мелодии, обычно в припеве. Это то, ради чего слушатель включает песню снова.

Характеристики хорошего хука:

| Параметр | Описание |
|---------|---------|
| **Краткость** | Обычно 2–8 нот |
| **Ритмическая четкость** | Ритм легко проговаривается |
| **Запоминаемость** | Слышишь — не можешь выкинуть из головы |
| **Эмоциональная точность** | Выражает суть песни |

**Примеры великих хуков:**
- «Yesterday» (Beatles) — «Yes-ter-daaaaay» — восходящая малая секста
- «Purple Rain» (Prince) — одна только нота, held
- «Smells Like Teen Spirit» (Nirvana) — риффовый хук, не только вокальный

## Мотивная разработка

**Мотив** — короткая ячейка из 2–5 нот, которая является «семенем» мелодии. Из одного мотива можно вырастить всю песню.

### Техники разработки

**Повторение**: мотив повторяется несколько раз.

```
Мотив: RE — МИ — ФА
Повторение: RE — МИ — ФА — RE — МИ — ФА...
```

**Секвенция**: мотив транспонируется на другую ступень.

```
Мотив на RE: RE — МИ — ФА
Секвенция выше: МИ — ФА# — СОЛЬ
```

**Инверсия**: мотив переворачивается — восходящие интервалы становятся нисходящими.

```
Мотив: RE — МИ(+2) — СОЛЬ(+3)
Инверсия: ЛЯ — СОЛЬ(-2) — МИ(-3)
```

**Ритмическое изменение**: те же ноты, другой ритм.

```
Мотив: ♩ ♩ ♩ (три равные)
Изменение: ♩. ♪ ♩ (с пунктиром)
```

## Вопрос и ответ (Call and Response)

Мелодии строятся на диалоге: **фраза-вопрос** (заканчивается на доминанте, неустойчиво) — **фраза-ответ** (заканчивается на тонике, стабильно).

```
Вопрос (такты 1-2): мелодия поднимается, останавливается на V степени
Ответ (такты 3-4): мелодия опускается, разрешается на I степень
```

Это структура большинства народных мелодий, хитов и классических тем.

## Вокальный диапазон

При написании вокальной мелодии нужно учитывать **диапазон исполнителя**:

| Голос | Типичный диапазон |
|-------|-----------------|
| Бас | E2–E4 |
| Баритон | A2–A4 |
| Тенор | C3–C5 |
| Альт | G3–G5 |
| Сопрано | C4–C6 |

**Хорус** обычно выше куплета — на нём должно быть легко «выплеснуть» эмоцию.

**Правило:** самые высокие ноты — в кульминационных моментах, не разбрасывай их по всей песне.

## Мелодия без слов

Инструментальная мелодия (соло, тема) работает по тем же принципам:
- David Gilmour — мелодии на гитаре с той же выразительностью, что вокал
- Mark Knopfler — гитарные мелодии как «разговор»
- BB King — в каждой фразе вопрос и ответ

## Инструмент для написания мелодий

Многие сонграйтеры **поют мелодию до того, как играют**. Голос первичен — гитара вторична. Сначала найди нужную эмоцию и контур мелодии голосом или напевом, затем переноси на инструмент.

> **Упражнение:** возьми уже написанную прогрессию и **пой поверх неё без слов** — слоги «ла-ла» или «на-на». Не думай, просто пой, что чувствуешь. Запиши. Это и есть твоя мелодия. Потом можно добавить слова.
