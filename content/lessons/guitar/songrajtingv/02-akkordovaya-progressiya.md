# Написание аккордовой прогрессии

## Прогрессия — это нарратив

**Аккордовая прогрессия** — это последовательность аккордов во времени. Каждый аккорд несёт **функцию**: тоника (дом), субдоминанта (движение), доминанта (напряжение). Прогрессия — это история на языке гармонии.

## Основы функциональной гармонии

В мажоре (тональность C):

| Степень | Аккорд | Функция |
|---------|--------|---------|
| I | C | Тоника (T) — дом, покой |
| II | Dm | Субдоминанта (S) |
| III | Em | Тоника (T) — нестабильный |
| IV | F | Субдоминанта (S) |
| V | G | Доминанта (D) — напряжение |
| VI | Am | Тоника (T) — «тёплый» |
| VII | Bdim | Доминанта (D) |

**Классическая формула:** T → S → D → T (I → IV → V → I)

## Популярные прогрессии

### I-V-VI-IV (самая популярная в поп-роке)

```
C — G — Am — F (в C мажоре)
```

Бесчисленное количество хитов используют эту прогрессию: «Let It Be», «With or Without You», «No Woman No Cry», «Someone Like You».

### VI-IV-I-V (вариация)

```
Am — F — C — G
```

Более «тёмная» версия той же прогрессии из-за начала с VI.

### I-IV-V (блюзовая основа)

```
E — A — B (в E)
```

12-тактовый блюз строится на этой прогрессии.

### II-V-I (джазовый кадэнс)

```
Dm7 — G7 — Cmaj7
```

Основа джазовой гармонии. Самое сильное разрешение в тональности.

## Выход за рамки клише

### Заимствования (modal interchange)

Аккорды из параллельного минора или другого лада, вставленные в мажорную прогрессию:

```
C — F — Ab — Bb — C
```
Ab и Bb взяты из C минора — это создаёт «неожиданный» поворот.

**Пример:** «Paranoid Android» (Radiohead), «Man on the Moon» (R.E.M.).

### Вторичные доминанты

Перед любым аккордом можно поставить его собственную доминанту:

```
C — A7 — Dm — G7 — C
```

A7 — вторичная доминанта к Dm (V7/II). Создаёт неожиданный яркий цвет.

### Педальный бас

Бас удерживает одну ноту (тонику), пока аккорды меняются:

```
Csus2/G — F/G — G7 — C
(G в басу под всеми аккордами)
```

Создаёт напряжение и движение при минимальных гармонических изменениях.

## Tension и Release

**Напряжение (tension)** — нестабильность, желание разрешения.
**Разрешение (release)** — возврат в стабильность.

Хорошая прогрессия играет с этим контрастом:

| Аккорд | Tension |
|--------|---------|
| I, VI | Низкий (дом) |
| IV, II | Средний |
| V, VII | Высокий |
| V7, VII dim | Максимальный |

### Задержание разрешения

Один из самых мощных приёмов: **не разрешать сразу**. Добавь ещё один аккорд с напряжением перед тоникой — это увеличивает «вес» разрешения.

```
I — IV — II — V7 — (пауза) — I
```

## Нестандартные подходы

- **Quartal harmony** (построение на квартах): характерно для джаза (McCoy Tyner, «So What» Miles Davis)
- **Педальные ноты с нарастающим напряжением**: Radiohead, Portishead
- **Один аккорд** (pedal point): funk, groove-метал — вся энергия в ритме и тембре

> **Практический совет:** возьми любую «скучную» прогрессию и замени один аккорд на заимствованный из параллельного лада или вторичную доминанту. Чаще всего это мгновенно делает прогрессию более интересной. Гармония — это ожидание и удивление. Дай слушателю оба.
