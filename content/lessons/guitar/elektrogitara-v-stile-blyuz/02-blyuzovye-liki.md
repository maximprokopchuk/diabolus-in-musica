# Блюзовые лики и фразы

## Что такое лик (lick)

**Лик** (lick) — короткая, запоминающаяся музыкальная фраза, которую гитарист использует как «словарное слово» в импровизации. Блюзовый словарь состоит из таких ликов, которые можно комбинировать, варьировать и развивать.

Обучение блюзу во многом — это **накопление словаря** из ликов и их вариаций.

## Структура блюзовой фразы

Типичная блюзовая фраза строится по принципу **вопрос–ответ** (call and response):

```
[Вопрос: 2 такта] — [Ответ: 2 такта]
[Вопрос: 2 такта] — [Ответ: 2 такта]
...
```

Этот принцип пришёл из африканской музыкальной традиции и лежит в основе блюзового повествования.

**«Фраза должна дышать»** — классический блюзовый принцип. Не играйте непрерывно. Паузы так же важны, как ноты.

## Call and Response (вопрос–ответ)

**Call and Response** — диалог между фразами, между гитарой и вокалом, между разными инструментами.

### Гитара и вокал:
В традиционном блюзе вокальная линия (call) «задаёт вопрос», гитара (response) «отвечает»:

```
Вокал: «I woke up this morning...»  [такты 1–2]
Гитара: [короткий лик, 2–3 ноты]    [такт 2–3]
Вокал: «Blues all around my door»   [такты 3–4]
Гитара: [лик-ответ]                  [такт 4]
```

**B.B. King** — абсолютный мастер этого диалога: его гитара буквально «разговаривает».

### Гитара с собой:
Соло может вести внутренний диалог — высокий регистр задаёт вопрос, низкий отвечает.

## Turnaround (разворот)

**Turnaround** — характерная фраза в конце блюзового хоруса (такты 11–12 в 12-тактовом блюзе), которая «разворачивает» гармонию обратно к началу.

### Классический turnaround в E:
```
Такт 11: E7 → (лик вниз)
Такт 12: B7 → (ожидание)
Повтор с I: E7
```

**Табулатурный пример (E-блюз turnaround):**
```
e: 0---0---3---2---0--------
B: 0---0---0---0---0--------
G: 1---1---1---1---1--------
(проходящие ноты по хроматике вниз к B7)
```

### Другой вариант turnaround (двойные ноты):
```
e: 12-11-10-9-
B: 12-11-10-9-
(параллельные октавы/унисоны вниз)
```

## Классические блюзовые лики

### Пентатоника E minor — базовый лик #1 (B.B. King стиль):
```
e: ---12b1---
B: 12--------
(бенд на 12-й лад E струны, предшествующая нота B)
```

### Базовый лик #2 (SRV стиль — открытая позиция E):
```
e: 0--3-0---
B: -----3-0-
G: ---------
(смешение открытых струн и 3-го лада)
```

### Shuffle лик:
```
A: 2-2-4-2-4-2-
E: 0-----------
(шаффл-ритм на A струне)
```

### Двойные ноты (double stops):
```
e: 5-5-
B: 5-7- (бенд B от 5 к 7)
   ↓ ↓ + бенд
```

Двойные ноты — характерный приём блюза и рокабилли.

## Intro Licks (вступительные лики)

**Intro lick** — фраза, открывающая блюзовое произведение. Часто это «визитная карточка» песни.

### «Johnny B. Goode» (Chuck Berry) — открывающий лик:
Нисходящая фраза в открытой позиции E с характерным bend-and-release. Один из самых узнаваемых ликов рок-н-ролла.

### «Pride and Joy» (SRV) — начало:
Характерный шаффл-ритм на двойных нотах — сразу задаёт грув Texas blues.

### «The Thrill Is Gone» (B.B. King) — мелодический вход:
Медленная, сдержанная фраза с характерными B.B. вибрато на отдельных нотах. Каждая нота — «слово».

## Блюзовый «лексикон»

Опытный блюзгитарист имеет **репертуар из 50–100+ ликов**, которые он:
- Варьирует (меняет ритм, бенды, ноты)
- Комбинирует (лик А + вариация В)
- Адаптирует к тональности
- Реагирует на контекст (громче/тише, быстрее/медленнее)

**Как учить лики:**
1. Выберите простой лик (2–4 ноты)
2. Играйте его медленно до автоматизма
3. Варьируйте ритм: вперёд, назад, паузы
4. Меняйте бенды: разная скорость и ширина
5. Переносите в другую позицию и тональность

## Блюзовые паузы и «воздух»

**«В блюзе важно то, что НЕ сыграно»** — классическое высказывание.

Паузы создают:
- Напряжение («чего ждать?»)
- Пространство для слушателя
- Контраст между фразами
- Ощущение groove и «разговора»

**B.B. King** — образец правильных пауз. Он мог взять 3 ноты за 4 такта — и это звучало лучше, чем 50 нот.

> **Практическое упражнение:** возьмите любой блюзовый бэкинг-трек (E blues, медленный шаффл). Начните импровизировать — но **запрещайте себе играть больше 3 нот в такт**. Только 3 ноты, много пауз. Это обучит вас говорить на «блюзовом языке» — экономно, выразительно и со смыслом.
