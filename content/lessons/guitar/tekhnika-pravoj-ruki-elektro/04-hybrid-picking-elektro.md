# Hybrid Picking

## Что такое hybrid picking

**Hybrid picking** (гибридный перебор) — техника, сочетающая медиатор и пальцы правой руки одновременно.

Медиатор держится стандартно (большой + указательный), а средний, безымянный и мизинец правой руки **свободны** и могут защипывать струны.

```
Правая рука при hybrid picking:
- Медиатор (большой + указательный) → нижние/басовые струны
- Средний палец (m)  → средние струны
- Безымянный (a)     → высокие струны
- Мизинец (c) — редко, для 4-пальцевого hybrid
```

## История и применение

**Hybrid picking** возник в **кантри-музыке** — стиле, где требуются одновременно:
- Басовая линия медиатором
- Мелодические «чикинг» (chicken picking) ноты пальцами
- Педал-стил имитация

**Классические кантри-гитаристы** с hybrid: Albert Lee, Brent Mason, Redd Volkaert, Brad Paisley.

В **рок и блюзе**: Robben Ford, Merle Travis, Carl Perkins.

В **современном прогрессиве и метале**: John Petrucci, Eric Johnson, Guthrie Govan, Andy Timmons.

## Chicken Picking

**Chicken picking** — характерный звук кантри-гитары, получаемый гибридным пикингом.

Средний палец захватывает струну и резко отпускает — получается яркий, щёлкающий звук, похожий на куриное кудахтанье (chicken cluck).

### Как сделать chicken picking:
1. Средний палец подхватывает струну снизу
2. Поднимает её чуть вверх
3. Резко отпускает — струна щёлкает об лад или кость порожка

Звук выразителен на чистом или лёгком клиппинге Telecaster — классика кантри-звука.

## Базовые паттерны hybrid picking

### Паттерн 1: Бас + мелодия
```
e: ------5m---------7m
B: ----5---m------5----
G: --4---------4------
D: 5---------5-------
   ↓         ↓
(медиатор на D, пальцы на G/B/e)
```

### Паттерн 2: Педал-стил имитация
```
Capo 2 или открытые:
e: -5m--3m--2m-
B: -3---3---3--
G: -0---0---0--
D: -----(медиатор)
   Педал-нота (G открытая) + мелодия пальцами
```

### Паттерн 3: Арпеджио с медиатором
```
Am arpeggio:
e: ----5m
B: --5m--
G: -5m---
D: ↓5----
   Медиатор бьёт D, пальцы перебирают G/B/e вверх
```

## Hybrid picking в прогрессивном металле

**John Petrucci** (Dream Theater) активно использует hybrid для:
- Плотных ритмических текстур (чередование медиатора и пальцев на соседних струнах)
- Арпеджио широкого диапазона (пальцы охватывают то, куда медиатор не дотягивается)
- Имитации фортепианных арпеджио

**Guthrie Govan** — один из самых технически развитых hybrid-пикеров:
- Фьюжн-стиль: плавное переключение между стилями
- Ноты в разных октавах одновременно (педаль-тон + мелодия)

**Andy Timmons:**
- Комбинирует hybrid с бендами
- Характерные «двойные ноты» — медиатор + палец одновременно

## Упражнения для развития hybrid picking

### Упражнение 1: Хроматика пальцами
```
e: 5m-6m-7m-8m (только средним пальцем, без медиатора)
B: 5m-6m-7m-8m
```
Цель: развить силу и независимость среднего пальца.

### Упражнение 2: Медиатор + палец одновременно
```
D + e одновременно:
D: 5↓  7↓  5↓  7↓
e: 5m  7m  5m  7m
   (одновременный удар медиатора и щипок пальцем)
```

### Упражнение 3: Кантри-пэттерн
```
E: 0↓ 2↓ 0↓ 2↓
B: --3m---3m
G: ----2m---2m
(чередование баса медиатором и мелодии пальцами)
```

## Преимущества hybrid picking

| Преимущество | Объяснение |
|-------------|-----------|
| **Большой охват** | Медиатор на нижних, пальцы на верхних — одновременно |
| **Разные тембры** | Медиатор звучит иначе, чем палец — тональное разнообразие |
| **Скорость арпеджио** | Пальцы двигаются независимо = более быстрые паттерны |
| **Динамика** | Пальцы дают более тихое, нюансированное звучание |
| **Уникальный звук** | Характерный «chicken» звук, недостижимый иначе |

> **Для развития hybrid picking:** начните с простых упражнений, где средний палец делает только один щипок в такт. Не пытайтесь сразу играть сложные паттерны. Первые недели средний палец будет «непослушным» — это нормально, нервные связи к нему менее развиты, чем к указательному. Терпение и ежедневная практика решат проблему.
