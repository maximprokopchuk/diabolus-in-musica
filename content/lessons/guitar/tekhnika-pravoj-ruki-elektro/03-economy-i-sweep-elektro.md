# Economy и Sweep Picking

## Economy Picking — экономичный штрих

**Economy picking** — техника, при которой при переходе на соседнюю струну медиатор продолжает движение **в том же направлении**, вместо того чтобы менять направление (как в alternate picking).

### Сравнение с alternate picking:

```
Alternate:
e: 5↓
B: 7↑ ← смена направления = дополнительное усилие

Economy:
e: 5↓
B: 7↓ ← то же направление = плавный переход
```

Это более «экономичное» движение — меньше сопротивления, меньше нагрузки на руку.

### Когда применять economy picking:

- **Восходящие гаммы** — переход на высокую струну после вниз-удара
- **Нисходящие гаммы** — переход на низкую струну после вверх-удара
- **Однонаправленные пассажи** (не арпеджио)

### Паттерн Фрэнка Гэмбале

**Frank Gambale** — австралийский гитарист, создавший систему sweep/economy picking как единую технику. Его книга «Speed Picking» (1988) — классика.

Ключевая идея Гэмбале: объедините sweep picking (для арпеджио) и economy picking (для гаммовых пассажей) в единую технику. Это позволяет играть любые линии минимальным количеством движений.

```
Гамма До мажор (одна октава, economy):
e: ---------------5-7-8
B: ----------5-6-8-----
G: ------4-5-7---------
D: --5-7---------------
   ↓  ↓ ↓  ↓ ↓  ↓ ↓  ↓
```

Обратите внимание: при переходе с D на G используется непрерывный downstroke.

## Sweep Picking — арпеджио свипом

**Sweep picking** — техника, при которой медиатор «прометается» (sweeps) по нескольким струнам одним непрерывным движением. Применяется преимущественно для **арпеджио**.

### Принцип:

Медиатор не «ударяет» каждую струну отдельно — он движется как смычок, одним непрерывным жестом через несколько струн.

```
Восходящее арпеджио Am (5 струн):
e: ----7↑
B: --8↑--
G: -9↑---
D: 7↓----
A: 5↓----
   ↓ = sweep down, ↑ = sweep up
```

Важно: каждая нота звучит **отдельно**, а не все вместе (не как аккорд). Левая рука **прокатывается** (rolling technique), нажимая по одной струне и сразу освобождая предыдущую.

### Rolling technique (левая рука):

При sweep picking на соседних струнах с одной позицией по ладу левая рука использует **rolling** — прокатывание одного пальца через несколько струн без баррэ:

```
D: 7 (указательный — полное касание)
G: 9 (безымянный)
B: 8 (средний)
e: 7 (указательный — только кончик после roll)
```

Указательный палец немного «прокатывается» с одной струны на другую, блокируя предыдущую.

## Базовые sweep-паттерны

### 3-струнное арпеджио (Am):
```
G: -9-
B: 8--
e: 7--
   ↓↑↓ или ↑↓↑
```

### 5-струнное арпеджио (A minor):
```
e: 5---
B: 5---
G: 6---
D: 7---
A: 7---
   sweep up (↑) все 5 струн подряд
```

### 6-струнное арпеджио с тэппингом (Extended sweep):
```
e: ---12t (tap)
B: --10↑--
G: -9↑----
D: 7↓-----
A: 5↓-----
E: 5↓-----
```

Тэппинг правой рукой на высоких ладах расширяет диапазон арпеджио.

## Типичные ошибки при sweep picking

| Ошибка | Следствие | Решение |
|--------|---------|---------|
| Все ноты звучат вместе | Получается аккорд, не арпеджио | Левая рука снимает каждую ноту перед следующей |
| «Щёлканье» при переходе | Слышны лишние удары | Плавность движения медиатора |
| Неравномерный темп | Одни ноты быстрее других | Медленная отработка с метрономом |
| Напряжение в правой руке | Не получается плавность | Расслабление, отработка движения без гитары |

## Economy vs Sweep: когда что использовать

| Ситуация | Техника |
|---------|---------|
| Гаммовые пассажи через несколько струн | Economy picking |
| Арпеджио по всем струнам | Sweep picking |
| Один тип нот на каждой струне | Sweep |
| 2–3 ноты на струне + переход | Economy |
| Смешанные линии | Комбинация |

> **Совет:** начинайте sweep с 3-струнных паттернов на очень медленном темпе (40–50 BPM). Когда движение станет «механическим», увеличивайте скорость. Sweep — одна из самых сложных техник для правильной постановки, и большинство самоучек «ломают» её на раннем этапе из-за спешки.
