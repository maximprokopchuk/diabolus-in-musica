# Высота струн (Action)

## Что такое action

**Action** (высота струн) — расстояние между нижней поверхностью струны и верхней поверхностью 12-го лада. Это ключевой параметр, напрямую влияющий на **играбельность** и **звук** инструмента.

## Влияние action на игру и звук

| Action | Плюсы | Минусы |
|--------|-------|--------|
| **Низкий** | Лёгкость нажатия, быстрая техника, меньше усилий | Дребезжание (buzz), потеря sustain |
| **Высокий** | Нет дребезжания, больше sustain, «живой» акустический звук | Тяжело играть, пальцы быстрее устают |

**Оптимальный action** — баланс: достаточно низкий для комфортной игры, достаточно высокий, чтобы избежать паразитного дребезжания.

## Стандартные значения action

Измерение производится на **12-м ладу** от верхушки лада до нижней поверхности струны:

| Тип гитары | Бас-струна (6-я) | Квинта-треблstruna (1-я) |
|------------|------------------|--------------------------|
| Электрогитара (стандарт) | 2.0–2.4 мм | 1.6–2.0 мм |
| Электрогитара (низкий) | 1.6 мм | 1.2 мм |
| Акустическая гитара | 2.8–3.2 мм | 2.0–2.4 мм |
| Классическая гитара | 3.0–4.0 мм | 2.5–3.0 мм |
| Бас-гитара | 2.5–3.0 мм | 2.0–2.5 мм |

**Примечание:** Гитаристы со слабым ударом могут использовать более низкий action без дребезжания; гитаристы с сильным — только с высоким.

## Где регулируется action

Action регулируется в **двух точках**:

### 1. Нулевой порожек (Nut)
Влияет на высоту струн на первых 3–4 ладах.
- Если первые лады трудно зажимать, а выше всё нормально — проблема в порожке
- Пазы порожка можно расширить специальными пилочками (nut files)
- Слишком глубокие пазы: струны дребезжат на открытых — нужна замена порожка

### 2. Бридж (Bridge / Saddle)
Влияет на высоту по всей длине грифа, но особенно заметно на 12-м ладу.

**Stratocaster:**
- Каждое седло регулируется двумя маленькими болтиками (hex key 1.5 мм)
- Поднять: закрутить болты по часовой стрелке
- Опустить: против часовой

**Les Paul (TOM):**
- Весь бридж поднимается/опускается двумя барашками
- Регулировка влияет на все шесть струн одновременно
- Для индивидуальной регулировки по струнам — только через порожек или подкладки

**Floyd Rose:**
- Болты на обеих стойках бриджа (hex key)
- Регулируется весь бридж целиком

## Связь action, трасс-стержня и интонации

Три параметра взаимозависимы:

```
Трасс-стержень → прогиб грифа → влияет на action в средней части грифа
Action → натяжение струн → влияет на интонацию
Интонация → настраивается после action
```

**Правильный порядок:**
1. Трасс-стержень (рельеф грифа)
2. Нулевой порожек
3. Бридж (action)
4. Интонация

## Дребезжание (Fret Buzz)

Если при нормальном action есть дребезжание, причины:

| Причина | Решение |
|---------|---------|
| Слишком мало рельефа (гриф прямой или обратный) | Настройка трасс-стержня |
| Неровные лады | Полировка/выравнивание ладов (fret leveling) |
| Слабое нажатие на струну | Тренировка техники |
| Неправильное расположение пальца (не у лада) | Исправление постановки руки |
| Слишком тонкие струны | Смена калибра |

> **Action — дело личных предпочтений.** Джон Мэйер предпочитает относительно высокий action на акустике для «органичного» звука. Шреддеры (Гилберт, Малмстин) — предельно низкий для скорости. Найдите свой баланс: начните со стандартных значений и корректируйте под свой стиль игры.
