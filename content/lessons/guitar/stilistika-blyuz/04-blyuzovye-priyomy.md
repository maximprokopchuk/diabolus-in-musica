# Блюзовые приёмы на гитаре

## Бенд (string bend)

**Бенд** — сдвиг струны вдоль лада, повышающий высоту ноты. Самый характерный приём блюза: через бенд передаётся вся экспрессия и «пение» гитары.

### Типы бендов

| Тип | Описание | Применение |
|-----|---------|-----------|
| **Half-step bend** | Подъём на полтона | Лёгкое напряжение |
| **Whole-step bend (full bend)** | Подъём на тон | Классический блюзовый ход |
| **1,5-step bend** | Подъём на полтора тона | Эмоциональный пик, SRV |
| **Bend and release** | Поднять и отпустить | Вокальный характер |
| **Pre-bend** | Согнуть до звука, потом снять | Эффект «вопрос» |
| **Unison bend** | Две струны — одна гнётся до высоты другой | Насыщенность, «сочность» |

### Техника бенда
- Используйте **три пальца** (указательный + средний + безымянный) для плавного, контролируемого бенда
- «Вращайте» запястье, а не тяните пальцем
- Поддерживайте большой палец за грифом как опора
- **Целевая нота** должна быть точной — настраивайте ухо

### Классический блюзовый ход (A, 8-й лад)

```
Нота: B-струна, 8-й лад (C) → bend → (D) → release
```

Подъём на тон с C до D — стандартный блюзовый ход.

## Вибрато

**Вибрато** — ритмическое колебание высоты ноты. Блюзовое вибрато шире и «нестабильнее», чем классическое — это часть языка.

### Стили вибрато

| Стиль | Описание | Гитарист |
|-------|---------|---------|
| **Классическое вибрато** | Узкое, равномерное | Скрипичный стиль |
| **Блюзовое вибрато** | Широкое, эмоциональное | B.B. King, Clapton |
| **Рокерское вибрато** | Агрессивное, с рычагом | EVH, SRV |

### Техника блюзового вибрато
- **Горизонтальное**: пальцем вдоль струны (не перпендикулярно) — для нот на тонких струнах
- **Вертикальное** (bend-based): быстрые маленькие бенды — блюзовый стандарт

**B.B. King** использовал уникальное «butterfingers» вибрато — быстрое, трепетное, без полного бенда.

## Call and Response

**Call and response** («вопрос–ответ») — основа блюзовой импровизации. Фраза-«вопрос» остаётся незаконченной, фраза-«ответ» завершает мысль.

```
Call:    |----бенд---вибрато----|  (пауза)
Response:|-----другая фраза-----|  (завершение)
```

Это прямое отражение блюзовых текстов: первая строка куплета повторяется (call), третья — отвечает (response).

### Практика call-and-response
1. Сыграйте 2-тактовую фразу, заканчивающуюся «в воздухе» (на неустойчивой ноте)
2. **Пауза** — это часть музыки, не пустота
3. Ответьте другой фразой, завершив «гармонически» (на тонике или квинте)

## Основные лики (licks)

### Лик 1: «Classic BB King» (A, 17-й лад)
Трепетное вибрато на E-струне, 17-й лад (A):
```
E |---17~~---|
```
Простота + вибрато = всё.

### Лик 2: Пентатоника с бендом (A, позиция 1)
```
E |---5---8b9r8---5---|
B |-------------------|
G |---5---7-----------|
D |-------------------|
```
(b = bend, r = release)

### Лик 3: Хроматический проход
```
G |---7---6---5---|
D |---------------|
```
Хроматический спуск через blue note.

### Лик 4: Double-stop (двойная нота)
```
B |---8---8---|
G |---7---9b--|
```
Две ноты одновременно с бендом нижней — характерный техасский блюз.

## Интро-лики

Блюзовое интро обычно состоит из **двух тактов** (иногда четырёх), после которых начинается основная форма. Интро задаёт тональность, темп и настроение.

**Классическое блюзовое интро (A):**
```
Такт 1: A7 — хроматическое движение вниз
Такт 2: E7 — turnaround-лик
→ Такт 3: начало 12-тактовой формы (I7)
```

## Закрытые (muted) ноты и рест-штрихи

Лёгкое касание ладонью струн у бриджа даёт **рest stroke** — приглушённые, «стаккатные» ноты. Это добавляет ритмический пунктир и «разговорный» характер.

```
Полная нота: |---7---8---7---|
С рestрoке:  |---7x--8---7x--|
(x = приглушённая нота)
```

> **Ключ к блюзу — это экономия.** Великие блюзмены не стремятся сыграть как можно больше нот. B.B. King мог взять одну ноту с вибрато и «сказать» больше, чем иной гитарист за весь соло. Пауза, пространство, одна правильная нота в нужный момент — вот язык блюза.
