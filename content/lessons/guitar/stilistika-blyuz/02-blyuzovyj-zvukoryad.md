# Блюзовый звукоряд

## Минорная пентатоника

**Минорная пентатоника** — пять нот, образующих основу блюзовой импровизации. Строится по формуле:

```
1 — b3 — 4 — 5 — b7
```

Для **A минорной пентатоники:**
```
A — C — D — E — G
```

### Первая позиция (box pattern) — A минорная пентатоника

```
E |---|---|---|---| 5 |---|
B |---|---|---|---| 8 |---|
G |---|---|---|---| 7 |---|
D |---|---|---|---| 7 |---|
A |---|---|---|---| 7 |---|
E |---|---|---|---| 5 |---|
      5       8
```

Цифры обозначают лады. Первая позиция начинается на 5-м ладу для A.

### Пять позиций пентатоники (CAGED)

| Позиция | Начало на ладу (для A) | Связь с аккордом CAGED |
|---------|----------------------|------------------------|
| 1 (box) | 5 | A-форма |
| 2 | 7 | G-форма |
| 3 | 9 | E-форма |
| 4 | 12 | D-форма |
| 5 | 14 | C-форма |

Все пять позиций покрывают весь гриф. Важно соединять их — не «застревать» в одной коробке.

## Blue Note — «синяя нота»

**Blue note** — уменьшённая квинта (b5), добавляемая к минорной пентатонике. Она создаёт характерную «блюзовую» напряжённость.

```
Минорная пентатоника: A — C — D — E — G
Блюзовый звукоряд:   A — C — D — Eb — E — G
                                  ↑
                              blue note (b5)
```

**Blue note** обычно не «стоит» на долгих нотах — её «проходят», тянут бендом, используют для перехода между D и E. Это придаёт характерную напряжённость и «блюзовость».

### Запись блюзового звукоряда для A (5-й лад)

```
E |---5---8---|
B |---5---8---|
G |---5---7---|
D |---5---6---7---|
A |---5---7---|
E |---5---8---|
```

6-й лад на D-струне — это blue note (Eb).

## Мажорная пентатоника

**Мажорная пентатоника** строится по формуле:

```
1 — 2 — 3 — 5 — 6
```

Для **A мажорной пентатоники:**
```
A — B — C# — E — F#
```

Мажорная пентатоника звучит **светлее, позитивнее**. Используется в кантри, кантри-блюзе и в «счастливых» блюзовых пассажах.

Важная связь: **A мажорная пентатоника = F# минорная пентатоника** (это одни и те же ноты, относительные тональности).

## Смешение мажорной и минорной пентатоники

Это **ключ к настоящему блюзовому звуку**. Блюз «плавает» между мажором и минором, не выбирая однозначно.

### Смешение в A блюзе

| Нота | Принадлежность |
|------|--------------|
| A | Обе |
| B | Мажорная |
| C | Минорная (b3) |
| C# | Мажорная (3) |
| D | Обе |
| Eb | Blue note |
| E | Обе |
| F# | Мажорная |
| G | Минорная (b7) |

**Техника смешения:**
- Начните пассаж с мажорной пентатоники (C#)
- «Соскользните» к минорной (C)
- Бенды C → C# (минорная 3 → мажорная 3) — классический ход

### Примеры в музыке

| Гитарист | Приём |
|---------|-------|
| **B.B. King** | Чередует мажорную и минорную в пределах одной фразы |
| **Eric Clapton** | Часто использует мажорную пентатонику в «светлых» частях |
| **SRV** | Смелое смешение, часто нарушает «правила» в пользу экспрессии |

## Практика: как «сломать коробку»

Распространённая ошибка — «застрять» в первой позиции. Как выйти:

1. **Горизонтальное движение:** сыграйте одну струну по нескольким позициям
2. **3-нотные струны:** три ноты на одной струне вместо двух — расширяет диапазон
3. **Соединяйте позиции:** переходите из 1-й в 2-ю через общие ноты
4. **Используйте neck + bridge:** не только «коробка» на 5-м ладу, но и позиции на 12–17 ладах

> **Главное откровение блюзового звукоряда:** важны не ноты, а то, **как** вы играете их. Один бенд, один вибрато на C# — и звук становится блюзом. Пять правильных нот без экспрессии — это просто гамма.
