# Компинг

## Что такое компинг

**Компинг** (comping — от *accompanying*) — ритмическое аккордовое сопровождение в джазе. Гитарист «поддерживает» солиста, создавая гармонический и ритмический фон.

Хороший компинг — это **разговор**: гитарист слушает солиста и «отвечает» аккордами, оставляя пространство для мелодии.

## Freddie Green стиль

**Freddie Green** (1911–1987) — гитарист оркестра Count Basie на протяжении 50 лет. Создал уникальный «тихий» стиль компинга.

### Принципы стиля Freddie Green:

1. **Четыре удара на такт** — строго, как метроном (walking guitar)
2. **Не усиливается** — акустическая арчтоп, без звукоснимателя (или с минимальным усилением)
3. **Только частичные аккорды** — чаще всего 2–3 струны
4. **Нет вибрато, нет украшений** — чистый, «сухой» удар
5. **Слияние с ударными и контрабасом** — часть ритм-секции, не сольный инструмент

### Позиции Freddie Green (3-нотные аккорды):

```
C7:    x-3-2-3-x-x  (на струнах 4-3-2)
F7:    x-x-3-2-1-x  (на струнах 3-2-1)
Cm7:   x-3-1-3-x-x
```

Фишка: аккорды **без открытых струн**, каждый — перемещаемый.

## Swinging Comping (свинговый компинг)

В отличие от Freddie Green, большинство современных гитаристов играют **не на каждую долю**, а «между» долями — создавая свинг.

### Базовые ритмические паттерны:

**Простой компинг (off-beat):**
```
Такт 4/4:  1  +  2  +  3  +  4  +
Гитара:       [>]       [>]      [>]
```
Акценты на «+» (ан) — характерно для свинга.

**Компинг с синкопой:**
```
Такт:      1  е  +  а  2  е  +  а  3  е  +  а  4  е  +  а
Гитара:       [>]     [>]        [>]     [>]
```

### Принцип «leaves and fills»:
- Когда солист играет — **оставьте пространство** (fewer chords)
- Когда солист «дышит» — **заполните** паузу аккордом или фразой

## Chord Comping vs Linear Comping

### Chord Comping (аккордовый):
- Полные Drop 2 / Drop 3 аккорды
- Более «плотный» звук
- Характерен для Wes Montgomery, Joe Pass

### Linear Comping (линейный):
- Аккорды, перемежаемые одноголосными фразами
- Более современный звук
- Характерен для Pat Metheny, Kurt Rosenwinkel

## Динамика компинга

**Четыре уровня динамики в компинге:**

| Уровень | Описание | Когда |
|---------|---------|-------|
| Piano (p) | Почти неслышно | Пение, баллады |
| MP | Мягко, поддерживающий | Стандартный компинг |
| MF | Чуть активнее | Акцент на важных точках |
| Forte (f) | Громко | Tutti, кульминации |

## Ритмические приёмы

### Backbeat comping:
Акценты на 2-й и 4-й долях — более «роковый» ощущение, используется в fusion.

### Bomb dropping:
«Взрывной» аккорд в неожиданном месте — характерно для Тэда Джонса, Элвина Джонса (ударные) — гитаристы переняли приём.

### Chord shots:
Короткий, «сухой» аккорд (staccato) — усиливает рифф или фразу солиста.

| Гитарист | Стиль компинга | Особенность |
|---------|---------------|------------|
| Freddie Green | Walking guitar | 4 удара, 3-нотные аккорды |
| Wes Montgomery | Block chords | Октавы + аккорды |
| Joe Pass | Chord-melody | Совмещает компинг и мелодию |
| Lenny Breau | Fingerstyle | Полифонический компинг |
| Kurt Rosenwinkel | Modern linear | Линейные фразы + аккорды |

> **Совет:** Лучший способ научиться компингу — играть с другими музыкантами, особенно с живым басистом. В первый месяц практики намеренно **играйте тише**, чем кажется нужным. Компинг должен служить — не мешать солисту.
