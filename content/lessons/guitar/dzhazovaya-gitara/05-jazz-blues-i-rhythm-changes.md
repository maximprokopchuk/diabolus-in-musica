# Jazz Blues и Rhythm Changes

## Jazz Blues — форма

**Jazz Blues** — 12-тактовая форма, производная от блюза, но с **усложнённой гармонией**. Используется как полигон для импровизации.

### Базовая схема jazz blues в Bb:

```
| Bb7    | Eb7    | Bb7    | Bb7    |
| Eb7    | Edim7  | Bb7    | G7     |
| Cm7    | F7     | Bb7  G7 | Cm7  F7 |
```

### Расширенный jazz blues (Bird changes):

Charlie Parker добавил хроматические субституции:

```
| Bb7      | Eb7      | Bb7      | Fm7  Bb7  |
| Eb7      | Edim7    | Bb7  Gm7 | Cm7   F7  |
| Fm7  Bb7 | Eb7  Edim| Bb7  G7  | Cm7   F7  |
```

## Воицинги для jazz blues в Bb

**Bb7 (тоника доминантовая):**
```
Drop 2, струны 2-5:
x-1-3-1-2-x (Bb-F-Ab-D, позиция I лада)
```

**Eb7 (субдоминанта):**
```
x-6-5-6-4-x (Eb-Bb-Db-G, позиция VI лада)
```

**Edim7 (хроматический проходящий):**
```
x-7-8-7-8-x (E-Bb-Db-G = уменьшённый 7 аккорд)
```

**Cm7 → F7 (II-V к Bb):**
```
Cm7: x-3-5-3-4-x
F7:  x-8-7-8-6-x
```

## Rhythm Changes

**Rhythm Changes** — 32-тактовая форма на основе «I Got Rhythm» Джорджа Гершвина (1930). После джазового блюза — вторая по популярности форма для джемов.

### Структура (AABA, тональность Bb):

**A секция (8 тактов, повторяется трижды):**
```
| Bb  Gm7 | Cm7  F7 | Bb  Gm7 | Cm7  F7 |
| Fm7 Bb7 | Eb  Edim| Bb/D  F7| Bb     |
```

**B секция (Bridge, 8 тактов):**
```
| D7     | D7     | G7     | G7     |
| C7     | C7     | F7     | F7     |
```

Bridge — это **доминанты по квинтам**: D7 → G7 → C7 → F7 (возвращается в Bb).

### Сокращённые аппликатуры Rhythm Changes:

**A секция (основные):
```
Bb: x-1-3-3-3-1 (Bb maj7 или Bb6)
Gm7: x-x-5-3-3-3
Cm7: x-3-5-3-4-3
F7:  x-8-7-8-6-x
```

## Подход к импровизации

### Jazz Blues:
- **Мажорная пентатоника:** безопасно, блюзово
- **Миксолидийский лад:** над каждым доминантовым аккордом
- **Bird-арпеджио:** Cm7→F7→Bb7 = II-V-I язык

### Rhythm Changes A-секция:
- **Bb мажор** над тоникой
- **G altered / G7b9** над Gm7 (джазовая фишка)
- **Bebop scale** (мажорная с хроматической восьмой нотой)

### Rhythm Changes Bridge:
Каждый доминантовый аккорд — **Mixolydian или Dominant Bebop**:
- D7: D Mixolydian
- G7: G Mixolydian
- C7: C Mixolydian
- F7: F Mixolydian

## Гитарные хоруэм воицинги — практика

**Упражнение 1: Rhythm Changes A в движении:**
Сыграйте каждый аккорд Drop 2 на одних струнах (2-5), минимально двигая руку:

```
Bb: x-1-3-1-2-x
Gm7: x-x-5-3-3-3 (смещение вверх)
Cm7: x-3-5-3-4-x
F7:  x-x-5-5-6-5 (позиция выше)
```

**Упражнение 2: Blues Substitution Chain:**
В такте Bb7 попробуйте: Bb7 → E7 (тритон) → Bb7

| Форма | Такты | Тональность | Характер |
|-------|-------|------------|---------|
| Jazz Blues | 12 | Bb, F, C, G | Блюзовый, гибкий |
| Rhythm Changes | 32 (AABA) | Bb | Bebop, «скоростная» |

> **Совет:** Rhythm Changes — это «полоса препятствий» для джазового музыканта. Charlie Parker написал десятки тем на эту форму («Anthropology», «Moose the Mooche», «Dexterity»). Выучите минимум 3-4 бопо-мелодии на Rhythm Changes и играйте их со скоростью оригинала — это лучший способ освоить форму.
