# Расширенные аккорды в R&B

## Почему R&B звучит «богато»

Особый «тёплый» и «сложный» звук R&B создаётся расширенными аккордами — аккордами, содержащими **ноты за пределами базовой трезвучной структуры** (нона, ундецима, терцдецима).

Если поп-музыка часто использует простые трезвучия (I–IV–V–I), то R&B «живёт» в мире **maj7, 9, 11, 13, sus2, sus4**.

## Структура расширенных аккордов

### Базовые 7-е аккорды

| Аккорд | Формула | Звучание |
|--------|---------|---------|
| **maj7** | 1-3-5-7 | Тёплый, «романтичный» |
| **m7** | 1-b3-5-b7 | Мягкий, задумчивый |
| **7 (dom7)** | 1-3-5-b7 | Напряжённый, «блюзовый» |
| **m7b5** | 1-b3-b5-b7 | Тёмный, загадочный |

### Нона (9)

**9-й аккорд** добавляет нону (9-я ступень = 2-я через октаву):

```
Cmaj9 (C + E + G + B + D):
E —0—  (D — нона)
B —0—  (B — maj7)
G —0—  (G — квинта)
D —2—  (E — терция)
A —3—  (C — основной тон)
E ———  (не играть)
```

```
D9 (доминантовый 9-й):
E —5—  (A — нона = 9)
B —5—  (F# — большая терция)
G —5—  (C — малая септима)
D —4—  (A — квинта)
A —5—  (D — основной тон)
E ———
```

### Sus2 и Sus4

**Sus** (suspended) — аккорд, в котором терция заменена второй или четвёртой ступенью:

```
Dsus2: D F# A  →  D E A  (F# → E)
Dsus4: D F# A  →  D G A  (F# → G)
```

Sus-аккорды создают **открытое, нейтральное** звучание — не мажор, не минор. Характерны для Neo-soul и R&B.

```
Dsus2 воицинг:
E —0—  (D)
B —3—  (D — октава)
G —2—  (A — квинта)
D —0—  (D)
A ———
E ———
```

### 11-й и 13-й аккорды

**11-й аккорд** добавляет ундециму (4-я через октаву). В практике чаще используется **m11** или **add11**:

```
Dm11: D F A C E G (все ноты — «стак» терций)
На гитаре — компактный воицинг:
E —3—  (G — ундецима)
B —1—  (C — септима)
G —2—  (A — квинта)
D —0—  (D — основной тон)
A ———
E ———
```

**13-й аккорд** — практически полная диатоническая гамма в одном аккорде:

```
G13 (без квинты и 11-й — как обычно на гитаре):
E —3—  (A — нона)
B —3—  (E — терцдецима = 13)
G —3—  (B — терция)
D —4—  (A — нона)
A —5—  (G — основной тон)
E ———
```

## Практические воицинги для R&B

### «Shell voicings» (скелетные воицинги)

Упрощённые воицинги, содержащие только ключевые тоны (основной тон, терция, септима):

```
Cmaj7 shell:
E ————
B —0—  (B — maj7)
G —0—  (G — квинта, опционально)
D —2—  (E — большая терция)
A —3—  (C — основной тон)
E ————
```

### «Drop-2 voicings»

Классические джазовые воицинги, часто применяемые в R&B:

```
Fmaj7 drop-2:
E —5—  (E — maj7)
B —5—  (C — квинта)
G —5—  (A — большая терция)
D —3—  (F — основной тон)
A ————
E ————
```

## Последовательности аккордов R&B

### ii–V–I (база джаза и R&B)

```
В C мажоре: Dm7 — G7 — Cmaj7
```

Расширенная версия:
```
Dm9 — G13 — Cmaj9
```

### Характерные прогрессии neo-soul

**iv–I** с расширениями (характерно для D'Angelo):
```
Gm9 — Cmaj9 (в ре миноре: iv9 — VII maj9)
```

**I maj7 — I7 — IV maj7 — iv m7** (смешение мажора и параллельного минора):
```
Cmaj7 — C7 — Fmaj7 — Fm7
```

## Как изучать расширенные аккорды

1. **Начните с maj7 и m7** — они встречаются в каждой R&B-песне
2. **Изучите drop-2 воицинги** — 4 инверсии каждого аккорда на всех группах струн
3. **Транспонируйте** — один воицинг в 12 тональностях
4. **Слушайте активно** — анализируйте аккорды в любимых R&B-треках

> **Ключ к R&B-звуку:** расширенные аккорды + правильный voicing (тесное расположение нот) + clean или слегка overdrive звук = мгновенно узнаваемый R&B. Не нужно знать все аккорды — нужно знать нужные.
