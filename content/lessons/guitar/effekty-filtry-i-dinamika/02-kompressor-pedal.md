# Компрессор (педаль)

## Что такое компрессор?

**Компрессор** — динамический процессор, уменьшающий разницу между тихими и громкими частями сигнала. Простыми словами: **тихое делает громче, громкое делает тише**.

### Принцип работы

Когда сигнал превышает заданный порог (threshold), компрессор снижает усиление в соответствии с заданным соотношением (ratio). После сжатия gain restoration (makeup gain) поднимает общий уровень.

```
Уровень без компрессора:         Уровень с компрессором:
  ↑ dB                             ↑ dB
  |  ↑↑ ↑↑↑                        |  ↑↑ ↑↑↑
  | ↑  ↑  ↑                        | ↑  ↑  ↑   ← «потолок»
  |↑    ↑                          |↑    ↑
  +──────────→                     +──────────→
     (большая динамика)              (меньшая динамика)
```

## Параметры компрессора

### Threshold

**Порог** — уровень сигнала, при превышении которого компрессор начинает работать. Ниже threshold — компрессор не влияет; выше — сжимает.

### Ratio (соотношение)

**Ratio** определяет, насколько сильно сжимается сигнал выше threshold.

| Ratio | Поведение |
|-------|---------|
| 2:1 | За каждые 2 дБ выше threshold — 1 дБ на выходе. Мягкое сжатие |
| 4:1 | За каждые 4 дБ — 1 дБ. Умеренное |
| 8:1 | Сильное сжатие |
| ∞:1 (limiting) | Уровень не превысит threshold — жёсткий лимитер |

### Attack (атака)

**Attack** — время, за которое компрессор реагирует на превышение threshold.

| Attack | Звуковой эффект |
|--------|----------------|
| Быстрый | Начальная атака звука тоже сжимается — более «сглаженный» звук |
| Медленный | Начальная атака проходит без компрессии — атака «пробивает» через компрессор, звук становится «щёлкающим» |

**Медленный attack** = характерный **chicken picking** звук кантри-гитары.

### Release (спад)

**Release** — время, за которое компрессор возвращается к исходному усилению после того, как сигнал упал ниже threshold.

| Release | Звуковой эффект |
|---------|----------------|
| Быстрый | Компрессор «отпускает» быстро, может создавать «помпование» (pumping) |
| Медленный | Плавный, незаметный, более «естественный» звук |

### Makeup Gain / Output Level

Поскольку компрессор снижает уровень, makeup gain компенсирует это снижение, поднимая выход до нужного уровня.

## Типы компрессоров

### OTA (Operational Transconductance Amplifier)

MXR Dyna Comp (M102), DOD 280 — используют чип CA3080. Характерный «squish»: заметная, ощутимая компрессия. Быстрое «схватывание».

### VCA (Voltage Controlled Amplifier)

Boss CS-3, Empress Compressor, Cali76 — используют VCA-чипы. Более прозрачный, студийный характер.

### Opto (Optical)

Используют оптопару (лампочка + фотоэлемент). **Teletronix LA-2A** — легендарный студийный оптический компрессор.

- Медленная, «музыкальная» реакция
- Характерное для Fender усилителей «дыхание»
- Педали: Diamond Compressor, Wampler Ego, Effectrode PC-2A

### FET (Field Effect Transistor)

UREI 1176 — легендарный FET-компрессор. Быстрый, точный.
- Педали: Cali76 (точная реплика 1176), Origin Effects Cali76 Compact

## Squish (Сквиш)

**Squish** — разговорный термин для описания характерного ощущения компрессии: «сжатый», «упругий», «резиновый» звук.

- Высокий squish: заметная компрессия, характерный «придавленный» звук Dyna Comp
- Низкий squish: незаметная компрессия, только sustain и уровень

**Для чистого, прозрачного результата** → Empress, Cali76, Wampler Ego (с Blend)
**Для характерного squish-звука** → MXR Dyna Comp, Keeley

---

## Chicken Picking

**Chicken Picking** — техника кантри-гитары, при которой медиатор и пальцы чередуют удары. Создаёт характерный «щёлкающий» звук.

Компрессор в chicken picking:
- **Медленный attack**: начальная атака «проходит» без компрессии → характерный «щелчок» остаётся
- Компрессор затем «давит» сустейн → каждая нота равная по уровню
- Результат: чёткие, «выбивающиеся» атаки + ровный, сжатый сустейн

**Эталоны:** Brad Paisley, Brent Mason, Albert Lee.

---

## Sustain (сустейн)

Компрессор значительно увеличивает sustain:

1. Тихие части ноты (затухание) поднимаются в уровне
2. Нота «продолжается» дольше, прежде чем исчезнуть
3. При высоком ratio и уровне — фактически «бесконечный» sustain (соседствует с пороговым feedback)

**Применение:** соло-гитара требует длинного sustain. Компрессор + умеренный overdrive = классический «поющий» lead-tone.

---

## Blend Control

Многие современные компрессоры имеют **Blend (Mix/Parallel Compression)** — контроль смешивания сжатого и несжатого сигнала.

**Параллельная компрессия:**
- Blend 100% → полностью сжатый сигнал
- Blend 50% → 50% сжатого + 50% оригинала
- Blend 0% → педаль отключена

Параллельная компрессия позволяет получить sustain и выравнивание уровней, **сохраняя динамику атак** оригинального сигнала.

## Применение по жанрам

| Жанр | Применение | Настройки |
|------|-----------|-----------|
| **Кантри** | Chicken picking, clean sustain | Dyna Comp, attack средний, ratio 4:1 |
| **Funk** | Ровный уровень chord strumming | Прозрачный, ratio 3:1, blend 70% |
| **Blues (clean)** | Sustain, ровный уровень | Оптический, мягкий, slow attack |
| **Jazz** | Минимальный, только для ровности | Прозрачный, ratio 2:1, blend 50% |
| **Rock (lead)** | Sustain для соло | Умеренный, makeup gain для буста |

## Расположение в цепи

**Компрессор обычно стоит в начале цепи** (после тюнера, перед Drive):

```
Гитара → Тюнер → Компрессор → Wah → Drive → Модуляция → Дилей → Ревер
```

**Почему в начале:** компрессор ровняет динамику гитарного сигнала перед остальными эффектами.

**Альтернатива:** компрессор **после Drive** — сжимает уже перегруженный сигнал. Даёт другой характер: более «компрессированный» и «сжатый» перегруз. Используется в некоторых metal-риг-сетапах.

> **Важный принцип:** хороший компрессор незаметен. Если вы слышите явный «squish» — это слишком много компрессии для большинства задач. Цель — поднять тихие ноты, опустить пики и добавить sustain без ощущения «сдавленности».
