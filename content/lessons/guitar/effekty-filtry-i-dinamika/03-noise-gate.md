# Noise Gate

## Что такое Noise Gate?

**Noise Gate** — динамический процессор, который **полностью убирает (отсекает) сигнал**, когда его уровень падает ниже заданного порога (threshold). По сути — автоматический выключатель.

### Принцип работы

```
Уровень сигнала:    ██████   ███████   ████
                   │      │  │       │  │    │
                  ON     OFF │      OFF │   ON
                        ↑              ↑
                    threshold       threshold
```

- Сигнал выше threshold → gate открыт, звук проходит
- Сигнал ниже threshold → gate закрывается, сигнал блокируется

---

## Параметры Noise Gate

### Threshold

**Threshold** — уровень, при котором gate переключается.

| Threshold | Поведение |
|-----------|---------|
| Высокий | Gate закрывается при громких звуках → частично обрезает сигнал |
| Слишком высокий | Gate срабатывает даже на тихие ноты → «заглатывает» сустейн |
| Оптимальный | Gate закрывается только при полной тишине (между нотами) |
| Слишком низкий | Gate не закрывается → шум проходит |

**Задача:** threshold должен быть **чуть выше** уровня шума, но **ниже** уровня самых тихих нот.

### Attack (Hinge) / Hold / Release

| Параметр | Функция |
|---------|---------|
| **Attack** | Скорость открытия gate (насколько быстро gate открывается при превышении threshold) |
| **Hold** | Время, которое gate остаётся открытым после падения сигнала ниже threshold |
| **Release / Decay** | Скорость закрытия gate (плавное или резкое «схлопывание») |

**Слишком быстрый Release** → резкое «отсечение» сустейна, неестественно.
**Слишком медленный Release** → шум «вылезает» в конце ноты.

### Attenuation / Range

Некоторые гейты позволяют настроить, **насколько** сигнал убирается при закрытии:
- **-∞ дБ (Mute)**: полная тишина
- **-40 дБ**: большое ослабление, но не полная тишина (мягче звучит)

## Зачем нужен Noise Gate

### Высокогейновый звук (Metal, Hard Rock)

Главная причина использования gate — **фоновый шум** при высоком усилении:

1. **Single coil pickups** — наводки 50/60 Гц (hum) от электрических полей
2. **Цепочка педалей** — каждый gain-каскад усиливает шум
3. **High gain amp** — усиливает не только сигнал, но и все помехи
4. **Длинные кабели** — ёмкостные утечки и наводки

При высоком гейне фоновый шум может быть очень громким. Noise gate убирает его в паузах.

### Проблемы без gate

| Ситуация | Без gate | С gate |
|---------|---------|--------|
| Пауза в игре | Слышен громкий шум/фон | Тишина |
| После брутального риффа | «Хвост» шума | Резкое закрытие |
| Сцена с плохим заземлением | Постоянный фон | Устранён в паузах |

## Классические педали Noise Gate

| Педаль | Характер | Применение |
|--------|---------|-----------|
| **Boss NS-2 Noise Suppressor** | Надёжный, много опций петли | Стандарт в метале |
| **ISP Decimator** | Очень прозрачный, не «заглатывает» сустейн | Профессиональный стандарт |
| **ISP Decimator Pro Rack G** | Стоечный, для высшего уровня | Студия, touring |
| **MXR Smart Gate** | 3 режима, компактный | Доступный, versatile |
| **Electro-Harmonix Silencer** | Gate + буфер + FX Loop | Компактный, функциональный |
| **TC Electronic Sentry** | TonePrint, mini | Современный |

**ISP Decimator** считается эталоном: его алгоритм «следит» за огибающей сигнала и практически не влияет на сустейн — gate открыт до тех пор, пока нота действительно звучит.

## Noise Gate в Metal-Rig

### Схема 1: Gate перед усилителем

```
Гитара → Тюнер → Drive → Noise Gate → FX Loop Return → Кабинет
```

Gate перед усилителем убирает шум **между педалями и преампом**.

### Схема 2: Gate в FX Loop

```
Гитара → Преамп усилителя → [FX Send] → Noise Gate → [FX Return] → Оконечник
```

Gate в FX Loop усиливает убирает шум **оконечного каскада и усилителя** в целом.

### Схема 3: Boss NS-2 — петлевая схема

Boss NS-2 имеет встроенную FX Loop (Send/Return). Это позволяет «обернуть» весь шумный сигнальный тракт в gate:

```
Гитара → NS-2 (Input) → NS-2 (Send) → Drive педали → NS-2 (Return) → NS-2 (Output) → Усилитель
```

NS-2 анализирует исходный сигнал гитары (до шумных педалей) как ключевой и на его основе управляет gate всей цепи.

## Правила настройки

### Шаг 1: Установите threshold правильно

1. Включите все Drive педали и усилитель
2. Не трогайте струны — слушайте шум
3. Медленно поднимайте threshold до момента, когда шум исчезает
4. Сыграйте ноту — если gate «заглатывает» её хвост, снизьте threshold чуть-чуть

### Шаг 2: Настройте Release

- Длинный Release: нота постепенно «угасает» с закрытием gate → более естественно
- Короткий Release: нота «обрезается» → характерный агрессивный звук (intentional в метале)

### Шаг 3: Проверьте Hold

Hold = минимальное время открытия gate. Помогает избежать «дёрганья» на коротких паузах внутри риффа.

## Noise Gate vs Noise Suppressor

Некоторые производители (Boss) называют свои педали «Noise Suppressor» вместо «Noise Gate»:

| Тип | Поведение |
|-----|---------|
| **Gate** | Резкое открытие/закрытие |
| **Suppressor** | Более плавная компрессия шума, не полное отсечение |

На практике NS-2 может работать в обоих режимах в зависимости от настройки.

> **Важно:** noise gate — **последнее средство**. Прежде чем использовать gate, устраните источник шума: замените кабели, улучшите заземление, поставьте humbuckers. Gate убирает симптомы, но не причину. Хорошая изоляция питания и качественные кабели могут дать тишину без gate.
