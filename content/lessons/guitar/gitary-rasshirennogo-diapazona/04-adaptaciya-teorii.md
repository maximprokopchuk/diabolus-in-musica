# Адаптация теории на расширенном грифе

## Основной принцип

**Теория музыки не меняется** при переходе на 7 или 8 струн — только **диапазон расширяется вниз**. Все те же гаммы, аккорды, интервалы, функциональные гармонии. Нужно только адаптировать **аппликатуры** (расположение пальцев на грифе).

## Гаммы на 7-струнной

Все гаммы можно продолжить на 7-ю струну. Паттерны просто «продолжаются вниз»:

```
A натуральный минор — все 7 струн:
e|--5-7-8----------|
B|--5-6-8----------|
G|--5-7------------|
D|--5-7------------|
A|--5-7------------|
E|--5-7------------|
B|--5-7-8----------|  ← 7-я струна
```

### Трёхнотные паттерны на 7 струн

На 7-струнной особенно удобны **паттерны с тремя нотами на струне** (3 notes per string):

```
E минор, 3 ноты на струне:
e|--------9-10-12--|
B|------9-10-12----|
G|----9-11-12------|
D|--9-11-12--------|
A|--9-10-12--------|
E|--9-10-12--------|
B|--7-9-10---------| ← 7-я струна добавляет нижний диапазон
```

## Аккорды с использованием 7-й струны

### Пауэр-аккорды

В Drop A строе пауэр-аккорд на нижних двух струнах — одним баррэ:

```
Drop A (A-E-A-D-G-B-E):
A5 пауэр-аккорд:
A|--0--|
E|--0--|
   (открытые 6-я и 7-я струны)

B5:
A|--2--|
E|--2--|
```

### Расширенные аккорды

7-я струна открывает возможность **сочетать низкий бас с высокими воицингами** — невозможное на 6-струнной:

```
Bm11 воицинг:
e|--7--|  (A — ундецима)
B|--7--|  (F# — квинта)
G|--7--|  (D — терция)
D|--9--|  (B — прима)
A|--9--|  (F# — квинта)
E|--7--|  (E — кварта)
B|--7--|  (B — прима, низкий бас)
```

## Арпеджио на 7 и 8 струнах

Sweep arpeggios можно строить через 6, 7 или 8 струн — это даёт огромные, полноокружные арпеджио:

```
Bm арпеджио через 7 струн:
e|--7----|
B|--7----|
G|--7----|
D|--9----|
A|--9----|
E|--7----|
B|--7----|  ← начало снизу
   ↑ sweep вверх
```

## Пентатоника на 7-струнной

Пентатоника не меняется структурно — добавь нижнюю струну по тому же принципу, что и другие:

```
Am пентатоника, 7 струн, позиция 1:
e|--5-8---|
B|--5-8---|
G|--5-7---|
D|--5-7---|
A|--5-7---|
E|--5-7---|
B|--5-8---|  ← 7-я струна
```

## Тональности и функциональная гармония

На 7- и 8-струнных можно строить **полноценные прогрессии** с использованием нижних струн как педальных нот или мощных бас-голосов:

```
Прогрессия в B минор:
B5 — D5 — A5 — E5
(все на нижних двух струнах в Drop A)
```

Функции те же: T (Bm), S (Em, G), D (F#m, A).

> **Практический совет:** при переходе на 7-струнную не нужно изучать всё заново. Возьми любую гамму или аппликатуру, которую знаешь на 6 струнах, и посмотри, как она продолжается на 7-ю. Начни с паттернов пентатоники — они самые интуитивные для расширения.
