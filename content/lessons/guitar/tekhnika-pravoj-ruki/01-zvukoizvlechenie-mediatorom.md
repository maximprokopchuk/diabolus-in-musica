# Звукоизвлечение медиатором

## Выбор медиатора

**Медиатор (plectrum, pick)** — небольшой кусочек пластика (или другого материала), которым извлекается звук. Выбор медиатора сильно влияет на тон и технику.

### Толщина

| Толщина | Применение |
|---------|-----------|
| **Thin (0.4–0.6 мм)** | Ритм-аккорды, бой, акустика |
| **Medium (0.7–0.85 мм)** | Универсальный |
| **Heavy (0.88–1.2 мм)** | Соло, точный пикинг, метал |
| **Extra Heavy (1.2+ мм)** | Джаз, точные техники |

**Правило:** для быстрого пикинга нужен жёсткий медиатор — тонкий будет «гулять» при атаке. Для ритмической игры тонкий даёт более мягкий звук.

### Материал

| Материал | Характер |
|---------|---------|
| **Celluloid** | Тёплый, стандартный |
| **Nylon** | Гибкий, мягкая атака |
| **Delrin / Tortex** | Чёткая атака, меньше скольжения |
| **Ultem** | Яркий, похож на черепаховый панцирь |
| **Металл / стекло** | Характерный «металлический» призвук |

## Хват медиатора

**Стандартный хват:** медиатор зажимается **между подушечкой указательного пальца и боковой поверхностью большого**. Выступает 3–5 мм.

### Типичные ошибки хвата

- Слишком много медиатора выступает → «гуляет» при атаке, теряется точность
- Слишком мало → задевает струны пальцами, теряется динамика
- Зажат слишком крепко → потеря гибкости, напряжение в руке
- Зажат слишком свободно → медиатор вращается при игре

## Угол удара

**Угол** между медиатором и струной влияет на тон:

| Угол | Звук |
|------|------|
| **Перпендикулярно** | Яркий, чёткий, «щелчок» атаки |
| **Под углом ~30–45°** | Более мягкий, «скользящий» |
| **Ребром** | Жёсткий, характерный для джаза |

Большинство гитаристов используют **слегка скошенный** удар — это компромисс между яркостью и плавностью.

## Точка атаки

Где по длине струны ты бьёшь медиатором, влияет на тембр:

- **Близко к бриджу** — яркий, резкий, много обертонов
- **Над звукоснимателем** — тёплый, «средний» тон
- **Близко к грифу** — мягкий, округлый, «ритмичный»

Этот приём используется как **художественный инструмент**: Robben Ford, Mark Knopfler намеренно меняют точку атаки во время игры.

## Динамика

**Динамика** — контроль громкости через силу удара. На гитаре динамика часто недооценивается начинающими.

| Стиль игры | Динамика |
|-----------|---------|
| Rhythm strumming | Широкий диапазон (pp → ff) |
| Lead / solo | Акценты на сильных долях |
| Fingerstyle | Очень тонкий контроль |
| Heavy metal | Равномерно сильная атака |

> **Совет:** запись себя — лучший способ услышать проблемы с динамикой. То, что кажется «динамичным» при игре, на записи часто оказывается монотонным. Упражнение: играй одну ноту с нарастающей и убывающей громкостью (crescendo/decrescendo), только правой рукой.
