# Выбор и стэкинг педалей перегруза

## Зачем стэкировать (stackировать) педали?

**Стэкинг** — использование двух и более педалей перегруза одновременно. Это не дублирование — это создание нового звука через взаимодействие педалей.

Логика стэкинга:
- Первая педаль (Drive) формирует «базу» перегруза
- Вторая педаль (буст или Drive) добавляет сатурацию, толчок, изменение характера
- Результат — новый тон, недостижимый одной педалью

---

## Базовые комбинации стэкинга

### Overdrive → Overdrive

**Два мягких клиппинга в серии** — суммируются в более плотный, насыщенный звук без жёсткости.

| Комбинация | Характер |
|-----------|---------|
| Tube Screamer → Blues Driver | TS добавляет mid, BD добавляет открытость |
| Klon → OCD | Прозрачный толчок + лёгкий distortion |
| TS → TS | «TS stack» — известный приём (SRV style) |

### Overdrive → Distortion

Overdrive «разогревает» вход distortion, делая его насыщеннее и «живее».

| Комбинация | Звук |
|-----------|------|
| TS → RAT | Мягкий мид + жёсткий клиппинг RAT |
| SD-1 → HM-2 | Шведский дэт-метал звук |

### Fuzz → Overdrive

Fuzz создаёт «фундамент» насыщенности, overdrive добавляет формирование.

### Boost → Drive

Самая распространённая схема:
- Clean Boost → Overdrive = усиленный, более сатурированный OD
- Clean Boost → Distortion = более «тайтный», насыщенный Distortion
- TS (0 drive) → Gain-канал усилителя = метал-буст

## Порядок педалей: почему важен порядок

При стэкинге порядок **принципиально меняет результат**.

### Педаль A → Педаль B

Педаль B «видит» уже искажённый сигнал от A. Дополнительная сатурация строится поверх существующей.

### Педаль B → Педаль A

Педаль A «видит» уже искажённый сигнал от B. Может дать «каша» из несовместимых гармоник, или неожиданно хорошо работать.

**Экспериментирование с порядком — стандартная практика.**

### Конкретные примеры

| Порядок | Результат |
|---------|----------|
| Fuzz → Overdrive | Fuzz-звук с «формированием» от OD, более контролируемо |
| Overdrive → Fuzz | Fuzz реагирует на форму волны OD, может дать «распад» |
| Boost → Drive | Drive получает усиленный сигнал, перегружается сильнее |
| Drive → Boost | Boost поднимает уровень уже перегруженного сигнала (соло-буст) |

## Взаимодействие педалей с усилителем

Стэк педалей — не замена усилителю, а **система**, где усилитель — последний и важнейший элемент.

### Clean amp + педальный стэк

- Весь характер перегруза — от педалей
- Предсказуемо, портабельно
- Стандарт для recording и touring с маленьким усилителем

### «Горячий» усилитель + буст

- Педаль (буст/OD) толкает преамп усилителя
- Усилитель добавляет свой характер
- «Живой», динамичный результат

### Drive-канал усилителя + буст

- Педаль только усиливает канал, не добавляет своего тембра
- Точный контроль над уровнем

## Как выбирать педали перегруза

### Шаг 1: определите базовый звук

Что вам нужно?

| Стиль | Рекомендуемая основа |
|-------|---------------------|
| Blues | TS9 / OCD / Fulltone GT500 |
| Classic Rock | RAT / DS-1 / OCD |
| Hard Rock | OCD / Wampler Pinnacle / Friedman BE-OD |
| Metal | BOSS MT-2 / Wampler Dracarys / Horizon Devices Precision Drive |
| Alternative / Shoegaze | Big Muff / Fuzz Face |
| Fuzz / Psychedelic | Fuzz Face / Tone Bender / Big Muff |

### Шаг 2: добавьте буст или второй Drive

Оцените, нужна ли вам:
- **Большая насыщенность?** → буст или второй Drive
- **Соло-буст?** → Clean Boost в конце цепи
- **Переключение между ритмом и соло?** → два разных Drive-педали

### Шаг 3: протестируйте с вашим усилителем

Один и тот же Drive на разных усилителях звучит по-разному. Обязательно тестируйте через свой усилитель.

## Распространённые ошибки при стэкинге

| Ошибка | Проблема | Решение |
|--------|---------|---------|
| Слишком много перегруза | «Каша», потеря артикуляции | Снизить Drive на одной из педалей |
| Несовместимые импедансы | Потеря высоких частот | Проверить буферизацию |
| Fuzz после буфера | Fuzz теряет характер | Fuzz — первым |
| Одинаковые педали в стэке | Нет нового тембра | Использовать педали с разным характером |

## Практические рекомендации по бюджету

**Минимальный сет (до 150$):**
- Boss SD-1 (или DS-1) + Ibanez TS9

**Средний сет (до 400$):**
- Tube Screamer + ProCo RAT + MXR Micro Amp (буст)

**Профессиональный сет:**
- Klon (клон) + OCD + Clean Boost (EP Booster)

> **Главный принцип:** меньше педалей с лучшим качеством лучше, чем много дешёвых. Начните с одной хорошей педали (TS9 или RAT) и усилителя — это даст вам 90% нужных звуков. Второй и третий Drive добавляйте, когда ясно понимаете, чего не хватает.
