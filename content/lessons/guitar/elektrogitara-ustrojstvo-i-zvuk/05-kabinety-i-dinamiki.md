# Кабинеты и динамики

## Роль кабинета в звуке

Кабинет (speaker cabinet) — не просто «ящик с динамиком». Он **принципиально влияет на характер звука**:
- Размер и материал корпуса меняют резонанс
- Конструкция (закрытый / открытый тыл) влияет на бас
- Тип динамика (speaker) определяет тембр

Один и тот же усилитель через разные кабинеты может звучать абсолютно по-другому.

## Конструкция кабинета

### Закрытый (Closed Back)
- Бас плотный, «собранный», направленный
- Хорошо для метала и хард-рока
- Меньше «рассеивает» звук по комнате

### Открытый (Open Back)
- Бас мягче, звук более «воздушный»
- Лучше для блюза, кантри, чистых звуков
- Характерен для Fender и Vox комбо

## Конфигурации кабинетов

| Конфигурация | Размер динамиков | Характер |
|-------------|-----------------|---------|
| **1×12** | Один 12" | Компактный, сбалансированный |
| **2×12** | Два 12" | Стереозвук, больше мощности |
| **4×12** | Четыре 12" | Максимальная мощность, «рок-стена» |
| **1×10, 2×10** | 10" | Яркий звук, быстрая атака (блюз) |
| **2×15** | 15" | Глубокий бас (реже для гитары) |

**4×12** — классика хард-рока и метала. Marshall 4×12 — иконический звук стека.

## Динамики

Самые важные производители гитарных динамиков:

### Celestion
Британский производитель, стандарт для рок- и метал-звука.

| Модель | Характер |
|--------|---------|
| **Vintage 30** | Характерный мид, «вой» в верхах, рок/метал |
| **Greenback** | Мягкий, «тёплый», British blues |
| **G12T-75** | Чёткий, высокий SPL, хард-рок |
| **Blue (Alnico)** | Редкий, мягкий, Vox AC30-характер |

### Eminence
Американский производитель, широкий диапазон применения.

| Модель | Характер |
|--------|---------|
| **Legend 1258** | Чистый, блюзовый |
| **Governor** | Аналог Vintage 30 |
| **Swamp Thang** | Американский звук, Southern rock |

### Jensen
Характерны для Fender-звука — чистые, яркие, «американские».

## Параметры динамиков

| Параметр | Описание |
|---------|---------|
| **Impedance (Ом)** | 4, 8 или 16 Ом — должен совпадать с усилителем |
| **Power Handling (Вт)** | Максимальная мощность без повреждения |
| **Sensitivity (дБ/Вт/м)** | Эффективность: чем выше, тем громче |
| **Диаметр** | 10", 12", 15" — влияет на частотный баланс |

## Импульсные характеристики (IR)

**Impulse Response (IR)** — цифровая «слепок» звучания реального кабинета с микрофоном в определённой позиции. Загружается в моделирующий усилитель или процессор эффектов.

Позволяет получить звук профессионального кабинета **без реального кабинета** — прямо в наушниках или на запись.

Популярные библиотеки IR: OwnHammer, Celestion, Ownhammer, York Audio.

## Согласование по импедансу

**Важно:** импеданс кабинета должен совпадать с номиналом выхода усилителя.

| Ситуация | Последствие |
|---------|------------|
| Кабинет слишком высокоомный | Усилитель работает «вхолостую», меньше мощности |
| Кабинет слишком низкоомный | Риск повреждения оконечного каскада |
| Совпадение | Оптимальная передача мощности |

> **Практический вывод:** для домашних репетиций подойдёт комбо 1×12 на 20–30 Вт с открытым тылом (Fender-стиль) или закрытым (Marshall-стиль). Кабинет 4×12 — для сцены, где нужна громкость и «физическое давление» звука.
