# Аккорды на гитаре: открытые позиции

## Что такое открытые аккорды

**Открытые аккорды** — аккорды, в которых используются открытые (незажатые) струны. Они звучат полно и резонансно, их легко брать.

## Основные мажорные

```
C           G           D           A           E
e|--0--|   e|--3--|   e|--2--|   e|--0--|   e|--0--|
B|--1--|   B|--3--|   B|--3--|   B|--2--|   B|--0--|
G|--0--|   G|--0--|   G|--2--|   G|--2--|   G|--1--|
D|--2--|   D|--0--|   D|--0--|   D|--2--|   D|--2--|
A|--3--|   A|--2--|   A|--x--|   A|--0--|   A|--2--|
E|--x--|   E|--3--|   E|--x--|   E|--x--|   E|--0--|
```

## Основные минорные

```
Am          Em          Dm
e|--0--|   e|--0--|   e|--1--|
B|--1--|   B|--0--|   B|--3--|
G|--2--|   G|--0--|   G|--2--|
D|--2--|   D|--2--|   D|--0--|
A|--0--|   A|--2--|   A|--x--|
E|--x--|   E|--0--|   E|--x--|
```

## Доминантсептаккорды (бонус)

```
G7          D7          A7          E7
e|--1--|   e|--2--|   e|--0--|   e|--0--|
B|--0--|   B|--1--|   B|--2--|   B|--3--|
G|--0--|   G|--2--|   G|--0--|   G|--1--|
D|--0--|   D|--0--|   D|--2--|   D|--0--|
A|--2--|   A|--x--|   A|--0--|   A|--2--|
E|--3--|   E|--x--|   E|--x--|   E|--0--|
```

## Правила постановки пальцев

1. Пальцы ставьте **ближе к ладовой перегородке** (но не на неё)
2. Пальцы должны быть **округлены** — не задевайте соседние струны
3. Большой палец — на тыльной стороне грифа, примерно напротив среднего пальца
4. **x** = струна не играется (глушится большим пальцем или не задевается)

## Какие тональности удобны

С открытыми аккордами удобно играть в:
- **C мажор / A минор**: C, Dm, Em, F, G, Am
- **G мажор / E минор**: G, Am, Bm(баррэ), C, D, Em
- **D мажор / B минор**: D, Em, F#m(баррэ), G, A, Bm
- **A мажор / F# минор**: A, Bm, C#m, D, E, F#m
- **E мажор / C# минор**: E, F#m, G#m, A, B, C#m

> **Первые аккорды:** начните с Em, Am, C, G, D — они самые простые и покрывают огромное количество песен.
