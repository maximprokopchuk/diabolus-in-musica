# Баррэ и подвижные формы

## Что такое баррэ

**Баррэ** (barre) — приём, при котором указательный палец прижимает все (или несколько) струн на одном ладу, выполняя роль «подвижного порожка».

Это превращает любой открытый аккорд в **подвижную форму** — её можно перемещать по грифу.

## Формы на основе E и A

### F (форма E на 1-м ладу)

```
e|--1--|  ← баррэ (указательный палец)
B|--1--|
G|--2--|
D|--3--|
A|--3--|
E|--1--|
```

Сдвигаем на 3-й лад = G, на 5-й = A, на 7-й = B.

### Bm (форма Am на 2-м ладу)

```
e|--2--|  ← баррэ
B|--3--|
G|--4--|
D|--4--|
A|--2--|
E|--x--|
```

Сдвигаем на 3-й лад = Cm, на 5-й = Dm, на 7-й = Em.

## Система CAGED

**CAGED** — пять основных форм открытых аккордов (C, A, G, E, D), которые покрывают весь гриф при перемещении с баррэ.

Для аккорда C мажор по грифу:

| Позиция | Форма | Лад баррэ |
|---------|-------|-----------|
| Открытая | C | — |
| 3-й лад | A | 3 |
| 5-й лад | G | (сложная) |
| 8-й лад | E | 8 |
| 10-й лад | D | 10 |

Все пять форм дают один и тот же аккорд C, но в разных регистрах грифа.

## Как работает CAGED

Формы идут в порядке: **C → A → G → E → D → C → ...**

Каждая следующая форма начинается там, где заканчивается предыдущая. Вместе они покрывают **весь гриф без пробелов**.

## Практические советы по баррэ

1. **Давите ребром** указательного пальца, не плоскостью
2. Палец должен быть **прямым** и близко к ладовой перегородке
3. Не пытайтесь прижать сильно — важнее **правильное положение**
4. Начинайте с **5-го лада** (лады уже, легче прижимать), затем двигайтесь к 1-му
5. Мизинец и безымянный палец формируют аккорд, указательный — баррэ

> **Не сдавайтесь:** баррэ — главный барьер для начинающих гитаристов. На привыкание уходит 2–4 недели регулярной практики. После этого весь гриф становится доступен.
