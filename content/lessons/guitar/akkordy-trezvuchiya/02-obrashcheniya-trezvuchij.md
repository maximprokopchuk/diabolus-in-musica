# Обращения трезвучий

## Что такое обращение

**Обращение** — перенос нижнего звука аккорда на октаву вверх. В результате в басу оказывается другая нота аккорда.

У трезвучия 3 звука — значит, **3 позиции**:

## Три позиции

| Позиция | Бас | Название | Обозначение |
|---------|-----|----------|-------------|
| Основной вид | Корень (1) | Трезвучие | C |
| 1-е обращение | Терция (3) | Секстаккорд | C/E |
| 2-е обращение | Квинта (5) | Квартсекстаккорд | C/G |

## Пример: C мажор

| Позиция | Ноты (снизу вверх) | Интервалы |
|---------|-------------------|-----------|
| Основной вид | **C** — E — G | терция + терция |
| 1-е обращение | **E** — G — C | терция + кварта |
| 2-е обращение | **G** — C — E | кварта + терция |

## Зачем нужны обращения

1. **Плавное голосоведение** — переход между аккордами с минимальным движением:
```
C → Am:   C-E-G → C-E-A  (меняется только один звук!)
```

2. **Мелодическая линия баса** — обращения позволяют создавать плавное движение баса:
```
C → C/E → F → F/A → G/B → C
Бас: C → E → F → A → B → C (поступенное движение)
```

3. **Разнообразие звучания** — один и тот же аккорд звучит по-разному в разных обращениях.

## Slash-аккорды на гитаре

На гитаре обращения записываются как **slash-аккорды**: C/E означает «аккорд C с нотой E в басу».

```
C/E:          C/G:
e|--0--|      e|--0--|
B|--1--|      B|--1--|
G|--0--|      G|--0--|
D|--2--|      D|--2--|
A|-----|      A|--3--|
E|--0--|      E|--3--|
```

> **Совет:** следите за нижней нотой аккорда — она определяет обращение. На гитаре для этого часто нужно заглушить некоторые струны.
