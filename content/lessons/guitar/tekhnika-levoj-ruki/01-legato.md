# Легато

## Что такое легато на гитаре

**Легато** (итал. legato — «связанно») — техника игры, при которой звук переходит от ноты к ноте **плавно и без разрывов**. На гитаре это достигается с помощью hammer-on и pull-off — без участия медиатора для каждой ноты.

## Hammer-On (хаммер-он)

**Hammer-on** (h) — «удар сверху». Пальцем левой руки резко и сильно ударяется по ладу, чтобы нота прозвучала без удара медиатора.

```
Обозначение: h
e|--5h7--|
        (медиатор бьёт только 5-ю ноту, 7-я звучит от удара пальца)
```

### Техника

- Удар должен быть **кончиком** пальца, прямо за ладом
- Палец «бьёт» с небольшой высоты (не поднимай слишком высоко)
- Нажимай сразу **финально** — без скольжения

### Ошибки

- Слишком мягкий удар — нота не звучит или звучит тихо
- Удар серединой пальца — нота звучит глухо
- Большая амплитуда подъёма — замедляет игру

## Pull-Off (пул-офф)

**Pull-off** (p) — «срыв». Палец «сдёргивается» со струны, слегка зацепляя её, чтобы прозвучала более низкая нота под ним.

```
Обозначение: p
e|--7p5--|
        (медиатор бьёт 7-ю, затем палец срывается → звучит 5-я)
```

### Техника

- Палец не просто поднимается, а **тянется вниз** (к полу) — это и создаёт звук
- Нижний палец должен уже стоять на ноте-цели
- Сила pull-off = сила звука следующей ноты

## Комбинированное легато

```
e|--5h7p5h7p5--|
   hammer, pull, hammer, pull...
```

Это базовый паттерн легато для гамм и пассажей.

## Легато-гаммы

```
Минорная пентатоника Am — легато:
e|--------5h8p5----------|
B|------5h8p5------------|
G|----5h7p5--------------|
D|--5h7p5----------------|
A|--5h7p5----------------|
E|--5h8p5----------------|
```

## Трели (Trills)

**Трель** — быстрое чередование двух нот с помощью hammer-on и pull-off.

```
e|--5~~tr~~7--|  (трель между 5 и 7)
  начни с 5, hammer 7, pull 5, hammer 7, pull 5...
```

Трели используются как орнаментация в:
- Классическом стиле (Yngwie Malmsteen)
- Блюзовых соло (B.B. King, Eric Clapton)
- Прогрессивном роке (Steve Howe, David Gilmour)

## Гитаристы легато

**Joe Satriani** — мастер плавного, «певучего» легато. «Always With Me, Always With You» — эталон легато на гитаре.

**Allan Holdsworth** — считается «богом легато» в джаз-фьюжне. Его пассажи практически неотличимы от духовых инструментов.

**Steve Lukather** — органичное использование легато в роке и фьюжне.

> **Совет:** чтобы проверить качество легато, сыграй упражнение **без медиатора вообще** — только hammer-on и pull-off. Если все ноты звучат ровно и чисто — легато поставлено правильно. Если некоторые ноты тихие или не звучат — значит нужно усилить работу пальцев.
