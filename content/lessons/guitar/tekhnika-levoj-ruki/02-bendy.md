# Бенды

## Что такое бенд

**Бенд** (bend, «сгиб») — техника, при которой нажатая струна **сдвигается поперёк** грифа, повышая натяжение и, соответственно, высоту ноты. Это основной инструмент экспрессии в блюзе, роке и большинстве современных стилей.

## Типы бендов

### Полутон (Semitone Bend)

Струна поднимается на **полтона** — на один лад.

```
e|--7b8--|
  (нота 7-го лада → звучит как 8-й лад)
```

Используется в блюзе для «bluesy» интонации, на ведущем тоне.

### Целый тон (Full Bend)

Струна поднимается на **целый тон** — на два лада.

```
e|--7b9--|
  (нота 7-го лада → звучит как 9-й лад)
```

Стандартный рок-бенд. Самый распространённый.

### Полтора тона (1.5 Step Bend)

```
e|--7b10--|
```

Используется в блюзе и хард-роке для драматического эффекта.

### Pre-Bend (предварительный бенд)

Струна сначала **поднимается молча** (без атаки), а затем звучит уже в поднятом положении.

```
e|--pb7--|
  (поднять до звука 9-го лада, затем ударить медиатором)
```

Даёт эффект «нота появляется сверху».

### Release Bend (спуск бенда)

После бенда струна **плавно опускается** обратно.

```
e|--7b9r7--|
  (поднять до 9, затем отпустить назад к 7)
```

### Pre-bend + Release

```
e|--pb7r--|
  (поднять молча, ударить, затем отпустить)
```

Характерный «плач» в блюзе — Stevie Ray Vaughan, BB King.

### Unison Bend

Две струны одновременно: одна поднимается бендом до высоты другой, которая уже звучит открыто или зажата.

```
e|---5---|
B|--7b9--|
  (B-струна поднимается до высоты e-струны → унисон)
```

Создаёт характерный «вой» и напряжение. Классика блюзового соло.

## Техника бенда

### Опора пальцев

**Никогда** не делай бенд одним пальцем. Используй **все доступные пальцы позади** рабочего:

- Бенд безымянным — помогают средний и указательный
- Бенд средним — помогает указательный
- Это удваивает и утраивает силу, контролирует точность

### Направление бенда

- На **тонких струнах** (e, B, G) — бенд **вверх** (к потолку)
- На **толстых струнах** (E, A) — бенд **вниз** (к полу)
- На G или D — оба варианта в зависимости от позиции

### Интонация

Бенд должен попадать **точно** в целевую ноту — не выше и не ниже. Наиболее распространённая ошибка — «недобенд» (останавливаются на полпути). Тренируй ухо: пой целевую ноту, затем поднимай бенд до совпадения.

## Бенды по стилям

| Стиль | Характерные бенды |
|-------|-----------------|
| Блюз | Pre-bend, release, медленный «плач» |
| Рок | Full bend, unison bend |
| Кантри | Быстрый, точный half/full bend |
| Метал | Быстрый wide bend + vibrato |

**Эталонные гитаристы:** BB King (экспрессивный «crying» бенд), David Gilmour (широкие, точные бенды), Stevie Ray Vaughan (интенсивные, «плотные» бенды на толстых струнах).

> **Совет:** проверяй точность бенда так — зажми целевую ноту (например, 9-й лад), запомни её высоту, затем попробуй бенд с 7-го лада. Нота должна совпасть идеально. Если недотягиваешь — тренируй силу пальцев и слух одновременно.
