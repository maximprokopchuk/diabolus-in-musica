# Re-amping

## Что такое Re-amping

**Re-amping** (переусиление) — процесс воспроизведения ранее записанного DI-сигнала **через реальный гитарный усилитель** с целью получения «живого» звука с микрофоном.

```
[DAW воспроизводит DI-трек]
         │
    [Выход DAW]
         │
   [Reamp Box]   ← преобразует Lo-Z → Hi-Z
         │
  [Гитарный усилитель]
         │
    [Кабинет]
         │
    [Микрофон]
         │
    [Запись в DAW]
```

## Зачем это нужно

### Свобода в момент записи

Гитарист приходит в студию, записывает **идеальное исполнение** как DI — не беспокоясь о звуке. Потом, в отдельную сессию, звукорежиссёр или продюсер занимается выбором звука:

- Пробует разные усилители
- Экспериментирует с педалями
- Меняет позицию микрофона
- Пробует разные кабинеты

### Невозможное без Re-amping

Гитарист живёт в другом городе? Студия с дорогими усилителями недоступна во время исполнения? Re-amping решает эти проблемы — DI записывается где угодно, а прогоняется через усилитель в профессиональной студии позже.

### Тестирование множества вариантов

С DI-записью можно **прогнать один и тот же дубль** через 5 разных усилителей и выбрать лучший — не прося гитариста играть снова.

## Оборудование

### Reamp Box

**Reamp Box** — специальный трансформатор для согласования уровней и импедансов.

| Параметр | Линейный выход DAW | Вход гитарного усилителя |
|---------|-------------------|------------------------|
| Уровень | +4 dBu (студийный) | -20 dBu (инструментальный) |
| Импеданс | 100–600 Ом | 1 МОм |

Без Reamp Box:
- Уровень сигнала слишком высокий — усилитель будет клиппировать даже без gain
- Неправильное согласование импедансов искажает тембр

**Популярные Reamp Box:**
- **Radial ProRMP** — профессиональный стандарт
- **Radial X-Amp** — активный, позволяет регулировать уровень
- **Vintage King Reamp** — бюджетный вариант
- **Самодельный** — трансформаторная схема несложна в изготовлении

### Альтернатива: два аудиоинтерфейса

Некоторые инженеры используют два интерфейса: один для воспроизведения, другой для записи. Работает, но менее удобно.

## Процесс Re-amping пошагово

### 1. Подготовка DI-трека

- Убедитесь, что DI-трек **чистый**: без клиппинга, без нежелательных шумов
- Уровень: примерно -12 dBFS (средний) — не слишком громкий

### 2. Маршрутизация в DAW

- Направьте вывод DI-трека на **отдельный выход** аудиоинтерфейса (не главный выход мониторов)
- Создайте новый **аудио-трек** и назначьте на него вход с микрофона

### 3. Подключение оборудования

```
Отдельный выход интерфейса
         │
    [Reamp Box]
         │
      [Педали] (опционально)
         │
    [Усилитель]
         │
     [Кабинет]
         │
    [Микрофон]
         │
   Вход интерфейса (для записи)
```

### 4. Регулировка уровня

- Начните с Gain усилителя на минимуме
- Включите воспроизведение DI-трека в DAW
- Постепенно поднимайте Gain до желаемого звука
- Проверьте, что записывающий микрофон не клиппирует

### 5. Синхронизация

Re-amped трек в DAW будет **с задержкой** относительно оригинала (время прохождения сигнала через оборудование). Необходимо:
- Сдвинуть re-amped трек назад во времени (align)
- Или использовать функцию Auto-Align в DAW

Практика: запишите короткий щелчок (click) одновременно на DI и re-amped трек — совместите щелчки вручную.

## Re-amping с Amp-Sim

**Виртуальный Re-amping** — то же самое, но вместо реального усилителя используется плагин Amp-Sim.

```
DI трек → [Amp-Sim плагин] → Обработанный сигнал
```

Преимущества:
- Мгновенно
- Без шума
- Без оборудования
- Можно вернуться и поменять звук в любой момент

Это стандартная практика в домашних студиях и при жёстких дедлайнах.

## Сравнение подходов

| Подход | Реализм | Гибкость | Скорость | Оборудование |
|-------|---------|---------|---------|-------------|
| Живой кабинет + микрофон | Максимум | Низкая | Медленно | Много |
| Re-amping через усилитель | Высокий | Высокая | Средняя | Средне |
| Amp-Sim в реальном времени | Высокий* | Максимальная | Быстро | Минимум |
| Amp-Sim постфактум | Высокий* | Максимальная | Быстро | Минимум |

*Современные Amp-Sim класса Neural DSP практически неотличимы от реального усилителя на записи.

> **Рабочая стратегия:** записывайте всегда параллельно — DI и живой кабинет одновременно. DI страхует: если звук кабинета не понравится при сведении, есть возможность re-amping'а. Живой трек — готовый звук без лишней работы.
