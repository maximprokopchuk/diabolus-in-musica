# DI запись гитары

## Что такое DI

**DI** (Direct Injection / Direct Input) — запись сигнала гитары **напрямую**, без усилителя и кабинета. Сигнал идёт прямо в интерфейс звуковой карты.

```
Гитара → [DI Box] → Аудиоинтерфейс → DAW
           (Hi-Z → Mic Level)
```

DI Box согласует высокое выходное сопротивление гитары (Hi-Z, ~1 МОм) с низкоомным входом (Lo-Z) предусилителя интерфейса. Без DI Box звук будет тонким и с потерями.

## Зачем записывать DI

### 1. Гибкость при сведении

DI-сигнал — «сырой», необработанный. Это означает:
- Можно в любой момент **переобработать** сигнал другим amp-sim
- Нет риска «сгореть» на выборе усилителя в начале сессии
- Можно экспериментировать со звуком после записи исполнения

### 2. Тихая запись

Без реального усилителя — **никакого шума**. Можно записывать поздно ночью в квартире.

### 3. Re-amping позже

Записанный DI можно потом прогнать через **реальный усилитель** (re-amping) для получения «живого» звука.

### 4. Мгновенная готовность

Нет нужды устанавливать микрофоны, настраивать акустику, ждать прогрева лампового усилителя.

## Типы DI Box

### Пассивные

- Трансформаторный тип
- Не требует питания
- Тёплый, немного «тёмный» характер
- Примеры: Radial J48 (passive), Countryman Type 85

### Активные

- Встроенная электроника с питанием (+48V Phantom или батарея)
- Более точное, нейтральное звучание
- Больший динамический диапазон
- Примеры: Radial J48 (active), BSS AR-133

### Hi-Z входы аудиоинтерфейсов

Большинство современных интерфейсов имеют **встроенный Hi-Z вход** (обозначается «INST» или иконкой гитары). Он и есть встроенный DI — можно подключить гитару прямо в интерфейс без внешнего DI Box.

## Параллельная запись: DI + кабинет одновременно

Профессиональная практика — записывать **оба сигнала одновременно**:

```
Гитара
   │
   ├──→ [DI Box] ──→ Вход 1 (сухой DI сигнал)
   │
   └──→ [Усилитель + Кабинет + Микрофон] ──→ Вход 2 (живой звук)
```

Это даёт:
- Готовый живой звук для немедленного использования
- DI-трек для последующего re-amping'а или amp-sim
- Максимальную гибкость

## Amp-Sim в реальном времени

**Amp-Sim** (симулятор усилителя) — плагин, имитирующий звучание реального усилителя. Позволяет работать с DI-сигналом так, будто это живая запись через кабинет.

### Популярные Amp-Sim плагины

| Плагин | Разработчик | Особенность |
|-------|------------|-------------|
| Neural DSP Archetype | Neural DSP | Авторизованные подписи музыкантов |
| Bias Amp 2 | Positive Grid | Технология Tone Matching |
| AmpliTube 5 | IK Multimedia | Огромная библиотека, физическое моделирование |
| Guitar Rig 7 | Native Instruments | Гибкий процессор эффектов |
| S-Gear | Scuffham Amps | Ламповое ощущение, простота |
| ToneX | IK Multimedia | Профилирование реального усилителя |

### Компоненты Amp-Sim

```
DI Сигнал
    │
[Стомп-боксы (педали эффектов)]
    │
[Модель усилителя (preamp + power amp)]
    │
[Модель кабинета (IR или симуляция)]
    │
[Микрофон + позиция (виртуально)]
    │
[Комнатный отзвук (опционально)]
    │
Готовый сигнал
```

### Cabinet IR в Amp-Sim

Многие современные Amp-Sim используют **Impulse Response** (IR) — математическую «слепок» реального кабинета с реальным микрофоном. Это значительно реалистичнее, чем имитация фильтром.

**IR-библиотеки:** OwnHammer, York Audio, Celestion — профессиональные библиотеки IR реальных кабинетов.

## Латентность при мониторинге

При записи с Amp-Sim в реальном времени возникает **задержка (latency)**: сигнал проходит через интерфейс → в DAW → через Amp-Sim → обратно в наушники. Задержка более 10–15 мс неприемлема для игры.

### Решения

- **Уменьшить буфер**: 32–64 samples (даёт < 5 мс), но увеличивает нагрузку CPU
- **Direct Monitoring**: слушать сухой DI через интерфейс, а Amp-Sim применять постфактум
- **DSP-ускорение**: аппаратные решения (UAD Apollo, Antelope) обрабатывают плагины на встроенном DSP чипе без задержки

> **Практическая стратегия:** при записи используйте Direct Monitoring + мысленно «слышите» звук Amp-Sim. После записи накладывайте Amp-Sim на DI трек. Так получите чистую запись без компромиссов по latency.
