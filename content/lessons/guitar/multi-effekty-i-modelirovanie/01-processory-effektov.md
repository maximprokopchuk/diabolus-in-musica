# Процессоры эффектов

## Что такое мульти-эффект

**Мульти-эффект** (multi-FX processor) — устройство, объединяющее в одном корпусе десятки или сотни эффектов и усилителей. Вместо отдельного перегруза, хоруса, задержки и реверба — один блок с полным набором инструментов.

Современные топовые процессоры — это уже не просто «сборники педалей». Это **полноценные звуковые платформы**, включающие моделирование усилителей, IR-кабинетов, сложных цепочек эффектов, MIDI-управление и прямую запись в DAW.

## Fractal Audio Axe-Fx

**Axe-Fx** (III, FM9, FM3) — профессиональный стандарт для студии и сцены.

| Характеристика | Описание |
|---------------|---------|
| Моделирование | 270+ усилителей и сотни кабинетов |
| Качество IR | Встроенные и пользовательские |
| Сигнальная цепь | До 512 блоков в библиотеке, гибкий роутинг |
| Обновления | Постоянные через Fractal Bot |
| Цена | Высокий сегмент ($1500–$3000+) |

**Сильные стороны:** реалистичность моделирования ламповых усилителей на высокой громкости, детальный контроль над каждым параметром. Любимый инструмент Марти Фридмана, Тоссина Абасси, Нили Пурт.

**Weaknesses:** крутая кривая обучения, требует времени на освоение редактора Axe-Edit.

## Kemper Profiler

**Kemper** — уникальный принцип: не моделирование, а **профилирование** (profiling) реальных усилителей.

Принцип работы профилирования:
1. Подключаем реальный усилитель к кабинету
2. Kemper посылает тестовые сигналы
3. Алгоритм строит математическую модель конкретного усилителя
4. Результат — «профиль», точно воспроизводящий тот усилитель

| Режим | Описание |
|-------|---------|
| Studio Profile | Усилитель + микрофон + кабинет |
| Direct Profile | Только усилитель (без кабинета) |
| Merged Profile | Можно разделить amp и кабинет |

**Rig Manager** — бесплатное ПО и огромная онлайн-библиотека профилей от пользователей. Тысячи профилей реальных усилителей Dumble, Matchless, Bogner, Mesa и т.д.

**Kemper Stage** — версия в форм-факторе педалборда с встроенным контроллером.

## Line 6 Helix

**Helix** (Floor, LT, HX Stomp) — самый доступный из топовых процессоров, отличный баланс цены и качества.

| Модель | Особенности |
|--------|------------|
| Helix Floor | Полноразмерный, 12 переключателей, 2 экспресс-педали |
| Helix LT | Версия без некоторых входов/выходов, дешевле |
| HX Stomp | Компактный, 3 переключателя, идеален как дополнение |
| HX Effects | Только эффекты без ампмоделирования |

**Сильные стороны:**
- Простой и интуитивный интерфейс
- Отличная экосистема (Helix Native — плагин для DAW)
- Хорошее качество моделирования по доступной цене
- Большое сообщество, много бесплатных пресетов

**Используют:** Joe Bonamassa, Pete Thorn, John Petrucci (в студии)

## Boss GT-Series

**Boss GT-1000 / GT-1000 Core / GT-100** — японский профессиональный уровень с фокусом на надёжность и удобство.

- **GT-1000** — флагман с AIRD (Augmented Impulse Response Dynamics) технологией
- Удобное управление с дисплеем
- Отличная MIDI-интеграция
- Встроенные петлёры

**GT-1** — бюджетная версия: компактная, лёгкая, подходит для начинающих.

## Neural DSP

**Neural DSP** выпускает и плагины (Archetype серия), и процессор **Quad Cortex**.

### Quad Cortex
- Touchscreen-интерфейс (первый в индустрии)
- Нейросетевое моделирование усилителей
- Встроенный профайлер (как Kemper)
- **Neural Capture** — захват усилителей прямо в процессоре
- Обновления через NDSP Cloud

**Плагины Neural DSP** (Archetype: Plini, Nolly, Tim Henson и др.):
- Работают в DAW (VST/AU/AAX)
- Идеальны для домашней студии
- Созданы в сотрудничестве с конкретными гитаристами
