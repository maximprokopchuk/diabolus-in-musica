# Студийное применение процессоров

## Запись напрямую (Direct In)

Главное преимущество современных процессоров в студии — **запись без микрофона и кабинета**.

### Сигнальный путь при прямой записи:

```
Гитара → Процессор (моделирование + IR) → USB/AES/S-PDIF → DAW
```

**Преимущества:**
- Никакой акустической обработки комнаты
- Запись в любое время (ночью, в квартире)
- Идеальный звук «с первого дубля» без настройки микрофонов
- Полный контроль над звуком после записи (если записан DI)

### Подключение к DAW

| Интерфейс | Процессоры |
|-----------|-----------|
| USB Audio | Helix, Quad Cortex, Boss GT-1000 |
| AES/EBU | Axe-Fx III (профессиональный) |
| S/PDIF | Kemper, Axe-Fx |
| Аналог через аудио-интерфейс | Все устройства |

**Helix как аудио-интерфейс:**
- Подключается по USB
- Работает как интерфейс 8-in/8-out
- Нулевая задержка мониторинга через процессор
- Записывает одновременно обработанный сигнал и DI

## Параллельная запись DI + обработанного сигнала

Профессиональная практика — записывать **одновременно**:
1. **DI (сухой сигнал)** — чистая гитара без обработки
2. **Обработанный сигнал** — через процессор в реальном времени

```
Гитара
  ↓
[DI-box (параллельный выход)]──────────────────→ DAW: Track 1 (DI)
  ↓
[Процессор → USB]────────────────────────────────→ DAW: Track 2 (обработанный)
```

**Зачем:** если звук обработанной дорожки не понравится позже, можно сделать **реамп** через DI-дорожку.

## Реамп (Re-amping)

**Реамп** — процесс пропуска записанного DI-сигнала через усилитель или процессор постфактум.

### Классический реамп (через реальный усилитель):

```
DAW: DI дорожка
  ↓
[Reamp Box (Radial ProRMP)] — конвертирует линейный уровень в инструментальный
  ↓
Ламповый усилитель
  ↓
Кабинет + микрофон
  ↓
Аудио-интерфейс → DAW: готовый звук
```

**Reamp box** необходим, потому что:
- DI-сигнал из DAW — линейный уровень (+4 dBu)
- Гитарный усилитель ожидает инструментальный уровень (-20 dBu)
- Без согласования импеданса теряются высокие частоты и характер

### Реамп через процессор:

Все современные процессоры поддерживают реамп:

**В Helix:**
1. Из DAW посылаем DI дорожку на Return вход Helix
2. В Helix настраиваем Input = Return
3. Helix обрабатывает сигнал и посылает обратно в DAW
4. Пробуем разные пресеты без перезаписи

**В Axe-Fx:**
- Аналогично через USB Audio Return
- Axe-Edit позволяет менять пресеты в реальном времени

### Реамп через плагины (Native/Offline):

- **Helix Native** — плагин VST/AU, идентичный Helix Floor
- **Neural DSP Archetype серия** — плагины с полным ампмоделированием
- **ToneX** (IK Multimedia) — плагин с нейросетевыми моделями
- **Kemper Kone** (через DAW интеграцию)

**Workflow:**
1. Записываем только DI (чистую гитару)
2. Вставляем плагин на дорожку
3. Меняем звук сколько угодно без перезаписи
4. Рендерим в финальный звук

## Запись без кабинета (Cabless Recording)

### Прямая запись в интерфейс через процессор
Самый простой вариант для домашней студии:

```
Гитара → Helix HX Stomp → USB → Mac/PC (GarageBand / Reaper / Pro Tools)
```

Helix настраивается как аудио-интерфейс, IR включён в пресете — готовый звук сразу в DAW.

### Speaker Simulator без кабинета

Для тех, кто использует реальный усилитель:
- **Two Notes Torpedo** — аппаратный нагрузочный эквивалент (load box) + IR
- **Universal Audio OX** — load box + моделирование комнаты
- **Boss Waza Tube Amp Expander** — для головы Boss Waza

Принцип: усилитель работает «в полную» через load box, сигнал снимается после трансформатора — гитарный кабинет не нужен.

## Латентность и мониторинг

**Проблема:** при записи через процессор + DAW возникает задержка (latency) — неудобно играть с мониторингом через компьютер.

**Решения:**
1. **Zero-latency мониторинг через процессор** — слушаем напрямую из Helix/Axe-Fx, не через DAW
2. **Low-latency аудио-интерфейс** — RME, Universal Audio, Focusrite Scarlett (с низким buffer size)
3. **ASIO4ALL (Windows)** — оптимизированные драйверы

**Рекомендуемый buffer size для записи:** 64–128 семплов (1–3 мс при 44.1kHz)

## Пример студийного сетапа (домашняя студия)

```
Гитара (Les Paul)
  ↓
Neural DSP Quad Cortex
  ↓ USB
Mac (Logic Pro X)
  - Дорожка 1: DI (чистый сигнал, через Reamp позже)
  - Дорожка 2: Обработанный (Quad Cortex пресет)
  ↓
Мониторинг: Yamaha HS8 или Sony MDR-7506
```

> **Профессиональный совет:** всегда записывайте DI-дорожку параллельно с обработанным сигналом. Диск не стоит ничего, а возможность переделать звук без переигрывания партии — бесценна. Многие знаменитые альбомы были переамплифицированы постфактум, когда продюсер менял видение звука уже после записи партий.
