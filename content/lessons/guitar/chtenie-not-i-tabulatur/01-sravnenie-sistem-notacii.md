# Сравнение систем нотации

## Два языка гитарной музыки

Гитаристы используют две основные системы записи:
- **Стандартная нотация** (standard notation) — нотный стан
- **Табулатура** (tab) — шесть линий = шесть струн

Обе системы имеют своё место. Профессиональный гитарист умеет читать обе.

## Стандартная нотация

### Принцип:
Нотный стан — 5 линий. Высота ноты = вертикальное положение. Длительность = форма ноты (целая, половинная, четверть, восьмая…).

### Для гитары:
Гитара — **транспонирующий инструмент**: ноты записываются на октаву выше реального звучания. Это сделано для удобства — иначе большинство нот уходило бы под стан.

```
Открытые струны в нотации (звучание на октаву ниже):
E4  —  первая струна
B3  —  вторая
G3  —  третья
D3  —  четвёртая
A2  —  пятая
E2  —  шестая (нижняя)
```

### Плюсы стандартной нотации:
| Преимущество | Описание |
|-------------|---------|
| Точный ритм | Длительности нот явно записаны |
| Высота точна | Видно точную ноту, независимо от аппликатуры |
| Совместимость | Читается всеми инструменталистами |
| Артикуляция | Лиги, штрихи, динамика — всё стандартизировано |
| Классическое образование | Необходима для академических учреждений |

### Минусы:
| Недостаток | Описание |
|-----------|---------|
| Не показывает расположение на грифе | Одна и та же нота — 5 позиций на гитаре |
| Требует обучения | Занимает время для освоения |
| Не показывает техники | Hammer-on, pull-off, бенд — только спецсимволами |

## Табулатура

### Принцип:
6 линий = 6 струн (нижняя линия = шестая струна E). Цифра на линии = лад. 0 = открытая струна.

```
e |--0--3--5--|
B |--1--3--5--|
G |--0--2--4--|
D |-----------|
A |-----------|
E |-----------|
```

### Плюсы табулатуры:
| Преимущество | Описание |
|-------------|---------|
| Показывает расположение на грифе | Сразу видно, куда ставить пальцы |
| Легко читается начинающими | Не требует знания нот |
| Специфические техники | Бенды, слайды, хаммеры — визуально понятны |
| Широко распространена | Тысячи сайтов с бесплатными табами |

### Минусы:
| Недостаток | Описание |
|-----------|---------|
| Ритм условный | Традиционные табы не показывают длительности точно |
| Не универсальна | Только для гитары/баса |
| Ошибки в сети | Большинство онлайн-табов сделаны «на слух» любителями — много ошибок |
| Нет высоты | Нельзя узнать, какие ноты, без дополнительной информации |

## Комбинированная нотация

Профессиональные нотные издания часто используют **TAB + стандартную нотацию** одновременно:

```
  [Нотный стан]
  ═══════════════
  [Табулатура]
```

Это лучший вариант: нотация даёт точный ритм и высоту, таблатура — аппликатуру.

## Когда что использовать

| Ситуация | Рекомендация |
|---------|-------------|
| Классическая гитара | Стандартная нотация (обязательно) |
| Рок/поп/блюз неформально | Табулатура |
| Студийная работа | Стандартная нотация |
| Джаз | Lead sheets (мелодия + цифровые гармонии) |
| Академическое образование | Стандартная нотация |
| Быстрое разучивание поп-песни | Таб + прослушивание оригинала |

## Цифровые гармонии (Nashville Number System)

В джазе и поп-музыке используются **lead sheets**: мелодия в нотах + аккорды обозначены буквами (C, Dm7, G7) или цифрами (I, ii, V7). Это быстро и функционально.

> **Рекомендация:** учите обе системы. Табулатура удобна для изучения рок-репертуара. Стандартная нотация открывает весь остальной музыкальный мир: классику, джаз, оркестровые партии, обучение детей. Гитарист, умеющий только табы, автоматически ограничен.
