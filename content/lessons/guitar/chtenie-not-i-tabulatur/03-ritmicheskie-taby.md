# Ритмические табы

## Проблема обычных табулатур

Классическая табулатура показывает **где играть** (лад и струна), но не **когда** и **как долго**. Это основной недостаток формата: без прослушивания оригинала невозможно понять ритм.

**Ритмические табы** (rhythmic tabs) решают эту проблему, добавляя ритмическую нотацию над или под линиями таба.

## Стандарт ритмических обозначений

Над каждой цифрой таба ставится ритмический символ, заимствованный из стандартной нотации:

| Символ | Длительность | Вид |
|--------|-------------|-----|
| Целая | 4 доли | Пустая нота без штиля |
| Половинная | 2 доли | Пустая нота со штилем |
| Четверть | 1 доля | Закрашенная нота со штилем |
| Восьмая | 1/2 доли | Закрашенная нота с флажком / беами |
| Шестнадцатая | 1/4 доли | Два флажка / двойная беама |
| Пунктирная | × 1.5 | Точка рядом с нотой |
| Пауза | молчание | Пауза соответствующей длительности |

## Беамы (beam) — группировки восьмых

Восьмые и шестнадцатые нередко **объединяются горизонтальной балкой (beam)** для удобства чтения:

```
  ♪ ♪ ♪ ♪   →   [████]   (4 восьмых = 1 доля в 4/4 ×4 = такт)

  ♬ ♬ ♬ ♬ ♬ ♬ ♬ ♬   →   [████████]   (8 шестнадцатых = 2 доли)
```

## Пример ритмического таба

```
Такт 1:
  ♩    ♩    ♩  ♩
e|--5--5--5--5--|
B|--5--5--5--5--|
G|--6--6--6--6--|
D|--7--7--7--7--|
A|--7--7--7--7--|
E|--5--5--5--5--|
```

## Спецсимволы в табах

Ритмические табы также включают обозначения техник:

| Символ | Значение |
|--------|---------|
| `h` | Hammer-on (левая рука, без удара правой) |
| `p` | Pull-off (снятие пальца) |
| `b` | Bend (загиб струны вверх) |
| `r` | Release (отпустить бенд обратно) |
| `~` | Vibrato |
| `/` | Slide вверх |
| `` | Slide вниз |
| `x` | Muted note (заглушённая нота) |
| `<>` | Harmonics (флажолеты) |
| `t` | Tap (тэппинг правой рукой) |
| `P.M.` | Palm mute (заглушение ладонью у бриджа) |

## Пример таба с техниками

```
  ♩  ♪ ♪  ♩      ♩  ♪ ♪ ♩
e|---------------------|
B|---------------------|
G|-------9b11~~--------|
D|--7h9p7------9\7-----|
A|---------------------|
E|---------------------|
```

Расшифровка:
- `7h9` — сначала нота на 7-м ладу, потом без удара хаммером на 9-й
- `9p7` — pull-off с 9-го лада на 7-й
- `9b11` — бенд с 9-го лада до высоты 11-го
- `~~` — вибрато
- `9\7` — слайд вниз с 9-го на 7-й лад

## Guitar Pro и цифровые редакторы

**Guitar Pro** — стандартный профессиональный редактор ритмических табов. Он показывает:
- Таб + стандартная нотация одновременно
- Точный ритм с метрономом
- Воспроизведение (MIDI или RSE — реалистичные инструменты)
- Все спецсимволы

Альтернативы:
| Программа | Платформа | Стоимость |
|----------|-----------|---------|
| Guitar Pro 8 | PC/Mac/iOS | Платно |
| TuxGuitar | PC/Mac/Linux | Бесплатно |
| MuseScore | PC/Mac | Бесплатно |
| Soundslice | Веб | Freemium |

## Расшифровка «немых» табов из интернета

Большинство бесплатных онлайн-табов — без ритмики. Как восстановить ритм:

1. Найдите оригинальную запись
2. Замедлите до 70% скорости
3. Отмечайте прослушивая, где каждый «штрих» правой руки
4. Записывайте ритмические символы прямо над табом
5. Сверяйте с метрономом

> **Практическое правило:** всегда слушайте оригинал вместе с табом, даже если в табе есть ритмическая нотация. Нотация показывает структуру — но groove, штрихи и фразировку можно понять только из записи.
