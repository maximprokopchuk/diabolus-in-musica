# Голосоведение на гитаре

## Специфика гитарного голосоведения

Гитара — **полифонический инструмент**, но с существенными ограничениями по сравнению с фортепиано:
- Максимум 6 голосов (струн)
- Физические ограничения растяжки
- Нельзя свободно держать любой голос любое время
- Строй (EADGBE) создаёт нестандартную геометрию

Тем не менее принципы голосоведения **полностью применимы** к гитаре — особенно в джазе и классике.

## Drop 2 как инструмент голосоведения

Drop 2 воицинги позволяют создать плавное голосоведение при смене аккордов. Ключевое: **третий и седьмой тон** обязательно ведутся плавно (на полтон или тон).

### Пример плавного голосоведения II-V-I в C:

**Dm7 → G7 → Cmaj7 (Drop 2, струны 2-5):**

```
Струна     Dm7   G7    Cmaj7
e (1):     —     —     —
B (2):     C     B     B     ← ведущий тон: B → C (ведущий→тоника)
G (3):     A     G     G     ← квинта остаётся
D (4):     F     F→E   E     ← септима G7 → терция Cmaj7
A (5):     D     G     C     ← прима (бас), движется функционально
```

Заметьте:
- **B → C** (B = ведущий тон G7, С = тоника)
- **F → E** (F = септима G7, E = терция Cmaj7)
- Эти два голоса — «двигатель» разрешения

## Ведение внутренних голосов

**Внутренние голоса** (альт и тенор в SATB = средние струны на гитаре) — часто игнорируются начинающими. Но именно они создают «качество» гармонии.

### Проблема стандартных аккордов:

Аккорд Am (x02210) при переходе к E (022100):

```
Струна    Am    E
e:        0     0  (нет движения)
B:        1     0  (полтон вниз — отлично)
G:        2     1  (полтон вниз — отлично)
D:        2     2  (нет движения)
A:        0     2  (движение — хорошо)
E:        x     0  (нет движения)
```

Это прилично! Внутренние голоса двигаются на полтон.

### Проблема при G → Am:

```
Струна    G      Am
e:        3      0  (большой скачок!)
B:        3      1  (секунда вниз)
G:        0      2  (секунда вверх)
D:        0      2  (секунда вверх)
A:        2      0  (секунда вниз)
E:        3      x  (бас теряется)
```

Внутренние голоса — нормально, но крайние прыгают. Решение: использовать **G/B (G с B в басу)**:

```
Струна    G/B    Am
e:        3      0  (скачок — можно оставить как мелодическое)
B:        0      1  (полтон вверх — хорошо)
G:        0      2  (тон вверх)
D:        0      2  (тон вверх)
A:        2      0  (тон вниз)
E:        x      x
```

## Хроматические движения внутренних голосов

**Хроматические проходящие** создают «голливудский» эффект плавного движения:

```
Cmaj7 → Cm7 (параллельный минор):
G: C-E-G-B → C-Eb-G-Bb
Движение:
  E → Eb  (полутон вниз — хроматика!)
  B → Bb  (полутон вниз)
  C и G — остаются
```

Этот тип голосоведения — «нео-соул», R&B, джазовые баллады.

## Pedal Point (педальный тон) на гитаре

**Педальный тон** — один голос остаётся неподвижным, пока остальные меняются:

```
e: 3-3-3-3-3-3  (педаль G)
B: 3-4-3-5-6-7  (движущийся голос)
G: 0-2-4-5-4-2  (движущийся)
D: 0-0-0-0-0-0  (педаль D — может быть открытой)
```

Открытые струны гитары — идеальный педальный тон: не нужно держать палец.

## Практическое голосоведение: смена аккордов

**Правило «держи, что можно»:**

При смене от C до Am:
```
C:  x-3-2-0-1-0  →  Am: x-0-2-2-1-0
     B остаётся (1-я позиция)
     G → A (остаётся G-струна)
     D → D (остаётся)
```

**Упражнение с реальными аккордами:**

Возьмите прогрессию C — Am — F — G7. Для каждой смены найдите:
1. Какие ноты общие?
2. Оставьте общие ноты на тех же струнах
3. Двигайте только те, что меняются
4. Минимизируйте движение каждого пальца

| Техника | Описание | Применение |
|---------|---------|-----------|
| Drop 2 | 4-голосный воицинг | Джаз-компинг |
| Педальный тон | Неподвижный голос | Латиноамериканские стили, ambient |
| Хроматические inner voices | Полутоновые движения | Neo-soul, R&B |
| Common tones | Удерживаемые ноты | Плавные переходы |

> **Совет:** Возьмите простую прогрессию (C-Am-F-G) и сыграйте её, меняя воицинги так, чтобы каждый голос двигался не более чем на секунду. Это может потребовать нестандартных аппликатур — но результат будет «певучим» и профессиональным. Голосоведение — главный секрет, отличающий «аккорды» от «гармонии».
