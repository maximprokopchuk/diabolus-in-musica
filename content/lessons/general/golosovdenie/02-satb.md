# SATB и четырёхголосие

## Что такое SATB

**SATB** — четыре основных хоровых голоса:
- **S** — Soprano (сопрано): высший женский голос
- **A** — Alto (альт): низший женский голос
- **T** — Tenor (тенор): высший мужской голос
- **B** — Bass (бас): низший мужской голос

SATB-хорал — **стандарт обучения гармонии** в академической теории. Бах написал сотни SATB-хоралов, которые до сих пор являются образцом для студентов всего мира.

## Диапазоны голосов

| Голос | Нижняя граница | Верхняя граница | Комфортная зона |
|-------|--------------|----------------|----------------|
| Сопрано | C4 | G5 | E4 – E5 |
| Альт | G3 | D5 | C4 – C5 |
| Тенор | C3 | G4 | E3 – E4 |
| Бас | E2 | C4 | G2 – G3 |

В реальной гармонии (и на гитаре) диапазоны условны, но **принципы расстановки** сохраняются.

## Расположение аккорда: тесное и широкое

### Тесное расположение (Close Position):

Три верхних голоса (SAT) находятся в пределах **одной октавы**.

```
C major тесное:
Сопрано: G (3-я октава сверху)
Альт:    E
Тенор:   C
Бас:     C (октавой ниже)
```

**Преимущества:** Плотный, «хоровой» звук; удобно для голосоведения.

### Широкое расположение (Open Position):

Расстояние между сопрано и тенором — **более октавы**.

```
C major широкое:
Сопрано: G (5-я октава)
Альт:    C (4-я октава)
Тенор:   E (3-я октава)
Бас:     C (2-я октава)
```

**Преимущества:** Более «воздушное», прозрачное звучание; характерно для оркестра.

## Правила SATB-изложения

### Расстояние между голосами:

| Пара голосов | Максимальный интервал |
|-------------|----------------------|
| Сопрано — Альт | Не больше октавы (желательно) |
| Альт — Тенор | Не больше октавы |
| Тенор — Бас | **До двух октав** (допустимо) |

Между тенором и басом больший разрыв — «лакуна» — допустима и характерна для широкого расположения.

### Удвоение нот:

В трезвучии (3 ноты) при четырёхголосии одну ноту нужно **удвоить**:

| Тип аккорда | Рекомендуемое удвоение |
|------------|----------------------|
| Тоника (I) | Удвоить примы или квинты |
| Субдоминанта (IV) | Удвоить примы или квинты |
| Доминанта (V) | Удвоить примы (не ведущий тон!) |
| Уменьшённое трезвучие | Удвоить терцию |

**Не удваивайте ведущий тон (VII ступень)!** — он создаёт диссонанс при удвоении и «тянет» к тонике.

## Ведущий тон и его разрешение

В тональности C major: **ведущий тон = B** (VII ступень).

В доминантовом аккорде G7 = G-B-D-F:
- **B** (ведущий тон) → обязан разрешиться в **C** (тоника) вверх
- **F** (септима) → обязан разрешиться в **E** (терция) вниз

```
G7 → Cmaj7:
Сопрано: F → E (вниз, полутон — разрешение септимы!)
Альт:    B → C (вверх, полутон — ведущий тон!)
Тенор:   G → G (остаётся — общий тон)
Бас:     G → C (вниз кварта / вверх квинта)
```

## Бах-хорал: пример голосоведения

Из хорала BWV 227 (упрощённо):

```
G major → C major → D7 → G major:
Сопрано: D → E → F# → D (мелодическое движение)
Альт:    B → G → A  → B (минимальное движение)
Тенор:   G → C → D  → G (опора)
Бас:     G → C → D  → G (функциональный бас)
```

Каждый голос — самостоятельная мелодия, все вместе — гармония.

## Практика в нотации

**Традиционная нотация SATB:**
- Сопрано и Альт — на скрипичном ключе (Soprano — штили вверх, Alto — вниз)
- Тенор и Бас — на басовом ключе (Tenor — вверх, Bass — вниз)

> **Главное упражнение:** Откройте любой Bach Chorale (есть на IMSLP бесплатно). Сыграйте каждый голос отдельно. Убедитесь, что все четыре звучат как мелодии. Затем играйте попарно: Soprano+Bass (крайние), Alto+Tenor (средние). Это «рентгеновский снимок» голосоведения.
