# Практическое применение голосоведения

## От теории к практике

Принципы голосоведения — это не академический груз, а **практические инструменты**, делающие гармонию живой. В этом уроке — конкретные техники, применимые в любом стиле.

## Common Tones (общие тоны)

**Общий тон** — нота, входящая в оба соседних аккорда. Это «связующее звено», обеспечивающее плавность.

### Как использовать:

1. Перед сменой аккорда определите общие тоны
2. Оставьте их в том же голосе (и по возможности на той же высоте)
3. Двигайте только «несовпадающие» голоса

### Пример: C major → E minor

```
C major: C — E — G
E minor: E — G — B
Общие: E и G!
→ E и G остаются на месте
→ C → B (один голос, движение на секунду вниз)
```

Вместо трёх движений — **одно минимальное**.

### Пример: G7 → Cmaj7

```
G7:     G — B — D — F
Cmaj7:  C — E — G — B
Общие:  B и G!
→ B остаётся (или переходит как ведущий тон в C)
→ G остаётся (прима G7 = квинта Cmaj7)
→ D → E (секунда)
→ F → E (полутон, разрешение септимы)
```

## Contrary Motion (противодвижение)

**Противодвижение** — наиболее «правильное» в полифоническом смысле: голоса расходятся или сходятся, но не движутся параллельно.

### Типы противодвижения:

| Тип | Описание |
|-----|---------|
| Convergent | Голоса сходятся (к унисону/октаве) |
| Divergent | Голоса расходятся |
| Oblique | Один стоит, другой движется |

**Пример в джазе (II-V-I):**
```
Dm7 → G7 → Cmaj7
Верхний голос: F → F → E  (вниз)
Нижний голос:  D → D → C  (вниз параллельно — допустимо для баса)
Средний:       A → B → C  (вверх — противодвижение к верхнему!)
```

Средний голос движется вверх, пока внешние движутся вниз — это противодвижение создаёт интерес.

## Минимальное движение (Minimal Voice Leading)

**Принцип минимального движения** — «парсимония» (parsimony) в теории голосоведения: каждый голос движется на **как можно меньший интервал**.

Это особенно важно в джазовых воицингах и прог-гармонии.

### Алгоритм:

1. Записать все ноты аккорда 1 и аккорда 2
2. Для каждой ноты аккорда 1 найти ближайшую ноту аккорда 2
3. «Распределить» ноты аккорда 2 по голосам с минимальным суммарным движением

### Пример: Cmaj7 → Fmaj7

```
Cmaj7: C — E — G — B
Fmaj7: F — A — C — E

Минимальное голосоведение:
C → C  (нет движения — общий тон!)
E → E  (нет движения — общий тон!)
G → A  (секунда вверх)
B → C  (полутон вверх — ведущий тон)
```

Только два голоса двигаются — и каждый минимально.

## Практические примеры по стилям

### Джаз: ii-V-I в Bb

```
Cm7 → F7 → Bbmaj7

Cm7:   Eb-G-Bb-D
F7:    Eb-A-C-Eb  (Eb остаётся, G→A, Bb→C, D→Eb)
Bbmaj7: D-A-C-F  (нет, лучше)

Оптимально:
Cm7:    Eb — G — Bb — D
F7:     C  — F — A  — Eb  (движение на 1-2 тона каждый голос)
Bbmaj7: D  — F — A  — C   (D возвращается, остальные на месте)
```

### Поп/R&B: I-V-VI-IV

```
C — G — Am — F

C:  E — G — C
G:  D — G — B  (E→D, G стоит, C→B)
Am: E — A — C  (D→E, G→A, B→C)
F:  F — A — C  (E→F, A стоит, C стоит)
```

Голоса движутся на секунду или меньше в каждой точке.

### Классика: Прерванная каденция

```
G7 → Am (вместо ожидаемой C major)

G7: G-B-D-F
Am: A-C-E (3 ноты — нужно добавить голос)

Решение:
G → A  (секунда вверх)
B → C  (полутон — ведущий тон!)
D → E  (секунда)
F → E  (полутон — разрешение септимы)
```

Прерванная каденция «обманывает» ожидание — C не приходит, приходит Am. Но голосоведение остаётся плавным.

## Голосоведение в аккомпанементе

В аккомпанементе (ритмическая гитара, фортепиано) голосоведение обеспечивает **«поющую» смену аккордов**:

| Плохо | Хорошо |
|------|--------|
| Все пальцы прыгают на новую позицию | Часть пальцев остаётся, часть двигается минимально |
| Бас прыгает на октаву | Бас движется по ступеням |
| Параллельное движение всех голосов | Хотя бы один голос — противодвижение |

## Чек-лист хорошего голосоведения

- [ ] Использованы общие тоны?
- [ ] Ведущий тон разрешён в тонику?
- [ ] Септима разрешена вниз?
- [ ] Нет параллельных квинт/октав?
- [ ] Каждый голос движется не более чем на терцию?
- [ ] Бас выполняет функциональное движение (не параллелен сопрано)?
- [ ] Хотя бы один голос использует противодвижение?

> **Финальный совет:** Голосоведение — это не правила ради правил. Это **язык плавности**: когда гармония течёт без усилий, слушатель получает наслаждение, а не «аккордовые удары». Послушайте медленные баллады — Bill Evans, João Gilberto, Дебюсси. Их гармония «поёт» именно потому, что каждый переход — это шаг, а не прыжок. Освойте этот принцип, и ваша музыка из набора аккордов превратится в живую речь.
