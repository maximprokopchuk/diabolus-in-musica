# Принципы голосоведения

## Что такое голосоведение

**Голосоведение** (voice leading) — принципы плавного и логичного движения отдельных голосов (мелодических линий) при смене аккордов. Это сердце классической полифонии, джазовой гармонии и практической аранжировки.

Голосоведение отвечает на вопрос: **как каждая нота аккорда «движется» к следующей ноте следующего аккорда?**

## Исторический контекст

Принципы голосоведения были сформулированы в **эпоху барокко** (XVII–XVIII вв.) и систематизированы в трудах теоретиков, опираясь на практику Баха, Палестрины, Монтеверди.

Иоганн Себастьян Бах — высшее воплощение полифонического голосоведения: каждый из его голосов (особенно в хоралах) движется как **независимая мелодия**, и все они гармонически согласованы.

## Основные принципы

### 1. Принцип экономии движения

**Каждый голос должен двигаться на минимально возможный интервал.**

Плохо: C → G (квинта вниз)
Хорошо: C → B (секунда вниз)

Применительно к аккордам:

```
C major → G major (плохое голосоведение):
Сопрано: E → D (секунда — нормально)
Альт:    C → B (секунда — нормально)
Тенор:   G → D (квинта — плохо!)
Бас:     C → G (квинта — нормально для баса)
```

### 2. Принцип независимости голосов

Каждый голос — это **мелодическая линия**, а не просто часть аккорда. Голос должен иметь собственное мелодическое «дыхание»: подъёмы и спуски, напряжение и разрядку.

Механически «переносить» аккорд вверх по параллели — плохое голосоведение.

### 3. Противодвижение (Contrary Motion)

Когда бас движется вверх, верхние голоса предпочтительно двигаются **вниз**, и наоборот.

```
Бас: C ↑ G
Сопрано: E ↓ D (или E ↓ B)
```

Противодвижение создаёт баланс и плавность.

### 4. Общие тоны (Common Tones)

**Тоны, общие для двух аккордов**, должны по возможности **оставаться в том же голосе** (не двигаться).

```
C major (C-E-G) → E minor (E-G-B)
Общие тоны: E и G
→ Оставляем E и G в тех же голосах, двигаем только C → B
```

Это создаёт связность — аккорды «вытекают» один из другого.

### 5. Разрешение неустойчивых интервалов

**Диссонансы** (септимы, нонаккорды) должны **разрешаться в консонанс** плавным движением:

- Септима вниз → секста (7-й тон → 6-й)
- Ведущий тон вверх → тоника (7-я ступень → 1-я)
- Нона вниз → октава

```
G7 → Cmaj7:
B (ведущий тон) → C (вверх, полутон)
F (малая септима G7) → E (вниз, секунда)
```

## Типы движения голосов

| Тип | Описание | Пример |
|-----|---------|--------|
| **Противодвижение** | Голоса движутся в противоположных направлениях | Бас ↑, сопрано ↓ |
| **Косое движение** | Один голос стоит, другой движется | Педальный тон |
| **Параллельное** | Голоса движутся в одном направлении | Терции, сексты (допустимо) |
| **Прямое** | Оба голоса движутся в одном направлении к одному интервалу | Осторожно — квинты/октавы запрещены |

## Практическое применение

**Тест на качество голосоведения:**

1. Сыграйте каждый голос по отдельности — он должен звучать как мелодия
2. Проверьте максимальный интервал каждого перехода — не больше кварты (исключение: октавный скачок в басу)
3. Найдите общие тоны — они должны оставаться на месте
4. Проверьте ведущий тон (VII) — он всегда разрешается в тонику вверх

> **Фундаментальный принцип:** Хорошее голосоведение делает гармонию **живой**. Плохое голосоведение делает даже правильные аккорды «мёртвыми» и механическими. Бах говорил (предположительно): «Хороший мастер должен уметь соединить любые два аккорда — вопрос лишь в том, насколько плавно это сделано».
