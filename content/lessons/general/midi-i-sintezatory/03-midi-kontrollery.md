# MIDI-контроллеры

## Что такое MIDI-контроллер

**MIDI-контроллер** — устройство, которое генерирует MIDI-сообщения, но **само не производит звук**. Звук создаёт подключённый синтезатор, звуковой модуль или DAW с плагином.

Это разделение позволяет использовать любой контроллер с любым источником звука.

## Клавишные контроллеры

### Клавиатуры без встроенного звука

**Основные параметры при выборе:**

| Параметр | Описание |
|---------|---------|
| **Количество клавиш** | 25, 37, 49, 61, 73, 88 |
| **Механика клавиш** | Полновзвешенная (фортепианная), полувзвешенная, синтезаторная |
| **Aftertouch** | Channel или Polyphonic (MPE) |
| **Pitch/Mod колёса** | Традиционные, или бесконечные энкодеры |
| **Дополнительные контроллеры** | Пэды, энкодеры, фейдеры, ленточный контроллер |

**Полновзвешенная механика** имитирует фортепиано: тяжёлые нижние клавиши, лёгкие верхние. Необходима для пианистов.

**Синтезаторная** — лёгкая, быстрая. Удобна для быстрых пассажей.

### Популярные клавишные контроллеры

| Модель | Клавиши | Особенности |
|-------|--------|------------|
| **Arturia KeyLab Essential 61** | 61 | Бюджетная, включает Analog Lab |
| **Novation 61SL MkIII** | 61 | Секвенсор, много контроллеров |
| **Native Instruments Komplete Kontrol S61** | 61 | Light Guide для подсказок нот |
| **Roland A-88 MKII** | 88 | Полновзвешенная, профессиональная |
| **Kawai MP11SE** | 88 | Фортепианная механика студийного уровня |

### MPE-контроллеры

**Roli Seaboard** — силиконовая поверхность вместо клавиш. Пять измерений экспрессии:
- Strike (удар — velocity)
- Press (давление — channel pressure)
- Glide (скольжение по горизонтали — pitch)
- Slide (движение по вертикали — timbre)
- Lift (отпускание — release velocity)

**Roger Linn LinnStrument** — сетка 200 чувствительных пэдов. Поддерживает MPE.

## Пэд-контроллеры

**Пэды** (velocity-sensitive pads) — нажимные площадки для программирования ударных, запуска сэмплов, арпеджиаторов.

### Ведущие пэд-контроллеры

| Устройство | Особенности |
|-----------|------------|
| **Akai MPC Live II** | Автономный: секвенсор, семплер, синтез — без компьютера |
| **Native Instruments Maschine MK3** | Интеграция с Komplete |
| **Ableton Push 3** | Предназначен для Ableton Live, автономный режим |
| **Novation Launchpad X** | Сетка 8×8, запуск клипов в Ableton |
| **Akai MPD232** | 16 пэдов, много энкодеров |

**Особенность Ableton Push:** каждая ячейка сетки — нота; полная интеграция с Live позволяет работать без монитора.

## Wind Controllers

**Wind controller** — MIDI-контроллер в виде духового инструмента. Музыкант управляет звуком дыханием, пальцами и давлением губ.

| Устройство | Особенности |
|-----------|------------|
| **Akai EWI 5000** | Беспроводной, встроенные звуки, USB MIDI |
| **Roland Aerophone AE-10** | Встроенные звуки саксофона, USB |
| **Yamaha WX5** | Классический вид, требует внешний модуль |
| **Lyrical Instrument Wersi Omega** | Профессиональный, кастомизируемый |

Wind controller передаёт Breath Control (CC 2) — давление воздуха управляет громкостью, фильтром, вибрато.

## Guitar MIDI

**Guitar MIDI** позволяет гитаристам управлять MIDI-синтезаторами игрой на гитаре.

### Roland GK-система

**Специальный датчик GK-3** крепится на гитару и считывает колебания каждой струны отдельно. Подключается к **MIDI-процессору (Roland GR-55, GP-10)**, который конвертирует сигнал в MIDI и генерирует синтезаторный звук.

Задержка конвертации: 3–10 мс — заметна, требует адаптации в технике игры.

### Цифровые гитары с MIDI

**Fishman TriplePlay** — системы, встроенные в гитару. **Стандартная гитара + Hex pickup** — более новые решения с лучшим tracking'ом.

**Fender American Ultra Telecaster** — с встроенным MIDI (через специальный датчик) в некоторых конфигурациях.

### Ограничения Guitar MIDI

- Задержка (latency) — unavoidable из-за анализа питча
- Бэнды и вибрато — сложно конвертировать точно
- Лучше всего работает на чётких нотах без сложных техник слайда

> **Практический совет:** для большинства музыкантов лучший MIDI-контроллер — тот, который соответствует уже известному инструменту. Гитарист начнёт с guitar MIDI или пэдов, а не с клавишной механики. Пианисты — с полновзвешенной клавиатуры. Барабанщики — с пэд-контроллеров.
