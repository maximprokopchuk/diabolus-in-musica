# Секвенсоры и аранжировщики

## Что такое секвенсор

**Секвенсор** (sequencer) — устройство или программа для записи, хранения и воспроизведения последовательностей MIDI-событий или аудио. Секвенсор «помнит» ноты, которые нужно сыграть, и воспроизводит их точно в нужное время.

В широком смысле любая **DAW** (Digital Audio Workstation) — это секвенсор.

## Piano Roll

**Piano Roll** — главный способ редактирования MIDI в современных DAW. Название происходит от перфорированных бумажных рулонов пианол XIX века.

### Интерфейс Piano Roll

```
Вертикальная ось = высота ноты (C1–C8)
Горизонтальная ось = время
Прямоугольник = нота (длина = длительность, положение = ритм, высота полосы = velocity)
```

### Возможности Piano Roll

- **Рисование нот** — мышью в нужной позиции
- **Квантизация** — привязка нот к сетке (<Q> в большинстве DAW)
- **Velocity редактирование** — высота прямоугольника или отдельная панель
- **CC automation** — рисование огибающих для любого MIDI-контроллера
- **Chord tools** — автоматическое добавление аккордов

**Piano roll в разных DAW:**
- Ableton Live: Notes view
- FL Studio: Piano Roll (считается лучшим)
- Logic Pro: Piano Roll editor
- Cubase: Key Editor

## Step Sequencer

**Step Sequencer** — программирование ритмических паттернов по «шагам» (steps). Каждый шаг — один момент времени (обычно 1/16 или 1/8 ноты).

### Классический шаг-секвенсор: Roland TR-808

**Roland TR-808** (1980) — ритм-машина с 16-шаговым программированием. Каждый ряд — один инструмент (Kick, Snare, Hi-Hat и т.д.). Нажать кнопку = включить шаг.

```
Kick:    [X][ ][ ][ ] [X][ ][ ][ ] [X][ ][ ][ ] [X][ ][ ][ ]
Snare:   [ ][ ][ ][ ] [X][ ][ ][ ] [ ][ ][ ][ ] [X][ ][ ][ ]
Hi-Hat:  [X][X][X][X] [X][X][X][X] [X][X][X][X] [X][X][X][X]
```

### Современные step sequencers

**Elektron (Octatrack, Rytm, Digitakt)** — развитые step sequencers с:
- **P-locks (Parameter Locks)** — каждый шаг может иметь уникальное значение любого параметра
- **Условные тригеры** — шаг воспроизводится только при определённых условиях (каждый 2-й такт, случайно и т.д.)
- **Микротайминг** — смещение шагов с сетки для человеческого feel

**Elektron workflow** называют «parameter locking» — это мощный инструмент для создания живых, нестатичных паттернов.

## Pattern-Based Секвенсоры

**Pattern-based** подход: музыка составляется из повторяемых паттернов (loops), которые запускаются в определённом порядке.

### Ableton Live: Session View

**Session View** — уникальная концепция Ableton:
- Клипы (clips) организованы в сетку: строки = треки, столбцы = сцены
- Каждый клип — короткий паттерн (1–8 тактов)
- **Запуск сцены** воспроизводит все клипы в ряду одновременно
- **Идеально для живых выступлений** — импровизированное переключение

### FL Studio: Pattern Sequencer + Playlist

- Pattern = группа нотных данных одного или нескольких инструментов
- Паттерны размещаются на Playlist как «кирпичики» аранжировки
- **Step Sequencer** внутри FL — для быстрого программирования ударных

### Аппаратные паттерн-секвенсоры

| Устройство | Особенности |
|-----------|------------|
| **Elektron Model:Cycles** | FM-синтез + step sequencer |
| **Teenage Engineering OP-1** | Компактный, уникальные алгоритмы |
| **Korg SQ-64** | Полифонический, аналоговый sync |
| **Arturia BeatStep Pro** | Два мелодических + один ударный секвенсор |

## MIDI Clock и синхронизация

Когда несколько устройств работают вместе, их нужно синхронизировать.

**MIDI Clock** — 24 импульса на четверть ноты. Передаётся по MIDI и синхронизирует BPM всех подключённых устройств.

**MIDI Start/Stop/Continue** — управляющие сообщения для запуска и остановки секвенсоров.

**DIN sync (sync24)** — старый стандарт синхронизации (Roland, используется для TB-303, TR-808).

**Ableton Link** — современный протокол синхронизации по Wi-Fi для нескольких устройств/компьютеров.

> **Практически:** для домашней студии и живых выступлений ключевой выбор — piano roll или step sequencer. Piano roll — максимальный контроль для сложных мелодий и аранжировок. Step sequencer — быстрое программирование ритмов и повторяющихся паттернов. Лучшие системы сочетают оба подхода.
