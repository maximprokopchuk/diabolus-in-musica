# Синтезаторы в живом выступлении

## Особенности живого выступления с синтезаторами

Студийная работа с синтезаторами отличается от живого выступления. В студии можно остановиться, перезаписать, настроить любой параметр. На сцене — всё должно работать мгновенно, надёжно и без сюрпризов.

**Главные требования к live-setup:**
- Быстрая загрузка пресетов
- Минимальный риск отказа
- Возможность быстро сменить звук между песнями
- Удобное управление без необходимости смотреть на дисплей

## Звуковые модули (Sound Modules)

**Звуковой модуль** — синтезатор без клавиатуры. Только генератор звука + выходы + управление. Управляется с внешнего MIDI-контроллера.

### Зачем использовать модули

| Причина | Объяснение |
|--------|-----------|
| Компактность | Меньше места на сцене |
| Специализация | Лучший рояль в одном модуле, лучший орган в другом |
| Резервирование | Несколько модулей = запасной вариант при отказе |
| Стоимость | Модуль дешевле полноценного синтезатора |

### Популярные звуковые модули

| Модуль | Специализация |
|-------|-------------|
| **Yamaha MX88** | Рояль, электропиано, оркестр |
| **Yamaha MODX+** | Широкий спектр тембров, FM + AWM2 |
| **Nord Rack 3** | Virtual analog, органоподобный звук |
| **Kurzweil PC4** | Профессиональные акустические инструменты |
| **Roland Integra-7** | Библиотека 6000+ тембров |

## Layering (слоевание звуков)

**Layering** — одновременное звучание нескольких тембров при нажатии одной клавиши. Создаёт богатые, «большие» звуки.

### Типы слоевания

**Горизонтальный layer** — оба тембра звучат одновременно на всей клавиатуре:
```
Layer A: Strings Pad
Layer B: Piano
→ Результат: фортепиано с атмосферными струнными
```

**Сплит (Split)** — разные тембры на разных зонах клавиатуры:
```
Клавиши C1–B2: Bass Synth
Клавиши C3–C7: Piano
→ Левая рука = бас, правая = мелодия
```

**Velocity Split** — тембр меняется в зависимости от силы удара:
```
Velocity 0–64:  Soft Piano (мягкий удар)
Velocity 65–127: Bright Piano (сильный удар)
→ Динамически реалистичный рояль
```

**Многие синтезаторы поддерживают до 4–8 слоёв одновременно.**

## MIDI Routing в live-setup

**MIDI routing** — маршрутизация MIDI-сигналов: какой контроллер управляет каким устройством по какому каналу.

### Простая схема (2 устройства)

```
MIDI-клавиатура (MIDI OUT) → Синтезатор A (MIDI IN, канал 1)
                           → Синтезатор B (MIDI IN, канал 2)
```

Клавиатура посылает по обоим каналам одновременно (dual mode) или переключает между ними.

### Профессиональная схема с MIDI-маршрутизатором

```
Клавиатура 1 → [MIDI Router / Merger] → Синтезатор A (CH 1)
Клавиатура 2 →                        → Синтезатор B (CH 2)
Педальный   →                         → Синтезатор C (CH 10 - drums)
контроллер
```

**MIDI-маршрутизаторы:** MOTU MIDI Express XT, iConnectivity mio10, RME Digiface MIDI.

## Смена пресетов на сцене

### Program Change Message

Когда исполнитель переходит к следующей песне, нужно сменить звук. Это делается через **Program Change** — можно назначить на педаль, кнопку или управлять с iPad.

**Looper + синтезатор:** перед переключением пресета зациклить последний аккорд → плавный переход.

### Patch Change с DAW или MIDI-процессором

**Kemper / Fractal + синтезатор:** гитаристы, использующие MIDI-совместимые процессоры, могут одновременно менять пресеты на гитарном процессоре и синтезаторе одной командой.

**iPad/iPhone:** приложения **TouchOSC**, **Lemur**, **MIDI Designer** позволяют создать пользовательский интерфейс для управления всем setup.

## Надёжность в живом выступлении

### Правило резервирования

- Всегда иметь резервный инструмент или модуль
- Проверить все соединения перед концертом
- Иметь запасные кабели MIDI, USB, XLR
- Батарея у беспроводных устройств — всегда заряжена

### Laptop на сцене: риски

Использование ноутбука с плагинами — гибкость, но риск:
- Сбои ОС
- Задержка (latency) при неоптимальных настройках ASIO
- Перегрев

**Рекомендации для ноутбука на сцене:**
- Отдельный «концертный» профиль ОС без фоновых процессов
- ASIO4ALL или профессиональный аудиоинтерфейс
- Buffer size 128 или 64 семплов для минимальной задержки
- Автосохранение проекта отключить (во избежание заморозки в момент записи)
- UPS или качественный Power Conditioner для стабильного питания

> **Итог:** надёжный и продуманный live-setup с синтезаторами строится на принципе минимализма и резервирования. Чем проще схема — тем меньше что-то пойдёт не так. Заранее репетировать смену пресетов и переключение между звуками — такая же часть подготовки, как репетиция музыкального материала.
