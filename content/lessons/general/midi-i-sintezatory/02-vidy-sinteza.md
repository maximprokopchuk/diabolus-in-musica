# Виды синтеза

## Что такое синтез звука

**Синтез** — создание звука электронными или вычислительными методами. В отличие от сэмплирования (воспроизведения записей), синтез генерирует звук «с нуля» на основе математических алгоритмов.

## Субтрактивный синтез (Subtractive Synthesis)

**Субтрактивный синтез** — самый распространённый и классический метод.

### Принцип

```
[Осциллятор] → [Фильтр] → [Усилитель (VCA)] → [Выход]
      ↑              ↑              ↑
   [LFO]         [EG фильтра]   [EG амплитуды]
```

**Осциллятор (VCO/DCO)** генерирует «богатый» сигнал с большим количеством гармоник (пила, прямоугольник, треугольник).

**Фильтр (VCF)** «вычитает» (срезает) ненужные гармоники, формируя тембр. Самый важный элемент субтрактивного синтезатора.

**Усилитель (VCA)** управляет громкостью.

**Envelope Generator (EG / ADSR)** — огибающая с параметрами Attack, Decay, Sustain, Release.

**LFO** — низкочастотный осциллятор для вибрато, тремоло, автопанорамирования.

### Классические субтрактивные синтезаторы

| Синтезатор | Особенности | Звук |
|-----------|------------|------|
| **Moog Minimoog** (1970) | Легендарный лестничный фильтр | Классический аналоговый бас, лид |
| **Roland Juno-106** (1984) | DCO (аналоговые осцилляторы с цифровым управлением), простой хорус | Pad'ы, поп-синтезаторы |
| **Sequential Circuits Prophet-5** (1978) | Полифония 5 голосов | Классика 80-х |
| **Dave Smith Instruments OB-6** | Современный аналог | Профессиональный |

## FM-синтез (Frequency Modulation)

**FM-синтез** использует один осциллятор (оператор) для модуляции частоты другого. Это создаёт сложные спектры, богатые металлическими и колокольными призвуками.

### Принцип FM

```
[Carrier (несущий)] ← частота модулируется ← [Modulator (модулятор)]
```

Результирующий спектр зависит от:
- Соотношения частот carrier и modulator (C:M ratio)
- Индекса модуляции (глубина модуляции)

**Чем выше индекс модуляции** — тем богаче и «металличнее» звук.

### Классика FM

**Yamaha DX7** (1983) — революционный синтезатор, определивший звук 80-х. Шесть операторов, 32 алгоритма их соединения. Характерные звуки: электропиано, блестящие пэды, колокола, «glockenspiel».

**Современные FM-синтезаторы:** Yamaha Reface DX, Native Instruments FM8, Elektron Digitone.

## Wavetable-синтез

**Wavetable-синтез** основан на наборах (таблицах) предзаписанных форм волны, между которыми можно плавно переключаться.

### Принцип

Осциллятор «читает» таблицу форм волны — при движении по таблице звук меняется. Можно задать анимацию (wavetable scan) для создания движущихся, эволюционирующих тембров.

**Преимущества:**
- Богатый, детализированный звук при небольших вычислительных затратах
- Огромные возможности для создания уникальных тембров

**Классические wavetable-синтезаторы:** PPG Wave (1981), Waldorf Wave, Waldorf Blofeld. Современные: Xfer Records Serum, Native Instruments Massive X, Waldorf Iridium.

**Serum** — стандарт de facto для EDM, future bass, dubstep. Позволяет создавать собственные wavetable, импортировать аудиофайлы и рисовать формы волны вручную.

## Гранулярный синтез (Granular Synthesis)

**Гранулярный синтез** — разбивает аудиосигнал на тысячи крошечных «зёрен» (грейнов) длиной 1–100 мс и воспроизводит их в изменённом порядке, с изменёнными параметрами.

### Ключевые параметры

- **Grain size** — размер зерна (1–200 мс)
- **Grain density** — количество зёрен в секунду
- **Scan position** — позиция считывания в исходном файле
- **Randomization** — случайное изменение параметров зёрен

### Что создаёт гранулярный синтез

- Замороженные, атмосферные текстуры
- Питч-шифтинг без изменения длительности (и наоборот)
- «Облака» звуков
- Трансформацию любых звуков в пэды и эмбиент

**Инструменты:** Native Instruments Absynth, Arturia Pigments, Max/MSP, AudioMulch; специализированные: Mutable Instruments Clouds (eurorack).

## Аддитивный синтез (Additive Synthesis)

**Аддитивный синтез** — синтез звука из множества независимых синусоидальных осцилляторов (гармоник). Основан на теореме Фурье: любой периодический сигнал можно представить суммой синусоид.

### Принцип

Каждая гармоника управляется независимо: своя частота, амплитуда, огибающая. 64 или 128 осцилляторов с разными огибающими дают бесконечно точный контроль тембра.

**Преимущества:** максимальный контроль над тембром, физически точное моделирование.

**Недостатки:** вычислительно сложно, требует большого числа параметров для настройки.

**Примеры:** Kawai K5000, Cameleon 5000, Roland V-Synth (частично).

## Физическое моделирование (Physical Modelling)

**Physical modelling** симулирует физические свойства реальных инструментов: струны, трость, мембрана.

**Примеры:** Arturia Piano V, Modartt Pianoteq (фортепиано), Applied Acoustics Chromaphone (струнные/перкуссия), Roland VG-99 (гитарный синтезатор).

> **Практический совет:** начинать знакомство с синтезом лучше с **субтрактивного** — он наиболее интуитивен и объясняет базовые принципы. После освоения ADSR, фильтра и LFO переходить к FM и wavetable. Гранулярный — для создания уникальных текстур и эмбиента.
