# Стереофонические техники

## Зачем нужна стерео-запись

Моно-запись не передаёт **пространственную информацию** — ширину, глубину, расположение инструментов в пространстве. Стерео-техники позволяют создать **реалистичную или художественную пространственность**.

Основные стерео-техники делятся на:
- **Интенсивностные** (coincident pair) — два микрофона в одной точке, разница в направлении создаёт стерео
- **Временны́е** (spaced pair, AB) — два микрофона на расстоянии, разница во времени создаёт стерео
- **Комбинированные** (ORTF, NOS)

## XY (Coincident Pair)

Два одинаковых микрофона **с кардиоидой** расположены в одной точке под углом **90° друг к другу** (капсюли максимально близко).

```
      ↗ L (0°–45°)
Источник
      ↘ R (0°–45°)
```

| Параметр | Значение |
|---------|---------|
| Угол между микрофонами | 90° |
| Тип микрофонов | Кардиоиды |
| Стереобаза | Умеренная |
| Совместимость с моно | Отличная (нет задержки) |
| Простота настройки | Высокая |

**Применение:** концертная запись, overheads на барабанах, акустические инструменты, интервью.

**Плюс:** полная моно-совместимость — при сведении в моно нет comb-фильтра.

## Spaced Pair (AB)

Два одинаковых микрофона на расстоянии **0.5–1.5 м** (иногда больше) друг от друга, смотрят вперёд (параллельно).

| Параметр | Значение |
|---------|---------|
| Расстояние между микрофонами | 50–150 см |
| Тип микрофонов | Омни или кардиоиды |
| Стереобаза | Широкая |
| Совместимость с моно | Осторожно (гребенчатый фильтр) |
| Захват комнаты | Больше, чем XY |

**Применение:** оркестр, хор, рояль, джазовые ансамбли — когда нужна широкая «живая» сцена.

**Минус:** при суммировании в моно возможны фазовые проблемы (comb filtering).

## ORTF

Разработан французским радио (Office de Radiodiffusion-Télévision Française). Золотая середина между XY и AB.

| Параметр | Значение |
|---------|---------|
| Расстояние между капсюлями | **17 см** (расстояние между ушами человека) |
| Угол между осями | **110°** |
| Тип микрофонов | Кардиоиды |
| Стереобаза | Широкая, естественная |
| Совместимость с моно | Хорошая |

ORTF имитирует **расположение человеческих ушей** — поэтому стерео-образ воспринимается наиболее естественно.

**Применение:** концертная запись, акустические инструменты, overheads, всё, где нужна реалистичная стереопанорама.

## Mid-Side (MS)

Элегантная техника, позволяющая **изменять ширину стерео после записи**.

### Компоненты

- **M (Mid)** — направленный микрофон (кардиоида), смотрит прямо на источник
- **S (Side)** — микрофон с характеристикой **восьмёрка**, ось направлена вбок (90° к Mid)

### Декодирование MS

```
L = M + S
R = M − S
```

В DAW это делается через плагин MS-матрицы или вручную:
1. Mid-микрофон → прямо на шину
2. Side-микрофон → две копии: одна в фазе, другая инвертированная (+180°)
3. Первая копия → панорама влево, инвертированная → вправо

### Преимущества

- **Регулировка ширины** стерео без переписи: увеличить gain Side — шире, уменьшить — уже
- **Идеальная моно-совместимость** (S-канал исчезает при суммировании в моно)
- Можно записать с одним микрофоном (Mid) и добавить Side позже

**Применение:** вещание, документальная запись, акустические инструменты, ситуации неопределённого конечного использования (моно/стерео).

## Blumlein Pair

Изобретён Аланом Блюмлейном (1931). Два микрофона с **восьмёркой** под углом **90°** в одной точке.

| Параметр | Значение |
|---------|---------|
| Угол между микрофонами | 90° |
| Тип микрофонов | Восьмёрки (figure-8) |
| Стереобаза | Очень широкая |
| Захват комнаты | Максимальный (восьмёрка захватывает и перед, и зад) |
| Совместимость с моно | Хорошая |

**Особенность:** Blumlein захватывает и звук перед, и звук сзади — это даёт **исключительно естественное трёхмерное ощущение пространства**.

**Применение:** оркестровая запись, запись в красивых концертных залах, джаз, классика.

## Сравнительная таблица

| Техника | Стереобаза | Моно-совместимость | Сложность | Применение |
|--------|-----------|-------------------|----------|-----------|
| XY | Умеренная | Отличная | Простая | Универсальная |
| AB | Широкая | Осторожно | Простая | Оркестр, хор |
| ORTF | Широкая | Хорошая | Средняя | Концертная запись |
| Mid-Side | Регулируемая | Идеальная | Сложная | Вещание, студия |
| Blumlein | Очень широкая | Хорошая | Средняя | Классика, джаз |

> **Рекомендация:** начинайте с **XY** — проста в настройке и даёт надёжный результат. Переходите к **ORTF** для более широкой и живой сцены. **Mid-Side** изучите как обязательный инструмент для профессиональной работы.
