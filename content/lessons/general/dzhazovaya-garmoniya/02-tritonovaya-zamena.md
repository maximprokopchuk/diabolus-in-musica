# Тритоновая замена

## Принцип

**Тритоновая замена** (tritone substitution) — замена доминантсептаккорда другим доминантсептаккордом, корень которого находится на **тритон** (6 полутонов) от исходного.

```
G7 → Db7  (G и Db — тритон друг от друга)
```

## Почему это работает

G7 содержит тритон **B — F**.
Db7 содержит тритон **F — Cb (B)**.

**Это один и тот же тритон!** Оба аккорда содержат одинаковую пару напряжённых нот, которая разрешается одинаково:
- B → C
- F → E

Поэтому Db7 может заменить G7 перед Cmaj7.

## Применение в ii-V-I

### Замена V

```
Исходное:  Dm7 — G7  — Cmaj7
Замена V:  Dm7 — Db7 — Cmaj7
```

Бас движется **хроматически**: D → Db → C — очень плавно!

### Замена и ii, и V

```
Dm7 — Db7 — Cmaj7    (замена только V)
Abm7 — Db7 — Cmaj7   (замена и ii — Ab = тритон от D)
```

## Хроматические линии

Тритоновые замены создают **хроматическое движение баса**:

```
Без замен:   Em7 — A7 — Dm7 — G7 — Cmaj7
С заменами:  Em7 — Eb7 — Dm7 — Db7 — Cmaj7
Бас:         E — Eb — D — Db — C (хроматически вниз!)
```

## На гитаре

Тритоновая замена = **тот же аккорд, сдвинутый на 6 ладов**:

```
G7 (3-й лад, E-форма):     Db7 (9-й лад, E-форма):
E|--3--|                     E|--9--|
B|--3--|                     B|--9--|
G|--4--|                     G|--10-|
D|--3--|                     D|--9--|
A|--x--|                     A|--x--|
E|--3--|                     E|--9--|
```

> **Подсказка:** каждый раз, когда видите доминантсептаккорд (V7), вы можете заменить его аккордом на 6 ладов выше/ниже. Попробуйте — и звучание станет «джазовее» мгновенно.
