# Расширенные и альтерированные аккорды в джазе

## Принцип обогащения

В джазе трезвучия практически не используются. Минимум — **септаккорд**. Далее добавляются расширения:

```
Трезвучие → Септ → Нона → Ундецима → Терцдецима
C        → Cmaj7 → Cmaj9 → Cmaj9#11 → Cmaj13
```

## Расширения по функциям

### Тоника (Imaj7)

| Аккорд | Воицинг | Характер |
|--------|---------|----------|
| Cmaj7 | Базовый | Чистый |
| Cmaj9 | + нона (D) | Тёплый |
| C6/9 | + секста + нона | Классический джазовый |
| Cmaj7#11 | + лидийская нота | Современный |

### Субдоминанта (ii)

| Аккорд | Характер |
|--------|----------|
| Dm7 | Базовый |
| Dm9 | Тёплый |
| Dm11 | Открытый |

### Доминанта (V7) — наибольшее разнообразие

| Аккорд | Характер |
|--------|----------|
| G7 | Базовый |
| G9 | Мягкий |
| G13 | Богатый |
| G7♭9 | Минорное разрешение |
| G7#9 | Hendrix, блюзовый |
| G7alt | Максимальное напряжение |
| G7#11 | Лидийский доминантный |

## Altered chord (подробнее)

**G7alt** — все изменяемые ступени альтерированы:

```
G7alt = G — B — (Ab или A#) — (Db или D#) — F
        1 — 3 — ♭9/♯9     — ♭5/♯5      — ♭7
```

Гамма для импровизации: **G altered** = Ab мелодический минор от 7-й ступени.

## Воицинги на гитаре

В джазе на гитаре используют **компактные воицинги** из 3–4 нот:

```
Cmaj9 (без корня):   G13 (без корня):   G7alt:
e|--x--|             e|--x--|           e|--x--|
B|--3--|             B|--3--|           B|--4--|
G|--4--|             G|--3--|           G|--3--|
D|--4--|             D|--3--|           D|--3--|
A|--3--|             A|--x--|           A|--x--|
E|--x--|             E|--3--|           E|--3--|
```

> **Корень можно пропустить** — его играет басист. Это освобождает гитариста для более интересных нот: терция, септима, расширения.
