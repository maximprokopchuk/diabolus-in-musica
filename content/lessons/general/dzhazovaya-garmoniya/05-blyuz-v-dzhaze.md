# Блюз в джазе

## От простого к сложному

Джаз трансформировал простой 12-тактовый блюз, добавив сложные гармонические замены.

## Базовый блюз → Джазовый блюз

### Базовый (рок/блюз)

```
| I7  | I7  | I7  | I7  |
| IV7 | IV7 | I7  | I7  |
| V7  | IV7 | I7  | V7  |
```

### Джазовый блюз (стандартный)

```
| I7     | IV7    | I7     | I7     |
| IV7    | #IVdim7| I7     | vi7    |
| ii7    | V7     | I7     | ii-V   |
```

В Bb:
```
| Bb7    | Eb7    | Bb7    | Bb7    |
| Eb7    | Edim7  | Bb7    | Gm7    |
| Cm7    | F7     | Bb7    | Cm7 F7 |
```

## Bird Blues (Charlie Parker)

Паркер максимально «нагрузил» блюз ii-V оборотами:

```
| Fmaj7  | Em7♭5 A7 | Dm7 G7  | Cm7  F7  |
| Bb7    | Bbm7 Eb7 | Am7 D7  | Abm7 Db7 |
| Gm7    | C7       | Fmaj7   | Gm7  C7  |
```

Каждый такт содержит ii-V или другое сложное движение. Однако **общий контур** остаётся блюзовым (I → IV → I → V → I).

## Минорный блюз

```
| Cm7    | Cm7    | Cm7    | Cm7    |
| Fm7    | Fm7    | Cm7    | Cm7    |
| Dm7♭5  | G7♭9   | Cm7    | Dm7♭5 G7|
```

С заменами:
```
| Cm7    | Cm7    | Cm7    | Cm7    |
| Fm7    | Fm7    | Cm7    | Am7♭5 D7|
| Dm7♭5  | G7alt  | Cm7 Abmaj7| Dm7♭5 G7|
```

## Гаммы для импровизации над джазовым блюзом

| Такты | Аккорд | Гамма |
|-------|--------|-------|
| 1–4 | I7 | Блюзовая, миксолидийский |
| 5–6 | IV7 | Миксолидийский от IV |
| 7–8 | I7, vi | Блюзовая, дорийский от vi |
| 9 | ii7 | Дорийский от ii |
| 10 | V7 | Миксолидийский, altered |
| 11–12 | I7, turnaround | Блюзовая |

> **Совет:** джазовый блюз — лучший материал для перехода от рок-гитары к джазу. Форма знакомая (12 тактов), но гармония постепенно усложняется. Начните с простого джазового блюза и постепенно добавляйте замены.
