# Personal monitor mixing

## Что такое personal monitor mixing

**Personal monitor mixing** — система, при которой каждый музыкант самостоятельно управляет своим мониторным миксом с помощью индивидуального устройства, не отвлекая мониторного инженера.

Традиционная схема: «Сделай мне громче бас» → мониторный инженер поднимает AUX 3 → часто угадывает не то.

Personal mixing: музыкант сам крутит нужную ручку на своём микшере.

## Системы personal monitor mixing

### Aviom A360

**Aviom** — пионер в области персонального мониторинга.

| Параметр | Значение |
|---------|---------|
| Количество каналов | 40 каналов на входе |
| Управление | Физические энкодеры + дисплей |
| Подключение | Cat-5e / Pro16e шина |
| Выход | Стерео (IEM или наушники) + балансный линейный |
| Питание | PoE (питание по кабелю) |

Каждый музыкант получает **персональный миксер Aviom A360**, который подключается к центральному **D800 Personal Mixer Hub**. Хаб принимает сигналы с консоли и распределяет их по всем персональным устройствам.

### Hear Technologies Hear Back

**Hear Back** — альтернатива Aviom с некоторыми преимуществами:
- До 8 каналов на базовый блок (расширяется до 32)
- Компактные персональные миксеры
- Интуитивное управление
- Более бюджетный вариант для клубных применений

### Roland M-48

**Roland M-48** — 40-канальный персональный микшер с сенсорным управлением.

| Плюсы | Минусы |
|-------|--------|
| Встроенные эффекты (реверб, хорус) | Дороже |
| Удобный интерфейс | Требует специального рэкового блока M-480 |
| Стерео IEM-выход + наушниковый выход | |

### Цифровые консоли с personal mixing

Современные цифровые консоли (Behringer X32/M32, Yamaha CL/QL, DiGiCo SD) часто имеют приложения для **iOS/Android**, которые позволяют музыканту управлять своим мониторным миксом со смартфона через Wi-Fi.

| Консоль | Приложение | Функции |
|--------|----------|--------|
| Behringer X32 | **Mixing Station** | Полный доступ к консоли (ограничивается по каналам) |
| Yamaha CL/QL | **MonitorMix** | Управление персональным миксом |
| Allen & Heath SQ | **SQ MixPad** | Полное управление персональными миксами |
| Presonus StudioLive | **UC Surface** | Персональные миксы + настройки |

**Wi-Fi personal mixing** — самое гибкое и дешёвое решение при наличии поддерживаемой консоли. Не требует дополнительного оборудования — только роутер у сцены.

## Workflow при personal mixing

### Базовый сценарий концерта

1. **Подготовка:** мониторный инженер создаёт «стартовый» микс для каждого музыканта перед саундчеком
2. **Саундчек:** музыканты корректируют свой микс с персональных устройств
3. **Выступление:** каждый управляет своим миксом самостоятельно в реальном времени

### Преимущества для мониторного инженера

- Освобождается от постоянных запросов «добавь/убери»
- Может сосредоточиться на глобальных проблемах (фидбэк, клиппинг)
- Музыканты довольны, потому что слышат то, что нужно именно им

### Обучение музыкантов

Personal mixing требует, чтобы музыкант понимал, что он делает. Первые шаги:
1. Определить, что нужно слышать в первую очередь (метроном? вокал? бас?)
2. Начать с минимального количества каналов — добавлять по необходимости
3. Не поднимать всё сразу — искать баланс
4. Использовать стерео — панорамировать инструменты

> **Золотое правило персонального миша:** лучше слышать меньше и разборчиво, чем больше и непонятно. Многие музыканты перегружают свой микс всеми каналами на максимум — в результате не слышат ничего.
