# Мелодия на гитаре

## Позиционная игра

**Позиция** — положение левой руки на грифе, при котором каждый палец отвечает за свой лад:

| Палец | Позиция 5 | Позиция 7 |
|-------|-----------|-----------|
| Указательный (1) | 5-й лад | 7-й лад |
| Средний (2) | 6-й лад | 8-й лад |
| Безымянный (3) | 7-й лад | 9-й лад |
| Мизинец (4) | 8-й лад | 10-й лад |

В одной позиции можно сыграть все ноты гаммы **без сдвига руки**.

## Связывание позиций (position shifting)

Для покрытия всего диапазона грифа необходимо переходить между позициями. Способы:

### Слайд

Один палец скользит по струне к новому ладу:
```
Позиция 5: ...7-й лад (3-й палец)
Слайд 3-го пальца → 9-й лад
Позиция 7: 9-й лад (3-й палец)...
```

### Растяжка

Мизинец или указательный тянется за пределы позиции:
```
Позиция 5: 5-6-7-8 + растяжка мизинца на 9
```

### Смена через открытую струну

Открытая струна звучит, пока рука перемещается.

## Легато (legato)

**Легато** — плавное соединение нот без повторного удара медиатором:

### Hammer-on (восходящее легато)

Палец «вбивается» в лад без участия правой руки:
```
5h7 = удар медиатором на 5-м ладу, hammer-on на 7-й
```

### Pull-off (нисходящее легато)

Палец «сдёргивается» со струны, оставляя звучать нижний лад:
```
7p5 = палец на 7-м сдёргивается, звучит 5-й лад
```

### Комбинации

```
5h7p5h8p5 — типичная легато-фраза
```

## Артикуляция

| Приём | Обозначение | Эффект |
|-------|-------------|--------|
| Стаккато | Точка над нотой | Короткий, отрывистый звук |
| Легато | Лига | Плавный, связный звук |
| Акцент | > над нотой | Более громкий удар |
| Palm mute | P.M. | Приглушённый звук |

> **Совет:** чередование легато и стаккато создаёт динамику во фразе. Не играйте всё одинаково — варьируйте артикуляцию, как певец варьирует подачу голоса.
