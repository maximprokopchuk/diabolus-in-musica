# Мелодия и гармония

## Связь мелодии с аккордами

Каждая нота мелодии находится в определённом отношении к текущему аккорду. Это определяет степень её «устойчивости».

## Типы мелодических нот

### Аккордовые тоны (chord tones)

Ноты, входящие в текущий аккорд — **самые устойчивые**. На них мелодия «отдыхает».

Для аккорда Cmaj7: C, E, G, B — все являются аккордовыми тонами.

### Проходящие ноты (passing tones)

Ноты **между** аккордовыми тонами, заполняющие поступенное движение:

```
Аккорд C: C — [D] — E   (D = проходящая нота)
```

Проходящие ноты звучат на **слабых долях** и быстро разрешаются.

### Вспомогательные ноты (neighbor tones)

Нота **рядом** с аккордовым тоном, которая возвращается обратно:

```
Аккорд C: E — [F] — E   (F = верхняя вспомогательная)
          E — [D] — E   (D = нижняя вспомогательная)
```

### Задержание (suspension)

Нота из **предыдущего аккорда** задерживается и затем разрешается:

```
F → C:  ... A — [A] → G   (A задерживается из F, разрешается в G)
```

### Предъём (anticipation)

Нота **следующего аккорда** появляется раньше:

```
G → C:  ... [C] | C ...   (C появляется на долю раньше смены аккорда)
```

## Правило сильной доли

На **сильных долях** мелодия обычно содержит аккордовые тоны. Проходящие и вспомогательные ноты попадают на **слабые доли**.

Нарушение этого правила создаёт **диссонанс** (задержание, апподжатура) — мощный выразительный приём.

> **Для импровизации:** если не знаете, какую ноту сыграть — играйте аккордовый тон на сильную долю. Это всегда будет звучать «правильно». Остальные ноты лада можно использовать между аккордовыми тонами.
