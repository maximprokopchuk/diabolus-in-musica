# Дистрибьюторы музыки

## Что такое дистрибьютор

**Цифровой дистрибьютор** — сервис, который доставляет вашу музыку на стриминговые платформы (Spotify, Apple Music, Tidal, YouTube Music и сотни других) и управляет получением роялти от проигрываний.

Без дистрибьютора вы не можете самостоятельно загрузить музыку напрямую на Spotify или Apple Music — для этого нужен аккредитованный посредник.

## Как работает дистрибуция

```
Ваш трек (WAV/FLAC) + метаданные
          ↓
    Дистрибьютор
          ↓
  Spotify / Apple Music / Tidal / Deezer / Amazon / YouTube Music / ...
          ↓
  Роялти → Дистрибьютор → Вам
```

**Роялти** — выплаты за каждое прослушивание. Ставки у платформ разные, обычно около $0.003–0.005 за стрим.

## DistroKid

**DistroKid** — один из самых популярных дистрибьюторов среди независимых артистов.

| Параметр | Условия |
|---------|--------|
| Модель оплаты | Годовая подписка (~$22–36/год) |
| Количество релизов | Неограниченно в рамках подписки |
| Роялти | 100% роялти артисту |
| Скорость загрузки | Обычно 1–5 дней до Spotify |
| Особенности | Spotify for Artists интеграция, разделение роялти |

**Плюсы:** дёшево при большом количестве релизов, быстрая загрузка, неограниченные релизы.

**Минусы:** подписка нужна постоянно — если не платить, релизы снимаются со всех платформ.

## TuneCore

| Параметр | Условия |
|---------|--------|
| Модель оплаты | Ежегодная плата за каждый релиз |
| Сингл | ~$9.99/год |
| Альбом | ~$29.99/год |
| Роялти | 100% роялти артисту |
| Особенности | Развитая аналитика, синхронизационное лицензирование |

**Плюсы:** музыка остаётся на платформах, пока платите за конкретный релиз (не подписка на всё).

**Минусы:** при большом количестве релизов становится дороже DistroKid.

## CD Baby

| Параметр | Условия |
|---------|--------|
| Модель оплаты | Единовременный платёж за релиз |
| Сингл | ~$9.95 (однократно) |
| Альбом | ~$49 (однократно) |
| Роялти | 91% артисту (9% берёт CD Baby) |
| Особенности | Физические CD/виниловые релизы, Publishing Administration |

**Плюсы:** однократная оплата, музыка остаётся навсегда, Publishing Administration (сбор паблишинг-роялти).

**Минусы:** берёт процент с роялти.

## Amuse

| Параметр | Условия |
|---------|--------|
| Бесплатный план | Базовая дистрибуция бесплатно |
| Pro план | $24.99/год |
| Роялти | 100% (Pro), задержки на бесплатном |
| Особенности | Мобильное приложение, фокус на развивающихся артистах |

**Плюсы:** можно начать бесплатно.

**Минусы:** бесплатный план имеет ограничения по скорости выплат и функционалу.

## Как выбрать дистрибьютора

**Вопросы для принятия решения:**

1. **Сколько релизов планируете?**
   - Много — DistroKid (подписка выгоднее)
   - Мало — CD Baby (разовая оплата)

2. **Нужен ли паблишинг (сбор мелодических роялти)?**
   - CD Baby Pro, TuneCore Publishing, SongTrust — специализируются на этом

3. **Важна ли скорость?**
   - DistroKid и TuneCore — одни из быстрейших

4. **Нужна ли физическая дистрибуция?**
   - CD Baby поддерживает физические релизы

> **Рекомендация для старта:** DistroKid — оптимальный выбор для большинства независимых артистов, которые планируют регулярно выпускать музыку. Если выпускаете редко — CD Baby (один раз заплатили и забыли).
