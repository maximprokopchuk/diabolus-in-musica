# Модальные аккорды и воицинги

## Проблема тритона

В модальной музыке тритон **нежелателен** — он создаёт функциональное тяготение (доминанта → тоника), которое «вытягивает» из модального мышления в тональное.

Поэтому в модальной гармонии:
- Избегают аккордов с тритоном (особенно доминантсептаккордов)
- Не используют каденцию V7 → I
- Предпочитают аккорды без септимы или с мажорной септимой

## Типичные модальные аккорды

### Квартовые аккорды (quartal voicings)

Аккорды, построенные по **квартам** вместо терций:

```
Терцовый: D — F — A — C  (Dm7, терции)
Квартовый: D — G — C — F  (кварты, нейтральный)
```

Квартовые воицинги звучат «открыто» и не определяют мажор/минор — идеально для модальной музыки.

На гитаре (типичный квартовый воицинг):
```
e|--3--|
B|--3--|
G|--2--|
D|--0--|  (D—A—D—G — квинта + кварта + кварта)
```

### Sus-аккорды

Sus2 и sus4 — «подвешенные», без терции:
- Dm7sus4, Gsus4, Asus2

### Аккорды с добавленными ступенями

- Dm(add9), Dm6, Cmaj7#11 — подчёркивают характерные ноты лада

## Типичные модальные прогрессии

### Дорийский (от D)

```
Dm7 — G     (i — IV)   ← G содержит B = характерная нота
Dm7 — Em7   (i — ii)
Dm7 — C/D   (педаль на D)
```

### Миксолидийский (от G)

```
G — F — G    (I — ♭VII — I)
G — F — C    (I — ♭VII — IV)
```

### Лидийский (от F)

```
Fmaj7 — G    (I — II)   ← II мажорный (с #4 = B)
Fmaj7#11     (один аккорд с педалью)
```

### Фригийский (от E)

```
Em — F    (i — ♭II)   ← F = характерная нота (♭2)
Em — Fmaj7 — Em
```

## Педальный бас

Главный приём модальной гармонии — **педаль** (drone): басовая нота держится, а верхние аккорды меняются:

```
Бас:   D — D — D — D
Аккорд: Dm — G/D — Am/D — C/D
```

Педаль «привязывает» гармонию к модальному центру.

> **На гитаре:** используйте открытые струны как педальный бас. Например, открытая E (6-я струна) и аккорды выше — мгновенно создаёт модальное звучание.
