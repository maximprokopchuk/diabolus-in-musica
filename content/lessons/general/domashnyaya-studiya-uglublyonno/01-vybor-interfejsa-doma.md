# Выбор аудиоинтерфейса для домашней студии

## Зачем нужен аудиоинтерфейс

Встроенная звуковая карта компьютера не предназначена для записи: высокая задержка, слабые предусилители, нет профессиональных входов. **Аудиоинтерфейс** — это специализированное устройство, решающее эти проблемы.

**Основные функции интерфейса:**
- Аналого-цифровое (A/D) и цифро-аналоговое (D/A) преобразование
- Предусилители (preamps) для микрофонов
- Фантомное питание (+48V) для конденсаторных микрофонов
- Мониторинг без задержки (direct monitoring)
- Подключение к компьютеру (USB, Thunderbolt, PCIe)

## Ключевые параметры аудиоинтерфейса

### Количество входов и выходов

| Ситуация | Рекомендуемые I/O |
|---------|-----------------|
| Вокал + гитара, один человек | 2 в / 2 вых |
| Запись малой группы | 4–8 вх / 4–8 вых |
| Полный барабанный сет | 8–18 вх |
| Профессиональная многоканальная запись | 16+ вх с расширением по ADAT |

**Практическое правило:** покупать интерфейс с чуть большим числом каналов, чем нужно сейчас — расширение потребуется быстрее, чем кажется.

### Качество предусилителей (преампов)

**Преамп** усиливает слабый сигнал микрофона (−60 dBu) до линейного уровня (−10...+4 dBu). Качество преампа напрямую влияет на шумовой пол и «прозрачность» записи.

**EIN (Equivalent Input Noise)** — главный технический параметр. Чем ниже значение, тем лучше:
- **Хорошо:** −129 дБу и ниже
- **Профессионально:** −130...−132 дБу
- **Бюджетно:** −120...−125 дБу (слышимый шум при высоком гейне)

### Конвертеры (A/D и D/A)

Качество конвертеров определяет точность оцифровки. Параметры:
- **Частота дискретизации:** 44,1 / 48 / 88,2 / 96 / 192 кГц
- **Разрядность:** 24 бит (стандарт) или 32 float (некоторые современные интерфейсы)
- **Динамический диапазон (DAC):** 112–124 дБ — чем выше, тем лучше

**THD+N (Total Harmonic Distortion + Noise)** — суммарные искажения. Должны быть ниже −100 дБ.

### Подключение к компьютеру

| Интерфейс | Пропускная способность | Задержка | Рекомендация |
|----------|----------------------|---------|-------------|
| **USB 2.0** | 480 Мбит/с | Хорошая | Для большинства домашних задач |
| **USB 3.0** | 5 Гбит/с | Отличная | Многоканальные интерфейсы |
| **Thunderbolt 3/4** | 40 Гбит/с | Минимальная | Профессиональные задачи, Mac |
| **PCIe** | Зависит от слота | Минимальная | Встраиваемые карты, студии |

**Focusrite OctoPre Dynamic** — 8 каналов с компрессорами.

## Задержка (Latency) и буфер

**Buffer size** — количество аудиосемплов, обрабатываемых за раз.

| Buffer size | Задержка @ 48 кГц | Применение |
|-----------|-----------------|-----------|
| 32 семпла | ~0,7 мс | Экстремально низкая, нагрузка на CPU |
| 64 семпла | ~1,3 мс | Запись в реальном времени |
| 128 семплов | ~2,7 мс | Сбалансированно |
| 256 семплов | ~5,3 мс | Микширование и сведение |
| 512+ семплов | 10+ мс | Только финальный рендеринг |

**При записи** — использовать 64–128 семплов. **При сведении** — можно 256–512 для уменьшения нагрузки на CPU.

> **Итоговая рекомендация:** Focusrite Scarlett 2i2 или MOTU M2 — лучший выбор для большинства начинающих. Первые 3–5 лет работы они не станут ограничивающим звеном. Апгрейд имеет смысл, когда понятно, чего конкретно не хватает.
