# Запись гитары

## Три подхода к записи электрогитары

### 1. Микрофонирование кабинета (Mic'ing)
### 2. DI + Amp Simulator (программный усилитель)
### 3. Re-amping (запись DI → усиление позже)

## Микрофонирование кабинета

**Классический метод** — поставить микрофон перед кабинетом и записать реальный звук усилителя.

### Выбор микрофона

| Микрофон | Тип | Характер |
|---------|-----|---------|
| Shure SM57 | Динамический | Индустриальный стандарт, недорогой, плотный мид |
| Sennheiser MD421 | Динамический | Больше баса, широкий тембр |
| Royer R-121 | Ленточный | Тёплый, «виниловый», дорогой |
| Neumann U87 | Конденсаторный | Воздушный, подробный, требует тихой комнаты |

**SM57** — абсолютный стандарт. Записаны тысячи рок-альбомов с единственным SM57.

### Позиция микрофона

```
              [Кабинет 4×12]
         ╔══════════════════╗
         ║  ╔════╗  ╔════╗  ║
         ║  ║    ║  ║    ║  ║
         ║  ╚════╝  ╚════╝  ║
         ║  ╔════╗  ╔════╝  ║   ← Центр конуса (яркий звук)
         ║  ║    ║  ║    ║  ║   ← Край конуса (тёплый звук)
         ║  ╚════╝  ╚════╝  ║
         ╚══════════════════╝
              ↑
           [SM57 ~2-5 см от гриля]
```

| Позиция микрофона | Звук |
|-------------------|------|
| Центр конуса, вплотную | Максимально яркий, резкий |
| Край конуса, вплотную | Теплее, меньше «бумажного» звука |
| Отодвинуть на 15–30 см | Добавляет «воздух» и комнату |
| Два микрофона (SM57 + конденсатор) | Можно смешивать в сведении |

**Правило фазы:** при использовании двух микрофонов убедитесь, что они **в фазе**. Перевернутая фаза = потеря баса и объёма.

## DI + Amp Simulator

**DI (Direct Input)** — запись «сухого» сигнала гитары напрямую без усилителя. Amp Simulator (плагин) имитирует звук реального усилителя и кабинета.

### Преимущества

- Нет шума комнаты и соседей
- Можно менять «усилитель» уже после записи
- Идеален для домашней студии

### Популярные amp-симуляторы

| Плагин | Производитель | Особенности |
|--------|--------------|-------------|
| **Neural DSP Archetype: Plini** | Neural DSP | Прогрессив, чистые и перегруз |
| **Neural DSP Tone King Imperial** | Neural DSP | Блюз и рок, ламповый характер |
| **Positive Grid BIAS Amp 2** | Positive Grid | Огромная библиотека, кастомизация |
| **IK AmpliTube 5** | IK Multimedia | Много моделей, включая редкие |
| **Logic Amp Designer** | Apple | Встроен в Logic, хорошее качество |
| **Line 6 Helix Native** | Line 6 | Тот же движок, что в педалборде Helix |

### Цепочка обработки DI-сигнала

```
DI-сигнал → [Noise Gate] → [Amp Sim] → [Cabinet IR] → [EQ] → [Mix
```

**Cabinet IR (Impulse Response)** — «слепок» реального кабинета. Большинство amp-симуляторов включают встроенные IR; можно загрузить сторонние (York Audio, OwnHammer).

## Re-amping

**Re-amping** — промежуточный подход:
1. Записывается **DI-сигнал** гитары (сухой, без обработки)
2. Позже DI-сигнал отправляется обратно через **re-amp box** в реальный усилитель
3. Звук усилителя записывается микрофоном

### Зачем это нужно

- Можно записать идеальный дубль без давления «играть с хорошим звуком одновременно»
- Можно отправить трек звукорежиссёру в другом городе
- Можно попробовать несколько усилителей на один и тот же дубль

### Оборудование для re-amp
**Re-amp box** (например, Radial X-Amp, Suhr Iso Line Out) преобразует линейный сигнал аудиоинтерфейса в инструментальный уровень для усилителя.

## Уровни записи

| Целевой уровень | Значение |
|----------------|---------|
| Пик при записи | -18 dBFS (оставить 18 дБ headroom) |
| RMS (средний уровень) | -20 до -24 dBFS |
| Клиппинг | **Никогда!** Цифровой клиппинг не исправить |

> **Правило:** записывайте тише, чем кажется нужным. В цифре лучше иметь запас громкости — усилить при сведении легко, исправить клиппинг невозможно. «Горячая» запись — миф аналоговой эпохи.
