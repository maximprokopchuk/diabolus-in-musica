# Основы микширования

## Что такое микширование

**Микширование** — процесс объединения отдельных записанных треков в единое, цельное и сбалансированное звучание. Цель не «сделать громче», а создать **пространство** для каждого инструмента.

Три измерения звука в миксе:
- **Вертикаль:** громкость (тихо → громко)
- **Горизонталь:** панорама (лево → право)
- **Глубина:** эффекты реверберации и задержки (близко → далеко)

## Уровни (Gain Staging)

**Gain staging** — управление уровнями сигнала через всю цепочку обработки, чтобы избежать перегрузки и сохранить оптимальное соотношение сигнал/шум.

| Этап | Рекомендуемый уровень |
|------|----------------------|
| Запись | Пик -18 dBFS |
| Отдельные треки в DAW | RMS около -18 до -20 dBFS |
| Мастер-шина перед мастерингом | Пик -6 dBFS (headroom для мастеринга) |
| Финальный мастер (стриминг) | Integrated loudness -14 LUFS |

### Начинайте с барабанов
Установите уровень барабанов как «якорь» микса. Остальные инструменты выставляются относительно них.

## Панорама

**Панорама** распределяет звук по стерео-полю (лево–право).

### Типичное распределение в рок/поп-миксе

| Инструмент | Позиция панорамы |
|-----------|----------------|
| Бас-барабан | Centre (0) |
| Малый барабан | Centre (0) или лёгкий сдвиг |
| Бас-гитара | Centre (0) |
| Вокал | Centre (0) |
| Гитара 1 (ритм) | L 70–80 |
| Гитара 2 (ритм, дубль) | R 70–80 |
| Хай-хэт | L 30–50 |
| Тарелки | L/R симметрично |
| Клавишные / пэды | L/R, зависит от аранжировки |

**Низкочастотные инструменты** (бас, бас-барабан) держите в центре — пространственное восприятие низких частот хуже, и смещённый бас звучит неустойчиво.

## Эквализация (EQ)

**EQ** управляет тембральным балансом — усиливает или срезает определённые частоты.

### Частотная карта инструментов

| Диапазон | Частоты | Инструменты |
|---------|---------|------------|
| Суббас | 20–80 Гц | Бас, бас-барабан (фундаментал) |
| Нижний мид | 80–250 Гц | «Тело» гитары, тепло |
| Средний мид | 250–2000 Гц | «Наполненность», мид гитары |
| Верхний мид | 2–6 кГц | Атака, «укус», разборчивость |
| Присутствие | 4–8 кГц | Присутствие, чёткость, «атака» |
| Воздух | 8–20 кГц | Блеск, сияние |

### Основные операции EQ

**High-Pass Filter (HPF):** срезает всё ниже заданной частоты. Применяется **почти на каждом треке** кроме баса и бас-барабана — убирает ненужный низкочастотный «мусор».

**Low-Pass Filter (LPF):** срезает всё выше заданной частоты. Полезен для подзвучки, создания «телефонного» эффекта.

**Bell EQ:** поднимает или срезает определённую полосу с настраиваемой шириной (Q). Основной инструмент формирования тембра.

**Shelf EQ:** равномерно поднимает или срезает всё выше/ниже заданной частоты.

### Принцип «вырезай, не поднимай»
Вместо того чтобы усиливать частоты, сначала ищите **лишние** и режьте их. Мощный инструментальный EQ — это EQ с убывающими, а не растущими полосами.

## Компрессия

**Компрессор** уменьшает динамический диапазон — громкие пики становятся тише, тихие части относительно громче.

### Параметры компрессора

| Параметр | Значение |
|---------|---------|
| **Threshold** | Уровень, при котором включается компрессия |
| **Ratio** | Степень сжатия (2:1 = мягко, 10:1 = сильно) |
| **Attack** | Скорость срабатывания (мс) |
| **Release** | Скорость отпускания (мс) |
| **Makeup Gain** | Компенсирующее усиление после компрессии |
| **Knee** | Hard knee = резко, Soft knee = плавно |

### Типичные настройки для гитары

| Задача | Threshold | Ratio | Attack | Release |
|--------|-----------|-------|--------|---------|
| Лёгкое «склеивание» ритм-гитары | -18 dB | 2:1 | 10–20 мс | 100–200 мс |
| Контроль пиков | -12 dB | 4:1 | 5 мс | авто |
| «Ламповый» характер | -20 dB | 3:1 | 20–30 мс | 200 мс |

**Быстрый Attack** срезает атаку ноты — звук становится мягче.
**Медленный Attack** пропускает атаку — звук «кусает», ударность сохраняется.

## Порядок обработки в канале

```
Входной сигнал
   ↓
[High-Pass Filter] — убрать низкочастотный шум
   ↓
[EQ] — коррекция тембра
   ↓
[Компрессор] — контроль динамики
   ↓
[Эффекты: Reverb, Delay] — на отдельных шинах (Aux Send)
   ↓
Выход на мастер-шину
```

> **Главное правило микширования:** слушайте микс в контексте, не в изоляции. Инструмент, который «отлично» звучит одиноко на треке, может теряться в миксе — и наоборот. Задача — чтобы весь микс звучал хорошо, а не каждый инструмент по отдельности.
