# DAW — Цифровая аудиорабочая станция

## Что такое DAW

**DAW** (Digital Audio Workstation) — программа для записи, редактирования, аранжировки и сведения музыки. Это цифровая версия аналоговой студии с многодорожечным магнитофоном, микшерным пультом и стойкой с эффектами.

Все современные DAW поддерживают:
- Запись аудио с аудиоинтерфейса
- MIDI-программирование инструментов
- Редактирование и обработку треков
- Плагины (VST/AU) — виртуальные инструменты и эффекты
- Микширование и мастеринг
- Экспорт в MP3, WAV, FLAC

## Обзор основных DAW

### Ableton Live
**Платформа:** Mac / Windows

| Аспект | Описание |
|--------|---------|
| Интерфейс | Два режима: Session View (живые петли) + Arrangement View (традиционный) |
| Сильная сторона | Живые выступления, электронная музыка, loop-based production |
| Слабая сторона | Нотная редакция, оркестровые аранжировки |
| Цена | ~750 € (Suite), ~450 € (Standard) |

**Используется:** Deadmau5, Skrillex, Daft Punk, Nine Inch Nails

### Logic Pro
**Платформа:** только Mac

| Аспект | Описание |
|--------|---------|
| Интерфейс | Традиционный timeline, удобный для гитаристов |
| Сильная сторона | Огромная библиотека звуков, отличные amp-симуляторы (Amp Designer) |
| Слабая сторона | Только Apple |
| Цена | 250 € (разовая покупка, без подписки) |

**Используется:** Billie Eilish (записана в Logic), Taylor Swift, многие Nashville-продюсеры

### Reaper
**Платформа:** Mac / Windows / Linux

| Аспект | Описание |
|--------|---------|
| Интерфейс | Гибкий, настраиваемый |
| Сильная сторона | Цена, гибкость, идеально для подкастов и многодорожечной записи |
| Слабая сторона | Нет встроенных инструментов мирового класса |
| Цена | ~60 $ (лицензия для некоммерческого использования) |

**Популярен** среди независимых музыкантов и подкастеров.

### FL Studio
**Платформа:** Mac / Windows

| Аспект | Описание |
|--------|---------|
| Интерфейс | Piano Roll + Step Sequencer |
| Сильная сторона | Создание битов, хип-хоп, EDM, простота работы с паттернами |
| Слабая сторона | Запись живых инструментов (не основной фокус) |
| Цена | ~200–500 $ (единоразово, бесплатные обновления навсегда) |

**Используется:** Martin Garrix, Avicii, Deadmau5 (ранний период), Lil Uzi Vert

### Pro Tools
**Платформа:** Mac / Windows

| Аспект | Описание |
|--------|---------|
| Интерфейс | Профессиональный, студийный стандарт |
| Сильная сторона | Стандарт для крупных студий, кино, TV |
| Слабая сторона | Дорого, сложно, подписка |
| Цена | ~300 € / год |

## Базовый workflow при записи гитары

```
1. Создать новый проект → установить BPM и тональность
2. Создать MIDI-трек с барабанами (или загрузить loop)
3. Создать аудио-трек для гитары (Input: аудиоинтерфейс)
4. Включить прямой мониторинг
5. Настроить уровень записи: пик -18 dBFS (оставить headroom)
6. Записать дубли, выбрать лучший
7. Добавить плагины: amp-sim → IR-кабинет → EQ → компрессор
8. Свести все треки
9. Экспорт: File → Export → WAV 24 bit / 48 kHz
```

## Ключевые понятия DAW

| Термин | Значение |
|--------|---------|
| **Track** | Дорожка — один инструмент или элемент |
| **Clip / Region** | Отрезок аудио или MIDI на треке |
| **Plugin (VST/AU)** | Виртуальный инструмент или эффект |
| **Routing** | Маршрутизация сигнала между треками |
| **Bus / Aux** | Вспомогательная шина для группировки треков |
| **Automation** | Автоматическое изменение параметров во времени |
| **Bounce / Export** | Экспорт финального аудио |
| **Latency** | Задержка между игрой и слышимым звуком |
| **Buffer Size** | Размер буфера: меньше = меньше задержка, но больше нагрузка на CPU |

## Настройки проекта

Перед записью установите:
- **Sample Rate:** 44100 Hz (CD-качество) или 48000 Hz (видео/студия)
- **Bit Depth:** 24 bit (16 bit только для финального экспорта)
- **Buffer Size:** 64–256 samples при записи (меньше задержка)
- **Tempo:** точный BPM вашего трека
- **Time Signature:** 4/4 или нужный метр

> **Совет:** не гонитесь за «лучшей» DAW — лучшая та, в которой вы работаете эффективно. Большинство профессионалов выбирают DAW один раз и изучают её глубоко, а не переключаются между программами. Для гитаристов, работающих в рок/метал, Logic Pro (на Mac) или Reaper — отличный старт.
