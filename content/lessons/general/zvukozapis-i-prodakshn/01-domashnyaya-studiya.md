# Домашняя студия

## С чего начать

Домашняя студия сегодня доступна каждому музыканту. Разница между профессиональной студией и домашней — не в количестве оборудования, а в **акустике помещения** и качестве цепочки записи. При правильном подходе можно получить коммерческий результат из спальни.

Минимальная цепочка:
```
Гитара / микрофон → Аудиоинтерфейс → Компьютер (DAW) → Мониторы / Наушники
```

## Аудиоинтерфейс

**Аудиоинтерфейс** — устройство, которое конвертирует аналоговый сигнал (микрофон, гитара) в цифровой формат для компьютера и обратно.

| Параметр | Что важно |
|----------|-----------|
| Количество входов | 2 входа достаточно для начала (микрофон + DI-гитара) |
| Предусилители (preamp) | Качество преампа влияет на тембр записи |
| Латентность | Чем ниже, тем меньше задержка при мониторинге |
| Разрядность / частота | 24 bit / 48 kHz — стандарт; 96 kHz для мастеринга |

### Популярные модели

| Модель | Уровень | Особенности |
|--------|---------|-------------|
| Focusrite Scarlett 2i2 | Начальный | Отличные преампы за свою цену |
| Universal Audio Volt | Начальный | Встроенный vintage-режим преампа |
| SSL 2+ | Средний | Профессиональные преампы, Legacy-режим |
| Audient iD14 | Средний | Прозрачный, студийный характер |
| Universal Audio Apollo Twin | Профессиональный | DSP-процессор для плагинов в реальном времени |

**Важно:** используйте интерфейс с **прямым мониторингом** (direct monitoring) — он позволяет слышать себя без задержки через интерфейс, минуя компьютер.

## Студийные мониторы

**Мониторы** (studio monitors) — активные акустические системы с **линейной АЧХ**: они воспроизводят звук максимально честно, без подкраски. Бытовая акустика специально окрашивает звук (добавляет бас и яркость), что мешает принимать правильные решения при сведении.

| Модель | Размер НЧ-динамика | Характер |
|--------|-------------------|---------|
| Yamaha HS5 / HS7 | 5" / 7" | Нейтральные, честные, «белые Ямахи» — стандарт |
| Adam Audio T5V | 5" | Яркие верха, хорошая детальность |
| Focal Alpha 50 | 5" | Детальные, прозрачные, французская школа |
| KRK Rokit 5 | 5" | Слегка окрашенные, популярны в поп/хип-хопе |
| Genelec 8020 | 4" | Студийный класс, исключительно точные |

**Расстановка мониторов:** равносторонний треугольник — мониторы и уши на одном уровне. Расстояние между мониторами = расстоянию от мониторов до слушателя (обычно 1–1,5 м). Тыл мониторов должен быть на 15–30 см от стены.

## Наушники

Для домашней студии нужны **два вида** наушников:

### Закрытые (для записи)
Не пропускают звук в микрофон. Музыкант слышит себя и метроном, не записывая их.

| Модель | Особенности |
|--------|------------|
| Sony MDR-7506 | Индустриальный стандарт, недорогие |
| Beyerdynamic DT 770 Pro | Детальные, точные |
| Audio-Technica ATH-M50x | Популярны, слегка окрашены |

### Открытые (для микширования)
Более точная звуковая сцена. Пропускают окружающий звук — только для студии в тишине.

| Модель | Особенности |
|--------|------------|
| Sennheiser HD 600 / 650 | Эталон мониторных наушников |
| Beyerdynamic DT 990 Pro | Яркие верха, подробные |
| AKG K702 | Широкая сцена, нейтральные |

## Акустическая обработка

Самый недооценённый аспект домашней студии. **Плохая акустика** — главный враг качественного микширования. Пустая прямоугольная комната создаёт стоячие волны и резонансы, которые мешают слышать реальную картину.

### Проблемы необработанной комнаты

| Проблема | Что происходит |
|---------|----------------|
| Стоячие волны | Некоторые частоты усиливаются, другие исчезают |
| Раннее отражение | Звук от стен сливается с прямым, смазывает стерео |
| Реверберация | Долгий «хвост» комнаты мешает слышать детали |

### Акустические панели

**Абсорберы** поглощают высокие и средние частоты:
- Панели из минеральной ваты (Rockwool / Basotect) толщиной 5–10 см
- Размещаются на первых точках отражения (боковые стены, потолок)

**Басовые ловушки (bass traps)** поглощают низкие частоты:
- Толстые абсорберы (10–30 см) в углах комнаты
- Углы — место накопления басовой энергии
- Критически важны для точности низкочастотного баланса

**Диффузоры** рассеивают отражения, сохраняя «живость» звука:
- Размещаются на задней стене за слушателем

### Минимальный бюджетный вариант
1. Бас-ловушки из Rockwool 100 мм в четырёх вертикальных углах
2. Панели 50 мм на первых отражениях по бокам
3. Толстый ковёр на полу
4. Книжные полки с книгами на задней стене (работают как диффузор)

> **Главный принцип:** вложите деньги сначала в акустику комнаты, потом в оборудование. Дорогие мониторы в плохой комнате дадут худший результат, чем средние мониторы в хорошо обработанной комнате.
