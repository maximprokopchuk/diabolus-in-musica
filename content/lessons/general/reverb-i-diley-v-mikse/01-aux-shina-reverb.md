# Aux-шина для реверба

## Два способа подключить реверб

### Insert (вставка)

Реверб ставится прямо в цепочку обработки трека. Всё, что идёт через трек, будет обработано ревербом.

```
Трек → [EQ] → [Компрессор] → [Реверб] → Выход
```

**Проблема:** если реверб нужен на 10 треках, вы создаёте 10 копий плагина. Это нагружает CPU и, что важнее, **каждый инструмент будет в своей «комнате»** — ощущение единого пространства теряется.

### Aux Send (параллельная шина)

Реверб размещается на **отдельной aux-шине** (return channel, FX channel). Каждый трек отправляет часть своего сигнала на эту шину через **send**.

```
Вокал ───────────────────────────── Выход
  └── [Send 30%] ─┐
                   ↓
Гитара ──────────── Выход         [Aux Reverb] → Выход
  └── [Send 20%] ──┘
```

**Преимущества aux-шины:**
- Один экземпляр плагина на все треки — экономия CPU
- Все инструменты находятся в **одном пространстве** — микс звучит цельно
- Можно независимо регулировать количество реверба для каждого трека

> **Главное правило:** реверб на aux-шине всегда должен быть настроен на **100% Wet** (только reverb-сигнал, без прямого). Dry-сигнал идёт через оригинальный трек.

## Pre-delay (пред-задержка)

**Pre-delay** — время между прямым сигналом и началом реверберации.

В реальных помещениях звук сначала доходит до вас напрямую, и только через несколько миллисекунд приходят первые отражения от стен.

| Pre-delay | Эффект |
|-----------|--------|
| 0 мс | Реверб начинается сразу — «склеивается» с прямым звуком |
| 10–20 мс | Небольшое разделение — звук остаётся разборчивым |
| 30–60 мс | Хорошая разборчивость, ощущение пространства |
| 80–120 мс | Большое разделение, «сзади» звучит |

**Pre-delay на вокале:** 20–40 мс — вокал остаётся чётким, реверб не «смазывает» согласные.

**Синхронизация с темпом:** pre-delay можно синхронизировать с темпом (например, 1/16 ноты). Это даёт более «музыкальный» результат.

## Decay (время затухания / RT60)

**Decay** (или **Reverb Time / RT60**) — время, за которое реверб затухает на 60 дБ.

| Decay | Ощущение |
|-------|---------|
| 0.3–0.8 с | Маленькая комната, студия |
| 1–2 с | Среднее помещение, live-room |
| 2–4 с | Большой зал, церковь |
| 4–8 с | Огромный собор, пещера |

**Для микса:** чем плотнее микс (быстрое BPM, много инструментов), тем короче должен быть decay. Длинный реверб на быстрой музыке создаёт «кашу».

**Правило BPM:** RT60 = 60 / BPM × 2 — приблизительный ориентир для «музыкального» времени затухания.

## Структура aux-шины реверба в DAW

### В Ableton Live

1. Создайте **Return Track** (Ctrl+Alt+T)
2. Разместите реверб-плагин на Return Track, установите Wet = 100%
3. На нужных треках нажмите кнопку **Sends** и поднимите уровень отправки на возвратный трек

### В Logic Pro

1. Создайте **Aux Channel Strip**
2. Добавьте реверб как insert на aux
3. На инструментальных треках настройте **Bus Send** в разделе Sends

### В Pro Tools

1. Создайте **Aux Input**
2. Установите реверб как insert
3. На инструментальных треках направьте sends на тот же bus

## Сколько реверба нужно?

**Общее правило:** реверба должно быть слышно меньше, чем кажется при настройке. Откройте фейдер aux до момента, когда реверб станет заметен, затем немного убавьте.

**Проверка:** заглушите aux с ревербом. Должно показаться, что чего-то не хватает. Если разница едва слышна — добавьте ещё немного.

> **Профессиональный приём:** высокочастотный EQ (high shelf cut около 8 кГц) на aux-шине делает реверб «тоньше» и менее навязчивым. Реверб занимает место в миксе — чем он темнее, тем меньше перекрывает прямые инструменты.
