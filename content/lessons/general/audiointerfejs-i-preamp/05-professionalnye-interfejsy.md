# Профессиональные интерфейсы

## Масштабирование студии

В профессиональных студиях одного интерфейса с 8–16 каналами недостаточно. Крупные сессии (оркестр, живая запись группы с барабанами) требуют **32–96 одновременных каналов** с минимальной латентностью и абсолютной надёжностью.

Для этого существуют **профессиональные форматы передачи аудио**.

## MADI (Multi-channel Audio Digital Interface)

**MADI** — профессиональный стандарт для передачи большого количества каналов по одному кабелю (коаксиальный BNC или оптоволокно).

| Параметр | Значение |
|---------|---------|
| Каналы | До 64 при 48 кГц, 32 при 96 кГц |
| Расстояние | До 100 м (коаксиал), до 2 км (оптоволокно) |
| Задержка | Минимальная (несколько семплов) |
| Применение | Крупные студии, broadcast, живой звук |

**MADI-устройства:** RME HDSPe MADI, Focusrite RedNet (MADI версии), SSL консоли

## Dante

**Dante** — современный сетевой аудиопротокол, передающий аудио по стандартной **Ethernet-сети** (Gigabit Ethernet).

| Параметр | Значение |
|---------|---------|
| Каналы | До 1024×1024 при 48 кГц |
| Расстояние | Неограничено (через сеть) |
| Задержка | ~0.15–10 мс (настраиваемая) |
| Подключение | RJ-45 (обычный Ethernet) |

### Преимущества Dante

- **Масштабируемость** — добавляйте устройства как к обычной сети
- **Гибкость** — любой канал на любое устройство в сети
- **Надёжность** — резервные пути, мониторинг сети
- **Интеграция** — широкая поддержка в Yamaha, Shure, Crown, Lab.Gruppen

**Применение:** крупные концертные площадки, театры, конференц-системы, broadcast.

## AVB (Audio Video Bridging)

Открытый стандарт (IEEE 802.1), альтернатива Dante.

- Поддержка в Motu, Presonus, Apple
- Стандарт — открытый, не требует лицензии
- Меньше устройств, чем в экосистеме Dante

## Pro Tools HD (HDX)

**Pro Tools** — стандарт профессиональных студий. Версия **HD / HDX** использует специализированные карты.

| Компонент | Описание |
|----------|---------|
| **HDX Card** | DSP-карта PCIe, обрабатывает плагины без нагрузки на CPU |
| **HD I/O** | 8-канальный интерфейс, подключается к HDX по DigiLink |
| **HD MADI** | MADI-интерфейс для Pro Tools HD |

**Особенность:** Pro Tools HD работает на фиксированной, минимальной латентности (~0.8 мс при 96 кГц) — что критично для сессий с живыми музыкантами.

**Стоимость:** $10 000–30 000 за полную HDX-систему. Стандарт для крупных коммерческих студий.

## Word Clock (Wordclock)

### Зачем нужна синхронизация

В системе с несколькими цифровыми устройствами **все ADC/DAC должны работать синхронно** — то есть производить семплы точно в один и тот же момент.

Если два устройства не синхронизированы, между ними возникают **клики, хрипы, нестабильность** в цифровом сигнале.

### Word Clock

**Word Clock** — стандартный сигнал синхронизации для профессиональных аудиоустройств. Передаётся по коаксиальному кабелю с разъёмами **BNC**.

```
[Master Clock] → [Аудиоинтерфейс 1] → [Аудиоинтерфейс 2] → [Конвертер 3]
      ↓ (Word Clock через BNC)
Все устройства работают по одному «метроному»
```

### Источники Word Clock

| Источник | Применение |
|---------|-----------|
| **Внутренний генератор интерфейса** | Простые системы с одним устройством |
| **Internal Clock другого устройства** | Одно устройство — master, другие — slave |
| **Dedicated Clock (Antelope, Black Lion)** | Эталонный мастер-клок для всей студии |

### Jitter

**Jitter** — нестабильность тактового сигнала. Даже небольшой jitter ухудшает качество конвертации: звук становится «резким», «напряжённым», теряет «воздух».

Качественные **master clock** (Antelope 10M, Apogee Big Ben, Black Lion Audio) минимизируют jitter и улучшают звук всей системы.

## Сравнение профессиональных форматов

| Формат | Каналы | Расстояние | Применение |
|--------|--------|-----------|-----------|
| ADAT | 8 (48 кГц) | До 5 м | Расширение студийного интерфейса |
| AES/EBU | 2 | До 100 м | Студийное соединение |
| MADI | До 64 | До 2 км | Broadcast, крупные студии |
| Dante | До 1024 | Неограничено | Сетевые инсталляции, PA |
| AVB | До 256 | Неограничено | Открытые сетевые системы |
| Pro Tools DigiLink | 32 | Внутри студии | Pro Tools HD системы |

> **Для начинающего:** всё это пригодится, когда вы дорастёте до нескольких синхронизированных устройств. Сейчас достаточно знать, что Word Clock — обязательный элемент при работе с несколькими цифровыми устройствами одновременно.
