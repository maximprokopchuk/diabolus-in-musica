# Преампы

## Зачем нужен преамп

Микрофон вырабатывает **очень слабый сигнал** — уровня mic level (−60 до −40 дБu). Для записи и обработки нужен уровень line level (+4 дБu в профессиональных системах). Задача преампа — **поднять уровень** микрофонного сигнала до рабочего.

```
Mic Level (~−50 дБu) → [ПРЕАМП +40–60 дБ gain] → Line Level (+4 дБu)
```

Но преамп делает не только это. Он **окрашивает** звук — или наоборот, старается не окрашивать.

## Прозрачные vs Окрашенные преампы

### Прозрачные преампы (Clean / Transparent)

Цель — максимально точно усилить сигнал без изменения тембра.

**Характеристики:**
- Очень низкий шум (EIN < −130 дБu)
- Ровная АЧХ
- Минимальные гармонические искажения (THD < 0.001%)
- Высокий headroom

**Применение:** запись, где важна точность — акустические инструменты, вокал, оркестр, когда вы хотите «честный» звук источника.

**Примеры:** Millennia HV-3D, Grace Design m101, Focusrite ISA One

### Окрашенные преампы (Colored / Character)

Добавляют к сигналу **характерный тембр** — гармоники, насыщение, особую «теплоту».

**Характеристики:**
- Трансформаторный вход/выход (добавляет гармоники)
- Класс A работа (другая динамика)
- Характерный mid-range или low-end

**Применение:** запись, где нужен «звук» — вокал рока/соула, барабаны с «характером», винтажный эстетический звук.

**Примеры:**
- **Neve 1073** — плотный low-mid, тёплый «британский» звук, стандарт рока и поп
- **API 312** — атакующий, пунш в mid, «американский» звук
- **Universal Audio 610** — ламповый, певучий, мягкий
- **Chandler Limited TG2** — плотный, агрессивный, студия Abbey Road

## Класс A (Class A)

**Class A** — схемотехническая конфигурация, при которой транзистор (или лампа) проводит ток **постоянно**, в течение всего периода сигнала.

| Характеристика | Class A | Class AB |
|---------------|---------|---------|
| Ток покоя | Высокий постоянно | Меняется |
| Искажения | Чётные гармоники (приятные) | Нечётные (неприятные при перегрузке) |
| КПД | Низкий (много тепла) | Выше |
| Звуковой характер | «Тёплый», «музыкальный» | Нейтральнее |

**Большинство «звучащих» студийных преампов работают в режиме Class A.**

## Трансформаторные преампы

**Трансформатор** на входе/выходе преампа — важная составляющая «винтажного» звука.

### Что даёт трансформатор

1. **Гармонические искажения** — трансформатор нелинеен, добавляет чётные гармоники (звучат «тепло» и «музыкально»)
2. **Частотная характеристика** — небольшой подъём мидов или низов в зависимости от конструкции
3. **Насыщение** — при перегрузке трансформатор «мягко» насыщается, подобно ламповому устройству
4. **Защита** — гальваническая развязка входа и выхода

**Примеры трансформаторных преампов:** Neve 1073/1084, API 312, Jensen 990, Chandler, Telefunken V72

### Бестрансформаторные преампы

- Более нейтральные, «прозрачные»
- Меньше гармонических искажений
- Лучше для измерительных задач
- **Примеры:** Grace Design, Millennia, RME

## Ламповые преампы

Лампы вместо транзисторов — классика студийного звука 1950-60-х.

| Плюсы | Минусы |
|-------|--------|
| «Тёплый», органичный звук | Дорогие |
| Мягкое насыщение при перегрузке | Требуют замены ламп |
| Восприятие как «живого» звука | Шумнее транзисторных |

**Примеры:** Universal Audio 610, Manley VOXBOX, Tube-Tech MP1A, Neve 1073 (гибрид)

## Преампы в аудиоинтерфейсах

Все аудиоинтерфейсы имеют встроенные преампы. Качество сильно различается:

| Класс | Примеры | Характер |
|------|--------|---------|
| **Бюджетный** | Behringer UMC, некоторые Focusrite начального уровня | Работоспособный, нейтральный |
| **Средний** | Focusrite Scarlett 3rd gen, PreSonus Studio 192 | Хороший, чистый |
| **Профессиональный** | Universal Audio Apollo, RME Fireface | Отличный, низкий шум |
| **High-End** | Apogee Symphony, Prism | Референсный уровень |

## Impedance Matching

**Импеданс входа** преампа влияет на то, как «нагружается» микрофон.

- **Стандартный вход микрофона:** 1–3 кОм
- **Высокоимпедансный вход (Hi-Z):** для прямого подключения гитары (>1 МОм)

> **Вывод:** для домашней студии качественный интерфейс со встроенными преампами (UA Apollo, RME, Apogee) закрывает большинство задач. Отдельные преампы (Neve, API) покупают, когда нужен конкретный «характер» звука — это инструмент выразительности, а не необходимость.
