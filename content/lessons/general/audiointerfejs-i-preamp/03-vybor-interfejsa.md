# Выбор аудиоинтерфейса

## Ключевые критерии выбора

При выборе аудиоинтерфейса важно ответить на несколько вопросов:
1. **Сколько каналов нужно одновременно?**
2. **Какое качество преампов требуется?**
3. **Нужно ли расширение через ADAT/MADI?**
4. **Какой интерфейс подключения: USB, Thunderbolt, PCIe?**
5. **Бюджет?**

## Количество каналов

| Ситуация | Минимум каналов |
|---------|----------------|
| Домашняя студия (один вокалист/гитарист) | 2 in / 2 out |
| Запись группы (вокал + гитара + бас) | 4–6 in |
| Полная запись барабанов | 8–12 in |
| Профессиональная студия с полным набором | 16–32+ in |

**Совет:** берите **чуть больше**, чем кажется нужным сейчас. Каналов всегда не хватает.

Один из самых популярных интерфейсов начального-среднего уровня.

| Параметр | Значение |

- Очень доступная цена

- Достаточное качество для старта

## Профессиональный уровень

### Universal Audio Apollo

**Apollo** — стандарт профессиональной домашней и средней студии.

| Особенность | Описание |
|-----------|---------|
| Преампы | UA/Neve Unison Technology — аппаратная имитация преампов |
| DSP | Встроенный процессор для UAD-плагинов без нагрузки на CPU |
| Каналы | Twin X (2), Solo, Quad (от 2 до 8 входов) |
| Подключение | Thunderbolt 3 |
| ADAT | Есть (расширение до 18 каналов) |
| Цена | $700–2000 |

**UAD-плагины** (Neve 1073, LA-2A, Pultec EQ и сотни других) работают на DSP-чипе внутри Apollo — CPU компьютера не нагружается.

### RME Fireface UCX II / 802

RME известна **исключительно низкой латентностью** и стабильностью драйверов.

- RME TotalMix FX — мощный встроенный микшер
- Стабильная работа на любой системе
- Исключительно точные конвертеры
- Цена: $1000–2500

### Apogee Symphony Desktop / Element

Apogee — компания, создавшая конвертеры для многих легендарных студий.

- Референсное качество конвертеров
- Тесная интеграция с macOS
- Цена: $800–3000

## ADAT — расширение каналов

**ADAT (Alesis Digital Audio Tape)** — цифровой оптический формат (Toslink кабель), передающий до **8 каналов** при 44.1/48 кГц или **4 канала** при 88.2/96 кГц.

Используется для расширения: к интерфейсу с ADAT-входом подключается **ADAT-конвертер** (или преамп с ADAT-выходом), добавляя каналы.

**Пример расширения:**
```
Focusrite Scarlett 18i20 (8 преампов + ADAT вход)
    + Behringer ADA8200 (8 преампов, ADAT выход)
    = 16 каналов суммарно
```

Популярные ADAT-расширители:
- **Behringer ADA8200** — 8 преампов, ADAT, бюджетный
- **Focusrite OctoPre** — 8 преампов, ADAT
- **MOTU 8pre** — 8 преампов, ADAT + USB

## Подключение к компьютеру

| Интерфейс | Пропускная способность | Латентность | Применение |
|----------|----------------------|-------------|-----------|
| **USB 2.0** | 480 Мбит/с | Хорошая | До 18 каналов |
| **USB-C / USB 3.0** | 5–10 Гбит/с | Отличная | Большинство современных интерфейсов |
| **Thunderbolt 3/4** | 40 Гбит/с | Минимальная | Профессиональные интерфейсы |
| **PCIe** | Прямо к материнской плате | Минимальная | Студийные системы (RME HDSPe, Pro Tools HD) |

**Thunderbolt** предпочтителен для работы с большим числом каналов и минимальной латентностью. **USB** — достаточен для большинства домашних задач.
