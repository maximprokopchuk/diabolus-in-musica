# Работа с регионами

## Что такое регион

**Регион (Region / Clip)** — прямоугольник на аудио-треке в DAW, представляющий собой фрагмент аудиофайла (или MIDI-данных). Регион — **не сам файл**, а «окно» в него с определёнными параметрами:

- Начальная позиция (на временной шкале)
- Стартовая точка внутри файла
- Длина (то есть сколько файла «видно»)
- Уровень, fade, name

**Ключевое свойство:** редактирование регионов **неразрушающее** — оригинальный аудиофайл никогда не изменяется.

## Основные операции

### Split (Разрезать)

**Split** делит один регион на два в точке playhead'а или в месте клика.

```
До:  [──────────────────────]
           ↑ точка разреза
После: [──────────] [───────]
```

**Зачем:**
- Выбрать лучший фрагмент из середины дубля
- Разрезать для индивидуального редактирования секций
- Подготовить к компингу

**Горячие клавиши:**
- Logic Pro: `Cmd + T`
- Pro Tools: `Cmd + E`
- Reaper: `S`
- Ableton: `Cmd + E`

### Trim (Обрезка)

**Trim** изменяет **начало или конец** региона без разрезания. Регион не разбивается на два — видимая часть файла уменьшается или увеличивается.

```
Оригинал: [──────────────────────]
Trim left: [             ─────────] ← начало сдвинуто вправо
Trim right:[──────────────        ] ← конец сдвинут влево
```

**Применение:**
- Убрать «пустое» начало или конец записи
- «Открыть» скрытую часть файла (если ранее был trim)
- Установить точный вход для кроссфейда

**Важно:** trim обратим. Можно «растянуть» регион обратно и увидеть «скрытую» часть файла.

### Slip (Скольжение)

**Slip** (также называют «slip edit») — перемещение **аудиосодержимого** внутри региона без изменения его положения или размера на временной шкале.

```
Регион не двигается: [──────────────────]
                      [  ← содержимое → ]

Slip:                 [──────────────────]
                       [  ← смещено →   ]
```

**Зачем:**
- Один и тот же регион на временной шкале, но слышим другой фрагмент файла
- Применяется при синхронизации аудио с видео (nudge content without moving clip)
- Тонкая настройка тайминга аудио без переноса региона

**Режим Slip в Pro Tools:** выберите Slip в toolbar или нажмите соответствующую клавишу.

### Move (Перемещение)

Стандартное перетаскивание региона на новую позицию во времени или на другой трек.

**Snap to Grid:** при перемещении региона «прилипает» к ближайшей точке сетки (такту, долям, 1/16 ноты) — удобно для точного выравнивания.

**Snap Off:** свободное перемещение с точностью до образца (sample).

### Duplicate (Дублирование)

Копирование региона:
- **Ctrl/Opt + drag** — перетащить с копированием
- **Cmd + D** (Logic) — дублировать на то же место
- **Cmd + D** (Logic) с Ripple — вставить следующую копию в конце предыдущей (удобно для ритмических петель)

### Resize (Изменение размера без Trim)

Изменение длины региона с растяжкой/сжатием аудио (Time Stretching):
- В большинстве DAW — тянуть за нижний правый угол региона
- Аудио растягивается или сжимается алгоритмически
- Полезно для подгонки длины сэмпла под нужное количество тактов

## Playlist / Take Folder

**Playlist** (Pro Tools) / **Take Folder** (Logic) — организация нескольких дублей на одном треке.

### Как работает Take Folder в Logic

1. Включите запись на трек
2. Запишите первый дубль
3. Нажмите Stop, затем снова Record — начинается **новый take** поверх предыдущего
4. Logic автоматически создаёт Take Folder с несколькими слоями

```
Take Folder:
  ├── Take 1: [──────────────────────────────]
  ├── Take 2: [──────────────────────────────]
  ├── Take 3: [──────────────────────────────]
  └── Comp:   [──Take1─][───Take3──][─Take2─]  ← собранный comp
```

### Компинг в Take Folder

1. Разверните Take Folder (кликните на стрелку)
2. Выделяйте нужные фрагменты в нужных takes (Paint/Marquee tool)
3. Выделенный фрагмент автоматически появляется в Comp-треке
4. Когда закончили: **Flatten** (объединить comp в один регион)

### Playlist в Pro Tools

- **Правый клик на треке** → Playlist → New Playlist
- Каждая сессия записи — отдельный playlist
- Сравнивайте дубли через Track Playlist Menu
- **Comp дорожки** создаются вручную из разных playlist'ов

## Группировка и Lock

### Группировка регионов

Несколько регионов можно **сгруппировать** для совместного перемещения:
- Выделите несколько регионов
- Cmd+G (Logic) или соответствующая команда → они двигаются вместе

**Применение:** double-tracking — группируем L и R треки, чтобы они всегда оставались синхронизированы.

### Lock (Блокировка)

**Lock** фиксирует регион на временной шкале — случайное перемещение невозможно. Используется для:
- Заблокировать MIDI синхронизацию (например, клик-трек к видео)
- Защитить финально отредактированные регионы

## Цветовое кодирование

Профессиональный workflow: **цвет регионов** = мгновенная навигация.

| Цвет | Инструмент |
|-----|-----------|
| Красный | Ударные / Kick / Snare |
| Синий | Гитара |
| Зелёный | Бас |
| Жёлтый | Вокал |
| Фиолетовый | Синтезаторы / клавишные |
| Оранжевый | Brass / духовые |

> **Итоговый принцип:** умение быстро работать с регионами — «грамота» аудиоредактирования. Знание Split, Trim, Slip и кроссфейдов на уровне автоматизма освобождает внимание для музыкальных решений.
