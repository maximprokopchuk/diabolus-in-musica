# Шумоподавление

## Источники шума в записи

Идеальная запись без шума практически недостижима, особенно в домашних условиях. Понимание источников шума помогает бороться с ними на этапе записи, а не редактирования.

| Источник шума | Тип | Решение при записи |
|-------------|-----|------------------|
| Фоновый гул (HVAC) | Постоянный | Выключить кондиционер |
| Электромагнитный фон (50/60 Гц) | Постоянный | Хорошее заземление, экранировка |
| Компьютер (кулеры) | Постоянный | Перенести в другую комнату |
| Трафик, улица | Непостоянный | Записывать ночью |
| Щелчки и хлопки | Случайный | Устранить причину |
| Дыхание вокалиста | Случайный | Компинг, ручное редактирование |
| Скрип стула | Случайный | Смазать или заменить стул |

## Noise Reduction (Шумоподавление)

**Noise Reduction** устраняет **постоянный фоновый шум** — шипение, гул, фон.

### Принцип работы

1. **Сбор шумового профиля (Noise Print)**: алгоритм анализирует фрагмент записи, где только шум без полезного сигнала (2–5 секунд «тишины»)
2. **Применение**: в остальной части записи алгоритм «вычитает» найденный шумовой спектр

```
Оригинал = Сигнал + Шум
Шумовой профиль = Шум
После обработки = Оригинал - Шумовой профиль ≈ Сигнал
```

### Плагины

**iZotope RX** — индустриальный стандарт для аудиореставрации. Набор включает:
- **Noise Reduction** — постоянный шум
- **Voice De-noise** — специально оптимизированный для вокала
- **Music Rebalance** — AI-разделение инструментов

**Другие инструменты:**
- Waves NS1 (автоматическое, «одна кнопка»)
- Audacity Noise Reduction (бесплатный)
- CEDAR Audio (профессиональный, дорогой)

### Побочные эффекты

**Чрезмерное шумоподавление** создаёт:
- «Бульканье» и «пузыри» вокруг транзиентов
- Металлический, «подводный» призвук
- Потерю высокочастотных деталей

Правило: **минимально необходимое шумоподавление**. Лучше немного шума, чем артефакты.

## De-Click (Удаление щелчков)

**Щелчки (clicks)** — короткие импульсные помехи: треск виниловой пластинки, случайный щелчок предусилителя, статический разряд, проблемы с кабелем.

### Алгоритм iZotope RX De-click

Обнаруживает аномально высокие кратковременные пики (импульсы) и **интерполирует** — заменяет проблемный образец расчётным значением, основанным на соседних образцах.

| Параметр | Описание |
|---------|---------|
| **Sensitivity** | Чувствительность к щелчкам: выше = больше находит |
| **Frequency Skew** | Частотный диапазон поиска |
| **Single Band / Multi-Band** | Полосная обработка для лучшего контроля |

### Ручное удаление щелчков

При одиночных щелчках:
1. Увеличьте масштаб до уровня образцов (sample view)
2. Найдите аномальный импульс
3. Нарисуйте исправление (Draw/Pencil tool) — «сгладьте» пик вручную

## De-Crackle (Удаление треска)

**Треск (crackle)** — непрерывные, плотно расположенные щелчки. Типичен для:
- Старых виниловых пластинок
- Потрескавшейся записи
- Плохих аналоговых подключений

**iZotope RX De-crackle** работает аналогично De-click, но оптимизирован для серии частых щелчков.

## Spectral Repair (Спектральное восстановление)

**Spectral Repair** (iZotope RX) — наиболее мощный и гибкий инструмент. Позволяет **визуально редактировать** запись в частотно-временном пространстве (spectogram).

### Как выглядит spectral view

```
Частота (Гц)
16000 │ ░░░░░░░░░░░░░░░
 8000 │ ░░█░░░░░░░█░░░  ← звуки выше 8 кГц
 4000 │ ░████░░░████░░
 2000 │ ████████████
 1000 │ ████████████
  500 │ ████████████
  100 │ ████████████
      └──────────────── время
             ↑
         Нежелательный звук (например, кашель)
         выделить и удалить
```

### Инструменты Spectral Repair

| Инструмент | Применение |
|-----------|-----------|
| **Attenuate** | Уменьшить выбранную зону |
| **Replace** | Заменить зону интерполяцией из соседних областей |
| **Inpaint** | AI-заполнение — лучший алгоритм для реставрации |
| **Erase** | Полностью удалить выбранную зону |

### Практические задачи Spectral Repair

- **Удаление кашля, чихания, посторонних звуков** из вокальной записи
- **Восстановление записей** с пропавшими частотными областями
- **Удаление feedback (обратная связь)** — часто узкополосный пик
- **Удаление отдельных нот** в полифонической записи

## De-Hum (Удаление гула)

**Гул 50/60 Гц** — электросетевой фон от неправильного заземления или наводок.

### iZotope RX De-hum

Настройте частоту гула (50 Гц для Европы, 60 Гц для США) и количество гармоник. Алгоритм убирает основную частоту и её кратные.

### Ручной подход

Хирургический EQ с узкими notch-фильтрами на 50, 100, 150, 200 Гц (для европейского гула).

## Workflow шумоподавления

**Порядок обработки важен:**

1. **De-clip** (если есть клиппинг) — сначала
2. **De-hum** — убрать постоянный гул
3. **De-click** — убрать щелчки
4. **Noise Reduction** — убрать фоновый шум
5. **Spectral Repair** — точечное восстановление

Применяйте в этом порядке: каждый следующий этап работает с «более чистым» материалом.

> **Правило профессионала:** лучшее шумоподавление — правильная запись. Тихая комната, хорошие кабели, правильное заземление, выключенный кондиционер. Чистота при записи = экономия часов редактирования.
