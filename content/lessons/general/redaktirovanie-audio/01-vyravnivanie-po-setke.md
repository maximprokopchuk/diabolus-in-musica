# Выравнивание по сетке

## Зачем выравнивать аудио

Живые исполнители — не машины. Барабанщик немного торопится в третьем такте, гитарист чуть запаздывает с атакой, вокалист «тянет» начало фразы. В большинстве жанров **небольшие неточности — норма и признак живости**. Но иногда неточности мешают groove, создают ощущение «разболтанности» или просто звучат неаккуратно.

**Выравнивание по сетке** — редактирование аудио для исправления тайминга. Ключевой принцип: **слушайте ухом, не смотрите на экран**. Идеальная сетка на экране ≠ хорошая музыка.

## Transient Detection (Определение транзиентов)

**Транзиент** — острое начало атаки звука. Именно по транзиентам алгоритм определяет, «куда» сдвигать аудио.

```
Аудиоволна:
          │ ← транзиент (атака)
          ██
         ████
        ██████
       ████████
──────────────────── время
```

### Качество обнаружения транзиентов

| Материал | Чёткость транзиентов | Сложность |
|---------|---------------------|---------|
| Ударные (kick, snare) | Очень чёткие | Лёгко |
| Гитара (med attack) | Чёткие | Лёгко |
| Вокал | Размытые | Сложнее |
| Подушки (pads), смычковые | Плавные | Трудно |

Настройки sensitivity позволяют регулировать количество обнаруживаемых транзиентов:
- **Высокая sensitivity**: находит больше транзиентов, может ошибаться
- **Низкая sensitivity**: только самые выраженные атаки — надёжнее, но меньше точек

## Flex Audio (Logic Pro) / Elastic Audio (Pro Tools)

**Flex/Elastic Audio** — технология «резиновой» растяжки аудиосегментов: между двумя зафиксированными точками аудио сжимается или растягивается для выравнивания.

```
До:
  [транзиент]───────[следующий транзиент]
  Оригинал: 505 мс

После (выравнивание на сетку):
  [транзиент]──[следующий транзиент]
  Требуется: 500 мс → аудио «сжато» на 5 мс
```

### Алгоритмы растяжки

| Алгоритм | Применение |
|---------|-----------|
| **Polyphonic** | Полифонический материал (гитара, пианино, пады) |
| **Monophonic** | Монофонические источники (вокал, бас, духовые) |
| **Rhythmic** | Перкуссия и ритмические инструменты |
| **Tempophone** | Эффект «радиофонирования» — намеренный артефакт |

### Logic Pro: Flex Time

В Logic:
1. Включите Flex View (Cmd+F или кнопка в меню Edit)
2. Выберите алгоритм для трека
3. Кликните по транзиентным маркерам для управления
4. Перетащите маркер на нужную позицию — аудио растянется автоматически

### Pro Tools: Elastic Audio

1. Включите Elastic Audio на треке (иконка в заголовке)
2. Выберите алгоритм (Polyphonic, Rhythmic, etc.)
3. Используйте Elastic Audio плагин или Warp tool

## Quantize Audio

**Quantize** — автоматическое выравнивание транзиентов по музыкальной сетке.

### Параметры Quantize

| Параметр | Описание |
|---------|---------|
| **Grid** | Базовое деление: 1/4, 1/8, 1/16 нот |
| **Strength** | Сила коррекции: 100% = точно на сетку; 50% = половина пути |
| **Swing** | Добавление swing-грува к результату |
| **Range** | Только транзиенты в пределах ±N мс выравниваются |

**Strength < 100%** — ключевой параметр. 75–85% дают «затянутый, но живой» результат. 100% — механически, роботизированно.

### Groove Template (Groove Quantize)

Вместо прямой сетки можно использовать **groove template** — паттерн тайминга живого исполнителя или известного грувового материала:

- Возьмите барабанную петлю с «правильным» groove
- Извлеките из неё timing template
- Примените к другим трекам

Этот подход даёт ощущение «все инструменты играют вместе», даже если записаны по отдельности.

## Ручное редактирование тайминга

Иногда автоматика ошибается — тогда нужно ручное вмешательство.

### Алгоритм ручного выравнивания

1. Найдите проблемный транзиент
2. Разрежьте аудио **до и после** транзиента (создайте регион)
3. Сдвиньте регион на нужную позицию
4. Добавьте crossfade на стыках
5. Опционально: fade-in и fade-out на краях региона

### Инструменты

- **Ножницы (Scissors / Split)**: разрезать аудио
- **Arrow / Grabber**: двигать регионы
- **Crossfade Tool**: создавать переходы
- **Magnify / Zoom**: крупный план для точного позиционирования

## Когда НЕ надо выравнивать

- **Джаз, блюз**: «нечеловеческая» точность убивает характер
- **Народная музыка, этника**: отклонения — часть аутентичности
- **Live-записи**: концертное «дыхание» ценно
- **Намеренный swing/человечность**: если исполнитель «играет с грувом», не трогайте

> **Золотое правило:** выравнивайте только то, что **звучит плохо**, а не то, что **выглядит плохо** на экране. Если закрыть глаза и слушать — нет ли проблемы?
