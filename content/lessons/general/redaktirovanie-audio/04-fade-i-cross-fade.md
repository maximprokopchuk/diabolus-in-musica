# Fade и Cross-Fade

## Зачем нужны fade и crossfade

**Правило:** каждый аудиорегион в вашем проекте должен начинаться и заканчиваться **фейдом**, если не является специально подрезанной атакой.

Причина: цифровой аудиосигнал, обрезанный в произвольном месте, часто заканчивается не на нуле. Резкое «отключение» сигнала = **щелчок** (click) — короткий импульсный артефакт, ясно слышимый при воспроизведении.

```
БЕЗ FADE:
────────████████████╴  ← резкий срез → ЩЕЛЧОК

С FADE OUT:
────────████████████╲
                     ╲____  ← плавное затухание, нет щелчка
```

## Типы Fade

### Fade In (Нарастание)

Сигнал плавно нарастает от тишины до полного уровня. Используется:
- В начале песни (вступление из тишины)
- В начале отдельного инструментального вступления
- Чтобы «замаскировать» шум в начале региона

### Fade Out (Затухание)

Сигнал плавно затухает до тишины. Используется:
- В конце песни (один из самых распространённых способов завершения)
- В конце аудиорегиона для предотвращения щелчков
- При микшировании треков между собой

### Формы кривых fade

| Форма | Характер | Применение |
|------|---------|-----------|
| **Линейная (Linear)** | Равномерное изменение | Общее применение |
| **Косинусная (Cosine/S-Curve)** | Медленно-быстро-медленно | Мягкое, натуральное |
| **Логарифмическая** | Быстро в начале, медленно в конце | Вокал, органические инструменты |
| **Экспоненциальная** | Медленно, потом быстро | Перкуссия |

Ухо воспринимает **громкость логарифмически**: линейный фейд из 0% в 100% кажется «быстрым в начале». Косинусная или логарифмическая кривая дают более «музыкальный» результат для большинства материала.

## Cross-Fade

**Cross-fade** — одновременное затухание одного региона и нарастание другого. Используется на **стыках двух регионов** для плавного перехода.

```
Регион 1:  ─────────────────╲
                               ╲__________
Регион 2:          __________╱
                   ╱─────────────────────

Результат: плавный переход, нет щелчков, нет провала
```

### Длина кроссфейда

| Материал | Рекомендуемая длина |
|---------|-------------------|
| Вокальные фразы | 10–50 мс |
| Гитарные регионы | 5–20 мс |
| Барабаны (атакующие) | 2–10 мс |
| Пады, атмосферы | 100–500 мс |
| Переходы между секциями | 500–2000 мс |

Короче = менее заметный переход (хорошо для стыков «в такт»), длиннее = более «плавный» переход.

### Форма кроссфейда

**Equal Power (Равная мощность):**
- Оба сигнала на -3 dB в точке пересечения
- Суммарная мощность остаётся постоянной
- Лучший вариант для большинства материала

**Equal Gain (Равное усиление):**
- Оба сигнала на -6 dB в точке пересечения
- Может создать небольшой «провал» в громкости
- Иногда лучше для очень похожего материала

**Асимметричный кроссфейд:**
- Разная длина для входящего и исходящего сигнала
- Используется когда один регион важнее другого

## Практические применения

### 1. Компинг вокала

При сборке comp-трека из нескольких дублей:
1. Разрезаем в нужных местах
2. Выбираем лучшие фрагменты из разных дублей
3. **На каждом стыке ставим кроссфейд** — без исключений

Без кроссфейдов comp-трек будет полон щелчков.

### 2. Зацикленные петли (loops)

Если зацикливаете аудиофрагмент (loop):
- Конец петли должен плавно «соединяться» с началом
- Кроссфейд в точке зацикливания устраняет резкие переходы

### 3. Замена плохих слов

Вокалист в одном дубле произнёс «ладно» неправильно — берём его из другого дубля, вставляем, добавляем кроссфейды. Правильно сделанный «splice» (вставка) абсолютно неслышим.

### 4. Начало и конец треков

**Обязательно** добавляйте fade in и fade out:
- Fade in: несколько миллисекунд, маскирует любые щелчки входа
- Fade out: от 10 мс (просто «чистый» конец) до нескольких секунд (художественный «уход»)

## Автоматические кроссфейды в DAW

Большинство DAW предлагают опцию **«Auto Crossfade»** — при пересечении двух регионов автоматически создаётся кроссфейд стандартной длины.

**Logic Pro:** Edit → Auto Create Fades → задайте длину по умолчанию

**Pro Tools:** Timeline → Crossfade Preference → Preset length

**Reaper:** Preferences → Editing → Default crossfade length

Рекомендуемая длина автокроссфейда: 5–20 мс для большинства материала.

## Fade при мастеринге

**Финальный fade out** альбомного трека — отдельный художественный вопрос:

| Тип концовки | Применение |
|------------|-----------|
| **Абruptный стоп** | Эффектное завершение, сильный последний аккорд |
| **Fade out** | Ощущение «музыка продолжается», классика поп-музыки 60–80-х |
| **Fade to room** | Затихает только инструментальная часть, слышна комнатная атмосфера |

> **Ключевой принцип:** fade — не просто «уход в тишину». Это музыкальный элемент. Скорость, форма кривой и момент начала fade влияют на ощущение завершённости. Пробуйте разные варианты и слушайте в контексте альбома.
