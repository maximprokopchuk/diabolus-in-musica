# Лимитирование

## Что такое лимитер

**Лимитер** — компрессор с очень высоким ratio (∞:1), который **не позволяет сигналу превысить установленный Ceiling**. Это «аварийный стоп» для уровня.

В мастеринге лимитер выполняет две функции:
1. **Защищает** от клиппинга (ceiling = −1 dBTP)
2. **Поднимает** воспринимаемую громкость

## Ключевые параметры лимитера

### Ceiling (потолок)

**Ceiling** — максимальный уровень выхода. Сигнал НИКОГДА не превысит этот уровень.

**Рекомендуемые значения:**
- Для стриминга: **−1.0 dBTP** (True Peak)
- Для конвертации в MP3: **−1.5 до −2.0 dBTP** (кодеки немного поднимают уровень)
- Для CD: **−0.1 до −0.3 dBFS**

> Важно: используйте лимитеры, поддерживающие **True Peak** (ISP — Inter-Sample Peak) ограничение. Не все лимитеры это делают.

### Input Gain / Threshold

**Input Gain** (или Threshold) — насколько вы «вдавливаете» сигнал в лимитер. Это главная ручка управления **громкостью**.

- Поднимаете Input Gain → лимитер срабатывает сильнее → громкость растёт
- Опускаете → лимитер почти не работает → тихо, но динамично

**GR (Gain Reduction)** на лимитере: обычно **1–4 дБ** для «прозрачного» мастеринга. Более 6–8 дБ GR — слышны артефакты.

### Attack и Release

На лимитере attack почти всегда **очень быстрый** (менее 1 мс) — он должен поймать транзиент.

Release влияет на характер:
- **Слишком быстрый release:** pumping, пульсация
- **Слишком медленный:** «задавленные» транзиенты, потеря динамики
- **Auto release:** плагин сам подбирает release по контенту — рекомендуется

## Прозрачность vs Окрашенность

### Прозрачные лимитеры

Цель — **добавить громкости без изменения характера**. Не слышно, что сигнал лимитируется.

**Примеры:**
- **FabFilter Pro-L 2** — лучший прозрачный лимитер
- **Limitless (DMG Audio)** — продвинутое многополосное лимитирование
- **Youlean Limiter (бесплатный)** — хороший для начала

**Когда использовать:** большинство жанров, когда важно сохранить характер микса.

### Окрашивающие лимитеры

Добавляют **гармонические искажения** и «аналоговый» характер. Это часть их звука, а не недостаток.

**Примеры:**
- **Slate Digital FG-X** — «насыщение» в лимитере
- **Waves L3** — немного «окрашивает», классика
- **Sonnox Inflator** — добавляет воспринимаемую громкость без изменения peaks

**Когда использовать:** рок, метал, электронная музыка — жанры, где некоторое насыщение желательно.

## Алгоритмы лимитирования

### Brick Wall (кирпичная стена)

Сигнал жёстко обрезается на уровне Ceiling. Простой и надёжный. При сильном срабатывании — слышны искажения.

### ISP (Inter-Sample Peak) True Peak

Лимитер контролирует не только цифровые пики, но и **межсэмпловые пики**. FabFilter Pro-L 2, Intersample Peaks в Waves L3-LL Ultramaximizer и других. Обязателен для современных релизов.

### Transient-aware (осознающий транзиенты)

Некоторые лимитеры умеют «различать» транзиенты и тело звука, применяя разные стратегии к каждому. Это позволяет поднять громкость без потери «удара».

**FabFilter Pro-L 2** имеет несколько алгоритмов: Transparent, Aggressive, Bus, Dynamic — для разных задач.

## Многополосное лимитирование

**Multiband limiting** применяет отдельный лимитер на каждый частотный диапазон. Это позволяет:
- Контролировать бочку и бас независимо
- Не «задавить» весь микс из-за пика в одном диапазоне

**Пример:** бочка создаёт пик, который заставляет обычный лимитер давить весь микс. Multiband лимитер давит только НЧ диапазон.

**Плагины:** iZotope Ozone Maximizer, Limitless (DMG Audio), Waves L3 Multimaximizer.

## Артефакты перелимитирования

**Distortion (искажения):** при слишком большом GR (>8–10 дБ) лимитер начинает «жевать» звук — слышно искажение, особенно на транзиентах.

**Pumping:** быстрый release + высокий GR → ритмичная пульсация громкости.

**Потеря транзиентов:** «удар» барабанов пропадает, звук становится «мягким» и «ватным».

**Fatigue (усталость слуха):** сверхкомпрессированный мастер быстро «надоедает» уху — нет динамики для «дыхания».

## Практический процесс лимитирования

1. Установите Ceiling = −1.0 dBTP (True Peak)
2. Поднимайте Input Gain, слушая результат
3. Останавливайтесь когда:
   - GR не превышает 4–6 дБ на пиках
   - LUFS достигает −14 до −16 LUFS
   - Нет слышимых артефактов
4. Проверьте в сравнении с bypass (с выравниванием уровней!)
5. Проверьте True Peak — не превышает −1 dBTP?

> **Важно:** при A/B сравнении (с лимитером и без) всегда **выравнивайте уровни**. Более громкая версия воспринимается лучше — это психоакустический эффект. Сравните при одинаковом уровне, и вы объективно оцените работу лимитера.
