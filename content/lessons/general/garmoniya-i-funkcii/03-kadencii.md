# Каденции

## Что такое каденция

**Каденция** — гармонический оборот, завершающий музыкальную фразу. Это «знак препинания» в музыке.

## Основные виды каденций

### Автентическая каденция (V → I)

Самая сильная, «утвердительная» — как точка в конце предложения.

```
G → C     (совершенная — оба аккорда в основном виде)
G7 → C    (ещё сильнее — с септимой)
```

**Совершенная автентическая**: V(7) → I, оба в основном положении, тоника в верхнем голосе мелодии.
**Несовершенная**: если одно из условий не выполнено.

### Плагальная каденция (IV → I)

Мягкая, «торжественная» — «аминь»-каденция (используется в хоралах).

```
F → C     (IV → I)
```

### Половинная каденция (→ V)

Фраза **останавливается на доминанте** — ощущение вопроса, незавершённости.

```
C → G     (I → V)
Dm → G    (ii → V)
F → G     (IV → V)
```

### Прерванная (обманчивая) каденция (V → vi)

Доминанта разрешается **не в тонику**, а в vi — неожиданный поворот.

```
G → Am    (вместо G → C)
```

Эффект «обманутого ожидания» — мы ждали C, а получили Am.

## Каденции в поп-музыке

| Каденция | Пример прогрессии | Ощущение |
|----------|-------------------|----------|
| Автентическая | ...G → C | Завершённость |
| Плагальная | ...F → C | Мягкое завершение |
| Прерванная | ...G → Am | Продолжение, сюрприз |
| Половинная | ...C → G (конец куплета) | «А дальше?» |

> **Прерванная каденция** — один из самых мощных приёмов: слушатель ожидает разрешение, но вместо I получает vi. Это создаёт эмоциональный эффект, который используется повсеместно.
