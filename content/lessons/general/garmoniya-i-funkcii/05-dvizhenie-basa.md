# Движение баса и голосоведение

## Voice leading (голосоведение)

**Voice leading** — искусство плавного перехода от одного аккорда к другому. Каждый голос (нота) в аккорде движется к ближайшей ноте следующего аккорда.

## Правило кратчайшего пути

Основной принцип: **каждый голос должен двигаться на минимальное расстояние** (желательно на полтона или тон, максимум — на терцию).

Пример хорошего голосоведения:
```
C:    E  G  C         Dm:   F  A  D
      ↓  ↓  ↓               ↓  ↓  ↓
      E→F(+1) G→A(+2) C→D(+2)
```

Все голоса движутся на 1–2 ступени — плавно.

## Типы движения баса

### Поступенное движение

Бас движется по ступеням гаммы — самое мелодичное:
```
C → Dm/C → Em/B → Am/A → ...
Бас: C — C — B — A (нисходящий)
```

### Квартовое/квинтовое движение

Бас прыгает на кварту вверх (= квинту вниз) — самое «гармоничное»:
```
C → F → Bdim → Em → Am → Dm → G → C
Бас: C—F—B—E—A—D—G—C (квартовый круг)
```

### Хроматическое движение

Бас движется по полутонам:
```
C → C/B → Am → Am/G# → Am/G → D/F# → F → G
Бас: C—B—A—G#—G—F#—F—G
```

## Общие тоны

При смене аккордов **общие ноты** остаются на месте:

| Переход | Общие ноты |
|---------|------------|
| C → Am | C, E |
| C → Em | E, G |
| C → F | C |
| C → G | G |
| Dm → G | D (в воицинге G) |

## На гитаре

Хорошее голосоведение на гитаре — залог «профессионального» звучания аккомпанемента:

1. Используйте **обращения** для плавного движения баса
2. Держите **общие ноты** между аккордами (не переставляйте всю руку)
3. Играйте **внутренние голоса** — не только бас и верхнюю ноту

> **Упражнение:** возьмите прогрессию C — Am — F — G и найдите воицинги, где между аккордами меняется минимум нот. Начните с одной позиции на грифе.
