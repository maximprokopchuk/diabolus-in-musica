# Базовый интерфейс DAW

## Общая архитектура интерфейса

Несмотря на визуальные различия, все DAW имеют **одинаковые функциональные зоны**. Понимание этих зон позволяет быстро ориентироваться в любой программе.

## Arrange View (Timeline / Song View)

**Arrange View** — главное рабочее пространство. Здесь видна вся структура песни во времени.

```
[Временная шкала (такты/секунды)]
┌──────────────────────────────────────────┐
│ Track 1 (Guitar)  │▓▓▓░░░▓▓▓▓▓░░░░░░░░│
│ Track 2 (Bass)    │▓▓▓▓▓▓░░░░▓▓▓▓▓▓▓▓│
│ Track 3 (Drums)   │▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
│ Track 4 (Vocals)  │░░░░░░▓▓▓▓▓▓░░░░░░│
└──────────────────────────────────────────┘
```

### Элементы Arrange View

**Транспорт** — панель управления воспроизведением:
- Play / Stop / Record
- Tempo (BPM) и размер (4/4, 3/4...)
- Метроном (click track)
- Позиция воспроизведения

**Область треков** — вертикальный список дорожек, каждая с:
- Названием
- Кнопками Mute (M) и Solo (S)
- Регуляторами громкости и панорамы
- Кнопкой записи (R / Arm)
- Иконкой цвета трека

**Регионы (клипы)** — прямоугольники на треках, представляющие аудиофайлы или MIDI-данные. Их можно:
- Перемещать (drag)
- Копировать (Alt/Opt + drag)
- Обрезать (trim с краёв)
- Зациклить (loop)
- Разрезать (Split, обычно S или Cmd+E)

## Mixer (Микшер)

**Mixer** — виртуальный аналог аналогового микшерного пульта. Каждый трек = один канальный стрип.

### Анатомия канального стрипа

```
         [INSERT EFFECTS]
              │
         [EQ]
              │
         [DYNAMICS]
              │
         [SENDS →]─────→ [Bus/Aux]
              │
         [FADER (громкость)]
              │
         [PAN (панорама)]
              │
         [METER (уровень)]
              │
         [MUTE / SOLO]
```

**Inserts (вставки)** — плагины, обрабатывающие весь сигнал трека последовательно: EQ, компрессор, сатурация.

**Sends (посылы)** — параллельная отправка сигнала на шину обработки: ревербератор, задержка. Позволяет одним ревербом обрабатывать несколько треков.

**Фейдер** — основной регулятор громкости. В большинстве DAW 0 dB = единица (unity gain).

**Pan** — панорамирование: положение звука в стереополе от Left до Right.

**Meter** — индикатор уровня сигнала. Следите, чтобы не было **клиппинга** (красный индикатор выше 0 dBFS).

## Piano Roll (MIDI-редактор)

**Piano Roll** — редактор MIDI-данных. Горизонтальная ось — время, вертикальная — высота нот (как клавиатура пианино, повёрнутая набок).

```
C5 │     ██         ██
B4 │  ██         ██
A4 │
G4 │        ████
F4 │██              ██
E4 │
D4 │
C4 │                    ████
   └────────────────────────── время →
```

### Основные операции в Piano Roll

| Действие | Инструмент |
|---------|-----------|
| Нарисовать ноту | Draw / Pencil |
| Удалить ноту | Правый клик / Eraser |
| Выделить и двигать | Selection / Arrow |
| Изменить длину ноты | Тянуть за правый край |
| Квантайз | Q (выровнять по сетке) |

**Velocity** — сила нажатия MIDI-ноты. Отображается столбиками внизу Piano Roll. Влияет на динамику: чем выше velocity, тем громче нота у большинства виртуальных инструментов.

**Velocity Ramp** — постепенное нарастание или спад velocity для создания крещендо/диминуэндо.

## Browser (Браузер)

**Browser** — проводник для поиска и загрузки ресурсов:

- **Пресеты плагинов** (наборы настроек)
- **Звуковые библиотеки** (сэмплы, петли)
- **Виртуальные инструменты**
- **Проекты** (шаблоны, предыдущие сессии)

Большинство DAW поддерживают **перетаскивание (drag & drop)** из браузера прямо на трек.

## Дополнительные окна

| Окно | Назначение |
|-----|-----------|
| **Tempo Map** | Автоматизация темпа (ускорение, замедление) |
| **Score Editor** | Нотный редактор (MIDI → ноты) |
| **Clip Launcher** | Нелинейный запуск клипов (Ableton Session View) |
| **Plugin Manager** | Управление установленными плагинами |
| **Marker Track** | Метки на временной шкале (куплет, припев, бридж) |

> **Совет:** освойте навигацию без мыши — только клавиатурой и тачпадом. Это ускоряет работу в 2–3 раза. Начните с изучения горячих клавиш транспорта (пробел, R, Cmd+Z) и базовых операций с регионами.
