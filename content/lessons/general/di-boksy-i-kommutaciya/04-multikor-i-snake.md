# Мультикор и snake

## Что такое мультикор

**Мультикор** (multicore) или **snake** — кабель, содержащий несколько (обычно 8–56) сигнальных пар в одном защитном кожухе. Используется для передачи многих каналов аудио между сценой и FOH-позицией.

Вместо 24 отдельных XLR-кабелей — один мультикор. Это удобно, надёжно и значительно сокращает время монтажа/демонтажа.

## Аналоговый мультикор

### Конструкция

Аналоговый snake состоит из:
- N пар витых проводников (балансные каналы) — каждая пара в отдельном экране
- Внешний экран для всего кабеля
- Прочный ПВХ или полиуретановый кожух

### Форматы

| Формат | Описание |
|-------|---------|
| **Stage box** | Разъёмная коробка с XLR-гнёздами на сцене → кабель → Fan-out (пучок) на FOH |
| **Y-snake** | Один кабель с fan-out на обоих концах |
| **Рэковый snake** | Stage box встраивается в рэк |

**Стандартная маркировка:** 24/4 означает 24 канала со сцены + 4 канала возврата (для мониторных миксов).

### Потери сигнала

Аналоговый сигнал в кабеле теряет качество из-за:
- **Ёмкость кабеля** — срезает высокие частоты (особенно при высоком источнике импеданса)
- **Сопротивление** — ослабляет уровень
- **Взаимная индуктивность** — перекрёстные помехи между каналами (crosstalk)

**Практические ограничения аналогового snake:**
- До 50 метров — без значимых потерь при профессиональном кабеле
- До 100 метров — допустимо при хорошем экранировании и правильном импедансе
- Более 100 метров — риск деградации; используйте цифровой snake

### Расчёт потерь высоких частот

Ёмкость кабеля и импеданс источника образуют **RC-фильтр** низких частот. Формула частоты среза:

```
fc = 1 / (2π × R × C)

R = сопротивление источника (Ом)
C = ёмкость кабеля (Ф) = ёмкость на метр × длина
```

Для микрофона с выходным импедансом 150 Ом и кабеля 100 пФ/м длиной 50 м:
C = 100 × 50 = 5000 пФ = 5 нФ
fc = 1 / (2π × 150 × 5×10⁻⁹) ≈ **212 кГц** — потери не слышимы

Для небалансного источника 10 кОм и того же кабеля:
fc ≈ **3,2 кГц** — уже заметная потеря верхов

## Цифровой snake

**Цифровой snake** — система передачи многоканального аудио в цифровом формате по одному кабелю (Ethernet, коаксиал или оптоволокно).

### Преимущества цифрового snake

| Параметр | Аналоговый | Цифровой |
|---------|-----------|---------|
| Кабель | Толстый мультикор | Один Ethernet CAT5e/6 |
| Максимальная длина | 50–100 м | 100 м (Ethernet), 1+ км (оптика) |
| Количество каналов | До 56 | 32–64+ |
| Задержка | Нет | 0,5–2 мс |
| Потери сигнала | Возможны | Нет (цифровой сигнал) |
| Развёртывание | Медленнее | Быстрее |

### Популярные форматы цифрового snake

| Формат | Производитель | Интерфейс |
|-------|-------------|---------|
| **AES50 (ULTRANET)** | Klark-Teknik/Behringer | CAT5e, 48 каналов |
| **Dante** | Audinate | Ethernet, до 1024 каналов |
| **AVB** | IEEE стандарт | Ethernet |
| **MADI** | AES10 | Коаксиал или оптика, 64 канала |
| **SoundGrid** | Waves | Ethernet |

**Dante** — наиболее универсальный и распространённый протокол в профессиональном звуке. Работает через стандартный гигабитный Ethernet-коммутатор.

### Практическое развёртывание цифрового snake

1. Подключить stage box к гигабитному switch (или напрямую к FOH-микшеру)
2. Убедиться, что switch поддерживает QoS (приоритизация аудиотрафика)
3. Настроить routing в программном обеспечении (Dante Controller для Dante)
4. Проверить задержку: для концертного применения должна быть < 1–2 мс

> **Рекомендация:** для постоянной инсталляции в клубе или театре — Dante, для туристического оборудования — AES50 (Behringer S32/S16 дёшевы и надёжны). Аналоговый snake остаётся актуальным для простых систем и бюджетных применений.
