# Инструменты для снятия

## Зачем нужны специальные инструменты

Снимать с обычным плеером крайне неудобно: нет возможности зациклить фрагмент, замедление большинства плееров искажает высоту. Специальные инструменты для транскрипции решают эти проблемы.

## Ключевые возможности

| Функция | Зачем нужна |
|---------|-------------|
| **Замедление без изменения высоты** | Слышать быстрые пассажи в деталях |
| **Зацикливание (A-B loop)** | Работать с одним фрагментом много раз подряд |
| **Изменение высоты без изменения темпа** | Транспонировать в удобную тональность |
| **EQ и изоляция частот** | Выделить гитару или бас из микса |
| **Замедление до 25–50%** | Снимать очень быстрые партии |

## Основные программы

### Transcribe! (Seventh String Software)

**Платформы:** Windows, macOS, Linux
**Цена:** ~$39 (разовая лицензия)

Профессиональный стандарт среди музыкантов для транскрипции. Возможности:
- Замедление с шагом 1%
- Спектральный анализ (визуализация нот)
- Петли, закладки, аннотации
- Анализ аккордов
- Экспорт в нотную запись (интеграция с Sibelius/Finale)

**Уникальная функция:** спектральный «piano roll» — вы видите на экране, какие ноты звучат в каждый момент времени. Это не заменяет слух, но существенно помогает при сложных созвучиях.

### Amazing Slow Downer

**Платформы:** iOS, Android, macOS, Windows
**Цена:** ~$20–30

Мобильный стандарт:
- Интуитивный интерфейс
- Работает с файлами и стриминговыми сервисами (через запись)
- Замедление от 25% до 200%
- Изменение тональности (-6 до +6 полутонов)
- A-B петли с высокой точностью

Идеален для работы с телефона или планшета во время игры на инструменте.

### Capo (macOS / iOS)

**Платформы:** macOS, iOS
**Цена:** ~$15

Отличается функцией **автоматического определения аккордов** (chord detection). Не всегда точно, но даёт отправную точку для анализа гармонии. Хорошо работает с записями с чётко слышной гармонией.

### Слаш-подход: YouTube + расширения

**YouTube по умолчанию** не позволяет замедлять ниже 0.25x. Для снятия это недостаточно. Расширения для браузера:

- **YouTube Speed Controller** — расширение Chrome/Firefox, позволяет замедлять с шагом 0.05x
- **Video Speed Controller** — аналог с более широкими настройками

**Недостаток:** YouTube компрессирует аудио, качество хуже, чем у оригинального файла.

### Moises

**Платформы:** iOS, Android, web
**Цена:** freemium

Революционный инструмент: **разделение треков на стемы** (вокал, гитара, бас, барабаны) с помощью ИИ. Качество не студийное, но часто достаточное для транскрипции. Позволяет выделить конкретный инструмент и слушать его отдельно.

Аналоги: **Lalal.ai**, **Spleeter** (open source).

## Практическая настройка рабочего процесса

### Для начинающих

```
1. Найдите трек в хорошем качестве (FLAC, WAV или 320 kbps MP3)
2. Загрузите в Amazing Slow Downer (телефон рядом с инструментом)
3. Выделите 2–4 такта, поставьте на цикл
4. Замедлите до 60–70% — не слишком медленно, иначе теряется характер
5. Слушайте, пойте, играйте — без записи нот на первом этапе
```

### Для среднего уровня

```
1. Transcribe! на компьютере, файл в хорошем качестве
2. Используйте спектральный анализ для проверки нот
3. Делайте аннотации прямо в программе
4. Для аккордов — параллельно Capo для первичной разметки
```

## Замедление: сколько достаточно

| Уровень сложности | Рекомендуемое замедление |
|-------------------|--------------------------|
| Медленный блюз, народная музыка | 80–100% (без замедления) |
| Стандартный рок/поп | 70–80% |
| Быстрое соло, bebop | 50–60% |
| Очень быстрый шред, Bird-style | 30–40% |

Не замедляйте ниже необходимого: слишком медленная скорость теряет характер исполнения — вы снимаете ноты, но не фразировку.

> **Инструменты не снимают за вас — они убирают технические барьеры.** Amazing Slow Downer или Transcribe! дают вам время разобрать быстрый пассаж, но слух и понимание развиваете только вы. Начните с простого материала в хорошем качестве — и инструменты станут незаменимыми союзниками.
