# Квинтовый круг

## Что такое квинтовый круг

**Квинтовый круг** (circle of fifths) — диаграмма, показывающая все 12 мажорных и 12 минорных тональностей, расположенных по квинтам.

По часовой стрелке — каждая следующая тональность на **квинту выше** (добавляется 1 диез):

C → G → D → A → E → B → F# (Gb) → Db → Ab → Eb → Bb → F → C

## Структура круга

```
          C (0)
     F (1♭)    G (1♯)
   B♭ (2♭)      D (2♯)
  E♭ (3♭)        A (3♯)
   A♭ (4♭)      E (4♯)
     D♭ (5♭)  B (5♯)
        G♭/F♯ (6♭/6♯)
```

Внутренний круг — параллельные минорные тональности (Am, Em, Bm...).

## Практическое использование

### 1. Определение знаков при ключе

Двигаясь по часовой стрелке от C: каждый шаг = +1 диез.
Против часовой: каждый шаг = +1 бемоль.

### 2. Подбор аккордов

Соседние тональности в круге дают аккорды, которые хорошо звучат вместе. В любой мажорной тональности основные аккорды расположены рядом в квинтовом круге:

Для C мажор: **F — C — G** (субдоминанта — тоника — доминанта)

### 3. Транспонирование

Чтобы транспонировать песню, достаточно повернуть все аккорды на одинаковое количество шагов по кругу.

### 4. Модуляция

Ближайшие тональности в круге — наиболее естественные цели для модуляции.

> **Запомните квинтовый круг** — это «карта» всей тональной музыки. Гитаристы часто вешают его на стену в комнате для занятий.
