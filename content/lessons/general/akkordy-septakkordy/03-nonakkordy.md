# Нонаккорды

## Нона — что это

**Нона** — 9-я ступень, то есть секунда на октаву выше. Добавление ноны к аккорду создаёт богатое, «открытое» звучание.

## Типы нонаккордов

| Обозначение | Структура | Пример от C |
|-------------|-----------|-------------|
| **Cadd9** | Трезвучие + 9 (без 7) | C—E—G—D |
| **C9** | C7 + 9 | C—E—G—Bb—D |
| **Cmaj9** | Cmaj7 + 9 | C—E—G—B—D |
| **Cm9** | Cm7 + 9 | C—Eb—G—Bb—D |
| **Cm(add9)** | Мин. трезвучие + 9 | C—Eb—G—D |

## Различие add9 и 9

Это важный нюанс:
- **Cadd9** = C + E + G + **D** (нет септимы, добавлена только нона)
- **C9** = C + E + G + **Bb** + **D** (есть доминантсептима + нона)

```
add9 = трезвучие + нона
9    = септаккорд + нона
```

## Звучание

- **add9** — свежий, поп/рок звук (The Police, Oasis)
- **9** — фанковый, блюзовый (Jimi Hendrix, James Brown)
- **maj9** — мечтательный, джазовый, утончённый
- **m9** — тёплый, глубокий, неоклассический соул

## На гитаре

```
Cadd9       G(add9)      A9 (E-форма)
e|--0--|   e|--3--|     e|--x--|
B|--3--|   B|--3--|     B|--5--|
G|--0--|   G|--2--|     G|--6--|
D|--2--|   D|--0--|     D|--5--|
A|--3--|   A|--2--|     A|--x--|
E|--x--|   E|--3--|     E|--5--|
```

Знаменитый «Hendrix chord» — E7#9 — это нонаккорд с альтерированной (повышенной) ноной.

> **Совет:** замените обычные C и G на Cadd9 и G(add9) в любой поп-песне — звучание станет заметно интереснее при минимальном изменении аппликатуры.
