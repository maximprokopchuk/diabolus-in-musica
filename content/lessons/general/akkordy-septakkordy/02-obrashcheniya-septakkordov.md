# Обращения септаккордов

## Четыре позиции

У септаккорда 4 звука — значит, **4 возможных позиции** (какая нота в басу):

| Позиция | Бас | Название | Slash-запись |
|---------|-----|----------|-------------|
| Основной вид | Корень (1) | Септаккорд | C7 |
| 1-е обращение | Терция (3) | Квинтсекстаккорд | C7/E |
| 2-е обращение | Квинта (5) | Терцквартаккорд | C7/G |
| 3-е обращение | Септима (7) | Секундаккорд | C7/Bb |

## Пример: G7 (G—B—D—F)

| Обращение | Ноты снизу вверх | Бас |
|-----------|-----------------|-----|
| Основной | G — B — D — F | G |
| 1-е | B — D — F — G | B |
| 2-е | D — F — G — B | D |
| 3-е | F — G — B — D | F |

## Практическое применение

### Плавное голосоведение

Обращения позволяют минимизировать «скачки» между аккордами:

```
Dm7 → G7 → Cmaj7
(без обращений — большие скачки баса)

Dm7 → G7/B → Cmaj7
(бас движется: D → B → C — плавно!)
```

### Нисходящий бас

Классический приём — хроматическое или поступенное движение баса через обращения:

```
C → C/B → C/Bb → C/A → ...
или
Am → Am/G → Am/F# → Am/F → ...
```

## На гитаре

На гитаре обращения септаккордов часто играются с пропуском некоторых нот (из 4 нот оставляют 3, дублируя нужные):

```
G7/B:        G7/F:
e|--1--|    e|--1--|
B|--0--|    B|--0--|
G|--0--|    G|--0--|
D|--0--|    D|--0--|
A|--2--|    A|--x--|
E|--x--|    E|--1--|
```

> **Совет:** для джазового аккомпанемента достаточно играть 3–4 струны. Не обязательно задействовать все 6 — компактные воицинги звучат чище.
