# Структура канала микшера

## Что такое канал микшера

**Канал** (channel strip) — базовый строительный блок любого микшера. Каждый инструмент или микрофон занимает один или несколько каналов. Все каналы имеют одинаковую структуру — набор обработок, применяемых последовательно к сигналу.

## Сигнальный путь канала

```
Вход (XLR/Jack) → Gain → HPF/LPF → Insert → EQ → Aux Sends → Pan → Fader → Выход на шину
```

## Gain (Предусиление)

**Gain** — первый элемент канала. Устанавливает **входной уровень** сигнала, приходящего с микрофона или инструмента.

**Задача gain:** привести сигнал к оптимальному рабочему уровню — достаточно громко для хорошего соотношения сигнал/шум, но не настолько, чтобы перегрузить последующие каскады.

### Правильная установка gain

1. Попросите источник воспроизвести типичный уровень сигнала (не максимум)
2. Поднимайте gain до тех пор, пока пик-индикатор не будет показывать −18 до −12 дБFS
3. Оставьте headroom для динамических пиков

**Ошибки:**
- Слишком малый gain → шум преампа слышен
- Слишком большой gain → перегруз, клиппинг, искажения

### Trim vs Gain

На многих пультах есть **два** уровневых контроля:
- **Gain** (или Mic Gain) — аналоговый, до АЦП
- **Trim** — цифровой, после АЦП

Всегда сначала устанавливайте аналоговый Gain.

## Фильтры HPF/LPF

**High-Pass Filter (HPF)** — срезает частоты **ниже** заданной точки. Используется практически на каждом канале, кроме бочки и баса.

| Инструмент | Рекомендуемый HPF |
|----------|------------------|
| Вокал | 80–120 Гц |
| Гитара | 100–150 Гц |
| Клавишные (мид-верх) | 200–300 Гц |
| Бас | Не нужен или 30–40 Гц |
| Kick drum | Не нужен |

**LPF** (Low-Pass Filter) — срезает выше заданной точки. Реже используется, но полезен для суббаса или специальных эффектов.

## Insert (Разрыв)

**Insert** — точка в сигнальном пути, куда можно **вставить внешнее устройство обработки** (компрессор, эквалайзер, гейт).

Физически — разъём TRS (tip-ring-sleeve), где:
- Tip — Send (выход из канала в внешнее устройство)
- Ring — Return (вход обратно из устройства в канал)

```
Канал → [INSERT SEND] → [Внешний компрессор] → [INSERT RETURN] → Канал (продолжение)
```

Сигнал проходит **последовательно** через вставленное устройство.

**Применение:** динамическая обработка (компрессоры, гейты, эквалайзеры) на отдельных каналах.

## EQ (Эквалайзер)

Большинство микшерных каналов имеют встроенный **параметрический или полупараметрический EQ**.

### Типичные полосы

| Полоса | Тип | Типичный диапазон |
|-------|-----|-----------------|
| **Low (Bass)** | Полочный (shelf) | Центр 80–100 Гц |
| **Low-Mid** | Параметрический (bell) | 250–500 Гц (мутность) |
| **High-Mid** | Параметрический (bell) | 2–5 кГц (присутствие) |
| **High (Treble)** | Полочный (shelf) | Центр 10–12 кГц |

### Параметрический EQ

Регулировки:
- **Frequency** — центральная частота полосы
- **Gain** — подъём/срез в дБ
- **Q** (bandwidth) — ширина полосы (узкая = хирургическая, широкая = мягкая)

## Aux Sends (Вспомогательные посылы)

**Aux Send** — отвод сигнала с канала на **вспомогательную шину**. Независимо от основного фейдера канала.

**Применение:**
1. **Эффекты (FX bus)** — посылаем сигнал на шину реверба, задержки
2. **Мониторный микс (Cue mix)** — персональный микс для музыканта в наушниках
3. **Запись** — отдельная запись mix

### Pre-Fader vs Post-Fader

| Тип | Зависит от фейдера | Применение |
|-----|-------------------|-----------|
| **Pre-Fader** | Нет (до фейдера) | Мониторный микс (музыкант должен слышать себя, даже если канал в миксе приглушён) |
| **Post-Fader** | Да (после фейдера) | Шины эффектов (убираешь канал — убирается и реверб) |

## Pan (Панорама)

**Pan** управляет расположением сигнала в стереополе — от полного левого (L) до полного правого (R).

**Стандартная расстановка:**
- Вокал — центр
- Бас — центр
- Kick drum — центр
- Снэр — центр или чуть влево
- Гитара 1 — L 30–50%
- Гитара 2 — R 30–50%
- Клавишные — широко L/R

## Fader (Фейдер)

**Фейдер** — основная регулировка громкости канала. Управляет уровнем, поступающим на основную шину (master).

- **Стандартное рабочее положение** — «0 дБ» (единичное усиление, unity gain, обычно маркировано «U» или линией)
- Поднять выше 0 дБ — усилить
- Опустить — убрать

### VU vs Peak метры

| Тип | Характеристика |
|-----|---------------|
| **VU (Volume Unit)** | Медленное время отклика (~300 мс), показывает «воспринимаемую» громкость |
| **Peak** | Мгновенный отклик, показывает абсолютные пики |
| **PPM (Programme Peak Meter)** | Компромисс, стандарт broadcast |

## Solo и Mute

| Кнопка | Действие |
|--------|---------|
| **Mute** | Отключает канал от основной шины |
| **Solo** | Слушать только этот канал (или несколько) |
| **Solo In Place (SIP)** | Соло в контексте панорамы |
| **AFL (After Fader Listen)** | Соло с учётом фейдера и панорамы |
| **PFL (Pre Fader Listen)** | Соло до фейдера (для установки gain) |

> **Итог:** понимание структуры канала — фундамент работы со звуком. Каждый элемент — инструмент, который нужно использовать осознанно. Хорошее сведение начинается с правильно выставленного gain на каждом канале.
