# Группы и матрица

## Зачем нужны группы

В большой сессии (концерт, оркестр, запись с полным составом) могут быть десятки каналов. Управлять каждым по отдельности — нереально. **Группировка** позволяет объединить несколько каналов и управлять ими как одним.

## Sub-Groups (Подгруппы)

**Sub-group** — объединение нескольких каналов на одну общую шину. Фейдер группы контролирует общий уровень всех входящих каналов.

```
Барабанные каналы (kick, snare, hh, toms, oh):
  Канал 1 (Kick)   ──┐
  Канал 2 (Snare)  ──┤
  Канал 3 (Hi-hat) ──┼──→ [GROUP «Drums» Fader] → Master Bus
  Канал 4 (Tom 1)  ──┤
  Канал 5 (Tom 2)  ──┘
```

**Преимущества:**
- Одним фейдером убрать или добавить **всю барабанную установку**
- Применить **вставной компрессор** на всю группу (компрессия барабанной шины)
- Управлять балансом больших секций (струнные, медные, хор)

### Типичные группы

| Группа | Содержимое |
|-------|-----------|
| Drums | Kick, snare, hats, toms, overheads, rooms |
| Bass | Бас-гитара, kick (для связи) |
| Guitars | Все гитарные каналы |
| Keys | Клавишные, синтезаторы |
| Vocals | Вокал, бэк-вокал |
| Strings | Секция струнных |

## VCA (Voltage Controlled Amplifier) группы

**VCA-группа** — принципиально иной способ группировки. VCA не является физической шиной — он **управляет усилением** (gain) нескольких каналов одновременно через управляющее напряжение.

| Параметр | Sub-group | VCA |
|---------|----------|-----|
| Физическая шина | Да | Нет |
| Маршрутизация | Канал → Group → Master | Канал → Master напрямую |
| Insert обработка | Да (общий компрессор) | Нет |
| Точность управления | Фейдер группы | VCA-фейдер управляет усилением каждого канала |
| Aux sends | Берутся из group bus | Берутся из оригинального канала |

**Преимущества VCA:**
- Aux-посылы каждого канала остаются независимыми (важно для мониторинга)
- Более гибкое управление при сложных маршрутизациях
- Стандарт в больших живых пультах (Digico, SSL, Yamaha CL/QL)

## Master Bus (Главная шина)

**Master Bus** (или Main Mix) — главная шина, куда приходят все каналы (или группы). Фейдер Master контролирует общий уровень выходного сигнала.

```
Все каналы/группы → [MASTER BUS] → [Master Fader] → Выход на усилители/запись
```

На Master Bus часто устанавливают **мастер-компрессор** (bus compressor) — он склеивает микс воедино, придаёт ему «единый» звуковой образ.

**Популярные мастер-компрессоры:**
- SSL G-Bus Compressor (встроен в SSL консоли)
- API 2500
- Neve 33609
- Manley Variable Mu

## Matrix Mix

**Matrix** — дополнительная гибкая система маршрутизации, позволяющая направлять любую шину на любой выход с независимой регулировкой уровня.

```
Шины/Группы:   Drums | Bass | Guitars | Vocals | Master
                 ↓      ↓       ↓        ↓        ↓
Matrix Out 1:  [x]    [ ]     [ ]      [ ]      [x]   → FOH Speakers
Matrix Out 2:  [x]    [x]     [ ]      [x]      [ ]   → Stage Monitor L
Matrix Out 3:  [ ]    [ ]     [x]      [x]      [ ]   → Stage Monitor R
Matrix Out 4:  [ ]    [ ]     [ ]      [ ]      [x]   → Recording System
Matrix Out 5:  [x]    [x]     [x]      [x]      [ ]   → Broadcast Feed
```

**Применение:**
- **FOH (Front of House)** — основной звук для зала
- **Мониторные миксы** — индивидуально для каждого музыканта
- **Broadcast feed** — отдельная подача для записи или трансляции
- **Сцена + ложи + фойе** — разные миксы для разных зон

### Matrix vs Aux

| | Aux | Matrix |
|-|-----|-------|
| Источники | Отдельные каналы | Шины / группы |
| Гибкость | Канальная | Шинная |
| Применение | Мониторы, FX | Многозонные системы, broadcast |

## Практический пример: живой концерт

```
[Каналы]           [Sub-Groups]    [Master]    [Matrix Outs]
Kick ──────┐
Snare ──────┤
Toms ──────┼→ [DRUMS] ──┐                    Matrix 1 → FOH L/R
OH ─────────┘           ├→ [MASTER] → FOH   Matrix 2 → Stage Mon 1
Bass ──────────────────→┤  Fader            Matrix 3 → Stage Mon 2
Guitar L/R ────────────→┤                   Matrix 4 → Broadcast
Vox ────────────────────┘                   Matrix 5 → Recording
```

> **Главное:** группировка — это организация. Чем сложнее сессия, тем важнее продуманная группировка. Начните с простых групп (drums, bass, guitars, vox) — это уже огромный шаг к удобному управлению.
