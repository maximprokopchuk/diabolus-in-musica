# Insert и Direct Out

## Insert — последовательная обработка

**Insert** — разрыв в сигнальном пути канала, позволяющий **последовательно** включить внешнее устройство обработки.

### Физический разъём

На аналоговых пультах Insert реализован через разъём **TRS (6.35 мм)**:

```
Tip   = Send   (выход из канала)
Ring  = Return (вход обратно)
Sleeve = GND

Кабель-разветвитель TRS → 2×TS:
  TRS [Insert] → TS (Send) → Вход компрессора
                TS (Return) ← Выход компрессора
```

### Что подключают через Insert

| Устройство | Канал / шина |
|----------|-------------|
| **Компрессор** | Вокал, бас, buss drums |
| **Гейт** | Барабаны (kick, snare, toms) |
| **De-esser** | Вокал (борьба с сибилянтами «С», «Щ») |
| **EQ** | Высококачественный параметрик |
| **Channel strip** (Neve 1073 и т.д.) | Специальный характер |
| **Мастер-компрессор** | Master bus insert |

### Нюансы аналогового Insert

- Длина кабелей Insert влияет на звук (высокоимпедансный сигнал)
- Плохой разъём = потеря сигнала или шум
- Подключение «пустышки» (normalled) — если Insert не используется, сигнал идёт без разрыва

## Direct Out (Прямой выход)

**Direct Out** — прямой выход сигнала с канала (обычно после gain и HPF, но до фейдера), направленный на внешнее устройство независимо от основного микса.

```
Канал → [Gain] → [HPF] → [Direct Out] ──→ Рекордер, Interface, Stage Box
                              ↓
                         Основной сигнальный путь продолжается...
                         [EQ] → [Fader] → [Master Bus]
```

### Применение Direct Out

1. **Многодорожечная запись** — каждый канал пишется на отдельную дорожку DAW, независимо от FOH-микса
2. **Backup запись** — параллельная запись на SD-рекордер
3. **Stage Box / Split** — деление сигнала между FOH и мониторными пультами
4. **Передача в другой пульт** — например, broadcast-миксер получает сигналы с FOH-пульта

### Pre-Fader Direct Out

**Pre-fader Direct Out** особенно полезен при **записи живого выступления**:
- Звукорежиссёр делает изменения в FOH-миксе (убирает каналы, изменяет уровни)
- Запись при этом получает **неизменённый** чистый сигнал каждого канала
- В студии можно свести «живой» концерт заново

## Параллельная обработка через Hardware Insert

**Параллельная обработка** на аппаратных пультах требует специальной маршрутизации:

```
Канал Send → [Компрессор] → Channel Return (смешивается с сухим сигналом)
                ↑
          Только часть сигнала
```

На **цифровых пультах** это встроено как функция (например, в Yamaha CL/QL — Parallel Insert).

## Digital Patch (Цифровые пульты)

В цифровых пультах Insert и Direct Out реализованы **программно** — физических разъёмов может не быть, только гибкая маршрутизация.

**Yamaha CL5 пример:**
- Любой вход можно направить на любой выход через **I/O Matrix**
- Virtual patches — прямо в программной матрице
- Direct Out можно поставить в любой точке сигнального пути (pre-gain, post-gain, pre-fader, post-fader)

## Splitter (Сплиттер)

При работе с несколькими пультами (FOH + Monitors + Broadcast) используется **сплиттер** — устройство, раздающее один микрофонный сигнал на несколько выходов.

| Тип | Описание |
|-----|---------|
| **Passive split** | Трансформаторный, один выход active, остальные passive |
| **Active split** | Усилитель на каждом выходе, буферизированный |
| **Digital split** | Через Dante/MADI: один источник → бесконечно много получателей |

**Стандарт большого концерта:**
- Один stage box
- Сплит на FOH-пульт (Yamaha CL5)
- Сплит на Monitor-пульт (DiGiCo SD10)
- Сплит на Broadcast-пульт или запись

## Импеданс и уровни при инсерте

При подключении через Insert важно согласование уровней:

| Устройство | Уровень |
|-----------|--------|
| Пульт (Insert Send) | +4 дБu (профессиональный) или −10 дБV (полупрофессиональный) |
| Компрессор/процессор | Должен соответствовать пульту |

**Проблема:** подключение −10 дБV компрессора в +4 дБu инсерт → уровень слишком высокий → перегруз и искажения.

> **Практический вывод:** Insert — инструмент точечной обработки. Direct Out — инструмент гибкой маршрутизации. Понимание обоих — основа построения любой звуковой системы, от маленькой студии до крупного концертного производства.
