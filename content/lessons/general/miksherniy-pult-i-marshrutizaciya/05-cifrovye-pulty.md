# Цифровые пульты

## Аналоговый vs Цифровой пульт

| | Аналоговый | Цифровой |
|-|-----------|---------|
| Каналы | Ограничены физически | Гибко, виртуальные каналы |
| Обработка | Внешние устройства | Встроенная (EQ, компрессор, гейт, эффекты) |
| Сохранение настроек | Невозможно | Сцены (snapshots) |
| Маршрутизация | Фиксированная | Программная матрица |
| Цена | Меньше за простую модель | Больше, но всё включено |
| Звук | Аналоговый характер | Зависит от конвертеров |
| Надёжность | Высокая | Зависит от производителя |

## Сцены (Scenes / Snapshots)

**Сцена** — полное сохранение всех настроек пульта в один момент: уровни фейдеров, EQ, компрессоры, маршрутизация, обработка.

### Что сохраняется в сцене

- Уровни всех фейдеров
- EQ всех каналов
- Компрессоры, гейты
- Aux-посылы
- Маршрутизация
- DCA/VCA группировки
- Mute-состояния
- Параметры эффектов

### Применение в живом звуке

**Театр / мюзикл:**
- Сцена 1: Увертюра — инструментальный состав
- Сцена 2: Акт 1, явление 1 — конкретные актёры активированы
- Сцена 3: Акт 1, явление 2 — другой состав
- ...
- Сцена 47: Финал

Оператор нажимает **Next Scene** — пульт мгновенно переходит к следующим настройкам.

### Recall Safe (Защита от recall)

**Recall Safe** — возможность «защитить» определённые параметры от перезаписи при смене сцены.

**Пример:**
- Вы откалибровали gain барабанной установки перед показом
- При смене сцен gain должен **не меняться** — защищаем его через Recall Safe
- EQ, компрессоры, фейдеры — меняются по сцене

## DAW-интеграция

### Запись из цифрового пульта в DAW

Все каналы пульта могут передаваться в DAW по **USB, Thunderbolt или сетевому аудиопротоколу** (Dante, MADI).

**Схема:**
```
Stage box → [Yamaha CL5] → Dante → [Dante Virtual Soundcard] → DAW (32 дорожки)
```

**Virtual Soundcard** (или аппаратный интерфейс Dante) позволяет DAW видеть Dante-потоки как обычные входы.

### Control Surface

Цифровой пульт как **Control Surface** для DAW:
- Фейдеры пульта управляют фейдерами в Pro Tools / Logic / Nuendo
- Kнопки Mute/Solo на пульте = Mute/Solo в DAW
- Протоколы: HUI, MCU (Mackie Control Universal), OSC

**Yamaha CL5 + Nuendo Steinberg** — тесная интеграция, позволяющая работать как с классическим пультом.

### Offline Editing

Большинство цифровых пультов имеют **offline-редактор** (программа на компьютере):
- Подготовить настройки до прихода на площадку
- Проверить маршрутизацию, загрузить в пульт
- **Yamaha CL Editor, DiGiCo SD-Range Offline, Allen & Heath dLive Director**

## Dante подключение

**Dante** — стандартный сетевой аудиопротокол в современных профессиональных пультах.

### Что даёт Dante в системе пульта

- Stage box → пульт по **одному Ethernet-кабелю** (вместо многоканальных аналоговых мультикоров)
- Расстояние до 100 м без деградации качества
- Backup маршруты через резервный сетевой порт
- Интеграция с другими Dante-устройствами (усилители, процессоры)

**Пример системы:**

```
[Stage Box: Yamaha Rio 3224-D]  → Dante (CAT6)  → [FOH: Yamaha CL5]
                                                  → [Monitor: Yamaha QL1]
                                                  → [Recording: MacBook + Dante VSC]
```

### Dante Controller

Программа **Dante Controller** управляет маршрутизацией Dante-сети:
- Визуальная матрица: кто кому что отдаёт
- Установка задержек (latency)
- Мониторинг сети (ошибки, пакеты)

## Touchscreen vs Knob управление

Современные пульты делятся на два лагеря:

| Управление | Плюсы | Минусы |
|-----------|-------|--------|
| **Физические ручки (Yamaha, DiGiCo)** | Быстрое, тактильное | Больше физических элементов |
| **Touchscreen (Behringer Wing, QSC)** | Компактный, гибкий | Нельзя работать «на ощупь» |
| **Гибрид** | Лучшее из обоих | Выше цена |

> **Итог:** цифровой пульт — это не просто микшер, а полноценная аудио-компьютерная система. Invest время в изучение конкретной платформы (Yamaha, DiGiCo, Allen & Heath) — навыки переносятся между моделями одной линейки. Начните с Yamaha QL или Allen & Heath SQ — они интуитивны, надёжны и широко распространены.
