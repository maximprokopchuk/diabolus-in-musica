# Простые интервалы

**Простые интервалы** — интервалы в пределах одной октавы (от 0 до 12 полутонов).

## Таблица простых интервалов

| Полутонов | Название | Обозначение | Пример (от C) |
|-----------|----------|-------------|---------------|
| 0 | Чистая прима (унисон) | ч.1 | C → C |
| 1 | Малая секунда | м.2 | C → Db |
| 2 | Большая секунда | б.2 | C → D |
| 3 | Малая терция | м.3 | C → Eb |
| 4 | Большая терция | б.3 | C → E |
| 5 | Чистая кварта | ч.4 | C → F |
| 6 | Тритон (ув.4 / ум.5) | ув.4 / ум.5 | C → F# / Gb |
| 7 | Чистая квинта | ч.5 | C → G |
| 8 | Малая секста | м.6 | C → Ab |
| 9 | Большая секста | б.6 | C → A |
| 10 | Малая септима | м.7 | C → Bb |
| 11 | Большая септима | б.7 | C → B |
| 12 | Чистая октава | ч.8 | C → C |

## Как запомнить звучание интервалов

Классический способ — ассоциации с известными мелодиями:

| Интервал | Восходящий пример | Нисходящий пример |
|----------|------------------|-------------------|
| м.2 | «Челюсти» (Jaws) | «Für Elise» |
| б.2 | «Happy Birthday» | «Mary Had a Little Lamb» |
| м.3 | «Smoke on the Water» (рифф) | «Hey Jude» (припев) |
| б.3 | «When the Saints» | «Summertime» |
| ч.4 | «Свадебный марш» | «Eine kleine Nachtmusik» |
| тритон | «The Simpsons» (тема) | «Blue in Green» |
| ч.5 | «Star Wars» (тема) | «Flintstones» |
| м.6 | «The Entertainer» | «Love Story» |
| б.6 | «My Bonnie» | «Nobody Knows» |
| м.7 | «Star Trek» (тема) | «Watermelon Man» |
| б.7 | «Take On Me» (припев) | — |
| ч.8 | «Somewhere Over the Rainbow» | — |

> **Совет:** создайте свой собственный список ассоциаций из песен, которые вы хорошо знаете — это будет работать лучше.
