# Качество интервалов

## Категории интервалов

Интервалы делятся на две группы по типу «качества»:

### Совершенные (чистые) интервалы

**Прима, кварта, квинта, октава** — могут быть:
- **Чистыми** (perfect) — стандартный размер
- **Увеличенными** (augmented) — на полутон шире чистого
- **Уменьшёнными** (diminished) — на полутон уже чистого

### Несовершенные интервалы

**Секунда, терция, секста, септима** — могут быть:
- **Большими** (major) — стандартный «мажорный» размер
- **Малыми** (minor) — на полутон уже большого
- **Увеличенными** — на полутон шире большого
- **Уменьшёнными** — на полутон уже малого

## Схема модификаций

```
Для совершенных:    ум. ← Чистый → ув.
Для несовершенных:  ум. ← Малый ← Большой → ув.
```

## Примеры

| Интервал | Полутонов | Качество |
|----------|-----------|----------|
| C → F | 5 | Чистая кварта |
| C → F# | 6 | Увеличенная кварта |
| C → Fb (=E) | 4 | Уменьшённая кварта |
| C → E | 4 | Большая терция |
| C → Eb | 3 | Малая терция |
| C → E# (=F) | 5 | Увеличенная терция |

## Правило определения качества

1. Посчитайте **ступени** (определите название: секунда, терция и т.д.)
2. Посчитайте **полутоны**
3. Сравните с эталоном (чистый/большой для данного интервала)
4. Определите качество (больше/меньше эталона — увеличенный/уменьшённый)

> **Важно:** название интервала определяется по ступеням, а не по полутонам. C→D# = увеличенная секунда (2 ступени, 3 полутона), а C→Eb = малая терция (3 ступени, 3 полутона) — одинаковое звучание, разные названия (энгармонизм интервалов).
