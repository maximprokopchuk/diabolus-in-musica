# Интервалы на гитаре

## Интервалы на одной струне

На одной струне всё просто — каждый лад = 1 полутон:

| Лады | Интервал |
|------|----------|
| 0 | Прима (унисон) |
| 1 | Малая секунда |
| 2 | Большая секунда |
| 3 | Малая терция |
| 4 | Большая терция |
| 5 | Чистая кварта |
| 6 | Тритон |
| 7 | Чистая квинта |

## Интервалы между струнами

На гитаре в стандартном строе интервал между соседними открытыми струнами — **чистая кварта** (5 полутонов), кроме пары G—B, где **большая терция** (4 полутона).

### Формы на соседних струнах (для пары с квартой: E-A, A-D, D-G, B-e)

| Интервал | Форма (лад нижней → верхней) |
|----------|------------------------------|
| Чистая кварта | Одинаковый лад (0, 0) |
| Чистая квинта | Нижняя, верхняя +2 |
| Малая терция | Нижняя, верхняя −2 |
| Большая терция | Нижняя, верхняя −1 |
| Октава | Нижняя, верхняя +7 (или через струну +2) |

### Поправка для пары G—B

Из-за терцовой настройки все формы на паре G—B **сдвигаются на 1 лад вверх** для верхней ноты.

## Пауэр-аккорд как интервал

**Пауэр-аккорд** (power chord) — это чистая квинта:

```
e|------|
B|------|
G|------|
D|------|
A|--2---|  ← квинта (E)
E|--0---|  ← основной тон (E)
```

Форма: корень на нижней струне, квинта на 2 лада выше на следующей струне. Эта форма перемещается по всему грифу.

## Практика: интервальные упражнения

1. **Терции по гамме**: играйте мажорную гамму терциями (C—E, D—F, E—G, F—A...)
2. **Сексты по гамме**: то же, но секстами
3. **Кварто-квинтовые ходы**: типичные басовые линии

> **Важно:** думайте об интервалах, а не о конкретных нотах. Если вы знаете форму большой терции на грифе, вы можете сыграть её от любой ноты. Это один из главных принципов гитарного мышления — **формы (shapes) переносятся**.
