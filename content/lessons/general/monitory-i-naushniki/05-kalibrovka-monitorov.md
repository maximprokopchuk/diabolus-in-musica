# Калибровка мониторов

## Почему комната искажает звук

Даже идеальные мониторы в реальной комнате звучат **не так, как замышляли производители**. Стены, потолок, пол и мебель создают отражения, которые складываются с прямым звуком — образуя **пики и провалы** в АЧХ.

```
Монитор → [Прямой звук] → Уши
        → [Отражения от стен] → Уши (с задержкой)
                                     ↓
                           Интерференция = пики/провалы АЧХ
```

**Типичные проблемы:**
- Огромный пик баса в углах (resonant modes, стоячие волны)
- Провал на частоте «null point» (первое отражение)
- Неровные верха от отражений от стола

## Акустические стоячие волны

**Стоячие волны** (room modes) — основная проблема маленьких студий. Возникают, когда длина волны кратна размеру комнаты.

**Формула для прямоугольной комнаты:**
```
f = 343 / (2 × L)
```
Где L — размер комнаты в метрах.

**Пример:** комната 4 м в длину → f = 343 / (2 × 4) = **42.8 Гц** — первая стоячая волна. В этой точке бас будет аномально громким.

## Измерение АЧХ комнаты

Для калибровки нужно **измерить** реальную АЧХ в позиции прослушивания.

### Что нужно

1. **Измерительный микрофон** — специальный микрофон с известной, ровной АЧХ:
   - **Behringer ECM8000** — бюджетный вариант (~$30)
   - **MiniDSP UMIK-1** — популярный USB-микрофон для калибровки (~$80)
   - **Earthworks M23** — профессиональный (~$700)

2. **Программа для измерения:**
   - **Room EQ Wizard (REW)** — бесплатная, профессиональный уровень
   - **Sonarworks SoundID Reference** — коммерческая с автоматической коррекцией
   - **Arc System (IK Multimedia)** — аппаратно-программный комплекс

### Процедура измерения (REW)

1. Поставьте измерительный микрофон в позицию прослушивания (на высоте ушей)
2. Подключите через аудиоинтерфейс
3. REW воспроизводит тестовый сигнал (sweep-тон) через мониторы
4. Программа записывает и анализирует измерительный микрофон
5. Строится график АЧХ, T60 (время реверберации), waterfall (время-частота)

## Sonarworks SoundID Reference

**SoundID Reference** — стандарт программной калибровки в индустрии.

### Как работает

1. Программа воспроизводит серию тест-сигналов
2. Измерительный микрофон записывает звук в нескольких позициях (обычно 37–256 точек)
3. Алгоритм строит FIR-фильтр, компенсирующий неровности АЧХ комнаты
4. Этот фильтр применяется к выходному сигналу в реальном времени (плагин в DAW или Systemwide-приложение)

### Результат

```
До калибровки:    ±15 дБ неравномерности АЧХ
После калибровки: ±2–3 дБ
```

**Режимы работы:**
- **Fullrange** — полная коррекция, рекомендуется для работы
- **Mixed** — частичная коррекция (оставляет часть «характера»)
- **Target** — коррекция под конкретную целевую кривую

**Поддержка наушников:** SoundID Reference включает профили для сотен моделей наушников — корректирует АЧХ наушников до нейтральной.

## IK Multimedia ARC System

**Arc** — аппаратно-программный комплекс:

1. Измерительный микрофон в комплекте
2. Программа для измерения и коррекции
3. Плагин Arc в DAW применяет коррекцию

Включает **коррекцию до и после точки прослушивания** (3D-коррекция).

## Аппаратные эквалайзеры

**Genelec SAM (Smart Active Monitoring)** — встроенная калибровка в мониторах Genelec.

- Подключаете Genelec к компьютеру через USB
- Программа SAM Manager проводит измерение с измерительным микрофоном
- Коррекция записывается в DSP внутри монитора
- Работает без DAW, в любом приложении

**Deqx HDP-5** — аппаратный DSP-процессор с коррекцией на уровне предусилителя.

## Ограничения калибровки

Программная коррекция — не панацея:

| Что исправляет | Что не исправляет |
|---------------|-----------------|
| Неравномерность АЧХ | Сильные стоячие волны (провалы −20 дБ) |
| Небольшие пики | Проблемы диффузного поля |
| Тонкие нюансы тембра | Раннее отражение (reverberation time) |

**Физическая акустическая обработка** (поглотители, диффузоры) эффективнее программных решений для тяжёлых случаев. Идеально — сочетание физической обработки + программной калибровки.

## Практический план калибровки студии

1. **Правильное размещение мониторов** (треугольник прослушивания)
2. **Акустические поглотители** в первых точках отражения (боковые стены, потолок)
3. **Бас-ловушки** в углах (поглощение стоячих волн)
4. **Измерение REW** — посмотреть, что осталось
5. **SoundID Reference / Arc** — программная доработка

> **Итог:** мониторы + комната + калибровка = триада профессионального прослушивания. Нельзя игнорировать ни один элемент. Даже $100 на акустические поглотители улучшат работу больше, чем $500 на апгрейд мониторов.
