# Активные vs пассивные мониторы

## Принципиальное различие

| | Активный монитор | Пассивный монитор |
|-|-----------------|-----------------|
| **Усилитель** | Встроен | Внешний (отдельное устройство) |
| **Кроссовер** | Активный (до усилителя) | Пассивный (после усилителя) |
| **Подключение** | XLR / Jack + питание 220V | Усилитель → акустический кабель |
| **Настройка** | Проще | Сложнее (выбор усилителя) |
| **Стоимость** | «Всё в одном» | Отдельно монитор + усилитель |

## Активные мониторы

**Активный (powered) монитор** — усилитель и динамики в одном корпусе. Стандарт современных студий.

### Активный кроссовер

В активном мониторе кроссовер стоит **до усилителей** — он делит аналоговый (или цифровой) сигнал на частотные полосы до усиления.

```
Входной сигнал → [Активный кроссовер] → [Усилитель ВЧ] → Твитер
                                       → [Усилитель НЧ] → Вуфер
```

**Преимущества:**
- Оптимальная настройка каждого усилителя под свою полосу
- Нет потерь в пассивных катушках и конденсаторах кроссовера
- Производитель точно «знает» свои динамики — оптимизирует под них
- Bi-amp и tri-amp — несколько усилителей

### Встроенные регулировки

Большинство активных мониторов имеют:
- **Регулировка уровня** (trim)
- **High-frequency shelf** (±2–4 дБ) — компенсация яркости комнаты
- **Low-frequency shelf** (±2–4 дБ) — компенсация баса у стены
- **Acoustic Space** switch — −2/−4/−6 дБ в нижних частотах (для размещения у стены)

## Пассивные мониторы

**Пассивный монитор** — только корпус с динамиками и пассивным кроссовером. Требует **внешнего усилителя**.

### Пассивный кроссовер

Стоит после усилителя, состоит из катушек индуктивности и конденсаторов. Делит уже усиленный сигнал.

```
Усилитель → [Пассивный кроссовер] → Твитер
                                   → Вуфер
```

**Недостатки:**
- Потери мощности в компонентах кроссовера (до 20–30%)
- Пассивные компоненты вносят свой тембральный след
- Нельзя точно оптимизировать без знания параметров усилителя

### Когда выбирают пассивные мониторы

1. **Уже есть качественный усилитель** — можно использовать существующее оборудование
2. **Возможность апгрейда** — заменить усилитель без замены колонок
3. **Специфические системы** — пассивные мониторы в комплекте с усилителями класса A (студийный fetish)
4. **Yamaha NS-10** — легендарный пассивный монитор, требующий конкретных усилителей (Crown, Hafler)

### Примеры пассивных студийных мониторов

- **Yamaha NS-10M** — классика, стандарт студий 1980–90-х
- **Auratone 5C** — маленький однополосный монитор для проверки «radio mix»
- **Avantone MixCubes** — современный аналог Auratone

## Субвуфер для студии

Пара ближнеполевых мониторов обычно не воспроизводит частоты ниже 40–50 Гц. Для контроля суббаса нужен **студийный сабвуфер**.

| Когда нужен субвуфер |
|--------------------|
| Работа с музыкой для клубных систем (EDM, trap, bass music) |
| Работа с кино и видео (LFE-канал) |
| Когда хотите слышать суббас при сведении |

**Важно:** добавление субвуфера требует тщательной **калибровки фазы** между сателлитами и сабом. Неправильная фаза даёт провал на частоте раздела.

**Примеры:** Yamaha HS8S, KRK 10S, Focal Sub One, Genelec 7050C

> **Рекомендация для начинающих:** начните с активных мониторов 5"–8" (Yamaha HS5/HS7, Adam T7V). Это проще, дешевле и даёт предсказуемый результат. Пассивные мониторы — для тех, у кого есть конкретная причина.
