# Вторичные доминанты

## Идея

**Вторичная доминанта** — аккорд, который является доминантой **не тоники**, а другой ступени. Это кратковременное «отклонение» — мы делаем вид, что следующий аккорд — это тоника, и ставим перед ним его собственную доминанту.

## Как это работает

В C мажор аккорд **Dm** (ii) не имеет своей доминанты в диатонике. Но мы можем «занять» её:

```
Доминанта Dm = A7 (V7/ii)

C — A7 — Dm — G — C
I — V7/ii — ii — V — I
```

A7 не принадлежит тональности C мажор (нота C# — чужая!), но звучит логично, потому что тяготеет в Dm.

## Запись

Записывается как **V7/** + целевая ступень:

| Запись | Значение | Аккорд (в C) | Разрешается в |
|--------|----------|--------------|---------------|
| V7/ii | Доминанта ii | A7 | Dm |
| V7/iii | Доминанта iii | B7 | Em |
| V7/IV | Доминанта IV | C7 | F |
| V7/V | Доминанта V | D7 | G |
| V7/vi | Доминанта vi | E7 | Am |

## Самые употребительные

### V7/V (доминанта доминанты)

```
C — D7 — G — C
I — V7/V — V — I
```

D7 → G — как будто на секунду попали в G мажор. Звучит очень естественно.

### V7/vi

```
C — E7 — Am — G
I — V7/vi — vi — V
```

E7 содержит G# — хроматическую ноту, создающую яркое тяготение в Am.

### V7/IV

```
C — C7 — F — G — C
I — V7/IV — IV — V — I
```

C превращается в C7 (добавляется Bb) перед F — типично для блюза и джаза.

## На гитаре

Вторичные доминанты — мощный способ «раскрасить» простую прогрессию:

```
Было:  C — Dm — G — C
Стало: C — A7 — Dm — G — C
```

> **Совет:** если аккорд «не из тональности» стоит перед диатоническим аккордом и является его доминантой — это вторичная доминанта. Этот анализ объясняет 90% хроматических аккордов в поп/рок музыке.
