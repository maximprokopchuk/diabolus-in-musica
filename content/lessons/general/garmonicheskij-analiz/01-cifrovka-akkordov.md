# Цифровка аккордов

## Римские цифры

**Цифровка** (Roman Numeral Analysis) — запись аккордов **римскими цифрами**, обозначающими ступень лада.

Это позволяет описать гармонию **независимо от тональности**:

```
C — F — G — C   →   I — IV — V — I
G — C — D — G   →   I — IV — V — I   (та же прогрессия!)
```

## Правила записи

| Запись | Значение |
|--------|----------|
| **I, IV, V** (заглавные) | Мажорный аккорд |
| **ii, iii, vi** (строчные) | Минорный аккорд |
| **vii°** | Уменьшённый аккорд |
| **III+** | Увеличенный аккорд |
| **V7** | Доминантсептаккорд |
| **IVmaj7** | Мажорный септаккорд |
| **ii7** | Минорный септаккорд |

## Пример анализа

Песня «Let It Be» (The Beatles), тональность C мажор:

```
C — G — Am — F
I — V — vi — IV
```

Песня «Smells Like Teen Spirit» (Nirvana), тональность F# минор:

```
F#m — B — A — D
i   — iv — III — VI
```

## Как определить ступень

1. Определите тональность песни (по ключевым знакам или на слух)
2. Найдите, какой ступени гаммы соответствует корень аккорда
3. Запишите римскую цифру с учётом качества (мажор/минор)

## Зачем нужна цифровка

- **Транспонирование**: зная I — V — vi — IV, можно сыграть в любой тональности
- **Анализ**: увидеть общие паттерны в разных песнях
- **Коммуникация**: музыканты часто говорят «1-5-6-4» вместо конкретных аккордов

> **На джемах:** если кто-то говорит «стандартный 1-5-6-4 в G» — это G — D — Em — C. Навык мгновенного пересчёта ступеней в аккорды — один из самых полезных.
