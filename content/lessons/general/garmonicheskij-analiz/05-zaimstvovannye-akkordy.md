# Заимствованные аккорды (Modal Interchange)

## Идея

**Modal interchange** (ладовый обмен) — использование аккордов из **одноимённого** лада. Чаще всего: аккорды из C минора используются в C мажоре.

Это не модуляция — тональность остаётся прежней, но «окрашивается» минорными красками.

## Какие аккорды заимствуются

Из C минора (натурального) в C мажор:

| Ступень | C мажор | C минор (заимствованный) |
|---------|---------|------------------------|
| I | C | Cm |
| II | Dm | Ddim |
| **III** | Em | **Eb** |
| **IV** | F | **Fm** |
| V | G | Gm |
| **VI** | Am | **Ab** |
| **VII** | Bdim | **Bb** |

Наиболее часто заимствуются: **♭III (Eb), ♭VI (Ab), ♭VII (Bb), iv (Fm)**

## Самые популярные заимствования

### ♭VII (Bb в тональности C)

```
C — Bb — F — C
I — ♭VII — IV — I
```

Рок-стандарт! Led Zeppelin, AC/DC, Guns N' Roses — ♭VII встречается повсюду.

### iv (Fm в тональности C)

```
C — F — Fm — C
I — IV — iv — I
```

Переход IV → iv — один из самых красивых приёмов. Минорная субдоминанта создаёт «щемящее» ощущение. Radiohead, Beatles, Elliott Smith.

### ♭VI (Ab в тональности C)

```
C — Ab — Bb — C
I — ♭VI — ♭VII — I
```

Эпический рок-оборот. Часто в припевах для «грандиозного» звучания.

### ♭III (Eb в тональности C)

```
Am — Eb — Bb — F
vi — ♭III — ♭VII — IV
```

## Как распознать заимствование

Если аккорд **не диатонический**, но и **не вторичная доминанта** (не разрешается как V7 в следующий аккорд) — скорее всего, это заимствование.

Ключевые признаки:
- Содержит бемольные ступени (♭3, ♭6, ♭7)
- Не является доминантсептаккордом
- Создаёт «минорный оттенок» в мажорной тональности

> **Заимствованные аккорды + вторичные доминанты** объясняют практически все «нестандартные» аккорды в поп- и рок-музыке. Освоив эти два инструмента анализа, вы сможете разобрать гармонию любой песни.
