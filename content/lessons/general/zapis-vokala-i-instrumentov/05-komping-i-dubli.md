# Компинг и редактирование дублей

## Что такое компинг

**Компинг (Comping)** — процесс создания идеального исполнения из **нескольких записанных дублей** (takes). Вместо того чтобы добиваться идеального исполнения за одну попытку, исполнитель записывает 3–10 дублей, из которых потом собирается «лучшая версия».

```
Дубль 1: [Хорошо] [ОШИБКА] [Хорошо] [Хорошо] [Хорошо]
Дубль 2: [Хорошо] [Хорошо] [Хорошо] [ОШИБКА] [Хорошо]
Дубль 3: [Ошибка] [Хорошо] [Хорошо] [Хорошо] [ЛУЧШИЙ]

Comp:    [Д1]     [Д2]     [Д1]     [Д1]     [Д3]
```

Итоговая «comp-дорожка» = лучшие фрагменты из всех дублей.

## Playlist (Take Folder) — организация дублей

**Playlist** (Pro Tools) / **Take Folder** (Logic) — способ хранить несколько дублей на одном треке:

- Все дубли записываются **один за другим** на один трек
- DAW автоматически создаёт «слой» (playlist/take) для каждого дубля
- Все дубли сохранены и доступны для компинга

### В разных DAW

| DAW | Название функции |
|----|----------------|
| Pro Tools | Playlist / Track Playlist |
| Logic Pro | Take Folder (Comp-дорожки) |
| Ableton Live | Ручное переименование треков |
| Reaper | Lanes / Sub-tracks |
| Cubase | VariAudio / Lane Recording |

## Процесс компинга

### Шаг 1: Прослушивание всех дублей

Прослушайте каждый дубль **целиком** и отметьте:
- Эмоциональные пики
- Точные технические места
- Ошибки и проблемы
- Уникальные характерные моменты

Используйте маркеры или заметки: «Дубль 3, такт 8 — лучший бридж».

### Шаг 2: Структурирование

Разбейте песню на **логические секции**: куплеты, припевы, бриджи, отдельные фразы.

Начните с **крупных блоков**: какой дубль лучший в каждой секции? Потом уточняйте до отдельных фраз или слов.

### Шаг 3: Вырезание и сборка

В DAW с поддержкой playlist:
1. Выберите инструмент выделения
2. Выделите нужный фрагмент в нужном дубле
3. Нажмите «Promote to Comp» или перетащите в comp-трек

Инструменты:
- **Cmd+E** (Logic) — разрезать в точке playback
- **Ножницы (Scissors)** — ручная резка
- **Smart Tool** (Pro Tools) — контекстные функции

### Шаг 4: Crossfades (переходы)

На стыке фрагментов из разных дублей неизбежны **щелчки и резкие переходы**. Решение — **crossfade**: плавное перекрытие между концом одного фрагмента и началом другого.

```
Дубль 1: ────────────────────╲
                                ╲___
Дубль 2:               _____╱╱
                      ╱────────────────

Crossfade: [Дубль 1 затухает] + [Дубль 2 нарастает]
```

Длина кроссфейда: 10–50 мс для вокала, 5–20 мс для гитары. Чем резче атака материала, тем короче должен быть crossfade.

## Критерии выбора лучшего фрагмента

### Для вокала

| Приоритет | Критерий |
|---------|---------|
| 1 | **Эмоция** — правильное «чувство» важнее технического совершенства |
| 2 | **Интонация** — точность нот (поддаётся pitch-коррекции при необходимости) |
| 3 | **Тайминг** — ритмическое попадание |
| 4 | **Тембр** — качество голоса в данном месте |

### Для инструментов

| Приоритет | Критерий |
|---------|---------|
| 1 | **Точность нот** — нет ошибок |
| 2 | **Тайминг** — попадание в groove |
| 3 | **Динамика** — уместная экспрессия |
| 4 | **Тон** — качество звука |

## Особые случаи

### «Достаточно хорошее» vs «Идеальное»

Стремление к абсолютному перфекционизму может уничтожить живость исполнения. Часто **дубль с небольшой интонационной неточностью, но с правильной эмоцией**, лучше технически идеального, но «мёртвого» дубля.

### Когда нет хорошего дубля

Если ни в одном дубле нет нужного фрагмента:
1. **Запишите дополнительные дубли** — лучше больше, чем начинать сведение с компромиссом
2. **Pitch correction + timing correction** — если технически проблема, не эмоционально
3. **Пересмотрите стандарты** — возможно, аранжировка слишком сложна

### Comp для инструментов

Компинг применяется не только к вокалу. Гитарное соло, барабанный Fill, вокальная фраза — всё это может быть собрано из дублей.

## Технические детали

### Undo и не-разрушающее редактирование

Все операции компинга в DAW **не разрушают** оригинальные файлы. Дубли всегда сохранены. Cmd+Z отменяет любое действие.

### Экспорт comp-трека

После завершения компинга **Flatten/Consolidate** comp-трек в единый аудиофайл:
- Logic: Flatten Take Folder
- Pro Tools: Consolidate Clip
- Reaper: Glue Items

Это упрощает последующую обработку (pitch correction, компрессия).

> **Профессиональный подход:** хороший звукорежиссёр не только «собирает» comp механически, но **рассказывает историю**. Правильный comp передаёт эмоциональную дугу исполнения — нарастание в припеве, нежность в куплете, срыв на финальной ноте.
