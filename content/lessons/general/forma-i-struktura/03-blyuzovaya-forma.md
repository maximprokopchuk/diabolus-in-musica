# Блюзовая форма

## 12-тактовый блюз

**12-тактовый блюз** — одна из самых влиятельных форм в истории музыки. Из неё выросли рок-н-ролл, R&B, соул и многое другое.

### Базовая схема

```
| I7  | I7  | I7  | I7  |
| IV7 | IV7 | I7  | I7  |
| V7  | IV7 | I7  | V7  |
```

В тональности A:

```
| A7  | A7  | A7  | A7  |
| D7  | D7  | A7  | A7  |
| E7  | D7  | A7  | E7  |
```

Последний такт (V7) называется **turnaround** — он «разворачивает» прогрессию к началу.

## Структура AAB

Текст 12-тактового блюза следует схеме **AAB**:
- Такты 1–4: фраза A (утверждение)
- Такты 5–8: фраза A (повтор, возможно с вариацией)
- Такты 9–12: фраза B (ответ, вывод)

## Варианты блюзовой прогрессии

### Quick change (быстрая смена)

IV7 появляется уже во 2-м такте:
```
| I7  | IV7 | I7  | I7  |
```

### Джазовый блюз

С доминантами и ii-V оборотами:
```
| I7    | IV7   | I7    | I7    |
| IV7   | #IVdim| I7    | vi7   |
| ii7   | V7    | I7    | ii-V  |
```

### Минорный блюз

```
| i7   | i7   | i7   | i7   |
| iv7  | iv7  | i7   | i7   |
| V7   | iv7  | i7   | V7   |
```

## Тональности для гитары

Самые удобные тональности для блюза на гитаре:
- **E** и **A** — используют открытые струны
- **G** — удобна для слайда
- **Bb** и **F** — типичны для джемов с духовыми

> **Блюз — основа основ.** Умение играть 12-тактовый блюз в нескольких тональностях — обязательный навык для любого гитариста. На блюзовом джеме это «общий язык» всех музыкантов.
