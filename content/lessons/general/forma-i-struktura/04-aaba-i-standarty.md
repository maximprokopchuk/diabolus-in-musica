# AABA и стандарты

## Форма AABA (32 такта)

**AABA** — классическая форма джазовых стандартов и песен Tin Pan Alley (1920–60-е):

```
A (8 тактов) — A (8 тактов) — B (8 тактов) — A (8 тактов)
```

Итого: 32 такта = один **chorus** (проход формы).

### Разделы

- **A** — основная тема (мелодия + гармония). Повторяется 3 раза.
- **B** — **бридж** (bridge, middle eight) — контрастный раздел. Обычно в другой тональности или с другой гармонией.

### Примеры AABA

| Стандарт | Тональность | Особенности |
|----------|-------------|-------------|
| «I Got Rhythm» | Bb | Основа «Rhythm Changes» |
| «Somewhere Over the Rainbow» | Eb | Классический Tin Pan Alley |
| «Take the A Train» | C | Тема Билли Стрейхорна (оркестр Эллингтона) |
| «Satin Doll» | C | С ii-V оборотами |
| «Yesterday» (Beatles) | F | Поп-адаптация AABA |

## Другие формы стандартов

### ABAC

```
A (8) — B (8) — A (8) — C (8)
```

Пример: «Stella by Starlight», «White Christmas».

### Through-composed (сквозная)

Без чёткого повторения секций. Каждый раздел — новый материал.

## Как играется джазовый стандарт

1. **Head** (тема) — мелодия проигрывается один-два раза
2. **Solos** — каждый музыкант импровизирует по форме (AABA повторяется)
3. **Head out** — тема повторяется в конце
4. **Coda** — окончание

Каждое соло = один или несколько полных «проходов» (choruses) по 32-тактовой форме.

> **Совет:** для изучения стандартов используйте **Real Book** — сборник с мелодиями и аккордами сотен джазовых стандартов. Начните с «Autumn Leaves», «Blue Bossa», «All of Me».
