# Нотный стан для барабанов

## Особенности барабанной нотации

В отличие от нотации для мелодических инструментов (где высота ноты соответствует звуку на стане), барабанная нотация использует стан для **обозначения разных элементов установки**. Положение ноты на стане говорит не о высоте, а о том, **какой барабан или тарелку** нужно ударить.

Это делает барабанную нотацию проще для начала чтения, но требует знания принятых соглашений.

## Ключ

Для барабанной нотации используется специальный **ударный ключ** — две вертикальные полосы в начале стана. Иногда вместо него ставят нейтральный ключ (одна вертикальная линия).

На некоторых нотных редакторах используется обычный скрипичный ключ с пометкой «Drums» или «Perc.» — в этом случае высоты нот не имеют тонального значения.

## Стандартное расположение инструментов на стане

Приведём наиболее распространённую систему (General MIDI Drum Map), которую используют большинство издателей и нотных программ:

### Линии и промежутки (снизу вверх)

| Позиция | Элемент установки |
|---------|-----------------|
| Ниже стана (нота с хвостиком вниз) | Бас-барабан (Kick Drum) |
| 1-я линия (нижняя) | Снэйр (основной удар) |
| Промежуток под 1-й линией | Snare rim click |
| 1-й промежуток | Hi-hat педалью (нога, закрытый) |
| 2-я линия | Hi-hat (палочкой, закрытый) |
| 2-й промежуток | Hi-hat открытый |
| 3-я линия (средняя) | High tom (первый том) |
| 3-й промежуток | Mid tom |
| 4-я линия | Floor tom |
| 4-й промежуток | Crash cymbal |
| 5-я линия (верхняя) | Ride cymbal |
| Выше стана | Ride bell / China cymbal |

### Важное примечание о системах

**Единой мировой стандартизации нет.** Разные издательства, школы и страны используют разные системы. Перед работой с нотами всегда проверяйте легенду (drum key) в начале произведения.

## Форма нот

В барабанной нотации используются два типа нотных головок:

### Обычная овальная головка (○)
Используется для всех **барабанов** (бас-барабан, снэйр, томы). Эти инструменты имеют «стандартный» звук.

### Крестообразная головка (×)
Используется для **тарелок** (хай-хэт, крэш, райд) и всех инструментов с неопределённой высотой звука.

### Ромбовидная головка (◇)
Используется для **особых техник**: удар по обечайке (rim shot), нажатие педали хай-хэта без удара, некоторые эффекты.

## Хвостики нот и распределение по рукам/ногам

Общепринятое правило:
- **Хвостики вверх** → руки (правая или левая)
- **Хвостики вниз** → ноги (правая = бас-барабан, левая = хай-хэт педаль)

Это позволяет разделить партии рук и ног визуально и записывать их в одном стане.

## Пример простого грува на стане

Типичный рок-грув (хай-хэт + снэйр + бас) записывается так:
- Верхний слой (хвостики вверх): крест на второй линии каждую восьмую — это хай-хэт
- Средний слой: овал на первой линии на долях 2 и 4 — это снэйр
- Нижний слой (хвостики вниз): овал под станом на долях 1 и 3 — это бас-барабан

Три «голоса» читаются одновременно в одном такте.

## Программы для чтения и написания нот

| Программа | Описание |

| **Sibelius** | Профессиональный нотный редактор |
| **Finale** | Альтернатива Sibelius, более гибкая |

Для чтения барабанных нот онлайн: **Drum Score World**, **Sheet Music Direct**, архивы **Modern Drummer**.

## Основное правило при чтении

Перед тем как играть, всегда:
1. Найдите drum key (легенду) в начале нот
2. Определите темп (метроном marking)
3. Определите размер (4/4, 3/4, 6/8 и т.д.)
4. Просмотрите структуру: сколько секций, есть ли повторы, коды

> **Важно:** барабанная нотация — это условный язык, а не абсолютный. Двое барабанщиков могут использовать разные нотные системы. Главное — понимать принцип: позиция ноты = конкретный инструмент + тип удара.
