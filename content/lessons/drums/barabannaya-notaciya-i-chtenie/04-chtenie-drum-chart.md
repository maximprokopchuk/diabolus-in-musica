# Чтение drum chart

## Что такое drum chart

**Drum chart** — упрощённая запись барабанной партии, используемая в профессиональной среде: сессионными музыкантами, на записях, в театрах и концертах.

В отличие от полной транскрипции, drum chart не всегда записывает каждую ноту. Чаще он содержит:
- Структуру (форму) песни
- Важные моменты (хиты, стопы)
- Базовый стиль и ощущение грува
- Изменения размера или темпа

Умение читать drum chart — **профессиональный навык**, отличающий сессионного музыканта от любителя.

## Форма песни

Профессиональная запись всегда начинается с обзора формы. Стандартные обозначения:

| Обозначение | Название |
|-------------|---------|
| **Intro** | Вступление |
| **V1, V2** | Куплет 1, куплет 2 (Verse) |
| **Pre-Ch** | Пре-хорус |
| **Ch** | Хорус (Chorus) |
| **Br** | Бридж (Bridge) |
| **Outro** | Концовка |
| **Solo** | Инструментальное соло |
| **Tag** | Дополнительный повтор последней фразы |

Пример формы:
```
Intro (4 такта) → V1 (16 тактов) → Ch (8 тактов) → V2 (16 тактов) → Ch (8 тактов) → Br (8 тактов) → Ch × 2 (16 тактов) → Outro (8 тактов)
```

## Знаки повтора (Repeat Signs)

### Двойная черта с точками (**||: ... :||**)
Повторить раздел между знаками. Если цифра над закрывающим знаком — сыграть столько раз.

### Вольта (1. и 2.)
Первый раз исполняется окончание под «1.», при повторе — под «2.». Часто используется для куплетов с разными окончаниями.

### Simile (sim.)
Сыграть аналогично предыдущим тактам. Обычно пишется, когда паттерн стабильный и повторять нотацию нет смысла.

### Slash marks (////)
Четыре наклонных черты в такте означают: **сыграй что-нибудь стилистически подходящее**. Это «приглашение к импровизации» в рамках стиля. Барабанщик должен держать грув, соответствующий жанру.

## DC, DS и Coda

Эти обозначения — «навигационные инструкции»:

| Обозначение | Расшифровка | Действие |
|-------------|------------|---------|
| **D.C.** | Da Capo | Вернуться к началу |
| **D.C. al Fine** | Da Capo al Fine | Вернуться к началу, закончить на Fine |
| **D.C. al Coda** | Da Capo al Coda | Вернуться к началу, при знаке Coda перейти к Coda |
| **D.S.** | Dal Segno | Вернуться к знаку § (Segno) |
| **D.S. al Coda** | Dal Segno al Coda | Вернуться к §, при знаке ⊕ перейти к Coda |
| **Fine** | Fine (финал) | Здесь заканчивается произведение (при D.C. или D.S.) |
| **Coda** | Хвост | Финальная секция, обозначается символом ⊕ |

### Пример прочтения

```
[Intro → V1 → Ch → V2 → Ch → D.S. al Coda]
Segno (§) стоит в начале V1.
Coda (⊕) — после второго Ch.

Порядок: Intro → V1 → Ch → V2 → Ch → (D.S.) → V1 → Ch → (Coda) → Outro
```

## Cue Notes

**Cue notes** («напоминающие ноты») — маленькие ноты в нотации, обозначающие **партию другого инструмента**. Они не играются — они информируют барабанщика о том, что происходит вокруг.

Зачем: барабанщик слышит вокалиста или гитариста и понимает, где он находится в форме — особенно важно при чтении с листа без предварительной репетиции.

Пример: маленькие ноты в первом такте куплета — это фраза вокала, которая помогает барабанщику «войти» в нужное место.

## Tempo Markings (темповые указания)

| Обозначение | Значение |
|-------------|---------|
| **♩ = 120** | Четвертная = 120 BPM |
| **Moderato** | Умеренно (96–120 BPM) |
| **Allegro** | Оживлённо (120–156 BPM) |
| **Andante** | Шагом (76–108 BPM) |
| **Adagio** | Медленно (66–76 BPM) |
| **Presto** | Очень быстро (168+ BPM) |
| **rit. / rall.** | Rallentando / Ritardando — постепенное замедление |
| **accel.** | Accelerando — постепенное ускорение |
| **a tempo** | Вернуться к исходному темпу |

## Практика чтения drum chart

**Упражнение 1: Анализ формы**
Возьмите любой drum chart. Прежде чем играть, пройдите по форме: найдите все повторы, коды, вернитесь к началу. Нарисуйте блок-схему формы.

**Упражнение 2: Чтение с листа**
Найдите незнакомый drum chart (Modern Drummer, онлайн-ресурсы). Поставьте метроном на 70% указанного темпа. Читайте «вживую» — без предварительного просмотра.

**Упражнение 3: Создание chart**
Возьмите любую знакомую песню и напишите drum chart для неё: обозначьте форму, расставьте знаки повтора, добавьте cue notes.

> **Профессиональный секрет:** сессионный барабанщик должен уметь сыграть новую песню с листа на первом прогоне. Это приходит только с практикой чтения. Читайте 15–20 минут каждый день — через несколько месяцев скорость чтения вырастет многократно.
