# Double bass

## Что такое double bass

**Double bass** (двойная бочка) — использование двух педалей бочки:
- Двух **отдельных барабанов-бочек** (классический вариант, popularized by Louie Bellson и Ginger Baker)
- **Одной бочки с двумя педалями** (double pedal — современный стандарт)

## Double Pedal vs Two Bass Drums

### Double Pedal (двойная педаль)
- Одна бочка, два механизма педали
- Левая педаль соединена с основной через тягу (drive shaft)
- Компактнее для транспортировки
- Оба битера бьют в одну точку
- **Современный стандарт** в метале и роке

### Two Bass Drums (два барабана)
- Две отдельных бочки
- Исторически первый вариант
- Разные позиции битеров дают чуть разный звук
- Традиционно в хэви-метале (Lars Ulrich, Vinnie Paul)
- Требует больше места

## Техника двойной педали

Принципы те же, что для одной педали:
- **Heel-up** — стандарт для скоростных рисунков в метале
- **Heel-down** — для контролируемых, тихих партий
- **Правильная посадка** критически важна — обе ноги должны быть в удобном положении

### Анатомия движения при double bass

При игре двумя ногами важно понять, что ноги работают **независимо**:
- Левая нога учится «заново» — она привыкла работать только на хэт-педали
- Первый месяц занятий с double bass — это в основном тренировка **левой ноги**

## Синхронизация ног

### Почему синхронизация сложна

Правая нога обычно сильнее и координированнее. Левая отстаёт — звук двойной бочки получается «хромым»: правый удар громче, левый тише и не в темп.

**Это нормально** — требует длительной тренировки.

### Упражнения на синхронизацию

**1. Четверти двумя ногами попеременно**
Метроном 50 BPM. Правая — левая — правая — левая. Фокус: каждый удар **одинаковой громкости** и **точно в темп**.

**2. Восьмые**
Та же идея на более мелкой длительности. Не ускоряйтесь, пока нет ровности.

**3. Sixteenth notes (шестнадцатые)**
Четыре удара на долю: RLLR RLLR или RLRL RLRL. Начните с 40 BPM.

**4. Только левая нога**
30 минут практики только левой ногой — четверти, восьмые, шестнадцатые. Выравнивает дисбаланс.

### Паттерны для развития

```
Паттерн 1 (чередование): R L R L R L R L
Паттерн 2 (двойки): RR LL RR LL
Паттерн 3 (тройки): RRL RRL RRL
Паттерн 4 (парадидл ногами): RLRR LRLL
```

## Интеграция с руками

Главная сложность — **синхронизация ног с руками** при игре полноценных паттернов.

### Базовый рок-паттерн с восьмыми на бочке

```
Хэт:     X X X X X X X X (восьмые)
Снейр:   - - O - - - O -
Бочка:   O - O - O - O - (двойная)
```

Начните **медленно**: 50–60 BPM. Руки и ноги должны быть синхронны прежде, чем добавлять скорость.

### Бластбиты и метал

Самый распространённый double bass паттерн в экстремальном метале:

```
Хэт/тарелка: X - X - X - X -
Снейр:       - O - O - O - O
Бочка:       O O O O O O O O (шестнадцатые)
```

Это требует серьёзной технической подготовки — скорость шестнадцатых на 180–200 BPM достигается **годами тренировок**.

## Физическая подготовка

Double bass — это серьёзная физическая нагрузка. Необходимо:

1. **Разминка** перед каждой тренировкой: вращение лодыжками, растяжка икр
2. **Постепенное увеличение нагрузки** — не играть часами с первых недель
3. **Паузы** — мышцы растут во время отдыха, а не во время игры
4. **Внимание к болевым ощущениям** — боль в колене или голеностопе = стоп

## Ориентиры темпа

| Уровень | Темп шестнадцатых на double bass |
|--------|--------------------------------|
| Начинающий | 80–100 BPM |
| Средний | 100–140 BPM |
| Продвинутый | 140–180 BPM |
| Профессиональный | 180–220 BPM |
| Экстремальный метал | 220+ BPM |

> **Реалистичный совет:** если вы только начинаете осваивать double bass — рассчитывайте на 6–12 месяцев серьёзных тренировок, чтобы левая нога стала надёжной. Это не быстрый процесс, но он того стоит.
