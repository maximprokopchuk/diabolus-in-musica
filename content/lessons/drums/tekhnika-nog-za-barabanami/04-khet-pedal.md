# Хэт педаль

## Роль хэт педали

**Хэт педаль** (hi-hat pedal) управляет смыканием и размыканием двух тарелок хэта. Это **левая нога** барабанщика (у правшей).

В отличие от педали бочки, хэт-педаль:
- Используется **постоянно** на протяжении большинства партий
- Является **ритмической опорой**, особенно в джазе
- Создаёт различные тембральные варианты одним инструментом

## Позиции хэта

### Закрытый хэт
Педаль нажата — тарелки сомкнуты. Удар палкой даёт **чёткий, острый звук** «tsk».

### Открытый хэт
Педаль отпущена — тарелки разомкнуты. Удар даёт **длинный, раскрытый звук** «tshhh».

### Полузакрытый (half-open)
Педаль слегка нажата. Звук между закрытым и открытым — характерный «sloshy» тембр.

## Ритмические рисунки ногой

Хэт педаль используется **ногой** (без палки) для создания ритмических акцентов.

### Четверти
Самый базовый паттерн — хэт педаль на каждую долю такта:

```
Хэт педаль: x - - - x - - - x - - - x - - -
            1       2       3       4
```

Создаёт подсознательное ощущение «большого такта», особенно заметно в джазе.

### Восьмые
Хэт на каждую восьмую:

```
Хэт педаль: x - x - x - x - x - x - x - x -
            1 + 2 + 3 + 4 +
```

Используется в регги, фанке, даёт характерную «пульсацию».

### Sixteenth (шестнадцатые)
Хэт на каждую шестнадцатую ноту — продвинутая техника, требует независимости левой ноги:

```
Хэт педаль: x x x x x x x x x x x x x x x x
            1 e + a 2 e + a 3 e + a 4 e + a
```

### 2 и 4
Классика джаза и биг-бэнда: хэт педаль на «слабые» доли 2 и 4:

```
Хэт педаль: - - - - x - - - - - - - x - - -
            1       2       3       4
```

## Splashing

**Splash** — быстрое открытие и закрытие хэта ногой для создания «хлопка»:

Техника:
1. Педаль нажата (хэт закрыт)
2. Резко отпустите педаль — тарелки разомкнутся и «чирикнут»
3. Немедленно снова нажмите — звук резко обрежется

Результат: характерный звук «chick» или «splash» — очень характерен для джаза и фьюжна.

**Применение:**
- Акценты в конце фраз
- Имитация снейра на хэте
- Джазовые «вопросы-ответы» между хэтом и снейром

## Bark

**Bark** («лай») — более агрессивная разновидность splash:

Техника аналогична splash, но:
- Педаль отпускается более резко
- Тарелки успевают издать более продолжительный звук перед закрытием
- Результат — звук с характерной «атакой и хвостом»

Используется в:
- Джазе (особенно в брейках)
- Фьюжне
- Инди-роке для нестандартных акцентов

## Хэт педаль в координации с руками

### Базовая независимость
Самое важное — отработать хэт педаль **независимо от рук**. Упражнение:

- Правая рука играет восьмые на хэте
- Снейр на 2 и 4
- Бочка на 1 и 3
- **Левая нога**: четверти (1-2-3-4) на хэт педали

Это базовое четырёхстороннее взаимодействие.

### Открытый хэт в рок-партиях

Популярный приём: основную часть хэт играется закрытым, в определённые моменты нога отпускает педаль:

```
Хэт рука:   x x x x x x x O x x x x x x x x
Хэт педаль: X X X X X X X _ X X X X X X X X
                           ^ открытый
```

Открытый хэт создаёт «всплеск» — характерный для рока и поп-рока приём.

## Настройка педали хэта

- **Высота тарелок** — влияет на удобство игры: средняя высота (около 30 см от пола) стандартна
- **Натяжение пружины** — чем тяжелее пружина, тем труднее держать хэт закрытым, но тем «злее» звук splice
- **Угол верхней тарелки** — небольшой наклон (2–5°) улучшает splash и даёт более чёткое закрытие

## Упражнения

**1. Только хэт педаль**
5 минут: четверти левой ногой, метроном 70–90 BPM. Фокус: ровность и точность.

**2. Хэт педаль + правая рука на хэте**
Правая — восьмые, левая нога — четверти. Следите за независимостью.

**3. Splash-упражнение**
Медленно отработайте splash на каждую вторую долю. Потом на каждую четвёртую.

**4. Открытый/закрытый переход**
Сыграйте партию, чередуя закрытый и открытый хэт через каждые 2 такта. Добейтесь плавного перехода.

> **Секрет хорошей хэт-педали:** левая нога должна быть такой же «музыкальной», как правая рука. Не игнорируйте её — работайте с хэт педалью отдельно каждую тренировку.
