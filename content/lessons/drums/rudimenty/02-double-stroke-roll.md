# Double Stroke Roll

## Формула и суть

**Double stroke roll** — два удара подряд каждой рукой:

```
R R L L R R L L R R L L ...
```

Это второй по важности рудимент после single stroke. На первый взгляд кажется проще — но на самом деле требует принципиально иной техники.

## Два метода исполнения

### 1. Контролируемый (open roll technique)

На медленных темпах каждый из двух ударов — осознанный, мышечный:
- Первый удар: бросок палочки вниз
- Второй удар: активное движение запястья или пальцев

Используется при медленном темпе (до ~80–90 BPM на шестнадцатых).

### 2. Отскоковый (bounce / closed roll)

На быстрых темпах второй удар — **естественный отскок** палочки:
- Первый удар: бросок с достаточной силой
- Второй удар: не мешайте палочке подпрыгнуть

Это техника «bounce» — основа закрытого ролла. Звук становится более «роллевым», слитным.

```
Медленно:  R  R  L  L  (два отдельных удара)
Быстро:    RR LL RR LL  (первый + отскок)
```

## Открытый и закрытый ролл

| Тип | Темп | Звук | Применение |
|-----|------|------|-----------|
| **Open roll** | Медленный | Слышны отдельные удары | Упражнения, акцентированная игра |
| **Closed roll** | Быстрый | Сливается в «шшш» | Crescendo, фоновый ролл |

**Закрытый ролл** (closed roll) — это то, что вы слышите, когда барабанщик плавно нарастает на тарелке или малом: непрерывный шелестящий звук. Он достигается очень быстрым double stroke с максимальным отскоком.

## Упражнения

### 1. Медленный double с акцентом

```
BPM 50, четверти:
> .  > .  > .  > .
R R  L L  R R  L L
```

Первый удар каждой пары — акцентированный (громче). Это помогает почувствовать структуру.

### 2. Переход open → closed

Начните медленно (60 BPM), постепенно ускоряйтесь. В какой-то момент естественно «поймаете» отскок и переключитесь в режим bounce. Запомните это ощущение.

### 3. Смена рук

```
RRLL RRLL → LLRR LLRR → RRLL LLRR (смешанный старт)
```

Тренируйте оба варианта — с правой и с левой руки.

### 4. Динамические градации

Сыграйте длинный double stroke roll:
```
pp → p → mp → mf → f → ff → f → mf → mp → p → pp
```
Это классическое упражнение для развития контроля динамики.

## Применение в музыке

### На малом барабане (snare)
- Украшение fill: два удара по малому перед переходом на том
- Ролл на весь такт с нарастанием динамики к crash

### На томах
```
Fill: RR LL (высокий том) → RR LL (средний) → RR LL (напольный)
```

### В джазе
Закрытый ролл на малом — фирменный элемент джазовых брейков. Начинается тихо, нарастает, заканчивается акцентом на «1».

### В рок-соло
Быстрые double stroke по томам создают эффект «грохота» — плотнее и мощнее, чем single stroke.

## Типичные ошибки

1. **Второй удар слабее первого** — нужно добиться полного равенства обоих ударов
2. **Зажатость при отскоке** — сжимание палочки «убивает» второй удар
3. **Неправильный угол** — палочка должна бить перпендикулярно поверхности, не косо
4. **Пропуск bounce-техники** — многие играют двойник всегда мышечно, не осваивая отскок

> **Практический совет:** поставьте метроном на 60 BPM и играйте четверти как RRLL. Затем медленно ускоряйтесь, наблюдая, в какой момент второй удар «выпрыгивает» сам. Это и есть bounce — не торопите его.
