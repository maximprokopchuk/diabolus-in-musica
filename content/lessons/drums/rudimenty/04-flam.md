# Flam и Flam Акценты

## Что такое flam

**Flam** — это удар, перед которым звучит очень тихая «украшающая» нота (grace note), практически слипающаяся с основным ударом. Визуально это выглядит как маленькая нота перед большой:

```
Правый flam:  lR   (маленькая левая + акцент правая)
Левый flam:   rL   (маленькая правая + акцент левая)
```

Строчная буква = grace note (тихая), прописная = основной акцент (громкая).

Flam создаёт «жирный», слегка расширенный звук удара. В маршевой музыке и оркестровой перкуссии это фирменный элемент.

## Техника исполнения

### Положение рук

Перед flamом:
- Рука с **grace note** находится **низко** (1–2 см над барабаном)
- Рука с **основным ударом** поднята **высоко** (15–20 см)

Обе руки падают одновременно, но из-за разной высоты grace note опережает основной удар на долю секунды.

### Точность расстояния

Слишком рано → слышны два отдельных удара (не flam, а «шлёп-удар»)
Слишком поздно → grace note сливается с основным (получается accent)
Правильно → grace note едва слышна, основной удар звучит «жирно»

```
Слишком открыто:  l - R   (слышны оба)
Правильно:        lR      (слипшийся)
Слишком закрыто: R        (только один)
```

## Базовые flam-рудименты

### Flam Tap
```
lR rL  lR rL  lR rL
```
Flam + одиночный удар той же рукой. Чередуется: правый flam → левый tap, левый flam → правый tap.

### Flam Accent
```
lR L R  rL R L
```
Flam + два одиночных удара. Группа из 3 нот, начинающаяся с flam. Отличный рудимент для триолей.

### Flamacue
```
lR L rL R  lR L rL R
```
Более сложный паттерн с несколькими flamами. Используется в маршевой музыке.

### Flam Paradiddle
```
lR L R R  rL R L L
```
Обычный paradiddle, но первая нота каждой группы — flam. Создаёт акцент и характер.

### Single Flammed Mill
```
rL R lR L  rL R lR L
```
Маршевый рудимент с чередующимися flamами.

## Flamacue подробно

**Flamacue** — один из 40 стандартных рудиментов PAS:

```
lR L rL R  lR L rL R
1  2  3  4  1  2  3  4
```

Особенность: акцент стоит не на первой ноте, а на **четвёртой**. Это создаёт синкопированный характер паттерна.

Применение: марши, drum corps (Blue Devils, Santa Clara Vanguard), оркестровая перкуссия.

## Применение в рок и поп музыке

Flam не ограничен маршевой музыкой:

### На малом барабане в рок
Вместо обычного удара на 2 и 4 — flam на малом. Звук становится «толще», акцент весомее.

```
Обычный:  . . S . | . . S .
С flamom: . . lS. | . . rS .
```

### В fills
```
Fill: tom → lR(малый) → том → crash
```
Flam в конце fill придаёт «удар» переходу в новый раздел.

### Ghost flam
Очень тихий flam как украшение ghost note на малом — характерный элемент фанк-барабанного языка.

## Упражнения

### 1. Один flam — медленно
Правый flam lR: поднимите правую высоко, левую опустите низко. Уроните обе одновременно. Слушайте — grace note должна едва слышаться.

### 2. Чередование
```
lR rL lR rL lR rL ...
```
На 60 BPM, каждый flam — четверть.

### 3. Flam accent в триолях
```
lR L R  rL R L  lR L R ...
```
Три ноты = триоль. Очень музыкальный паттерн.

## Типичные ошибки

1. **Flam слишком открытый** — grace note слышна как отдельный удар
2. **Обе руки одной высоты** — физически невозможно получить правильный flam
3. **Grace note громкая** — она должна быть максимально тихой, почти неслышимой
4. **Неправильная смена рук** — после lR следующий flam должен быть rL

> **Тест на правильность:** запишите свой flam на телефон. Если в записи слышны явно два удара — он слишком открытый. Если слышен один «жирный» — это правильный flam.
