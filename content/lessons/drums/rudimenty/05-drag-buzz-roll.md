# Drag и Buzz Roll

## Drag (Dragg)

**Drag** — рудимент, похожий на flam, но вместо одной grace note перед основным ударом звучат **две** быстрые ноты одной рукой:

```
llR  (два тихих левых + акцент правый)
rrL  (два тихих правых + акцент левый)
```

Два предварительных удара играются максимально быстро — почти как отскок. В отличие от flam (один grace note), drag «тянет» звук чуть дольше.

### Два способа исполнения

**Открытый (ruff):** два отдельных grace note — ll или rr — отчётливо слышны. Используется на медленном темпе.

**Закрытый (closed drag):** два grace note сливаются в единый «дребезг» — практически как очень короткий buzz. Используется на быстром темпе.

```
Медленно:  l l R  (слышны три удара)
Быстро:    [ll]R  (drag + основной)
```

## Ruff

**Ruff** — другое название открытого draga. Часто используется как синоним. В точном определении PAS:

- **Ruff** = два grace note + основной удар (3 ноты)
- **Drag** = закрытый вариант того же паттерна

Иногда термины путают, но суть одна: украшение основного удара двумя быстрыми предварительными нотами.

## Drag Tap

**Drag Tap** — один из стандартных рудиментов:

```
llR L  rrL R  llR L  rrL R
```

Drag + одиночный удар противоположной рукой. Структура напоминает flam tap, но с двумя grace notes.

### Single Drag Tap
```
llR L  rrL R
```

### Double Drag Tap
```
llR llR L  rrL rrL R
```

## Lesson 25 (Dragadiddle)

```
llR R L R  rrL L R L
```
Drag в начале paradiddle-структуры. Сложный, но очень выразительный рудимент.

## Buzz Roll

**Buzz roll** (также **multiple bounce roll** или **press roll**) — принципиально другой тип ролла. Вместо контролируемых ударов — **максимальное количество отскоков** с каждого удара:

```
Обычный double: R R L L (контролируемые 2 отскока)
Buzz roll:      RRRRrr LLLLll (неконтролируемые отскоки)
```

### Техника buzz roll

1. Удар вниз с нормальной силой
2. **Прижмите** палочку сразу после удара — не сильно, но достаточно
3. Палочка «дребезжит» о мембрану несколько раз
4. Быстро чередуйте руки

Контроль давления — ключ: слишком сильное прижатие глушит, слишком слабое — обычный отскок.

### Open buzz vs Closed buzz

| Тип | Давление | Звук |
|-----|---------|------|
| **Open buzz** | Минимальное | Много отскоков, ярко |
| **Closed buzz** | Умеренное | Плотный «шшш» |
| **Dead stroke** | Максимальное | Глухой удар без отскока |

## Применение в марширующем оркестре

Маршевые барабанщики (drumline) — главные хранители традиции рудиментов. В drum corps (DCI):

- **Drag** используется для украшения мелодических линий малого барабана
- **Buzz roll** — обязательный элемент solo и ensemble writing
- **Ruff** создаёт текстуру в tutti-секциях
- Сложнейшие комбинации рудиментов — hallmark стиля Blue Devils, Bluecoats, Crown

В маршевой музыке drag всегда открытый (ruff-стиль) — иначе украшение не слышно на улице.

## Применение в джазе

В джазе drag и buzz roll используются совершенно иначе:

### На малом барабане
**Drag** как акцент: bar 3 или 4 одиночного такта — llR вместо обычного удара. Добавляет «вес».

### Buzz roll в джазовых brakes
Барабанщик играет нарастающий buzz roll в конце фразы перед паузой. Классический прием в swing и bebop.

**Музыканты:** Papa Jo Jones, Buddy Rich, Philly Joe Jones — все виртуозно владели drag и buzz roll как языком джазовой экспрессии.

### На ride-тарелке
Тихий buzz roll на edge тарелки — шелестящий фоновый текстурный элемент в тихих jazz ballads.

## Упражнения

### 1. Базовый drag
```
BPM 50: llR · llR · llR ...
```
Убедитесь, что оба grace note слышны и быстры.

### 2. Чередование
```
llR rrL llR rrL ...
```
Четверти на 60 BPM.

### 3. Buzz roll crescendo
Начните тихо (pp), нарастайте к ff за 8 тактов. Следите, чтобы «дребезг» оставался равномерным.

### 4. Drag в fill
```
Обычный грув × 3 такта → Fill: том llR(малый) том crash
```

## Типичные ошибки

1. **Grace notes слишком громкие** — drag превращается в три одинаковых удара
2. **Неравномерный buzz** — одна рука дребезжит больше другой
3. **Слишком сильное прижатие при buzz** — звук глохнет вместо дребезга
4. **Путаница drag/flam** — drag = ДВА grace note (ll), flam = ОДИН (l)

> **Историческая справка:** традиция рудиментов восходит к европейским военным барабанщикам XVII–XVIII веков. Drag и buzz roll использовались для подачи команд на поле боя — громко, выразительно и узнаваемо. Сегодня это живая традиция, передаваемая через drum corps и маршевые ансамбли.
