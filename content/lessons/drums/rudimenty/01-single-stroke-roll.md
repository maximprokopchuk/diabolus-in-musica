# Single Stroke Roll

## Что такое рудимент

**Рудименты** (rudiments) — это стандартизированные паттерны чередования рук, которые составляют алфавит барабанной техники. Так же, как пианист отрабатывает гаммы и арпеджио, барабанщик строит технику на рудиментах. NARD (National Association of Rudimental Drummers) и PAS (Percussive Arts Society) стандартизировали 40 основных рудиментов.

Single stroke roll — **первый и важнейший рудимент**: поочерёдные одиночные удары правой и левой рукой.

## Нотация и формула

```
R L R L R L R L ...
```

Буква **R** (Right) — правая рука, **L** (Left) — левая. Удары чередуются строго поочерёдно, без пропусков.

## Правильная техника удара

### Захват палочки (grip)

Прежде чем отрабатывать скорость — поставьте правильный захват:

| Тип захвата | Описание | Применение |
|-------------|----------|-----------|
| **Matched grip** | Обе руки держат одинаково, как отвёртку | Рок, поп, фанк |
| **Traditional grip** | Левая рука снизу (как у маршевых барабанщиков) | Джаз, марш |

Для начала рекомендуется **matched grip** — он интуитивно понятен и равномерно развивает обе руки.

### Принцип отскока (rebound)

Секрет быстрого single stroke — **не давить**, а **использовать отскок**:

1. Бросьте палочку вниз — она отскочит сама
2. Позвольте ей вернуться, лишь слегка контролируя движение
3. Следующий удар — снова бросок

При правильной технике рука затрачивает минимум мускульных усилий. Именно это позволяет играть быстро без зажатости.

### Высота подъёма

Высота определяет **динамику** (громкость):

| Высота подъёма | Динамика | Обозначение |
|---------------|---------|-------------|
| 1–2 см | Тихо | pp / p |
| 5–7 см | Средне | mp / mf |
| 20–25 см | Громко | f / ff |
| Полный замах | Максимально громко | fff |

## Упражнения на развитие

### 1. Медленный контроль (BPM 40–60)

Начните на 40 BPM, каждый удар — восьмая нота. Следите за:
- Полным отскоком
- Одинаковым звуком правой и левой руки
- Расслабленностью плеч и запястий

### 2. Пирамида темпа

```
40 BPM → 50 → 60 → 70 → 80 → 90 → 100 → 110 → 120
```

Переходите к следующему темпу только когда текущий звучит ровно и без усилий.

### 3. Акцентирование

Один из самых полезных вариантов упражнения — **акцент на каждую четверть**:

```
> - - - > - - - > - - - > - - -
R L R L R L R L R L R L R L R L
```

Затем смещайте акцент:
- Акцент каждые 3 ноты (создаёт ощущение полиритмии)
- Акцент только правой рукой
- Акцент только левой рукой

### 4. Развитие слабой руки

Большинство людей — правши, и левая рука отстаёт. Упражнение:
- Играйте 5 минут только левой рукой
- Играйте зеркальные паттерны: **LRLR** вместо RLRL

## Применение в грувах и соло

Single stroke roll — это не просто техническое упражнение. В реальной игре вы используете его:

- **В fills** — быстрые пробежки по томам
- **На hi-hat** — скоростные паттерны в фанке и метале
- **В соло** — непрерывные потоки шестнадцатых

### Пример fill с single stroke:

```
Такт 1-3: обычный грув
Такт 4 (fill): R L R L R L R L по томам → crash на "1"
```

## Типичные ошибки

1. **Зажатость** — сжимание палочки слишком крепко блокирует отскок
2. **Неравномерность** — правая громче левой; тренируйте отдельно
3. **Торопливость** — переход на быстрый темп без контроля ровности
4. **Опускание локтей** — локти должны свободно «лететь», не прижиматься к телу

> **Ключевой принцип:** Single stroke roll — фундамент всех остальных рудиментов. Потратьте на него столько времени, сколько нужно, чтобы обе руки звучали **идентично**. Это инвестиция, которая окупится во всём остальном.
