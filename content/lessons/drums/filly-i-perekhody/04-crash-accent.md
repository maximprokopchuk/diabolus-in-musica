# Crash Accent

## Роль crash тарелки

**Crash cymbal** (крэш) — тарелка с широким, громким, быстро нарастающим и затем затухающим звуком. Функция в музыке:

1. **Акцентирование начала раздела** — «1» нового куплета/припева
2. **Финал fill** — резкое завершение
3. **Акцент по тексту** — совпадение с сильным словом или нотой вокала/гитары
4. **Динамический пик** — самая громкая точка в музыкальной фразе

## Анатомия правильного удара crash

### Неправильный удар (распространённая ошибка)

Многие начинающие барабанщики бьют crash **сверху вниз**, как по малому. Результат:
- Тарелка «задерживается» (не «летит»)
- Звук короткий и «деревянный»
- Рука получает отдачу и риск травмы

### Правильный удар: скользящий (glancing blow)

Правильный crash — **скользящий удар краем палочки по краю тарелки**:

```
Палочка движется в сторону и чуть вниз
        ↘
    тарелка (удар вскользь, не прямо вниз)
```

1. Удар **кончиком** или **плечом** палочки (shoulder)
2. Движение **вперёд и вниз** — не просто вниз
3. Тарелка «летит» от удара и свободно вибрирует
4. Рука продолжает движение вниз, не задерживается

### Crash with ride hand

Большинство crash тарелок располагается слева (у правши). Но удар можно наносить:
- **Правой рукой** — основной способ (рука «летит» поверх малого)
- **Левой рукой** — при определённых fill-паттернах
- **Обеими одновременно** — в финальных аккордах, очень мощный звук

## Crashc accent на «1»: техника

### Ситуация: конец fill, crash на «1»

```
Fill:   R L R L R L R L (такт 4)
Crash:  ↑ (на «1» следующего такта)
```

При этом обычно:
- Правая рука делает crash
- Левая рука может ударить малый одновременно («crash-snare unison»)
- Бочка тоже бьёт на «1» — тройной акцент

```
«1» нового раздела:
  Crash (правая) + Малый (левая) + Бочка (нога)
  = максимальный акцент
```

## Демпфирование crash (чокинг)

**Чокинг** (choking) — захват тарелки рукой сразу после удара, останавливающий вибрацию. Звук резко обрывается: «тщщ!»

Применение:
- В конце фраз для резкой остановки
- В джаз-фьюжн для эффекта
- В записи — если crash «размазывается» в микс и мешает следующей фразе

Техника: сразу после удара **сжать тарелку пальцами** (большой + указательный) у края.

## Типы crash тарелок

| Тип | Размер | Звук | Применение |
|-----|--------|------|-----------|
| **Thin crash** | 14–16" | Быстрый, яркий | Лёгкие акценты, поп |
| **Medium crash** | 16–18" | Сбалансированный | Универсально |
| **Heavy crash** | 18–20" | Громкий, тёмный | Рок, метал |
| **China crash** | 16–20" | Грязный, агрессивный | Метал, хардкор |
| **Splash** | 8–12" | Короткий, острый | Лёгкие акценты в джазе/фанке |

## Использование нескольких crash

Большинство профессиональных установок имеет **2–3 crash тарелки**:

- **Левый crash** (ближе к хэту): обычно меньше, тоньше — для «обычных» акцентов
- **Правый crash** (над ride или слева): больше, тяжелее — для мощных финалов
- **China** (по желанию): специальный звук

### Преимущество двух crash

```
Fill → левый crash: умеренно
Fill → правый crash: мощно
Оба одновременно: максимально
```

Барабанщик может управлять **степенью акцента**, просто выбирая тарелку.

## Crash accent в джазе: ride bell

В джазе crash используется редко. Аналог сильного акцента — **удар по колокольчику** (bell) ride тарелки:

```
Ride bell accent: чёткий «динь!» — акцент без огромного крэш-взрыва
```

Также в джазе часто используется **crash на edge ride** — удар по краю ride тарелки, который даёт звук, похожий на crash, но тоньше.

## Crash с бочкой: «downbeat punch»

Одновременный crash + kick drum — «downbeat punch». Это физически мощный удар, который слушатель **ощущает телом**:

```
«1»: CRASH (правая) + KICK (нога правая)
```

Это стандарт в роке и метале. Crash без бочки звучит «незавершённо».

## Упражнения

### 1. Изолированный crash
Просто отрабатывайте удар crash: правильный скользящий, без задержки руки. 20 ударов с паузами.

### 2. Грув + crash на «1»
Три такта грув → crash на «1» четвёртого такта (без fill). Просто удар на «1». Это учит попадать точно.

### 3. Fill + crash
Три такта грув → fill → crash на «1». Начните с простого fill (4 восьмых по томам).

### 4. Чокинг
Сыграйте crash → сразу заглушите рукой. Контролируйте длину sustain.

## Типичные ошибки

1. **Удар сверху вниз** — тарелка «застревает», звук деревянный
2. **Crash без бочки** — слабый финал
3. **Crash слишком часто** — теряет значение
4. **Запоздалый crash** — попадает на «е-1» вместо «1», разрушает синхрон

> **Принцип:** crash — это **восклицательный знак**. Используйте осознанно. Каждый crash должен быть оправдан музыкально. Барабанщик, который бьёт crash на каждой доле, — как человек, кричащий каждое слово: внимание очень быстро пропадает.
