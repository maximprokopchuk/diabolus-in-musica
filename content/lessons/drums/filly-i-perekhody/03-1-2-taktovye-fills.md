# 1-тактовые и 2-тактовые Fills

## Классификация fills по длине

Fills можно классифицировать по тому, сколько тактового пространства они занимают:

| Длина fill | Когда используется | Характер |
|-----------|-------------------|---------|
| **1/2 доли** | Короткий акцент | Минималистично, неожиданно |
| **1 доля** | Короткий переход | Лаконично |
| **2 доли** | Стандартный конец такта | Часто используется |
| **1 такт** | Полный fill | Классика |
| **2 такта** | Большой переход | Мощно, требует мастерства |
| **4 такта** | Барабанное вступление | Специальные случаи |

## 1-тактовый fill: детали

Один такт в 4/4 = 4 доли = 8 восьмых = 16 шестнадцатых

### Варианты заполнения 1 такта

**Вариант А: Шестнадцатые по томам**
```
T1 T1 T1 T1 T2 T2 T2 T2 T3 T3 T3 T3 FL FL FL FL → CRASH
1  e  +  a  2  e  +  a  3  e  +  a  4  e  +  a
```
4 ноты на каждый том, последовательно от высокого к напольному.

**Вариант Б: Восьмые ноты (медленнее)**
```
T1 T2 T3 FL T1 T2 T3 FL → CRASH
1  +  2  +  3  +  4  +
```
Более «тяжёлый», подходит для хэви-рока.

**Вариант В: Смешанный — малый + тома**
```
S  S  S  S  T1 T1 T2 T2 T3 T3 FL FL → CRASH
1  e  +  a  2  e  +  a  3  e  +  a
```
Малый в начале создаёт «разгон», тома — «посадку».

**Вариант Г: Триоли**
```
T1 T2 T3  T1 T2 T3  T1 T2 T3  T1 T2 T3  → CRASH
  триоль    триоль    триоль    триоль
```
12 нот = 4 триоли. Shuffle-характер.

**Вариант Д: Смешанные длины**
```
S S S S T1 T1 T3 FL → CRASH
1 e + a 2  e  +  a
(4 шестнадцатых малый + 4 восьмых томы)
```

## 2-тактовый fill

2-тактовый fill — серьёзное заявление. Он занимает **8 долей** и требует:
1. Чёткой внутренней структуры (иначе звучит как «шум»)
2. Нарастания динамики к концу
3. Ясного попадания в «1»

### Структура 2-тактового fill

**Логика «вопрос-ответ»:**
```
Такт 1 fill: «вопрос» — введение, не слишком насыщенно
Такт 2 fill: «ответ» — кульминация, нарастание к crash
```

**Пример:**
```
Такт 1: S  S  T1 T1 T2 T2 T3 T3 (восьмые, умеренно)
Такт 2: FL FL FL FL FL FL FL FL → CRASH (шестнадцатые, мощно)
```

### Динамическое нарастание в 2 тактах
```
Такт 1: pp → mp (нарастает внутри такта)
Такт 2: mp → ff (нарастает к crash)
```

## Роль томов

**Тома** (тоm-томы) — главный инструмент fills. Установка обычно включает:

| Том | Позиция | Тон |
|-----|---------|-----|
| **Rack tom 1** | Слева над бочкой | Высокий |
| **Rack tom 2** | Правее | Средний |
| **Floor tom** | Справа от барабанщика | Низкий, глубокий |

Стандартный fill движется **сверху вниз** (от высокого к низкому) — это соответствует «падению» и «завершению» фразы. Обратное движение (снизу вверх) — эффект «взлёта».

### Настройка томов

Тома должны быть настроены **в интервалах**. Классика: квинта или кварта между томами:
```
Tom 1: D (ре)
Tom 2: A (ля)  — квинта ниже
Floor: E (ми) — квинта ниже
```

## Комбинации малого и тарелок

Fill не обязан использовать только тома. Интересные комбинации:

### Малый + открытый хэт
```
S  S  oH S  S  oH S  S  → CRASH
1  e  +  a  2  e  +  a
(oH = открытый хэт)
```

### Кросс-стик + тома
```
CS CS T1 T1 T2 T2 FL FL → CRASH
(CS = cross stick — тихий щелчок по ободу малого)
```

### Ride bell + тома
```
RB RB RB RB T1 T2 T3 FL → CRASH
(RB = ride bell — колокольчик ride)
```

## China Cymbal в fills

**China cymbal** — тарелка с «грязным» резким звуком, повёрнутая вверх ногами. В метале и хардкоре используется как альтернатива crash в конце fill.

```
Fill → China (вместо Crash) — более агрессивный финал
```

## Упражнения

### 1. Библиотека fills
Придумайте и отработайте 10 разных 1-тактовых fills. Записывайте их в нотацию или видео.

### 2. Контекст
Каждый fill из библиотеки — сыграйте 3 такта грув + fill. Меняйте fill каждые 4 такта.

### 3. 2-тактовый fill
Сыграйте 6 тактов грув + 2 такта fill. Убедитесь, что «1» после fill — точная.

### 4. Импровизация
BPM 90, клик, 4 такта грув + fill, в петле 10 минут. Каждый fill — новый, спонтанный.

## Типичные ошибки

1. **Все fills одинаковые** — один паттерн на все случаи. Развивайте разнообразие
2. **Fill «уходит» из темпа** — на сложном fill tempo drift. Тренируйте с кликом
3. **Crash слишком тихий** — после fill crash должен «взрываться»
4. **Слишком много fills** — каждые 2 такта fill мешает слушать музыку

> **Принцип:** думайте о fills как о **пунктуации в речи**. Запятая (короткий fill), точка (длинный fill), восклицательный знак (2-тактовый fill с crash). Слишком много восклицательных знаков — текст теряет смысл.
