# Subdivisions и Клик

## Что такое subdivision

**Subdivision** (деление) — это разбивка основной доли на более мелкие единицы. Понимание subdivision — ключ к стабильному темпу и ритмической точности.

```
Такт 4/4:
Доли (четверти):  1    2    3    4
Восьмые:          1  + 2  + 3  + 4  +
Шестнадцатые:     1 e + a 2 e + a 3 e + a 4 e + a
Триоли:           1 - - 2 - - 3 - - 4 - -
```

## Зачем считать subdivision

Представьте, что вы идёте и считаете шаги: 1, 2, 3, 4. Это четверти.

Теперь считайте каждый маленький шажок внутри: 1-и-2-и-3-и-4-и. Это восьмые.

Ещё мельче: 1-э-и-а-2-э-и-а. Это шестнадцатые.

Когда subdivision «включён» в голове, темп **автоматически стабилизируется** — потому что между долями больше нет «пустого» пространства.

## Subdivision с метрономом: техники

### 1. Клик на четвертях, subdivision в голове

Самое распространённое: метроном бьёт четверти, вы **мысленно** делите их на восьмые или шестнадцатые.

```
Клик:       1         2         3         4
Вы думаете: 1  e  +  a  2  e  +  a  3  ...
Вы играете: x  x  x  x  x  x  x  x  x  ...
```

### 2. Клик на восьмых

Метроном бьёт восьмые. Это удвоенный темп — вы слышите «между долями» как клик.

```
Клик: x x x x x x x x (каждая восьмая)
```

Полезно для медленных темпов: при BPM 50 (четверти) между кликами — огромный промежуток. Если поставить восьмые (клик 100 BPM) — легче держать пульс.

### 3. Клик на шестнадцатых

Максимальная детализация — клик бьёт каждую шестнадцатую. Это «сетка» для точного попадания.

Осторожно: при высоком BPM шестнадцатые-клик очень быстрый и может запутать.

### 4. Клик на триолях

```
Клик: 1 - - 2 - - 3 - - 4 - - (каждая нота триоли)
```

Используется при работе с shuffle и swing паттернами.

## Восьмые (8th notes)

**Восьмые** — самая распространённая subdivision в рок и поп музыке.

```
1 + 2 + 3 + 4 +
↑   ↑   ↑   ↑   (доли)
  ↑   ↑   ↑   ↑ («и» — между долями)
```

«И» («+» в нотации) — восьмые между долями. Барабанщик, который точно слышит «и», гораздо стабильнее, чем тот, кто думает только о долях.

### Упражнение: восьмые вслух
Играйте грув и **считайте вслух**: «один-и, два-и, три-и, четыре-и». Сначала это неудобно, но со временем счёт уходит внутрь.

## Шестнадцатые (16th notes)

```
1 e + a 2 e + a 3 e + a 4 e + a
```

Четыре ноты на долю. Обозначение: **1, e, +, a** (один, э, и, а).

Шестнадцатые subdivision критически важны в фанке, где ghost notes, kick и хэт все расположены в шестнадцатых.

### Тест: считать шестнадцатые под метроном
BPM 60. Считайте «1-э-и-а, 2-э-и-а, 3-э-и-а, 4-э-и-а» — ровно, без ускорения. Это сложнее, чем кажется.

## Триоли

**Триоль** — три ноты на долю:
```
1 - - 2 - - 3 - - 4 - -
(три равных промежутка на каждую долю)
```

Триоли создают **swing feel** и используются в блюзе, джазе, кантри.

### Swing = триольные восьмые

Swing feel буквально означает: восьмые играются как первая и третья ноты триоли:
```
Триоль:  1 - - 2 - - (три ноты)
Swing 8th: 1   2   (первая и третья)
```

Именно поэтому работа с триольным кликом помогает освоить swing.

## Переключение между subdivision

Важный навык: **переключаться между subdivision** в реальном времени.

### Упражнение: смена subdivision
```
4 такта восьмые хэт (x x x x x x x x)
4 такта шестнадцатые хэт (x x x x x x x x x x x x x x x x)
4 такта триоли хэт (xxx xxx xxx xxx)
```

Клик на четвертях — не меняется. Меняется только ваше внутреннее деление.

## Subdivision и tempo drift

**Главная функция subdivision** — предотвращать tempo drift.

Когда вы думаете «между долями», каждый «промежуток» заполнен. Некуда «проваливаться» или «проскакивать».

```
Без subdivision: 1 ... ... ... 2 ... (пустота → дрейф)
С subdivision:   1 e + a 2 e + a (заполнено → стабильно)
```

## Упражнения

### 1. Клик + subdivision вслух
Метроном 70 BPM. Считайте вслух шестнадцатые 5 минут без остановки.

### 2. Ghost notes как subdivision
Играйте грув, но каждую шестнадцатую добавьте ghost note на малом (очень тихо). Это «физически» включает subdivision.

### 3. «Выключить subdivision»
После 10 минут игры с subdivision-мышлением — переключитесь на счёт долей. Чувствуете, как темп становится менее стабильным?

### 4. Смена уровня subdivision
Одна песня, три повтора:
- Повтор 1: думаете восьмыми
- Повтор 2: думаете шестнадцатыми
- Повтор 3: думаете триолями
Сравните ощущения.

> **Мудрость ветерана:** «Думай мелко, играй точно» — принцип, который старые джазовые барабанщики передавали молодым. Чем мельче subdivision у вас в голове — тем точнее попадание каждой ноты.
