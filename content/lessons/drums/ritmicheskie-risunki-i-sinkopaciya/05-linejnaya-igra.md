# Линейная игра (linear drumming)

## Что такое linear drumming

**Linear drumming** («линейная игра») — техника, при которой **ни один инструмент не звучит одновременно с другим**. Каждый удар занимает уникальную позицию во времени — нет никаких «нот в унисон».

Это прямая противоположность стандартной барабанной игре, где, например, хэт и бочка часто звучат вместе (на первой доле).

## Принцип линейной игры

**Традиционный паттерн (не линейный):**
```
Хэт:    X X X X X X X X
Снейр:  - - O - - - O -
Бочка:  O - - - O - - -
```
На первой доле хэт и бочка совпадают — это **вертикальный** (аккордовый) ритм.

**Линейный паттерн:**
```
Хэт:    X - X - X - X -
Снейр:  - X - - - X - -
Бочка:  - - - X - - - X
```
Ни одно из событий не совпадает. Каждая нота занимает своё уникальное место — это **горизонтальный** (мелодический) ритм.

## Откуда взялась линейная игра

Концепция linear drumming развивалась в:

**Фанке:** Barney McKay, David Garibaldi (Tower of Power) — синкопированные паттерны без дублирования.

**Электронной музыке:** секвенсоры программируют каждый звук в своё место — человеческие барабанщики стали имитировать эту точность.

**Современном роке и поп-роке:** линейные паттерны дают «воздух» и пространство в современных продюсированных треках.

## Todd Sucherman

**Todd Sucherman** — барабанщик группы Styx, один из ведущих пропонентов и популяризаторов линейной игры.

### Его вклад

Sucherman опубликовал образовательный материал **«Methods & Mechanics»** (DVD и книга) — один из лучших ресурсов по линейной игре.

**Ключевые принципы Sucherman:**

1. **Музыкальность прежде техники:** линейная игра — это не трюк, а музыкальный язык
2. **Плавность:** переходы между инструментами должны быть мелодичными, как мелодия на барабанах
3. **Независимость конечностей:** каждая рука и нога должна быть абсолютно независимой
4. **Динамика:** линейные паттерны особенно красивы с тонкой динамической нюансировкой

**Характерные приёмы Sucherman:**
- Ghost notes снейра между каждым «основным» ударом
- Бочка в неожиданных, но точных позициях
- Открытый хэт в off-beat позициях для тембровой вариации
- Переходы (fills) построены по тому же линейному принципу

## Базовые линейные паттерны

### Паттерн 1: Простой линейный фанк

```
Шестнадцатые: 1  e  +  a  2  e  +  a  3  e  +  a  4  e  +  a
Хэт:          X        X        X        X
Снейр:           X        X        X        X
Бочка:              X        X        X        X
```

Три голоса делят 16 шестнадцатых на три группы по одной.

### Паттерн 2: Линейный с ghost notes

```
1  e  +  a  2  e  +  a
H  s  H  B  H  s  H  B
```
(H = хэт, s = ghost note снейра, B = бочка)

Хэт, снейр, хэт, бочка — цикл из 4 шестнадцатых.

### Паттерн 3: Asymmetric linear

```
1  e  +  a  2  e  +  a  3  e  +  a  4  e  +  a
H  s  B  H     s  H  B     H  s     B  H  s  H
```

Асимметричный — разные длины пауз создают синкопированный характер.

## Нотация и разработка паттернов

При создании линейных паттернов полезен **метод заполнения:**

1. Нарисуйте сетку шестнадцатых (16 клеток для такта 4/4)
2. Назначьте каждой клетке один из голосов: H (хэт), S (снейр), B (бочка), R (ride) и т.д.
3. Убедитесь, что никакие два голоса не совпадают
4. Сыграйте паттерн

**Правило:** в простейшем линейном паттерне из 16 шестнадцатых каждая позиция либо занята одним инструментом, либо пауза.

## Сложность линейной игры

### Почему это трудно

**Традиционная игра:** руки и ноги часто движутся вместе (хэт + бочка, снейр + хэт). Это **проще координационно** — конечности синхронизированы.

**Линейная игра:** каждая конечность движется независимо, никогда не совпадая с другой. Это требует **истинной четырёхсторонней независимости**.

### Развитие независимости

Упражнение «пирамида»:

1. Только руки (хэт + снейр), линейно
2. Только руки + правая нога (хэт + снейр + бочка)
3. Все четыре конечности

На каждом уровне — полная линейность (ничего не совпадает).

## Линейные переходы (linear fills)

Переходы в линейном стиле — мощный приём:

```
Линейный fill через барабаны:
R  L  B  R  L  B  R  L  (чередование рук и бочки)
```

Или через тарелки:
```
Crash  Snare  Kick  Ride  Tom  Snare  Kick  Crash
```

Каждый «голос» не повторяется сразу — это создаёт мелодичный, «разговорный» характер переходов.

## Упражнения

**1. Хэт-снейр линейно (две руки)**
Только руки: хэт правой, снейр левой, строго чередуя. Никакого совпадения. 60 BPM, шестнадцатые.

**2. Добавление бочки**
К предыдущему добавьте бочку там, где нет рук. Например: хэт — снейр — бочка — хэт — снейр — бочка.

**3. Линейный паттерн по Sucherman**
Выучите один конкретный паттерн из «Methods & Mechanics». Медленно, по конечностям. Темп 50 BPM, потом 60, 70...

**4. Импровизация с правилом**
Играйте свободно, но с одним правилом: никакие два инструмента не звучат одновременно. Это развивает линейное мышление интуитивно.

> **Линейная игра** меняет то, как вы думаете о барабанной установке — не как о наборе инструментов, которые совпадают, а как о **мелодическом инструменте**, где каждый звук имеет свою позицию, как нота в мелодии. Освойте это мышление — и ваша игра станет значительно более изощрённой и музыкальной.
