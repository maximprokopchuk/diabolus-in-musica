# Овердрайв и фузз для баса

## Специфика перегруза для баса

Главная проблема при использовании гитарного перегруза на басу — **потеря низких частот**. Большинство гитарных педалей обрезают бас при перегрузе, что делает звук тонким и «картонным».

Поэтому существуют специальные **бас-педали перегруза**, которые сохраняют низкие частоты.

## Основные подходы

### Parallel (параллельный) сигнал

Самый популярный подход для баса:

```
Бас → [Разделитель] → [Чистый сигнал]  ┐
                    → [Перегруз]         ┤ → [Сумматор] → Усилитель
```

Чистый сигнал сохраняет **полноту и плотность баса**, а перегруз добавляет характер и агрессию. Можно регулировать баланс между ними.

Этот принцип встроен в большинство качественных бас-педалей перегруза.

### Стандартный (series)
Весь сигнал проходит через перегруз. Требует очень грамотно настроенных педалей, иначе теряется low-end.

## Овердрайв для баса

**Овердрайв** даёт мягкое, «тёплое» насыщение — от лёгкого хруста до плотного драйва.

### Darkglass Electronics
Финский бренд, ставший стандартом в современном метале и прогрессивной музыке.

| Педаль | Характер |
|-------|---------|
| **Darkglass B3K** | Агрессивный, чёткий, «стеклянный» перегруз |
| **Darkglass B7K** | B3K + встроенный EQ и DI |
| **Darkglass Alpha·Omega** | Два типа перегруза в одной педали |
| **Darkglass Microtubes X** | Топ линейки, многополосная обработка |

**Характерный звук Darkglass:** слышен у Adam Nolly Getgood (Periphery), Steve DiGiorgio, многих djent и прог-метал-басистов.

## Фузз для баса

**Фузз** — более агрессивный и «грязный» тип перегруза. Звук ближе к синтезатору или саксофону. Особенно популярен в:
- Психоделическом роке
- Стоунер-роке и думе
- Фанке (с октавером)
- Краут-роке

### Bass Big Muff Pi (Electro-Harmonix)

**Bass Big Muff** — один из самых известных фузз-педалей специально для баса.

**Отличие от гитарного Big Muff:**
- Сохраняет больше низких частот
- Имеет переключатель **Dry/Wet blend** или встроенный clean blend
- Мягче на атаке

**Версии:**
- **Bass Big Muff Pi** — классика с clean blend
- **Nano Bass Big Muff** — компактная версия
- **Bass Russian Big Muff** — «советский» фузз с более тёмным характером

**Используют:** St. Vincent (для агрессивных партий), Tame Impala (Kevin Parker на записях), многие стоунер-группы.

## Сохранение low-end: практические советы

1. **Всегда ищите педали с clean blend** — это самый простой способ сохранить бас
2. **Контролируйте Bass-ручку на педали** — не добавляйте слишком много баса, это может создать «грязь»
3. **EQ после педали** — срезайте частоты вокруг 200–400 Гц (там скапливается «каша»), поднимайте 60–80 Гц для плотности
4. **Тестируйте в группе** — то, что хорошо звучит соло, может исчезнуть в миксе

## Стековые комбинации

Многие басисты используют несколько педалей вместе:

**Классическая схема:**
```
Бас → Компрессор → Овердрайв (всегда включён, лёгкий) → Фузз (для акцентов) → Усилитель
```

**Современная схема (djent/progressive):**
```
Бас → Darkglass B7K (всегда) → Alpha·Omega (для тяжёлых секций) → Усилитель
```

> **Совет:** начните с одной качественной педали с clean blend. Освойте её полностью, прежде чем добавлять вторую. Лучше хорошо знать одну педаль, чем плохо пользоваться пятью.
