# Арпеджио на басу

## Что такое арпеджио

**Арпеджио** — это ноты аккорда, сыгранные поочерёдно (не одновременно). Для бас-гитаристов арпеджио — основной инструмент гармонического мышления: вместо «игры корней» музыкант проходит через весь аккорд, выявляя его качество.

Знание арпеджио позволяет:
- Создавать интересные мелодические линии над гармонией
- Чётко указывать тональный центр
- Импровизировать осмысленно в любом жанре

## Основные типы арпеджио

### Мажорное арпеджио (1 - 3 - 5)

Построено из корня, большой терции и чистой квинты.

**Аппликатура от ноты A (открытая A-струна):**

```
G|------2--|
D|------2--|
A|0-4------|
E|---------|
```

- A (открытая A) → C# (4-й лад A) → E (2-й лад D)

Звучит: светло, устойчиво, «мажорно».

### Минорное арпеджио (1 - b3 - 5)

Малая терция вместо большой.

**Аппликатура от A (открытая A-струна):**

```
G|------2--|
D|------2--|
A|0-3------|
E|---------|
```

- A (открытая A) → C (3-й лад A) → E (2-й лад D)

Звучит: тёмно, меланхолично.

### Доминантовое арпеджио (1 - 3 - 5 - b7)

Мажорное трезвучие + малая септима. Ключевое для V-ступени.

**От G (3-й лад E):**

```
G|---------|
D|---0--3--|
A|---2-----|
E|3--------|
```

- G (E-3) → B (A-2) → D (D-0) → F (D-3)

Звучит: напряжённо, требует разрешения.

### Maj7 арпеджио (1 - 3 - 5 - 7)

Мажорное трезвучие + большая септима. Характерно для тоники в джазе.

**От C (8-й лад E):**

```
G|------4--|
D|---5-----|
A|------7--|
E|8--------|
```

- C (E-8) → E (A-7) → G (D-5) → B (G-4)

Звучит: мягко, романтично, «джазово».

### m7 арпеджио (1 - b3 - 5 - b7)

Минорное трезвучие + малая септима. Ii-ступень в мажоре, i-ступень в миноре.

**От D (5-й лад A):**

```
G|------5--|
D|---3--7--|
A|5--------|
E|---------|
```

- D (A-5) → F (D-3) → A (D-7) → C (G-5)

## Горизонтальные аппликатуры (вдоль грифа)

Все арпеджио можно играть **по одной или двум струнам**, перемещаясь вдоль грифа. Это удобно для длинных фраз и runs.

### Мажорное арпеджио по одной струне (E-струна)

```
E|8--12--15--20--|
   C   E   G   C
```

Полное арпеджио C-мажор на E-струне.

## Вертикальные аппликатуры (поперёк грифа)

В одной позиции (4–5 ладов) можно сыграть полное арпеджио в четыре ноты на четырёх струнах.

### Мажорное арпеджио, позиция «CAGED-образная»

| Струна | Лад | Нота |
|--------|-----|------|
| E | 8 | C |
| A | 7 | E |
| D | 5 | G |
| G | 5 | C |

Четыре ноты в пределах 3 позиций — удобно для быстрых фраз.

## Применение арпеджио в линии баса

### Approach & Arpeggiate

Один из самых эффективных приёмов: подойти к аккорду снизу хроматически и пройтись по арпеджио.

```
Am → Dm:
A  → A  → C  → E  → D  → F  → A
```

Это превращает простой аккорд-переход в мелодическую фразу.

### Верхняя нота арпеджио как мелодия

Если партия баса заканчивается на **квинте** или **септиме** арпеджио, это создаёт «незавершённость», которая тянет вперёд. Профессиональные бас-гитаристы выбирают конечную ноту фразы осознанно.

## Практические упражнения

1. **Хроматический арпеджио-тур:** играйте maj7 арпеджио от C, затем поднимайтесь на полутон (C# maj7), потом D maj7 и т.д.
2. **ii-V-I арпеджио:** Dm7 → G7 → Cmaj7 — через арпеджио. Стандарт джаза.
3. **Ритмическое арпеджио:** играйте арпеджио не монотонно, а с ритмическим акцентом (синкопы, пунктирный ритм).

> **Ключевой принцип:** арпеджио — это не просто «упражнение». Это **словарный запас** вашего гармонического языка. Чем свободнее вы играете арпеджио всех типов в любой тональности, тем богаче ваши возможности в любом жанре.
