# Октавы и аккордовые пассажи

## Техника октав

**Октавы** — один из мощнейших эффектов фортепиано. Правая рука берёт октавы пальцами **1 и 5**, левая — **5 и 1**.

### Виды движения октавами

**Legato-октавы:** запястье остаётся относительно неподвижным, работают пальцы. Используются в медленных лирических пассажах.

**Staccato-октавы:** бросок запястья. Рука «отпрыгивает» после каждой октавы — как мячик. Применяются в быстрых, мощных пассажах (Бетховен, Лист).

**Tremolo октав:** быстрое чередование двух октав. Создаёт эффект непрерывного звука.

### Частые ошибки

| Ошибка | Последствие | Решение |
|--------|-------------|---------|
| Зажатое запястье | Боль, усталость | Мягкое, свободное запястье |
| Только пальцы, без руки | Нет силы | Подключите предплечье |
| Только рука, без пальцев | Нет точности | Фиксируйте расстояние 1–5 |

## Аккордовые пассажи

Последовательность аккордов в быстром темпе требует:
1. **Фиксации формы руки** — пальцы заранее принимают форму следующего аккорда
2. **«Полёта»** — рука двигается к следующему аккорду, пока ещё звучит предыдущий
3. **Расслабления** между ударами — напряжение только в момент удара

### Упражнение на подготовку

Играйте аккорд — отпустите и держите пальцы в форме следующего — «летите» к новой позиции — берите. Сначала очень медленно, потом быстрее.

## Двойные ноты

**Двойные ноты** (терции, сексты, октавы) — два звука в одной руке. Для беглости в двойных нотах нужна специальная аппликатура и тренировка.

Гамма в терциях правой руки (До мажор):
```
C–E / D–F / E–G / F–A / G–B / A–C / B–D / C–E
1–3 / 1–3 / 1–3 / 1–3 / 1–3 / 1–3 / 2–4 / 1–3
```

> **Предупреждение:** октавная техника — серьёзная нагрузка на запястье. Начинайте с маленьких порций (5–10 минут), останавливайтесь при первом намёке на боль. Разминайтесь перед игрой октавами.
