# Основы синтеза звука

## Четыре блока синтезатора

Любой синтезатор — это цепь из четырёх основных элементов:

```
Осциллятор (VCO) → Фильтр (VCF) → Усилитель (VCA) → Звук
                                          ↑
                              Огибающие (ADSR) и LFO управляют всем
```

## 1. Осциллятор (VCO/DCO)

**Осциллятор** генерирует исходную волну. Основные формы:

| Форма волны | Характер |
|------------|---------|
| **Синусоидальная** | Чистый, без обертонов, мягкий |
| **Прямоугольная (Square)** | Полый, «деревянный», богатый нечётными гармониками |
| **Пилообразная (Sawtooth)** | Резкий, яркий, богатый всеми гармониками |
| **Треугольная** | Мягче прямоугольной, напоминает флейту |
| **Шум (Noise)** | Перкуссия, спецэффекты |

Большинство синтезаторов имеют несколько осцилляторов — их детюнинг (небольшое расстройство) создаёт «толстый» хорусный звук.

## 2. Фильтр (VCF)

**Фильтр** убирает определённые частоты.

- **Low-pass filter (LPF)** — пропускает низкие, срезает высокие. Самый распространённый.
- **High-pass filter (HPF)** — наоборот.
- **Band-pass** — пропускает полосу вокруг центральной частоты.

**Cutoff** — частота среза. Чем ниже — тем «темнее» звук.
**Resonance** — подъём вокруг точки среза. При высоком значении фильтр «свистит».

## 3. Огибающая (ADSR)

**ADSR** — Attack, Decay, Sustain, Release. Управляет тем, как громкость (или фильтр) изменяется во времени:

- **Attack** — время нарастания (0 = мгновенно, долго = медленное появление)
- **Decay** — время спада до уровня Sustain
- **Sustain** — уровень во время удержания клавиши
- **Release** — время затихания после отпуска клавиши

## 4. LFO (Low Frequency Oscillator)

**LFO** — медленный осциллятор, модулирующий другие параметры:
- LFO на высоту тона → **вибрато**
- LFO на громкость → **тремоло**
- LFO на фильтр → «пульсирующий» звук

> **Практика:** возьмите любой синтезатор (или виртуальный — Vital, Serum, бесплатный Surge XT). Начните с preset. Постепенно двигайте Cutoff — слушайте, как меняется звук. Потом Attack — сделайте его длиннее и короче.
