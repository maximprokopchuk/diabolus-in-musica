# Программирование звуков

## Подход к созданию звука

Вместо случайного вращения ручек — думайте о звуке **аналитически**:
1. Какой тип звука? (бас, лид, пэд, плак, перкуссия)
2. Какая форма волны подходит?
3. Сколько атаки нужно?
4. Как должен звучать хвост (release)?

## Рецепт: классический пэд (Pad)

**Пэд** — медленный, атмосферный, «висящий» звук.

```
Осциллятор 1: Пила (Saw), детюн -5 центов
Осциллятор 2: Пила (Saw), детюн +5 центов (создаёт хорус)
Фильтр: LPF, Cutoff ~40%, Resonance ~20%
ADSR (амплитуда): Attack = 1 сек, Decay = 0.5 сек, Sustain = 80%, Release = 2 сек
ADSR (фильтр): Attack = 1.5 сек, Sustain = 60% (фильтр открывается со временем)
LFO: на высоту тона, глубина 10%, скорость 4 Hz (лёгкое вибрато)
Эффекты: Reverb (большой зал), Chorus
```

## Рецепт: лид (Lead)

**Лид** — яркий, поющий звук для соло.

```
Осциллятор: Прямоугольник (Square), +Пила
Фильтр: LPF, Cutoff ~70%, Resonance ~30%
ADSR (амплитуда): Attack = 0, Decay = 0.1 сек, Sustain = 90%, Release = 0.3 сек
Pitch Bend: включите (± 2 тона)
LFO: на высоту, задержка 0.5 сек → вибрато появляется после атаки
Portamento: небольшой глайд между нотами (если есть)
```

## Рецепт: синт-бас

```
Осциллятор: Пила + Прямоугольник (sub-октавой ниже)
Фильтр: LPF, Cutoff ~25%, Resonance ~50%
ADSR (фильтр): Attack = 0, Decay = 0.2 сек → фильтр быстро закрывается (создаёт «плюх»)
ADSR (амплитуда): Attack = 0, Decay = 0.1 сек, Sustain = 70%, Release = 0.1 сек
```

## Рецепт: плак (Pluck)

**Плак** — атакующий, быстро затихающий звук (арфа, щипковые).

```
Осциллятор: Пила или Шум
Фильтр: LPF с большой энергией огибающей
ADSR (фильтр): Attack = 0, Decay = 0.1–0.3 сек, Sustain = 0% (фильтр полностью закрывается)
ADSR (амплитуда): Attack = 0, Decay = 0.3–1 сек, Sustain = 0%, Release = 0.1 сек
```

> **Принцип:** большинство звуков создаются через **конверт фильтра** — как быстро и насколько открывается LPF. Это главный инструмент характера синтезаторного звука.
