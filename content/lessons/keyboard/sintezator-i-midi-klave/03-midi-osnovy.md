# MIDI: основы

## Что такое MIDI

**MIDI** (Musical Instrument Digital Interface) — протокол передачи музыкальных данных. MIDI не содержит звука — он передаёт **инструкции**: «нажата нота 60 (До), с силой 80, на канале 1».

Звук создаёт **получатель** (синтезатор, компьютер с VST, сэмплер).

## Основные MIDI-сообщения

| Сообщение | Что делает |
|-----------|-----------|
| **Note On** | Нота нажата (pitch + velocity) |
| **Note Off** | Нота отпущена |
| **Control Change (CC)** | Параметр (педаль, модуляция, громкость...) |
| **Pitch Bend** | Отклонение высоты тона |
| **Program Change** | Смена тембра/пресета |
| **Aftertouch** | Давление на нажатую клавишу |

## Velocity

**Velocity** (от 0 до 127) — скорость нажатия клавиши = громкость и тембр ноты. 0 = тихо, 127 = максимально громко.

Это аналог динамики на пианино — сила удара.

## MIDI-каналы

MIDI работает по **16 каналам**. Каждый канал — отдельный «инструмент». MIDI-клавиатура обычно отправляет на канал 1.

**General MIDI (GM):** стандарт, где канал 10 — всегда перкуссия, а остальные — мелодические инструменты.

## Подключение MIDI-клавиатуры

Современные клавиатуры подключаются через **USB** — самый простой способ. Старые инструменты — через **5-pin DIN MIDI** (нужен MIDI-интерфейс).

**Цепочка:**
```
MIDI-клавиатура → USB → Компьютер (DAW) → VST-синтезатор → Колонки
```

## MIDI в DAW

В DAW (Ableton, Logic, FL Studio) MIDI-данные записываются как **MIDI-трек**:
- Видны как «кирпичики» (ноты в Piano Roll)
- Можно редактировать, транспонировать, менять скорость
- Можно направить на любой синтезатор

> **Основной плюс MIDI:** гибкость. Записали партию — потом можете изменить темп, транспонировать, исправить ноты, применить другой инструмент — всё без перезаписи.
