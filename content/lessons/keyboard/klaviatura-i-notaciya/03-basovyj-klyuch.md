# Басовый ключ

## Ключ Фа

Басовый ключ — это **ключ Фа**: две точки обозначают четвёртую линейку снизу, на которой стоит нота **Фа** малой октавы (F3).

**Левая рука** пианиста играет преимущественно в басовом ключе.

## Ноты на линейках (снизу вверх)

```
Линейка 1: Соль (G2)
Линейка 2: Си (B2)
Линейка 3: Ре (D3)
Линейка 4: Фа (F3)
Линейка 5: Ля (A3)
```

Мнемоника: **«Собака Бежит Домой Фомину Алексею»** (С–Б–Д–Ф–А)

## Ноты в промежутках (снизу вверх)

```
Промежуток под 1-й линейкой: Фа (F2)
Промежуток 1–2: Ля (A2)
Промежуток 2–3: До (C3)
Промежуток 3–4: Ми (E3)
Промежуток 4–5: Соль (G3)
```

## Средний До в басовом ключе

**Средний До (C4)** записывается **первой добавочной линейкой сверху** в басовом ключе — той же нотой, что записана первой добавочной линейкой снизу в скрипичном ключе. Это точка соединения двух систем.

## Большой нотный стан (Grand Staff)

Для фортепиано используется **большой нотный стан** — скрипичный и басовый ключи, соединённые вертикальной линией и фигурной скобкой (акколадой). Правая рука читает верхний стан (скрипичный), левая — нижний (басовый).

```
┌──── Скрипичный ключ ────┐   ← правая рука
│                          │
└──── Басовый ключ ────────┘   ← левая рука
```

Средний До находится **между** двумя станами — на добавочной линейке.

> **Упражнение:** сыграйте гамму До мажор левой рукой (C3–C4), называя каждую ноту. Потом сыграйте обеими руками одновременно — правая октавой выше левой.
