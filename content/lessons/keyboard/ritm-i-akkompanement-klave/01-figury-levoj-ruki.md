# Фигуры левой руки

## Три основные фигуры аккомпанемента

### 1. «Бас + аккорд» (простейший паттерн)

```
Бас (сильная доля) — Аккорд (слабая доля)
```

В 4/4: C2 | E3–G3–C4 | C2 | E3–G3–C4
Пальцы: 5–1 | 3–2–1 | 5–1 | ...

Применяется в поп, рок, простых романсах. Чётко выделяет гармонию.

### 2. Альберти-бас

Паттерн: бас — верхний — средний — верхний

```
Для C мажора (C–E–G):
C–G–E–G–C–G–E–G...
5–1–2–1–5–1–2–1...
```

Классический паттерн сонат Моцарта, Гайдна. Создаёт движение без смены гармонии.

### 3. Арпеджио (разбитый аккорд)

Ноты аккорда берутся поочерёдно снизу вверх:

```
Для Am (A–C–E):
A–C–E–A–C–E–A...  (в 3/4 — идеальный вальс)
5–3–1–5–3–1...
```

Более лирично, чем Альберти-бас. Хорошо для баллад, романтической музыки.

## Выбор паттерна под жанр

| Паттерн | Подходит для |
|---------|-------------|
| Бас + аккорд | Поп, рок, марши, простые пьесы |
| Альберти-бас | Классика, сонаты, этюды |
| Арпеджио | Баллады, ноктюрны, романтика |
| Вальсовый (бас+акк+акк) | Вальсы, 3/4 |

## Вальсовый паттерн (3/4)

```
Доля 1: Бас-нота (5)
Доля 2: Аккорд (3–2–1)
Доля 3: Аккорд (3–2–1)
```

C мажор в вальсе: C2 | E3–G3–C4 | E3–G3–C4

> **Упражнение:** сыграйте прогрессию C–Am–F–G в трёх вариантах: «бас+аккорд», Альберти-бас и арпеджио. Правая рука держит простой аккорд или играет мелодию.
