# Обращения аккордов

## Что такое обращение

**Обращение** — перенос нижней ноты аккорда на октаву вверх. Состав нот тот же, но порядок меняется.

Для трезвучия:
- **Основной вид** (тесное расположение): Корень — Терция — Квинта
- **Первое обращение** (секстаккорд): Терция — Квинта — Корень
- **Второе обращение** (квартсекстаккорд): Квинта — Корень — Терция

**Пример на До мажоре:**
```
Основной вид:   До — Ми — Соль  (C3–E3–G3)
Первое обр.:    Ми — Соль — До  (E3–G3–C4)
Второе обр.:    Соль — До — Ми  (G3–C4–E4)
```

## Зачем нужны обращения

Обращения позволяют:
1. **Остаться в одной позиции** — не прыгать по клавиатуре при смене аккордов
2. **Создать плавное голосоведение** — минимальное движение каждого голоса
3. **Изменить «вес»** — квартсекстаккорд звучит неустойчивее основного вида

## Плавное голосоведение

**Цель:** при смене аккорда каждый голос движется на минимальное расстояние.

Пример — прогрессия C–F–G в До мажоре:
```
C (основной): C3–E3–G3
F (2-е обр.):  C3–F3–A3   (только E→F, G→A)
G (1-е обр.):  B2–D3–G3   (или C3–D3–G3)
```

## Аппликатура обращений (правая рука)

| Вид | Аппликатура |
|-----|------------|
| Основной | 1–3–5 |
| Первое обращение | 1–2–5 |
| Второе обращение | 1–3–5 |

> **Упражнение:** сыграйте трезвучие До мажор в трёх позициях подряд, вверх и вниз. Потом чередуйте аккорды C–Am–F–G, используя обращения для минимального движения рук.
