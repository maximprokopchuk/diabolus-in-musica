# Аккорды в левой руке

## Функции левой руки

На фортепиано левая рука отвечает за:
- **Бас** — нижний голос, гармоническая основа
- **Аккомпанемент** — ритмическая и гармоническая поддержка
- **Внутренние голоса** — средний пласт фактуры в полифонии

## Простейшая позиция: бас + аккорд

Самая удобная позиция для начинающих:
- Большой палец (1) или мизинец (5) — бас-нота (корень или квинта)
- Пальцы 2–3–4–5 или 1–2–3 — аккорд выше

**Пример в C мажоре:**
```
Бас: C2 (5-й палец)
Аккорд: E3–G3–C4 (пальцы 3–2–1)
```

## Альберти-бас

**Альберти-бас** — классический паттерн левой руки: бас–аккорд–средний–аккорд.

```
Для C мажора (C–E–G):
C — G — E — G — C — G — E — G...
5   2   3   2   5   2   3   2
```

Этот паттерн встречается у Гайдна, Моцарта, Бетховена — он создаёт движение, не перегружая гармонию.

## Разбитый аккорд (арпеджио)

Ноты аккорда берутся последовательно снизу вверх:
```
C–E–G–C (пальцы 5–3–2–1)
```

Более певучий вариант — хорошо подходит для баллад.

## «Бас + аккорд» в разных ритмах

**Вальс (3/4):**
```
Доля 1: Бас (C)
Доля 2: Аккорд (E–G)
Доля 3: Аккорд (E–G)
```

**Поп (4/4):**
```
Доля 1: Бас
Доля 2: Аккорд
Доля 3: Бас
Доля 4: Аккорд
```

## Позиционная игра левой руки

Старайтесь **не менять позицию** при каждой смене аккорда. Используйте обращения, чтобы рука оставалась в одном месте клавиатуры.

> **Упражнение:** сыграйте прогрессию C–F–G–C левой рукой паттерном «бас + аккорд» в ритме 4/4. Правой рукой играйте простую мелодию. Это базовый навык аккомпанемента.
