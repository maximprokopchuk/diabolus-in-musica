# Педаль для создания легато

## Когда пальцы не справляются

Есть ситуации, когда пальцевое легато физически невозможно:
- **Широкие скачки** в мелодии (дека + нота, терция + нота через октаву)
- **Переход между позициями** в гамме через всю клавиатуру
- **Аккордовые последовательности**, где нужно плавное голосоведение, но позиции далеко

В этих случаях педаль «склеивает» звуки.

## Педальное легато: техника

1. Нажмите первую ноту/аккорд
2. **Нажмите педаль** (сразу или с чуть запаздыванием)
3. Уберите пальцы и переместите руку к следующей позиции
4. **Нажмите** следующую ноту/аккорд
5. **Смените педаль** в этот же момент (или чуть после)

Звук первой ноты «живёт» в педали, пока рука переходит к следующей.

## Пример: широкий скачок

```
До1 (левая рука) → Ля2 (скачок на большую сексту)
```

Без педали: пауза во время перехода.
С педалью: До1 держится педалью, рука переходит к Ля2, педаль меняется в момент взятия Ля2 — получается легато.

## Певучие мелодии в верхнем регистре

Для лирических мелодий правой руки педаль помогает достичь «певучести» — звук каждой ноты плавно перетекает в следующую, как вокальная линия.

Ключевой момент: **мелодия всё равно должна играться легато пальцами** — педаль лишь усиливает связность, не заменяет её.

## Аккорды legato

Последовательность аккордов тоже может быть «педальным легато»:
- Берёте аккорд C
- Нажимаете педаль
- Переходите к аккорду F (пальцы уже в форме F)
- Берёте F и меняете педаль одновременно

Между аккордами нет паузы — педаль «мост».

> **Упражнение:** сыграйте первые 8 тактов ноктюрна Шопена op. 9 № 2 (правая рука мелодия, без левой) — без педали, потом с педалью. Разница в «певучести» должна быть очевидной.
